「这世界」苹果招工程师，第一步写启事，第二步把它藏起来_智能_好奇心日报

孙若空2017-08-22 15:09:06

快去找啊

苹果公司，应该是挺多大牛工程师希望去工作的一家公司吧？可以想象，如果苹果今天说我们打算招一个关键岗位的工程师，那不少人都在摩拳擦掌跃跃欲试准备投简历。

但让人没想到的是，这个不按常理出牌的公司在招聘的时候也很不按常理出牌。其实在提交申请前，苹果的面试已经开始了。

前两天，科技新闻网站 ZDNet 的编辑 Zach Whittaker 在苹果网站上闲逛（大概）的时候，一不小心发现了一个苹果公司藏在官网深处的秘密。就是下面这段内容啦。

在一个用代码组成的大苹果 logo 下面，居然藏着一段招聘启事。“嘿！你找到我们了。我们正在找一位有才华的工程师，为我们开发一个关键的基础设施组件，这是苹果生态系统的关键部分。”

热情地打完招呼之后，就是一连串充满热诚的工作介绍。苹果透露，这位工程师将与数百万台驱动器、数万台服务器和 Exabyte 数据一起工作。 这份工作要求申请人具有设计、实施和支持高度可扩展的应用程序和 Web 服务的经验。申请人最好还要懂 Java 8，现代服务器技术和分布式系统。然后在这段介绍最后，最关键的部分来了，它表示感兴趣的工程师可以发简历到 blog-recruiting@group.apple.com 这个邮箱。

“酷啊。如果你发现了苹果的这个隐藏页面，那就有一份工作了！” Zach Whittaker 在 Twitter 上说。但很显然，找到这个页面只是第一步，接下来还得有严格的考核。

从 Whittaker 晒出的截图来看，这个招聘启事藏在了一个地址为 us-east-1.blobstore.apple.com 的页面里。但现在，再寻着这个网址去找，只能看到一段话：This XML file does not appear to have any style information associated with it. The document tree is shown below.

显然，之前的那个招聘启事已经被删除了。当然，它也可能被转移去了别的地方，静静地等待高手去发现。

当然啦，作为发现者供职的公司， ZDNet 也适时地打了一波广告，让 Zach Whittaker 介绍了下发现这个隐藏网站的经过——不过标题用的是 how we。

经过是这样的。当时，Whittaker 正在使用流量分析器 Burp Suite 来拦截几个 iPhone 应用之间的数据，以查看向广告客户发送的个人数据。“在跟踪用户 App 消费情况的分析网站和广告商的连接流中，有几个 outbound 连接到苹果公司的‘blobstore’服务器之一，该服务器用于托管 iCloud 数据，例如客户照片和视频。”然后他输入了那个地址，就看到了那段招聘启事。

Whittaker 说，其实他也向那个邮箱递了简历，他说对方回复说会有进一步联系已经让他很高兴了。至于苹果该招的人，应该更聪明、能干、专业。
