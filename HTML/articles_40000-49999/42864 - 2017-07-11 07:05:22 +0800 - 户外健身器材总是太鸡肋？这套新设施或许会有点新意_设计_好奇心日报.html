户外健身器材总是太鸡肋？这套新设施或许会有点新意_设计_好奇心日报

胡晓琪2017-07-11 07:05:22

把专业的健身房设施搬到了户外

陈旧又无聊的社区健身器材总是很难让人产生兴趣，这在全世界几乎都一样——有人提出了新点子：如果把更为专业的健身器材设计成更适合户外运动的样子呢？

Cross System 健身系统或许能改变你对户外健身器材的认知。

比起老一套的户外健身器材，Cross Stystem 最大的优势在于它的灵活适应性和专业性，它就像是把健身房的器材搬到了户外一样，能够让你随时随地进行训练。

Cross System 系统由六个不同的框架系统组成，从悬吊训练带到拉力器，再到壶铃和双杠。根据各类型的使用场地和使用者，这六个系统可以独立放置或者被连接在一起。

它在设计中还加入了一个可控的阻力磁断裂系统，在没有专人进行指导的情况下，也能保证户外的自由重量练习的安全。比如 Cross System 的壶铃就可以在垂直钢管上自由移动， 因为有阻断系统，会逐渐给壶铃减速然后缓慢掉落，防止砸伤人。

如果你不太清楚 Cross System 应该怎么操作的话，KOMPAN 还开发了一套支持性的应用程序。可以为用户演示器械的使用方法，并连接社交网络来更新用户的状态。激励用户以个人或分组的形式，进行正确的训练。

设计出这套户外健身器材的，是来自丹麦的 KOMPAN，它是世界第一的儿童游乐场制造商，有着四十多年的历史。其产品包括儿童游乐场、运动健身器材及室内外桌椅。在 KOMPAN 的网站上，你可以根据占地面积大小、适用对象年龄范围、素材种类来挑选合适的健身器材。

KOMPAN 的这项设计获得了两项设计大奖：欧洲产品设计奖和澳大利亚优秀设计奖。在澳大利亚，Cross Systems 赢得了“体育与生活产品设计类别”的奖项。设计者希望能够通过它来增强社区居民的身体素质，改善亚健康的问题。

今年年初，新加坡的裕廊组屋区成为了亚洲首个引进 Cross System 户外公共健身设施的地区。这套设施包括壶铃、悬吊训练带、引体向上杆等，初学者、专业健身人士，甚至以轮椅代步的人士，都能使用这套设施做运动。
