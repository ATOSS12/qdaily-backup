特斯拉交付了第一批 Model 3，为自动驾驶连仪表盘都不要了_智能_好奇心日报

马若飞2017-07-31 07:14:03

一辆没有仪表盘的车。

北京时间 7 月 29 日 中午，特斯拉在加州弗利蒙特总装厂召开了一场发布会，老板伊隆·马斯克出场，交付了首批 30 台新型量产车 Model 3。

Model 3 于 2016 年 3 月发布，2017 年 7 月初第一台量产下线，车主是马斯克本人。这次虽说是交付首批车的发布会，但实际上这 30 台车的车主全都是特斯拉员工，发布会之后的试驾会开的也是这 30 台车。

和最初发布的概念车相比，Model 3 的外形和内饰变化不大。从车身侧面的线条来看，Model 3 基本上就是一台 Model S 的缩小版，它们的腰线、车门折线、后轮上方的轮拱折线、后门形状和后三角窗都几乎一模一样。

和概念车相比，量产车 B 柱之前的距离有加长，这大概是为了增加车内空间。按照马斯克的说法，Model 3 在看起来不大的车身外形内提供了最大的空间。按照前往现场试驾的媒体的体验，特斯拉 Model 3 的内部空间可以让四个成年人不局促的乘坐。

但因为车身长度更小，Model 3 的行李储存空间当然比 Model S 更小，大约只有 Model S 的一半，车头行李舱能放一个背包，车尾箱能放两个登机箱。

最令人惊讶的是 Model 3 没有仪表盘

和概念车几乎一模一样，位于司机前方，所有汽车应该都有的仪表盘都不存在。

Model 3 前排只有在最中间的位置设计了一个 15 英寸的大屏幕，所有的行车信息都在这个屏幕上。

在方向盘后部，只有两个控制拨杆，左侧的控制车灯、雨刷器，右侧的拨杆控制五个档位，倒车、停车、空档、行驶和自动（辅助）驾驶：自动驾驶被单独设计成了一个档位。

Model 3 也没有传统的空调控制面板和空调出风口。空调面板自然是集成在中控屏幕里面，空调出风口变成了挡风玻璃下方的一个横跨整个车厢的长条状开口，风向也是在中控屏幕里控制。

和这个惊世骇俗的内饰相比，Model 3 的玻璃车顶、隐藏式门把手都显得不值一提了。

不用多说，特斯拉 Model 3 看上去非常特别，但有一些非常简单的设计还值得怀疑。

比如中央显示屏这个最大的内饰设计突破，虽然看上去整个车厢氛围因此显得非常现代、精简，但驾驶员获得车速等基本信息会更麻烦，需要从路面到中央屏幕大幅度转移视线。

按照马斯克的说法，未来 Model 3 支持全自动驾驶以后，用户可以在车里看电影、交谈或者睡觉。所以可以认为，Model 3 的内饰是一种为了全自动驾驶的超前设计。但就像这辆车还不得不装上方向盘一样，它还需要驾驶员，这种完全没有仪表盘的设计实在非常激进。

配置更精简

性能方面，特斯拉提供了两种配置，基本版续航 350 公里，百公里加速 5.6 秒，售价 35000 美元起；长续航版续航 500 公里，百公里加速 5 秒，售价 44000 美元起。这两种车型暂时只提供后轮驱动车型，全部 Model 3 都将使用全新的 21700 锂电池芯级。

此外还有部分配置的选装：解锁自动辅助驾驶 5000 美元。包含电动座椅、座椅加热、玻璃除雾、打孔木内饰在内的高级选装包 5000 美元、特殊油漆 1000 美元等。未来如果特斯拉更新全自动驾驶软件，解锁还需要 3000 美元。

总体来说，可选的部件相对 Model S 和 X 少得多，目的也是为了降低成本，精简生产。

特斯拉把首批车交付给员工，是更方便特斯拉找出量产车阶段的问题，方便后续的产能爬坡。

目前，Model 3 的累积订单已经超过了 50 万台，每一张订单都预付了 1000 美元或等值货币。 但目前 7 月只生产了 30 台。马斯克说，Model 3 的产能将在电池工厂“千兆工厂”和弗里蒙特总装厂的改进下提升，目标在今年 12 月达到月产 20000 辆的目标。

在 11 月之前，特斯拉将仅生产长续航车型，标准版车型 12 月开始生产，明年春天开始生产 4 轮驱动车型。美国之外的其他国家车辆交付将从 2018 下半年开始。

特斯拉在活动上宣传，Model 3 是“世界上第一款大众市场电动车”。实际上 Model 3 并不是世界上第一款大规模量产的电动汽车，也不是最便宜的。

但以现在已经量产的电动车产品来说，Model 3 是同价格中续航最长的电动车型：最畅销的日产聆风价格便宜，只要 30000 美元起，但续航最长只有 170 公里；Model 3 最大的对手雪佛兰 Bolt 虽然售价更便宜，但按照 37000 美元的建议售价算，每公里续航的平均价格 Model 3 更低。

最大问题还是量产

本月初，特斯拉中国在把办公室搬到华茂中心之后组织了一场媒体活动，当时 Model 3 还没有量产交车，《好奇心日报》询问，Model 3 何时会在国内交车，何时有试驾车可以供潜在顾客体验。

特斯拉中国的回答都是：暂时没有时间表。

3 月份，特斯拉宣布 2016 年在中国营收 10 亿美元，占全年总收入的 15%。但这个数字里面至少有 42% 是税费（关税 25% 和增值税 17%）。

减去税的数字只有大约 6 亿美元。按照最便宜的 Model S 车型来计算，也就是不超过 8500 辆。

中国市场的税对 Model S 这样的豪华车来说可能不是问题，因为买一辆 100 万人民币以上的车的人不太在乎车子是不是再贵 20%。

但 Model 3 未来在中国的销售中，税一定是非常重要的因素：按照特斯拉的定价规则，未来进口 Model 3 在中国的起售价格估计会在 30 万到 31 万人民币之间。

这个价格正好是中国市场上争夺最激烈的豪华三厢车的价格区间。宝马 3 系、奔驰 C 级，奥迪 A4 国产车型的价格都定在 28 万上下。

Model 3 如果国产，减去 20% 的进口关税，售价将降低至 26 万人民币左右。在这个价格下，Model 3 将因为不需要额外取得车辆牌照，在一线城市的潜在消费人群里明显吸引力更强。

在周末的发布会上，马斯克出场了 15 分钟，神情轻松，笑料不断。他在展示了一段 Model 3 和沃尔沃 S60 侧面立柱碰撞试验的对比视频之后，一度强忍笑意不得不停下演讲几秒钟：视频里 S60 在侧面立柱碰撞试验下车身侧面凹陷严重，大幅入侵驾驶舱空间，而 Model 3 因为底盘是整体电池的先天优势，碰撞形变非常小。

沃尔沃 S60 正是传统豪华三厢车的代表车型之一，沃尔沃也一直是以安全作为最重要的卖点。马斯克还调侃了现在的汽车安全测试，说所谓的“5 星安全”也就是这样，就差直接说“特斯拉才是汽车的未来”了。

在奔驰、宝马、大众、本田、丰田这一大堆传统车企量产电动车的 2019 年之前，特斯拉 Model 3 就是这个价位最好的电动汽车。

现在最大的问题是，特斯拉的量产计划能否实现。

特斯拉从成立至今，每一款车的量产计划都比特斯拉最初公布的时间表晚得多。Model S 延期三年，Model X 一年半。

随着欧洲、中国的政策调整，传统车厂都在筹备电动车。这一次，它不能再晚了。
