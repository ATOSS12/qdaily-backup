「本周新片」《侠盗联盟》合格，《心理罪》也还行_娱乐_好奇心日报

王珊珊2017-08-11 18:32:29

还有尴尬的《鲛珠传》，以及只能看画面的《地球：神奇的一天》。

《心理罪》

  • 导演：谢东燊
  • 主演：廖凡 / 李易峰 / 万茜 / 李纯 / 张国柱
  • 片长：110 分钟
  • 格式：2D

如果你对于国产“悬疑”的印象欠佳。从完整度和表演自然上来看，这部电影可能会比你原本的预期强一些。

影片改编自《心理罪》第一部《画像》中的吸血鬼案：杀手马凯的亲人因遗传性血液病接连去世，对自己的健康产生了担忧。虽然医院告知马凯血液完全正常，但他却产生妄想症，坚信自己的血液将慢慢干涸，因此走上了猎杀女人的犯罪之路，开始取血“续命”。

柏林电影节影帝廖凡继《白日焰火》后，将继续挑战警察角色，在片中饰演行事风风火火的刑警队长邰伟。敏感、内向的犯罪心理学天才大学生方木将由李易峰扮演。凭《军中乐园》夺得金马奖最佳女配的万茜在里面扮演新增加的角色女法医乔兰。

电影最大的优点是有一个相对流畅的叙事节奏，廖凡和万茜表演完整，李易峰实际上没有引发过多的尴尬。

但是人物矛盾的塑造仍然停留在刻板印象上，悬疑感也只是浅尝辄止。如果你追求的是紧张、震撼的黑暗悬疑，显然这部仍然不适合你。

《侠盗联盟》

  • 导演：冯德伦
  • 主演：刘德华 / 舒淇 / 张静初 / 杨祐宁 / 让·雷诺
  • 片长：108 分钟
  • 格式：2D/IMAX/中国巨幕

它本身是一个非常老套的故事，中规中矩的特工电影，主题没有新颖之处。

但是它同样在基础节奏和演员表演上都拿到了基础分。冯德伦把商业爆米花该有的元素都用了进去：颜值、动作戏、各类高科技道具。

一个很有保障性的因素是，它的演员阵容：刘德华、舒淇、杨佑宁、曾志伟、让·雷诺——都给出了非常标准的表演。

《鲛珠传》

  • 导演：杨磊
  • 主演：王大陆 / 张天爱 / 任达华 / 盛冠森 / 王迅 
  • 片长：108 分钟
  • 格式：2D/3D/IMAX3D/中国巨幕

导演杨磊上一部电视剧《九州天空城》的成绩就欠佳，这部电影尽管在场面调度上有视觉上的进步，但是故事上仍然非常生硬、突兀。

电影剧情给人最大的感受就是“硬尬”——每一个场景都非常突然，例如在船上坐着好好的，突然就吵架了；刚刚打架打得还挺累的，突然就要感情戏进展了。（就不能过渡一下吗？）

最可怕的是一只用电脑动画做的“穿山甲”。既然效果糟糕成这样，为什么还非要有这只也不是很可爱，很可能成本还很高的宠物呢？

《地球：神奇的一天》

  • 导演：理查德·戴尔 / 彼得·韦柏 / 范立欣
  • 配音：成龙 / 罗伯特·雷德福
  • 片长：94分钟
  • 格式：2D

鉴于这部片子主要是《地球脉动》第二季的素材重新剪辑（官方的说法是“共享部分素材”）而成，从画面配乐上说，它是能看的。

但如果你是一个资深 BBC 自然纪录片爱好者，这部人类沙文主义视角的纪录片会让你觉得失望（类似于强调“人类是地球主人”，“唯有人类可以罔顾日夜交替”）。如果你冲着合家欢的目的带着小朋友去，你最好在走出影院的时候告诉ta：“其实人类只不过是地球上的一种数量比较多的生物而已。”

《地球：神奇的一天》是“中英合拍协议框架”下的第一部中英合拍电影，合作方 SMG 尚世影业很显然花了功夫，因为他们请中国管弦乐合唱团现场演奏以制作 BGM，请严歌苓做编剧（负责旁白），让成龙来配了音（并努力练习了普通话）。在《地球脉动》“主角”之外添加的熊猫和白头叶猴也很能唤起中国观众的地域自豪感。

如果从“这样已经很不容易了”的角度，《神奇的一天》是合格的产品，但考虑到它的合作伙伴 BBC ，以及在推广时反复强调“中国独有的天人合一”等宣传语，还是觉得十分遗憾。

此外，想念赵忠祥老师。
