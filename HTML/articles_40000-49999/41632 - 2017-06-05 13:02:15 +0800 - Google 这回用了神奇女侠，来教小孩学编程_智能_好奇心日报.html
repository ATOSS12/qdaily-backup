Google 这回用了神奇女侠，来教小孩学编程_智能_好奇心日报

谢若含2017-06-05 13:02:15

Made with code 是 Google 推出的一个帮助女性学习编程的项目

根据 DC 漫画改编的《神奇女侠》正在影院上映，Google 也来蹭了一把热点， 他们用上了《神奇女侠》的剧情来教女性学习编程。

在这个项目中，Google 选用了《神奇女侠》电影中的三幕剧情来分解为不同动作。因为这是一个针对低龄群体的编程项目。小孩不需要编写出具体代码，而是通过采用包含了动作、对象和变量的代码块，进行组合和拖放。

这些编程新手们能看到不同的代码是如何组合起来，并最终实现程序目标。学习者将通过不同的代码块来帮助“神奇女侠”完成在剧情中面临的一系列困难，在天堂岛黛安娜和同伴决斗的场景就需要 5 个代码块组合才能完成。

这来源于 Google 去年推出的一个叫做 “Made with Code”的项目，它旨在激发并培养女孩对编程的兴趣。这个为了让更多女孩接触编程技术的项目获得了来自切尔西克林顿（前美国总统比尔·克林顿之女）、麻省理工学院媒体实验室等多个机构于个人的支持。

Made with code 项目的培训课程包括：学习基于 Blockly 语言的初级实用编程技术。你可以用这个程序自主设计一款手镯，并交给 Shaoeways 公司，用 3D 打印技术打印出来。美国童子军以及 Girls.lnc 等机构都是 Made with code 的支持者，他们希望借此鼓励女生完成他们人生中的第一次编程体验。此外，Google 还为女程序员项目提供 5000 万美元的赞助，包括奖励那些到 Codeacademy 上参加计算机科学的教师们。

Google 不是第一次采用人物形象来吸引小孩学习编程。此前，Google 还联合过 Cartoon Network 卡通频道，让儿童通过飞天小女警的卡通形象来学习编程项目，比如制作飞天小女警的动图应用，儿童可以通过设置不同的动图参数，来制作动画角色“花花”、“泡泡”“毛毛”的动图。

在 Google 介绍这一项目时，列举了三位在各自领域表现杰出的女性，她们中有的是嘻哈舞者、有的是电影摄影师、甚至还有与全球疟疾作斗争的人道主义战士。她们的共同点在于在工作中都使用到了编程。

但一项数据表明，在美国只有不到 1% 的高中女生表示对计算机科学感兴趣。世界上最大的开源代码社区 Github 在调查了 5500 名开源代码开发者后发现，其中只有 5% 为女性。

除了 Google 之外、微软和苹果也在不断推出让青少年学习编程的趣味项目。而让女性参与和学习编程也开始成为了越来越多科技公司、以及公益组织在促进性别平等上的举措。
