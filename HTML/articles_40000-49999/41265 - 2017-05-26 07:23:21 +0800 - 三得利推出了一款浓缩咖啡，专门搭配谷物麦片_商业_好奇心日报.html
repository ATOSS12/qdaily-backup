三得利推出了一款浓缩咖啡，专门搭配谷物麦片_商业_好奇心日报

陈思吟2017-05-26 07:23:21

三得利和卡乐比希望能为女性上班族在早上不花很多时间却能做出美味的早餐出点力。

为了搭配卡乐比谷物麦片 Furugura，三得利将推出一款枫糖味浓缩咖啡，据称将两者搭配后在家也能做出咖啡店般的早餐。

我们曾提到过推出 20 多年的卡乐比谷物麦片 Furugura 直到近年才在日本市场受到欢迎。趁胜追击的卡乐比提出了“第三种早餐”理念，希望早餐吃谷物能够成为继米饭、面包后第三种早餐形式。可以搭配牛奶、酸奶、色拉、豆腐食用的 Furugura 颇受日本女性消费者欢迎。

此次合作的另一方三得利 BOSS LATTE BASE 是一款浓缩咖啡，倒入牛奶后一瓶 490 毫升的 LATTE BASE 大约可以调制 10 杯拿铁。一直以来三得利 BOSS 系列咖啡在日本的主要消费者是男性，但于 2016 年推出的 LATTE BASE 却意外受到了女性消费者的欢迎。

三得利在调查后发现， LATTE BASE 约 70% 的购买者是女性，且其中近六成购买者会在早餐时段进行饮用。由于消费者群体接近，三得利 LATTE BASE 与卡乐比 Furugura 决定开展合作一同开拓早餐市场。

Furugura 与 LATTE BASE 的搭配方法十分简单，在杯中先置入 Furugura，再倒一定量牛奶，最后按照自己的需要淋上 LATTE BASE 即可。“我们追求最佳的口味，搭配 Furugura 后风味更佳。”三得利执行役员柳井慎说，三得利专门研发了能够增加 Furugura 香味的枫糖味浓缩咖啡。

为了宣传合作产品，两家公司决定在日本全国 5000 多处零售门店设置共同卖场。他们还将以快闪形式出现在 WIRED CAFE 在东京的三家分店的菜单上，不管是芭菲还是拿铁都采用了两家公司的产品。

早些时候三得利推出了一款针对男性上班族的透明柠檬红茶，不难看出，这次的新品主要是面向女性消费者的。两家公司都认为 Furugura 与 LATTE BASE 同属省时省力型产品，他们希望能为女性上班族在早上不花很多时间却能做出美味的早餐出点力。

由于枫糖味 LATTE BASE 还没有正式上市，只有相关日本媒体参加了试吃会。然而这次的评价又令人有些意外， Mynavi 的一位记者表示：“本来目标消费者就是女性，但觉得也值得向男性推荐。”
