「本周新片」《建军大业》和《战狼 2 》，都一言难尽_娱乐_好奇心日报

魏倩2017-07-28 16:53:56

还有一部叫《豆福传》的动画片。

《战狼 2》

  • 导演：吴京
  • 主演: 吴京 / 弗兰克·格里罗 / 吴刚 / 张翰 / 卢靖姗
  • 片长：123 分钟
  • 格式：2D / 3D / 中国巨幕

2015 年，吴京自编自导自演了“国内首部 3D 动作电影”《战狼》。两年后，他又拍了一部升级版的《战狼 2》。

简单描述一下剧情：身处非洲的主角冷锋被开除军籍后，遭遇了一场非洲国家叛乱。为了军人的使命，他孤身犯险冲回沦陷区，带领身陷屠杀中的同胞和难民，展开生死逃亡。

在逃亡过程中，影片出现了坦克、无人机、直升机、军舰、导弹等所有你能想象到的军事武器。比起第一部来说，场面更大，动作戏也有升级，据说影片投资超过 2 亿。然而在特效和动作场面超越前作的情况下，故事反而变得更加凌乱，尤其令人不适的是毫无存在感的女主人公和莫名其妙的感情戏。

如果你喜欢“中国硬汉拯救同胞”的戏码，或是喜欢“犯我中华者虽远必诛”等词汇，又或者对“坦克漂移”技术和主人公快速回血的能力感兴趣，《战狼 2》会是本周的首选。

但如果只是想快速了解吴京在该片中的表现，不如回忆一下近期在影院里看到的《我的中国梦》贴片。

《建军大业》

  • 导演：刘伟强
  • 主演: 刘烨 / 朱亚文 / 黄志忠 / 王景春 / 欧豪 / 刘昊然 / 马天宇 / 刘浩 / 张艺兴 / 李易峰 /  ……

  • 片长：133 分钟 
  • 格式：2D / IMAX / 中国巨幕

上映 3 天的《建军大业》，正在“主旋律”和“娱乐化”的两极间激烈摇摆。

直到 22 日凌晨，这部影片的宣传定位还是“热血青春战争片”。当夜，国家新闻出版广电总局副局长张宏森在朋友圈发文，寄语《建军大业》，为主旋律电影正名。在长文中，他特别将“小鲜肉”称为“中国新一代青年演员，是中国电影的未来”。

7 月 25 日下午，@电影建军大业 发出点映影评，称该片为“中国版漫威”，影评出现了“中国队长叶挺、金刚狼粟裕”等宣传词，引发质疑后，该海报很快被撤下。

当天下午，叶挺将军之孙，导演叶大鹰在微博发声，称片中叶挺的饰演者欧豪为“腿都站不直女里女气的小鲜肉”。并质问导演是否“别有用心的想借重大历史事件来发娱乐财”；26 日，叶大鹰又发微博批评片中瞿秋白的扮演者白客；27 日下午，叶大鹰以未收到制片方创作授权申请为由，向广电总局领导致信抗议。

至此，原本附着在《建军大业》之上的“主旋律献礼片”标签被置换为“娱乐化革命历史影片”。影片上映后，观众在微博发出片中叶挺炸毁军火库的动图，称欧豪表演轻浮。实际观影时，观众席中也确实出现多次笑场。

2009 年上映的《建国大业》曾集结了唐国强、张国立等 172 位明星，2011 年上映的《建党伟业》同样有 109 位明星出演。作为“建国三部曲”的《建军大业》，在宣发中更多次采用“趁年轻，干大事”等口号，意在吸引和教育青年爱国者，同时也紧盯着他们手里的票房。

“建国三部曲”的前两部票房成绩均在 4 亿元左右，目前《建军大业》票房已过 7000 万。目前大众讨论依然集中在“谁演谁比较像”、“小鲜肉能不能演革命先烈”等话题上，而它们显然不是这部影片真正的问题。

毕竟说起具体影片内容时，它在豆瓣页面中是禁止评论的。

《豆福传》

  • 导演：邹燚
  • 主演: 陈佩斯 / 季冠霖 / 李立群
  • 片长：85 分钟 
  • 格式：3D

导演邹燚应该是个忠实的黄豆爱好者。从 2014 年开始，他陆续拍摄了《吃饭睡觉打豆豆》、《豆特么黑》、《神经豆》等一系列以黄豆为主角的动画片。

2017 年，他又以汉代淮南王刘安修习仙术的传说为蓝本，创造了一个黄豆幻化而成的“豆福”形象。现在看来，这个圆胖的小豆人还是挺可爱的。

这部影片最大的亮点是中国风的修仙故事，以及相当不错的画面渲染效果。不过“中国风”和“修仙”的符号也极易引起争议，不少观众已经在片中看出了《功夫熊猫》和《怪物史莱克》的影子，而草根逆袭拯救小镇以及后半部分外星人乱入的桥段，成年人观看还是难免尴尬。

值得一提的是片中的“神经豆”一角请来了陈佩斯做配音。没什么选择的情况下，可以家庭观影。当然，如果有排片的话。
