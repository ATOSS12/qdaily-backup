特斯拉二季度亏损加大，但外界还是因为 Model 3 看好它_商业_好奇心日报

龚方毅2017-08-03 16:20:37

特斯拉说到年底的时候，一周能生产 5000 辆 Model 3，尽管现在只有 25 辆。

今晨特斯拉公布 17 年第二季度财报，公司实现汽车销售收入 22.86 亿美元，是去年同期的两倍，其主要原因是特斯拉产能提升，在报告期内向车主交付更多汽车。

特斯拉在 4-6 月交付 22026 辆 Model S 和 Model X，上半年共交付 47077 辆。它们预计下半年这一数字还会继续增加。

在投资人更关心的 Model 3 产能方面，特斯拉表示项目正在有序生产，“有信心”达到既定目产能标 —— 第三季度生产 1500 辆 Model 并在年底的时候将产能提升至每周 5000 辆。

特斯拉创始人马斯克在上周末的 Model 3 首发仪式上对外说过这个数字。

但特斯拉还有更大期望，“我们还打算继续提升 Model 3 产能，预计在 2018 年某个时间达到每周 10000 辆。”特斯拉在财报中表示。

尽管特斯拉第二季度亏损扩大至 4.01 亿美元、在任 11 年的电池技术负责人 Kurt Kelty 在 Model 3 等待交付这一关键时刻离职，但投资人看好 Model 3，特斯拉盘后股价大涨 7.42%。

目前特斯拉每天净增近 1800 辆 Model 3 订单、总订单数突破 50 万辆。按照每月 2 万辆产能计算，特斯拉至少得花 25 个月才能将截止到目前为止的所有账面订单，转成一辆辆停在流水线上的电动车，然后交给客户。

由于特斯拉订车是先支付押金、快交付的时候才向车主收车款，中途可以取消预订、押金退回。它能以多快的速度挣到卖车钱，取决于车辆交付速度有多快。

谈到本季度亏损加剧的现象，特斯拉解释是因为公司固定资产扩大投入，新增的 9 亿美元资本支出主要用在比如超级工厂、Model 3 佛蒙特工厂等重要生产项目的投资上。

下半年这笔支出预计达到 20 亿美元，用途仍旧是上述几个项目。马斯克还在“认真思考”在哪里投建第 3、4、5、6 座超级工厂，中国和欧洲都在考虑范围内。截止至第二季度末，特斯拉现金储备为 30 亿美元。

“现阶段我们不考虑股权融资，我们正在考虑债权融资。”马斯克说。债务融资就是借贷，到期返还本金支付利息，它不会稀释特斯拉股东的股份。

马斯克的电动车公司倾向于这种融资方式，可以理解为特斯拉有信心加快车辆交付速度，提高现金流周转率。

一些分析师由此已经开始期待特斯拉不远的将来实现盈利。但这具体还是要看 Model 3 的产能。

作为特斯拉最重要的一款大众化电动车，它直接决定了特斯拉是否能实现真正盈利。
