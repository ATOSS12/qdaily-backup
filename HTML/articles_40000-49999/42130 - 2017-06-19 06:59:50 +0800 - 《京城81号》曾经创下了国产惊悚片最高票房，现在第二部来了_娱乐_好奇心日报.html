《京城81号》曾经创下了国产惊悚片最高票房，现在第二部来了_娱乐_好奇心日报

陈心怡2017-06-19 06:59:50

这次会不再走狗血言情路线，而是变得像一部真正的惊悚片吗？

2014 年上映的《京城 81 号》是当时国产惊悚电影中投资最高的影片，影片的投资近亿，而它的最终票房突破了 4 亿元。这个成绩刷新了国产惊悚片的最高票房纪录。

不过影片的故事剧情遭到了疯狂吐槽，它在豆瓣上只有 4.9 分的评价。有观众对它的评价是：不像一部恐怖惊悚片，反倒是一个狗血情感剧。“愿得一人心，白首不相离”这句出现在各大言情剧中的万金油也在《京城 81 号》中登场，显得相当尴尬。其中最惊悚的，可能就是一惊一乍的音效了。

但这个成绩自然让出品与发行方感到相当满意。于是，续作《京城 81 号 2》也要来了。影片将于 7 月 6 日上映，由恒业影业、万达影视、爱奇艺影业等联合出品。

第二部的主演为张智霖、梅婷、钟欣潼、耿乐、邬君梅，电影由陈嘉上监制，谭广源、蔡骏、黄慧惠编剧，钱人豪执导。

由吴镇宇、林心如主演的第一部影片讲述了女作家许若卿来到京城老宅后发生的一系列诡异事件，从而引出前世陆蝶玉与霍家三兄弟之间发生的爱恨情仇。而《京城 81 号 2》的恐怖关键词是“尸婴”，依旧有前世今生两条故事线。

民国末期，正逢乱世，少帅张骘生（张智霖饰）被逼迎娶大军阀之女纪金翠（钟欣潼饰）。随即府内怪事丛生，人心惶惶。不久后京城涌现大批“尸婴案”，一切矛头指向少帅原配夫人钮梦鹤（梅婷饰演）。随后钮梦鹤离奇失踪，一夜之间京城 81 号惨遭灭门，只留下一座破败鬼宅。

百年后，文物修复师宋腾在修复古宅过程中挖出了大量婴儿尸骨与诡异符咒，血淋淋的旧事再次浮现。

《京城 81 号 2》电影的主题曲《回魂》由费玉清演唱，这首歌根据第一部中的插曲《沉醉》改编。

除了 3D，《京城 81 号 2》还将加入 4D、VR 等体验来增强恐怖氛围。而这一次，发行方对于《京城 81 号 2》的预期目标票房是 5 亿。

但是，惊悚片在国内的市场还相对小众。当时的《京城 81 号》打着取材自“朝阳门内大街 81 号院”真实历史事件的旗号，再加上不错的排片与宣传，拿到了突破性的成绩。而比起惊悚，《京城 81 号》的重点明显是放在了男女角色狗血的感情故事上。

前作口碑不佳却赚足了噱头和票房，这一次的《京城 81 号 2》是否会更像一部真正的恐怖惊悚作？它还能再次刷新纪录吗？
