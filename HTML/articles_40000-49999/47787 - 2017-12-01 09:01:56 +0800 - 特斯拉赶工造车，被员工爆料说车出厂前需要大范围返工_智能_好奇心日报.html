特斯拉赶工造车，被员工爆料说车出厂前需要大范围返工_智能_好奇心日报

马若飞2017-12-01 09:01:56

Model 3 的生产还是没有好消息。

路透社援引 9 个特斯拉员工和前员工的描述称，特斯拉在加州弗里蒙特总装厂生产的 Model S 和 Model X 遇到了严重的质量问题，10 月份的内部生产数据显示，有 90% 的车从生产线上下来之后没有办法通过质检，必须经过修理才能出厂。

这些需要维修的故障有大有小，比如门关不严、软质材料修剪不齐、缺少零件、零件松散带来异响、漏水，等等问题。在传统汽车厂商那里，这样的问题不常见，比如丰田出厂前需要返工的概率不高于 10%。

爆料者还说，为了处理这些需要维修的新车，特斯拉专门开辟了一个可以停 2000 辆车的停车场，来中转这些待出厂的产品。

路透社采访的这 9 位和特斯拉有关的人中，有现任员工，同时也有上个月因为 Model 3 生产不力被开除的人。

特斯拉没有正面回应车辆缺陷的问题，只是声明称生产后维修的环节是为了“追求完美”，特斯拉说他们关注的问题是“几毫米的间隙差异和油漆划痕”，同时否认了有一个 2000 辆车的停车场来周转缺陷车辆的新闻。

但无论特斯拉再怎么强调产品“追求完美”，特斯拉轿车也从来都不是因为质量可靠被消费者欣赏的。实际上在 2012、2013 年，特斯拉轿车曾经在媒体那里收到了完全分裂的评价。一方面 Model S 被称为最革命性的汽车产品，一方面却被评价质量稳定性不佳，与高昂的售价不符。

虽然这次被爆料的是 Model S 与 Model X 两款老车型，但特斯拉的问题根本不在这：无论这个出厂前修理的环节是否大规模存在，用户收到的车都基本符合预期，近期没有出现大规模的质量问题，诸如《消费者报告》这样的独立评测媒体对特斯拉轿车质量的评价也比从前调高了。

最大的问题还是新车 Model 3。

按照特斯拉的规划，12 月应该能交货 5000 辆 Model 3，但实际上这个计划严重落后，10 月份只完成了预期的百分之一，也就是 145 辆 Model 3。因为计划推迟，特斯拉上个月解雇了几百个员工，调低了生产预期。和 10 月交付 Model 3 的热闹相比，过去一个月特斯拉几乎没有说过关于 Model 3 生产的任何细节。特斯拉一位前总监对路透社说，就是因为不断返工，特斯拉花掉了大量的钱。

同时，老产品 Model X 和 S 如果都有返工问题，新产品 Model 3 的状况可能更加不容乐观。

特斯拉现在的状况其实有些微妙。一方面，这家公司的老产品展示了部分实力，获得了大批粉丝，但商业上迟迟没有进展，制造盈利的重要产品 Model 3 遇到困难，而且状况很不透明；另一方面，特斯拉还在不断的制造噱头吸引注意力，让投资人放心：前不久发布的电动卡车特斯拉 Semi 宣称了一个全新的商业模式，但交付只有笼统的两年后；新跑车 Roadster 宣称“10000 牛顿·米轮上扭矩”，但是实际上这是一个宣传小把戏，这个数据是把汽车企业通用的扭矩性能参数乘以变速器齿轮比得到的，事实上它没有问题，但行业里没有人这么说，这仅仅是一种宣传手段罢了。

换句话说，特斯拉现在极度依赖伊隆·马斯克的公信力，投资人和公众对这家公司的期待和容忍度不寻常的高。
