微软强化 Game Pass 订阅服务，主机销量不再重要_游戏_好奇心日报

顾天鹂2018-01-25 07:18:21

后主机时代？

去年 6 月开始，微软推出了名为 Xbox Game Pass 的订阅服务，用户每月付费 10 美元就可以在 Xbox One 主机上玩到 100 多款游戏。不过这些产品大多是古早游戏和独立游戏，无论对于核心用户还是刚接触主机的玩家来说，都算不上有吸引力。

如今微软计划改变这一情况，大幅提升 Game Pass 的含金量——主管 Xbox 的 Phil Spencer 在博客中写道，这个计划就是让新游戏在全球发售日同期登陆 Game Pass，这不但包括《盗贼之海》、《除暴战警 3》和《腐烂国度》这样的新作，还有微软工作室还未官宣的作品，以及未来的《光晕》、《极限竞速》、《战争机器》等系列。

也就是说，以后所有微软第一方工作室的作品都会在第一时间进入订阅服务。“有了这些新添加的大作和新的订阅套餐，我们给了粉丝更多的选择和价值——不但可以永久购买游戏，也可以低廉的价格通过 Game Pass 获取它们。”

订阅价格确实很具吸引力。目前推出的套餐是 6 个月 59.99 美元。相当于你付出一部主机大作的费用，就可以畅玩 6 个月游戏，其中可能还有很多新作。如果你本就打算购入某部新作，那么这个附加值惊人的套餐看起来很划算。

与其说是微软意图拯救 Xbox One 的销量，不如说他们选择了逐渐看淡主机之争。

在本世代主机的竞争中，相比于卖出 7360 万台的 PS4 ，Xbox One 的市场占有率显然落于下风，现在想在销量上赶超已经不现实。因此微软最近在想别的办法避开这一销量劣势。从去年起，微软就开始挖掘其他模式，不再用主机出货量衡量得失，而是着眼于游戏销量、月活用户和订阅。

当然，他们仍然想卖出更多主机，但后者已不是首要目标。如果强化后的 Game Pass 真的带动了主机销量，那其实也只是附带福利。从最近的决策来看，硬件和平台的影响被淡化了，他们不太在意用户来自哪儿，主机和 PC，此时没什么差别，只要你在玩微软的游戏，你就是他们想留住的对象；游戏部门不再以单卖主机为己任，他们同时还在计划扩张和收购工作室，资源也向非硬件领域倾斜，可以说是意图回归销售“软件”和“服务”的公司核心。

让大作登陆 Game Pass 不是多么革命性的举动，但是一个可持续的计划。它将强化微软的价值，用低价月费的方式让第一方游戏始终保持着曝光度，也会让不少主流玩家为了优惠重归 Xbox。微软似乎在 PS4 和 Switch 统治市场的情况下找到了一个间隙。

也许他们正在为步入后主机时代做准备——如果微软的云端服务最终能在技术上让游戏在线加载达到 Netflix 的水平，那么游戏分发的方式将被彻底改变。现在让新作进入订阅模式，也是逐渐改变主流玩家消费游戏的习惯。
