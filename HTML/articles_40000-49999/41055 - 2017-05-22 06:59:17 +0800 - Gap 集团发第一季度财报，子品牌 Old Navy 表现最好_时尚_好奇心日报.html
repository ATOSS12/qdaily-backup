Gap 集团发第一季度财报，子品牌 Old Navy 表现最好_时尚_好奇心日报

张田小2017-05-22 06:59:17

Old Navy 的销售增长主要来自美国市场。

5 月 18 日， GAP 集团发布了 2017 年财年第一季度财报。净销售额为 34 亿美元，与 2016 年同期持平。但净收入有上升，为 1.43 亿美元，2016 年同期这个数字为 1.27 亿美元。

“我们很满意这一季度集团的表现，” GAP 集团总裁兼首席执行官 Art Peck 说，“我们已经在产品质量和合身上做了调整，我们也能更能对流行趋势有反应，并满足消费者需求。”

财报特别提到了 GAP 集团 3 个主要品牌 Old Navy Global 、Gap Global 和 Banana Republic Global 的表现。不过没有公布具体的销售数字，只说了 3 个品牌的同店销售增长率。

其中，表现最好的是 Old Navy Global ，同店销售额上升 8%，在 2016 年财年同期这个数字下降了 6%。

另外两个品牌 Gap Global 和 Banana Republic Global 依然表现不太好，同店销售额都分别下降了 4%。不过，Banana Republic Global 还算有点好转，在 2016 年财年同期这个数字下降了 11%。

美国市场（包括了美国、波多黎各，关岛）仍然是 GAP 集团最大的市场。根据其 2014 财年到 2016 财年的财报数据来看，美国市场的销售额都占到其总销售的 77 % 。

这一季度， GAP 集团美国市场的销售额为 27.3 亿美元，占到其总销售额的 79%。

另外，对于 GAP 集团来说，一直单独列出来的加拿大市场也比较重要。在 2017 年财年第一季度，加拿大市场以 2.34 亿美元占到总销售的 7%。

尽管 GAP 财报里没有直接写出表现比较好的 Old Navy Global 具体是在哪里卖得最好。但从 GAP 集团各个市场的销售情况来看， Old Navy Global 的销售增长主要来自美国市场。

只有在美国市场和加拿大市场 Old Navy Global 的销售额超过了 Gap 品牌。在美国市场，Old Navy Global 的销售额是 14.26 亿美元，Gap Global 的销售额是 6.68 亿美元；在加拿大市场，Old Navy Global 的销售额是 1.11 亿美元，Gap Global 的销售额是 7700 万美元。

但在为其总销售额贡献较少的欧洲市场、亚洲市场和其他市场，拉动 GAP 集团销售额的还是 Gap Global 。

不过，这也不算什么意外的结论。从财报给出的关于店铺面积的数据来看，GAP 集团在美国市场和加拿大市场在重点发展 Old Navy Global ，其次才是 Gap Global 。

在 2017 年财年第一季度，Gap Global 在北美市场一共有 835 个店铺，店铺总面积有 860 万平方英尺。

而 Old Navy Global 在北美市场一共有 1047 个店铺，店铺总面积则有 1750 万平方英尺。
