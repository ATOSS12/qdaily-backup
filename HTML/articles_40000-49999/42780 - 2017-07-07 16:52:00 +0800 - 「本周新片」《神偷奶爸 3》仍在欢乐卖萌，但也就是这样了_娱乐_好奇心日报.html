「本周新片」《神偷奶爸 3》仍在欢乐卖萌，但也就是这样了_娱乐_好奇心日报

王珊珊2017-07-07 16:52:00

祝你在欣赏《光荣与梦想——我们的中国梦系列公益片》后看得愉快。

《神偷奶爸 3》

  • 导演: 凯尔·巴尔达 / 皮艾尔·柯芬
  • 主演: 史蒂夫·卡瑞尔 / 克里斯汀·韦格 / 崔·帕克 / 米兰达·卡斯格拉夫 / 拉塞尔·布兰德
  • 片长：90 分钟
  • 格式：3D/IMAX3D/中国巨幕

正统系列的第三部，比纯粹卖萌的番外《小黄人大眼萌》要完整，但几条主线剧情也都和第二部相似，是为了让角色魅力持续发挥的衍生，一方面经不起严密的推敲，没有崭新的核心主题；但另一方面也从头到尾都散发着热闹、缤纷、完全无害的轻松气场。

系列最令人愉悦的仍然是那些角色们：小黄人的每次出现，几乎可能都会令喜欢它们的人微笑或大笑；三个女孩这次的任务，是学习和“妈妈”露西相处；而格鲁则负责科技产品展示、进行华丽的童趣冒险、以及控制整体节奏。

在中文配音版本中，大张伟会为“大反派”巴萨扎·布莱德配音。这实际上就是个傲娇的大男孩，大张伟的形象几乎能完全地代入。

给一个建议：如果你喜欢第二部，那么第三部也很适合你，反之也是如此。

《绝世高手》

  • 导演: 卢正雨
  • 主演: 卢正雨 / 郭采洁 / 范伟 / 蔡国庆 / 陈冲
  • 片长: 116 分钟
  • 格式：2D

《绝世高手》可以被看作是一部周星驰的粉丝电影——导演卢正雨，此前曾担任《西游·降魔篇》的编剧和《美人鱼》的编剧及副导演，此前拍摄过短片版《绝世高手》和《大侠卢小鱼》。

剧情讲述的是混混卢小鱼，在美食江湖的冒险故事。卢正雨自己饰演男主角小混混，配角包括郭采洁、范伟、陈冲 。

《绝世高手》对于周星驰的搞笑方式痕迹太过浓厚，从剧情上来说也几乎是《食神》和《功夫》的混合体。它的一些笑点的确是成功的，甚至也有零星不俗的创意，但是它从核心来说缺乏自己本质性的原创。如果卢正雨能够把自己提到致敬偶像前面，结果可能会好一些。

蔡国庆饰演的头号反派“和膳集团蔡老板”，其实是一个亮点。

《京城 81 号Ⅱ》

  • 导演: 钱人豪
  • 主演: 张智霖 / 梅婷 / 钟欣潼 / 耿乐 / 邬君梅
  • 片长: 93 分钟
  • 格式：2D/3D

又一部“鬼片”。

2014 年上映的《京城 81 号》是当时国产惊悚电影中投资最高的影片，影片的投资近亿，而它的最终票房突破了 4 亿元。这个成绩刷新了国产惊悚片的最高票房纪录。

不过影片剧情遭到了疯狂吐槽，豆瓣上只有 4.9 分的评价。有观众对它的评价是：不像一部恐怖惊悚片，反倒是一个狗血情感剧。

这句话放在第二部上同样也是适用的。电影围绕一个所谓的“尸婴案”展开，由于太过混乱的剧情表达，和为了吓人而生搬硬造的尴尬画面，整部片相对于令人觉得恐怖，可能还更加令人觉得好笑。
