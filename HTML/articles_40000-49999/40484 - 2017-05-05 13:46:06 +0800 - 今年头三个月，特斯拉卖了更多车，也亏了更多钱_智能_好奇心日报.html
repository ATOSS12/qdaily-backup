今年头三个月，特斯拉卖了更多车，也亏了更多钱_智能_好奇心日报

马若飞2017-05-05 13:46:06

投资人都在等特斯拉新车 Model 3。

美国电动汽车公司特斯拉公布了截至 3 月 31 日的 2017 年第一季度财报。总的来说，特斯拉目前卖的 Model S 和 Model X 毛利润很高，但公司为新车花出去和即将花出去的成本也很高。

财报显示，一季度特斯拉营收 26.96 亿美元，同比去年的 11.47 亿美元涨 235%。其中汽车业务增长 123%，收购 Solarcity 后组建的太阳能部门 Tesla Energy 营收增长 841% 营收 2.14 亿美元，服务和其他业务增长 196%。

营收增长是因为卖出了更多的车和太阳能电池。第一季度，特斯拉交付了超过 25051 辆车，同比增长 69%，包括 1.35 万辆 Model S 和 1.12 万辆 Model X。

同时，特斯拉也在亏更多的钱。从毛利率来看，一季度汽车业务为 27.4%，比去年同期 24.0% 略高。整体毛利润 6.68 亿，也高于去年同期的 2.52 亿美元。但是，因为特斯拉正在花更多的钱造电池工厂，以及为量产廉价车 Model 3 做准备，一季度整个公司净亏损 3.97 亿美元，其中包括收购 Solarcity 带来的 1 亿美元亏空，与去年同期的 2.82 亿美元相比继续扩大。

CEO 伊隆·马斯克在财报电话会议上反复强调，公司的未来是新车 Model 3。他说，为了快速生产金属车身，公司从德国进口的 Schuler 牌巨型冲压机正在组装，这种机器安装需要六个月，就是在给快速生产 Model 3 做准备，他同时说，用来生产电池的千兆工厂 1 号（Gigafactory 1）也已经为 Model 3 的动力部件生产做好准备。特斯拉表示，Model 3 会从 7 月开始量产，初期产能可能会是每月 1000 台，但是每个月会按照 2000、4000 台的指数速度增长，最终达到月产 20000 台的水平。

从财报的现金流情况来看，特斯拉现在手里有很多钱，做了很多准备工作，但是还没看到成果。一季度财报显示，特斯拉自由现金流为 -6.22 亿美元，这其中在运营上花了 7000 万，在 Model 3 量产准备、弗里蒙特总装厂和千兆工厂的扩容上花了 5.53 亿美元。同时，包括 3 月份在股市上新募集的 11.5 亿美元，特斯拉拥有现金超过 40 亿美元，是历史最高水平。

但很明显的是，投资人对特斯拉这种很激进的策略有点摇摆，马斯克也明白这一点。在电话会议上，马斯克可能是为了给投资人更多信心，披露了更多新车的信息。

马斯克第一次表示，公司的下一款新车 Model Y 会在 2019 或 2020 年推出。Model Y 是特斯拉之前公布的新车，但目前信息很少，只知道是一款紧凑型的电动 SUV。曾有媒体针对 Model Y 做出了假想图，认为就像 Model X 是从 Model S 上衍生出来的，这个假想图看起来就像一台抬高了底盘的 Model 3。但马斯克这一次说，Model Y 会是一款全新的车，和 Model 3 没有关系。另外，他还表示预计 9 月份发布的电动货车利润不错，因为和 Model 3 会有很多共用的零件。

财报发布之后的交易日，特斯拉股价下跌 15.56 美元，报 295.46 美元，跌幅为5%。

有分析师表示，新车 Model 3 没有明确的路线图令投资人担心。但巴克莱分析师 Brian Johnson 说的问题似乎更直接：特斯拉可能面临奥斯本效应(Osborne Effect)，也就是新产品的热烈反响会影响老产品的销售。尤其是 Model 3，这个名字可能让消费者失望，此前他们认为这车是 Model S 的下一代，然而实际上并不是。

实际上，马斯克在电话会议里提到了这一点。他说，这款车原本的名字 Model E 是一个梗，Model S、Model X 和 Model E 刚好组成了 SEX，但福特占有了 Model E 这个名字，而新名字 Model 3 是个坏主意。
