电商做时尚的又一次失败，亚马逊真人秀被砍了_智能_好奇心日报

唐云路2017-05-31 14:12:05

亚马逊的电视剧和电影都做得风生水起，怎么沾上时尚就不行了？

亚马逊内部的一个电视购物团队，上周五突然被告知全员都被裁了。这个团队虽然不太大，但是也汇集了《早安美国（Good Morning America）》的主持人Rachel Smith,，MTV 的直播节目《绝对现场（Total Request Live）》的主播 Lyndsey 等知名的电视制作人。

那是亚马逊去年三月推出的首档时尚真人秀 Style Code Live，这档节目时长 30 分钟，每周的周一到周五，美国东部时间晚间，在亚马逊的视频平台 Amazon Video 上免费播出。

上周五，这档节目删除了在社交网络上的账号，员工则被一个临时召集的全员会议上被告知这份工作没有了。

Pagesix 援引知情人士消息称，不再有新的节目播出，不过团队的薪水会付到七月，来处理现有的订阅以及其他的后续事宜。

对于员工们来说，这个消息令人震惊，整个项目的结束也相当突然。

Style Code Live 算是在电视购物上的开拓性尝试，每期会邀请不同的嘉宾，设计师、时尚博主、时尚顾问、明星等，就是那些在 Twitter、Instagram、YouTube 上有一大批粉丝的时尚界意见领袖。请他们做客节目访谈，提供美容时尚建议。

另外观众可以实时提问，和主持人以及嘉宾互动交流，节目中推荐的商品在视频下方就可以顺带点击购买。

《欲望都市》的主演莎拉·杰西卡·帕克、考特妮·卡戴珊、《摩登家庭》的主演之一阿芮尔·温特等都被请来上过节目。这也是亚马逊为流媒体视频平台 Amazon Video 推出的首档脱口秀节目。

不过，亚马逊也有大把的原创节目被砍掉过，比如播了一季就被砍掉的电视剧 Mad Dog。

亚马逊还从 Condé Nast Entertainment 手中买过一个真人秀节目“The Fashion Fund”的版权，《Vogue》主编 Anna Wintour 领衔导师团，当然，亚马逊的用户也可以直接买到真人秀里设计师的作品。

在 Style Code Live 之前，亚马逊在影视上的尝试都还算成功，剧集《透明人生》拿下了两项金球奖和 11 项艾美奖提名，电影项目今年还拿下了奥斯卡最佳影片的提名，但是在时尚领域，亚马逊一直处于“很用力却就是差点意思”的状态。

过去两年，亚马逊先后赞助了印度时装周及纽约时装周，并且表达出希望与 LVMH 等奢侈品集团合作的意愿。最近它们还推出了自有的内衣品牌，亚马逊正在尝试一条更符合大众电商平台定位的路线：尽量多地增加时尚产品的细分品类，并且主打基本款和性价比。
