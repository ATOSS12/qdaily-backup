特斯拉想让 Model 3 预订者转买 Model S，可能因为后者更贵_智能_好奇心日报

张智伟2017-08-19 07:07:21

售价更贵的豪华车型 Model S 利润更高，产能也没问题。

美国小有名气的社区网站 Reddit 上最近有部分 Model 3 预订用户发帖称，收到了来自特斯拉推荐购买 Model S 的邮件。

Model 3 是特斯拉上月初宣布进入量产阶段的平价款电动车，最低其售价只要 35000 美元，比特斯拉的豪华车型 Model S 要便宜不少。

按照这封邮件中的描述，特斯拉的销售人员向 Model 3 预订者罗列出了几个不该继续等待量产车交付的原因。比如 Model 3 的平均售价为 57000 美元。

这个数字其实是默认算上 Model 3 所有选装配件之后的平均售价，具体到单个预订者情况并不一样。

另一个表述不详的点是，邮件发送人称到 2017 年底，每台 Model 3 补贴的 7500 美元将会失效。事实上，只有当特斯拉在美国的电动车销量超过 20 万之后，政府每台车补贴的 7500 美元才会减少一半到 3750 美元。

对于不了解详情的人来说，这封邮件可能造成诸多误导。特斯拉向汽车博客 Electrek 回应称，公司并不认同这封邮件里的说法，但也没有就此事有更多动作。

可以确定的一点是，特斯拉的员工并不会从每一台售出的电动车中赚取佣金。不过这封邮件的存在还是给特斯拉带来了一些负面影响。那些早早预订了 Model 3 的用户随时可以撤销订单。

7 月底，首批 30 辆 Model 3 量产车交付之后，特斯拉很快就宣称预定量超过了 50 万台，但每天有大约 1 万人取消预订。

到现在为止，特斯拉还没彻底解决 Model 3 的量产问题。
