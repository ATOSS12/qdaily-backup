推迟一年，亚马逊的无人零售店终于开业了_智能_好奇心日报

张智伟2018-01-22 11:50:14

想从 Amazon Go 无人店里偷东西几乎是不可能的。

位于亚马逊西雅图总部的无人零售店 Amazon Go 本周一正式开业。

这个占地不到 170 平米的店面，原计划在 2017 年初对公众开放。一年时间过去，现在的 Amazon Go 店面和整套系统仍然处于 beta 测试阶段。

按照多家海外媒体的购物体验，到 Amazon Go 商店买东西先要在手机上下载安装 Amazon Go 应用。登录个人的亚马逊账号之后，会自动生成一个专属二维码。

跟国内某些城市地铁开通的微信扫码进站相似，这个二维码要在 Amazon Go 商店入口处的闸机上刷一下才能进到店内。

和普通超市不同，Amazon Go 并不提供购物车，也没有需要排队的收银台和收银员。进到店里的消费者可以自助选用纸袋或印着 logo 的手提袋，从货架上取下的任何商品都会被店内部署的摄像头和传感器系统自动识别并添加到对应的亚马逊线上购物车中。

如果把手提袋里的商品放回到货架上，购物车里对应的商店也会被自动删除。

购物结束后走出店外，Amazon Go 会通过已经绑定的银行卡自动结账。同时手机 App 里也会将此次购物的详细清单列出来。

这背后的整套技术由亚马逊 AWS 的 Amazon Rekognition 提供支持。它可以根据面孔、人体动作、商品标签和位置等信息实现批量处理和实时分析，从而保证结账时不会出错。

科技媒体 TechCrunch 记者在店内拍摄的照片显示，Amazon Go 商店的天花板上布满了摄像头。这让店内的任何一个角落都处于实时监视之中。

纽约时报的一位记者试图从店中“偷”一排四联包的香草味苏打水，走到店外发现他的账户还是被扣除了 4.35 美元。

目前，Amazon Go 商店主要提供三明治、沙拉、饮料等即食食品，餐具也是亚马逊定制的。除此之外，店里还能买到小瓶装的啤酒、葡萄酒和部分肉类。杂货商 365 Everyday Value 提供的薯片、饼干和坚果也有一个专门的货架。

虽然没有收银员，但 Amazon Go 还是有员工常驻店内的。其中包括入口处的一位迎宾人员，酒类货架附近的身份证核查员（美国法定饮酒年龄是 21 岁），以及厨房内的 6 名制作即食食品的工作人员。

总起来看，亚马逊还在测试 Amazon Go 背后的整套系统。这家电商巨头并没有公布接下来的策略是继续开更多 Amazon Go 商店，还是将这套系统应用到收购来的 Whole Foods 店面中。
