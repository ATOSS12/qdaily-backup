特斯拉提前半年发货 Model 3，造车当真那么顺利吗？_智能_好奇心日报

罗骢2017-07-04 17:41:13

更像是缓解难看销售报告的一个秀

以往每款新车都要跳票一两年的特斯拉，终于早到了一次。

特斯拉 CEO 马斯克昨天在推特上宣布：备受市场期待的廉价车 Model 3 比预计提前了两周通过监管审批。特斯拉开始计划在本周五完成第一辆量产车的生产。

这时间点比去年发布 Model 3 的时候，足足早了半年时间。

Model 3 是特斯拉最重要的产品，决定着它是不是能从一个年产 5 万的小公司，变成福特、通用等年产过百万辆的巨头。

此前 Model S 和 Model X 虽然都是电动车，但也主打性能，而非环保。定价也相对较高，最低配置的 S75 的售价基本和奔驰 E 级以及宝马 5 系在同一级别。

而 Model 3 的售价仅为 3.5 万美元，这比通用公司和福特公司的新车平均售价都要低，还没有算上各州的政府补贴。较低的售价让 Model 3 备受市场期待，美国的新车售价平均为 3.1 万美元，在北美市场上只有一款车型在大小和售价上与 Model 3 相当，那就是年销量达到 10 万辆的宝马 3 系。由此可见 Model 3 背后的市场。

作为特斯拉最重要的一款大众化乘用车，Model 3 的产量直接决定了特斯拉是否能实现真正盈利。

吸取了量产 Model X 多次跳票的教训，马斯克减少了首批量产的 Model 3 的定制选项。只有基本的颜色和轮毂规格可以选择。

从生产来说，生产电池的特斯拉超级工厂建设顺利。6 月份，2170 新型电池就已经在工厂里投入生产，随着 Model 3 大规模量产后电池组成本还将下降，逼近 125 美元/kwh。

但 Model 3 并没有真正提半年大批到来就是了。

虽然第一批在 7 月 28 日就会到，但首批只有 30 名辆车。

之后一整个 8 月，Model 3 的量产计划仅仅是 100 辆，到 9 月才会计划爬坡到 1500 辆。

而雪佛兰已经在一季度卖出了 2735 辆号称 Model 3 杀手的 Bolt 电动汽车。

特斯拉在赶时间。对比之前两款车的量产计划，特斯拉这次明显加快了 Model 3 的生产计划，甚至不惜牺牲第二季度的销售量。把其他生产线让出来提高 Model 3 的电池组产量。

这次宣布提前量产的计划正赶在特斯拉发布不好看的二季度销售数字。今年 4 - 6 月，特斯拉交付了 2.2 万辆电动车，低于一季度的 2.5 万辆。

特斯拉解释说不足的原因是为了 Model 3 上使用的新电池加速生产腾出空间。

这不是特斯拉第一次根据财务数据调整生产节奏。

根据《好奇心日报》早先对特斯拉汽车销量和季报做了一组对比：整个 2016 年，特斯拉在每个财报截止的那个月，交付数量都会有不同程度的激增，然后下个月陡跌。而这个现象在其他电动车销量上并没看见。合理的解释就是马斯克会有意将原本下季度交付的车往前赶，让当季的数字更好看。

迎合财报节点，争取股价影响是几乎所有上市公司都做的事，但特斯拉格外需要一个高股价。

虽然早先几款车都很贵，但成立至今，特斯拉一直没有盈利。因为特斯拉在完成现有订单上的投入很大，建造电池工厂、筹备 Model 3 量产。一季度整个公司净亏损 3.97 亿美元，其中还包括收购 Solarcity 带来的 1 亿美元亏空。这和去年同期的 2.82 亿美元相比来说继续扩大了。

相比起不断投入的成本，特斯拉几乎没有利润可言。为了不影响 Model 3 的顺利量产，特斯拉在今年 3 月通过增发 2.5 亿美元的普通股和 7.5 亿美元在 2022 年到期的优先兑现债券筹备了 11.5 亿美元。对于特斯拉来说，继续保持股价较高的增长才能通过出让股份来筹到更多的资金。

但这些技巧影响财报、融资都是一时的。最终决定特斯拉能不能成为一线大车企，还是看 Model 3 的量产情况。一年造 50 万辆大众车型，可比一年三五万辆高端车难多了。
