奔驰推出最新超跑，基本上就是一台装了外壳的 F1 赛车_智能_好奇心日报

马若飞2017-09-13 14:19:42

1.6 升排量，动力 1000 匹马力。

正在举行的法兰克福车展上，梅赛德斯·奔驰展出了旗下最新的超级跑车 AMG Project One。简单的说，这辆车就是一台装了普通车外壳的 F1 赛车。

奔驰 AMG Project One 已经披露了很久，但之前一直没有公布完整的车辆外形细节，只是推出了很多段广告宣传材料。这些广告里的主要形象都是梅赛德斯 AMG F1 车队的现役车手路易斯·汉密尔顿，显然是强调这辆车和 F1 赛车的紧密关系。

法兰克福车展正式公开的 AMG Project One，外形似乎成了最不重要的部分：和其他品牌的超级跑车相比，AMG Project One 外形看起来并不出格，论惊艳绝对比不上法拉利的 LaFerrari，Project One 引擎盖上的奔驰徽标甚至都只是印上去的。

最重要的部分是，这台车的动力来源就是脱胎于 F1 赛车。

AMG Project One 的内燃引擎只有 1.6 升排量，V 型六缸结构，带有涡轮增压。曲轴和涡轮增压器上分别安装了一台电机，车辆不仅可以回收刹车时的动力，还可以回收涡轮增压器多余的能量，回收的能量可以在较低转速时驱动涡轮消除涡轮迟滞，也可以直接驱动曲轴增加车辆动力。

和 F1 稍有不同的是，这辆车的电池部分会大一些，在前轴安装了额外两个电动机，组成了混合动力四驱系统。AMG Project One 的排量可能比有些家用车还小，但在 11000 转每分钟的工况，以及前轴电机的帮助下，整车可以输出 1000 匹以上的马力，200 公里加速成绩只需要 6 秒。

实际上，除了前轮的电动机，AMG Project One 的动力系统就是一台降低了部分性能要求的 F1 引擎。它的转速从赛场里的 15000 转最高下调到 11000 转，不再需要专用燃料，不再需要提前热机和特殊润滑油，能满足拥有者把 F1 赛车开上路的心愿。但是即使降低了性能需求，这台发动机也需要在 50000 公里里程时进行彻底拆解维护，一般的家用车引擎一般终身是不需要这么做的。

AMG Project One 的变速箱不是 F1 的序列式变速箱，是一台专门设计的 8 速手动变速箱。这台车售价 272 万美元，配额 275 台，早已预定一空。

AMG Project One 基于 F1 技术，但也提供 F1 没有的功能，比如纯电动行驶。AMG Project One 可以在纯电动模式下行驶 25 公里，但和一般的“混合动力”汽车其实不一样，家用车的动力回收一般只有刹车，没有涡轮废气动力回收；Project One 的回收的电力主要用来驱动引擎的涡轮增压器，以便让这台“小排量”引擎在较低转速输出足够的动力。

虽然 AMG Project One 可以按照法规上牌上路行驶，不过大概不会有车主真的拿它来游车河。和过去的超级跑车相比，AMG Project One 更像是一台实验作品，验证的是 F1 技术上路的可能性。然而实际上，现行的 F1 规则下，赛场中使用的，也就是 AMG Project One 上使用的混合动力技术，其实对民用车没有太多借鉴意义。尤其是全世界车企都在转向电动化的未来，F1 的技术前瞻意义会越来越小。
