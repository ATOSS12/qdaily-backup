耐克发布新跑鞋，鞋底是一整块没有粘合剂的泡棉_商业_好奇心日报

朱凯麟2018-01-30 07:01:36

看起来是用来回击阿迪达斯 boost 的重要产品

1 月 23 日，耐克邀请媒体发布了一双花 3 年时间研发的跑鞋 Nike Epic React Flyknit，定价 1299 元。

创新的部分主要在于中底泡棉的材料 “Nike React”，据耐克表示，这种材料兼具了缓震、回弹、轻质、耐久 4 种特性。

事实上，Nike React 早在去年 6 月就被运用在篮球鞋上，当时我们就曾报道，这款鞋底看起来会是耐克用来回击阿迪达斯 Boost 的重要底牌。但由于篮球鞋需要外壳保护，这双勇士队球员 Draymond Green 上脚的篮球鞋底外包过了一层外壳，你看不到里面材料的样子。

休闲跑步鞋才是如今运动市场最主要的增长动力，因此 Nike React 被应用到跑步鞋上更令人期待一些。

和 Boost 技术来自化工巨头巴斯夫不一样，Nike React 是一种耐克内部自行研发的材料科技，经过数百次化学配方的微调。

“坚硬的材料往往能获得最大的能量反馈，这就是为什么传统的泡棉只能提供缓震或能量反馈。也是为什么只能满足轻盈或是耐久。但是通常它无法将这些性能集于一身。”耐克跑步鞋类产品创新总监 Ernest Kim 在接受《好奇心日报》采访时解释，Nike React 可以兼具这些特性。Kim 一边解释，一边在纸上画了一个火上烤的棉花糖。

发布会现场，耐克用枕头、海绵、弹簧这些道具试图视觉化鞋底材料带来的体验，这些道具捆绑在一起的造型也成为 Nike Epic React Flyknit 的广告宣传画面。徐濠萦、乔欣、李荣浩、运动员韦永丽、演员瑞玛席丹则出席了发布会，并为这双鞋拍摄了一些广告发布内容。

事实上，Nike React 材料是对此前运用在耐克跑鞋新品 LunarEpic 上的鞋底技术 Lunarlon 的一次升级，增加了 13% 的能量反馈，更重要的是，Nike React 据称最大的提升是在耐久度这一特性，而 Lunarlon 在耐久这一块做得不算太好。

Nike React 用到的泡棉，实际上和去年耐克发布的 ZoomX 马拉松跑鞋使用的都是泡棉，但配方略有不同，主要提升了耐久度。这和两双鞋的定位有关，Nike Epic React Flyknit 旨在面向那些日常的跑者，而 ZoomX 目前应用的 Vaporfly 马拉松鞋依然货量稀少，定位相对更高端。

在设计上，这双跑鞋的目标是要“像鞋垫上的一双拖鞋”，看上去越轻越好，颜色则选择了粉、蓝、白这些中性色，这是为了覆盖更多的消费者。“Nike React 比之前耐克跑步所测试过的任何泡棉都更耐久，意味着它将能够满足长距离跑者的需求。”Kim 说。

如果你最近两年光顾耐克的门店，会发现它们的跑鞋产品被许多中低价格的鞋款充斥，1000 元以上的 Zoomfly、Vapormax 则都推出市场不超过一年。这也导致了耐克在阿迪的 Boost 火了三年的时候，一直都没有一款可以与之抗衡的、应用在全线跑步产品上的中底科技。

Ultra Boost 1.0 刚刚发售时的定价是 1399 人民币，后来换上了马牌大底，价格又涨了 100 元。而所有应用了 Boost 科技的阿迪运动鞋，价格都在 1000 元以上。创新的中底科技是运动品牌鞋类产品高毛利的基础，而耐克显然对 Nike React 寄望挺高。

从鞋底纹路的辨识度来说，Nike React 看上去没有 Boost 球面颗粒那种完全令人耳目一新的感觉。但 Ernest Kim 说，这些纹路实际上是借助了计算机设计团队，进行流体几何学设计的结果。“过去最多上市前最多只能做四次（不同设计）的测试，现在可以做几百次。”

Nike Epic React Flyknit 跑鞋将自2月22号起对 Nike.com 会员发售，3 月 1 日开始上架零售渠道。耐克希望把 Nike React 打造成像 Boost 那样价值几十亿美元的平台，但它的表现还需要市场来验证。
