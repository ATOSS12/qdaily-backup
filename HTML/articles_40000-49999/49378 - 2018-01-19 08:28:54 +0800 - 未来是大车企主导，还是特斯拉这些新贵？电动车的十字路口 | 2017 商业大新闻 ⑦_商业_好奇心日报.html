未来是大车企主导，还是特斯拉这些新贵？电动车的十字路口 | 2017 商业大新闻 ⑦_商业_好奇心日报

龚方毅 马若飞2018-01-19 08:28:54

巨大资本推动的成长和泡沫，发展过程中积累的财富与问题，捉摸不定的政治走向——20 余年繁荣之后，三者纠结在这一年的商业社会中。虽然热点颇多，但细究起来，商业逻辑却并非公司成败的主导。这是 2017 年商业大新闻的第 7 篇。

上周末结束的 CES 消费电子展，丰田把它的大展位基本都给了新发布的概念车 e-Palette。它大约长 5 米，配 8 个轮子，不分车头车尾，没有方向盘和油门，外形像一个卧倒的电脑机箱。

按照丰田的设想，不同尺寸的 e-Palette 卖给不同合作公司，可以接送人、可以成为送货的工具，也可以充当自动贩售机。

电动、无人驾驶、卖给企业客户而不是个人消费者，这是丰田构想的未来汽车。而不同于以往展出后就消失的概念车，e-Palette 有个时间表——2020。

丰田是目前全球市值最高的车厂，超过 2000 亿美元市值，相当于大众汽车两倍。它也是 2017 年宣布转向电动车的车企之一。

此前丰田虽然也是目前新能源车销售最好的企业，但在 2017 年以前，它谈的“新能源”基本都是油电混合或者氢能。

随着政策迁移，电动已经不可避免。2017 年，多个欧洲国家提出禁止销售燃油车时间表，最激进的荷兰、挪威建议把节点定在 2025 年。中国工信部也在这一年承诺将制定禁售燃油车的时间表。

政策刺激下，大公司赶上来了。大众、丰田、通用、雷诺日产四大车厂合计宣布超过 1000 亿美元的电动车投资，最快在 2019 年量产纯电动车。

早先由于需要顾及传统燃油车业务，大车企在电动车领域动作缓慢。现在随着政策变动，生产、销售渠道都不愁的大车企在 2017 年都转向了电动车，时间表也都定在了 2019 - 2020 左右。

这样一来，留给新公司的时间不多了。

2017 年也是特斯拉量产 Model 3 的节点，这款起价 3.5 万美元的新车是特斯拉最重要的一款大众化电动车，它直接决定了特斯拉是否能实现真正盈利、决定了特斯拉能否成为一个新的大车厂。

然而直到目前，特斯拉依然在其 CEO 马斯克所说的“生产地狱”当中，产量不到计划的 1/10。限制它的，更多是汽车生产的上游。

在想靠电动车“弯道超车”的中国，互联网的钱早已进入了汽车相关的每一个产业。就像《好奇心日报》去年的梳理，从早几年的新车和二手车电商、后装配件生意，再到互联网车险和上门洗车保养……互联网公司无处不在。

到 2017 年，这些配套生意不再是重点，创业公司拿互联网的钱效仿特斯拉造电动车、做无人驾驶。

蔚来汽车是第一个把车拿出来的互联网车企，在 12 月发布了 ES8。但它还没有证明这个车可以批量造出来。其它公司距离生产更远。

大公司要进入电动车市场，但新公司还没准备好。

从刚结束的 CES 可以看出，“电动”对于大车厂已经不再是概念，而是生产计划的一部分

今年 CES 汽车馆里，丰田的展位面积最大。它刚发布的 e-Palette 是其展区主舞台上唯一的展示物。

这款车全程不要人干预，自动驾驶。丰田希望今后能将它用于拼车、送外卖、共享汽车、短途接驳、自动售货或者工厂机器人等多个场景。

车辆实现自动驾驶后，丰田从所有权变化开始，考虑汽车业的未来。届时机器取代人类司机，在一个布满无人驾驶车的社会里，人可能不再需要拥有汽车。

少数几家大公司已经同丰田建立了合作关系以推动这个项目，包括 Uber、滴滴、必胜客和亚马逊等已是其启动客户。丰田打算在 2020 年东京奥运会上启用该车型。

主舞台下，停着一辆“Project-爱”。那辆会给人情感支持的概念车代表着丰田一年前谈论“电动”的基调：天马行空，短时间内没有实现的可能。现在，丰田有了更实际的计划。

类似的还有其它几个大车厂也带去电动概念车。奔驰电动车子品牌 EQ 的概念车在整个展会期间一直停在主舞台上。它前后各有一个电机，外形和尺寸与奔驰现款运动轿跑接近。设计续航里程跟特斯拉一个水平，约 500 公里。

日产的展区里放着一辆最新款的电动车 Leaf（聆风）。去年 9 月，日产为这个全球销量最高的电动车型号升级换代，新款 240 公里续航里程大约是 Model 3 一半，现场工作人员称该车型已经在全球取得超过 3 万辆订单。

现代汽车展示其在自动驾驶、电动车和模块化生产方面的研究，包括有可能在未来一到三年内大规模生产的产品，其中一项是为电动车设计的模块化平台。

据现代汽车介绍，该平台将电机、电子阻尼器、线控制动技术和线控转向技术结合在一起，安装在汽车的每个车轮内。采用这种技术的电动车不需要置于电动机和传动轴来驱动车轮，底盘腾出来的位置可以放置更多电池。由于没有了轴距限制，设计和工程团队可以用一套动力方案设计出不同车型。

曾经更多在幕后的零部件厂商也因为电动化和自动驾驶走向前台。

博世继续展现其前几年在停车场和车辆端两方面的技术储备：车主只需将车停于停车场入口处，通过智能手机程序发出相应指令，汽车便会自动寻找空位停车。

这套系统可以自动识别并测量已停泊车辆之间的空隙距离，然后将这些信息实时传送到电子停车地图中，司机可以通过该地图找到最近的空位。

还有其他如采埃孚、英伟达和百度共同开发了给自动驾驶车用的行车电脑；从德尔福分拆出来的安波福（Aptiv）公司，与 Lyft 联合推出了无人驾驶服务；大陆集团展出新型轮毂电驱系统，通过集成在轮毂内的电动系统进行刹车，降低传统机械刹车装置的负担。

博世自动驾驶部门副总裁 Kay Stepper 对《好奇心日报》表示，他们内部自动驾驶研发部门，从传感器、油门、刹车、电子控制系统已经开发数年。

而这些零件商的研发进度，决定了大车厂能到哪一步。

“在如今自动驾驶和电动车浪潮中，博世跟其他一级供应商一样，正在与各大车厂沟通，为后者提供包括自动驾驶解决方案、电动车电源管理系统等在内的零部件。”Stepper 说。

世界变电动了，对先行的特斯拉完全不是好事情。

特斯拉拿出首个长续航的平价电动车 Model 3，但它没能调动汽车产业链大批量造出来

电动车最大的问题还是充电后的续航，特斯拉 Model S 是第一辆能让人开上 300 多公里再充电的电动车。

《金融时报》这张图清晰凸显了特斯拉曾经的优势。直到 2016 年，才有传统车厂拿出续航和特斯拉媲美的电动车，此时距离 Model S 发货已经 4 年时间。

同样也是在这张图里，你能看到现在全世界都在谈电动以后，先行的特斯拉将在两年后遇到巨大挑战。

捷豹 i-Pace、奥迪 Q6 e-tron、奔驰 EQ 等车型预计都会在 2019 年能量产，这些车无论是品牌还是续航，都将和特斯拉正面竞争。

以前大公司因为顾及成熟的汽油车业务，所以缓慢转向电动车。但到今天它们一头扎进电动车市场，很显然特斯拉已经没有几年领先空隙了。

Model S 太贵，永远不可能成为主流。面对新的竞争对手，特斯拉的希望在 Model 3 大规模量产上。

2017 年 7 月马斯克宣布 Model 3 量产的时候，为其定下 8 月产 100 辆、12 月产 2 万辆的生产计划。彭博称“马斯克有一个大计划”，并表示在这种生产节奏下 Model 3 产能将取得爆发式增长，而这并不符合现在的造车规律。

但马斯克可能不得不这么做了。因为起价 3.5 万美元的 Model 3，是特斯拉的关键车型。

特斯拉能否在大车厂赶到前兑现产能，决定了它是否有从一个年产 5 万辆汽车的小公司，变成大众、通用、丰田等年产千万辆的汽车巨头。

但 2017 年第三季度以来，特斯拉 Model 3 的量产计划就一直没有达标，年末月产 500 辆只相当于预期的 1/40。

这种实际生产远落后于预期的情况到 2018 年还是没有好转。特斯拉千兆工厂合作方、日本松下在 CES 上透露，特斯拉已经下调 2018 年 Model 3 周产能至 2500 辆，缩水一半。

即便这样，赶工 Model 3 还是影响了特斯拉 Model X、Model S 的生产。前特斯拉员工去年 12 月爆料说这些车型出厂前需要大范围返工，故障大小不一，比如门关不严、软质材料修剪不齐、缺少零件、零件松散带来异响、漏水，等等问题。

而大车企生产新车，产能总是没问题。雪佛兰的 Bolt 已经累积了很可观的销量：2017 年全年，Bolt 交付了超过 2.3 万辆。它的产能也正按照以往汽油车的造车习惯逐步攀升中。

造车很早就不只是车厂一个环节的事。

今天除了发动机、底盘、车身的核心部件，其他如变速箱、刹车盘/片基本都不是车厂自己制造，它们选择成熟的配件商开展合作。

特斯拉没能调用整个汽车业的技术储备。它自己生产电机、铝合金车身和底盘部分，以及塑料部件模具、仪表板和压铸金属部件。轮胎、玻璃、车载、整车线束等较通用的零部件则来自外部供应链。

马斯克自喻为“比今天的福特更福特”，曾经的福特什么都自己做，现在特斯拉也差不多。

不过这更多可能是特斯拉没有那么多可以依靠的生产合作伙伴。汽车业的供应商还没有成熟的方案，让一个车企直接造出续航 500 公里的电动车。因此特斯拉不得不自己做一切。

一家公司带来更好的产品之后更重要的是得造出来。就像早几年的苹果，靠 iPhone 打开了一个新市场，赶在黑莓、诺基亚们反应过来前拿下了正在显著增长的智能手机市场。如果苹果不能大规模生产它，今天它也不会是市值最高的公司。

同样，带来更好产品的特斯拉，现在还没能证明自己可以把车批量造出来。

电动车产能低导致特斯拉即便有再好的设计也难以全部转换成实际收入，马斯克的公司从未实现过年度盈利。随着 Model 3 的生产问题，特斯拉一季度亏 5 亿美元成了家常便饭，债务也越来越多。

电动车的需求毋庸置疑，Model 3 现在有 40 万订单积压。

但以特斯拉现在的产能，这个需求最后不一定是它来满足。

互联网来的钱投遍了中国跟车有关的生意，而它们新看中的整车制造还在追赶特斯拉的阶段

在现在这波持续了两三年的互联网造车热中，中国互联网资本把跟造车有关的每个生意几乎都做了遍。

比如阿里巴巴收购高德地图、在淘宝卖车、开发车联网服务、投资二手车平台大搜车。一样的事情腾讯也做了多年。

但现在这些都消停了。连去年你来我往广告费数以亿计的二手车生意也变了。瓜子二手车开始用“毛豆”的品牌卖起了新车，盈利寄望于再贷款买车的利息上。

互联网公司在汽车行业投资最后还是落到核心，即电动车整车制造。

腾讯 2015 年投资蔚来汽车。这家公司由汽车类垂直网站易车网 CEO 李斌、汽车之家创始人李想等人共同创立，是中国新车企里面最早宣布量产的。

它上月发布的 ES8 是一台主打家庭出行的 7 座 SUV，355 公里 NEDC 续航里程，价格 44.8 万元起。这款车已经拿到工信部的颁布的生产资质，是互联网资金扶持车企里的第一个。

但从一辆车出来，和它被大规模造出来是两回事。特斯拉 2008 年就发布了 Model S。直到 2012 年才造出来，产能还有限。直到 2018 年，特斯拉还在为新平价车型的生产发愁。

这是特斯拉超越汽车业平均技术水平之后，所付出的代价。

蔚来的生产现在交给江淮汽车。很难说江淮汽车能不能造好高价车，但可以肯定的是江淮并没有造 50 万价位车的经验。

在蔚来宣布新车的发布会上，车以外的点子更像是汽车科技新闻集锦，特斯拉试过又放弃的换电池取代充电、亚马逊申请专利的送货到后备箱都被搬了出来。但这些都对解决生产问题毫无帮助，还是概念而已。

而这已经是互联网资金扶持下的中国新兴车企里走得最远的了。其他公司比如威马、拜腾刚展示过准备量产的车型，它们选择自建工厂。还有一些公司比如小鹏、奇点到现在也没有说造车时间。

随着大车企逐渐进入电动车市场、开始无人驾驶的合作，留给新公司的时间是越来越少了。
