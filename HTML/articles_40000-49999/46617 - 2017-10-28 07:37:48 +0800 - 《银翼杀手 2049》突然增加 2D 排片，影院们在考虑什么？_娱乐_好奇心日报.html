《银翼杀手 2049》突然增加 2D 排片，影院们在考虑什么？_娱乐_好奇心日报

韩方航2017-10-28 07:37:48

但他们在这部电影上犯的最大错误，并不在于 2D 还是 3D。

在《银翼杀手 2049》的 3D 版本引发争议之后，不少影院迅速撤下了原本排映的 3D 版本电影，改为放映电影摄影师罗杰·迪金斯推荐的普通 2D 版本。

根据《电影山海经》的报道，在争议出现之后，上海影城的东方巨幕厅将原本已经排定的 3D 场次全部更换为 2D 场次。此后，上海的其他影院，例如天山电影院和朵云轩，也都增加了 2D 场次的排片。

对于这些电影院来说，2D 场次事实上已经成为了招揽观众的一种方式，就像一年前李安的《比利林恩的中场战事》上映时，4K、3D、120 帧得到的待遇一样。位于北京的博纳影院悠唐店，就发微博打出了 2D 版本上映的噱头。

不过，并不是所有影院都做出了改换 2D 排片的选择。

以下三张图分别显示了三家不同的电影院。虽然都是 10 月 29 日周日的排片，但是第一家影院选择了 2D 和 3D 混排的形式，第二家影院则仍然是全 3D 排片，而只有最后一家影院将所有的排片都换成了 2D 版本。

在这其中，第一家影院的策略有些耐人寻味。从下图可以看到，这家影院在周六将所有的大厅都留给了 3D 版本，小厅则放映 2D 版本。到了周日，排片就完全反了过来，大厅改放 2D，小厅改放 3D 。

在该影院，3D 票价要比 2D 票价贵了 15 元到 20 元不等。而从出票情况来看，也并不是所有观众都自发抵制 3D 排片。在这种情况下，影院面临的是一个两难的选择。2D 排片也许上座率高，但票价低。3D 排片也许上座率低，但票价高。究竟哪一种策略会带来最大的收益，目前看来并不明确。这可能也是这家影院将 2D 和 3D 版本混合编排的一个原因。

当然，也有影院通过其他方式规避这一问题。如下图所示，这家影院在同一时间的杜比全景声影厅的 3D 和 2D 票价保持相同。而在 10 月 28 日这一天，同为 VIP 厅的 2D 和 3D 票价也保持一致。

理论上来说，影院的排片和定价天然具有自主权。只排 3D 或者 2D 与 3D 同价这些行为本身并无问题。

事实上，从《银翼杀手 2049》的 2D 排片增多的情况来看，影院也受到市场规则的限制。当越来越多的观众表达对于 2D 电影的需求时，自然会有一部分影院响应需求并进行调整。只不过，许多影院并没有类似的经验，因此采取了各种不同的应对措施。

不过，在《银翼杀手 2049》这件事情上，影院犯的最大错误可能还不是 3D 或者 2D 的问题。10 月 27 日，电影正式上映的第一天，尽管获得了 20.9% 的排片，但其票房占总票房的比例却仅为 12.5%。而同日上映的灾难片《全球风暴》却以 33.2% 的排片获得了超过一半的票房。

《银翼杀手 2049》并没有影院经理们想象中那样受大众欢迎。
