跟 AMD 合作后，英特尔迅速挖走了对方架构师自己造 GPU_智能_好奇心日报

徐弢2017-11-09 17:04:16

为了人工智能

周一刚跟 AMD 达成新的合作，英特尔转身就挖走了对方 GPU 芯片设计的重要人物。

昨天，英特尔官网上公布，AMD 前首席 GPU 架构师 Raja Koduri 将加入英特尔，进入新成立的核心和视觉计算部门（Core and Visual Computing Group），担任首席架构师、副总裁，并负责推进一个新的边缘运算解决方案的开发。他的正式上任是在 12 月份。

英特尔没有公布 Raja Koduri 所负责的新的部门是做什么的。但英特尔在官网上称，Raja Koduri 将会推进计算、图像和机器学习上的开发，并且这些技术会覆盖到个人 PC 和数据中心。看上去，很可能主要是用开发新的 GPU 芯片做人工智能。

从履历上来说，Raja Koduri 是 GPU 架构设计方面最重要的一小部分人，有 25 年开发 GPU 以及相关领域的经验。他先后在被 AMD 收购的 ATI、AMD 以及苹果公司工作，从事的领域覆盖了 GPU 芯片设计的硬件和软件算法。

在加入到英特尔之前，Raja Koduri 在 AMD 担任 Radeon Technologies Group 的资深副总裁和首席架构师，全面负责 APU 芯片、GPU 计算产品等项目上的图像技术。也是过去多年来，第一次有人在 AMD 同时负责 GPU 的硬件和软件两个部分。

AMD 旗下的这个部门的基础也就是此前收购的 ATI 公司，Raja Koduri 随着这笔收购在 2001 年进入到 AMD 工作，并负责推进包括 Radeon 图像处理器在内的开发工作。

英特尔挖人相当迅速。周一，英特尔刚跟 AMD 达成合作、开发集成了 AMD 的 Radeon GPU 芯片的处理器，这也是时隔几十年英特尔将竞争对手的 GPU 芯片集成到处理器内。周二，Raja Koduri 就宣布了从 AMD 离职的消息，并在后续两天被怀疑、最后被证实加入了英特尔。

英特尔挖来 Raja Koduri 无疑是想加强其在 GPU 芯片上的实力。

虽然靠着集成 GPU 到处理器内，英特尔占据了大量的 GPU 芯片市场份额。但对于游戏有需求的玩家通常会购买 AMD、英伟达的独立显卡。

独立显卡的市场远不及 CPU 大，英特尔原本并不在意。但随着个人电脑业务增长停滞、这也成了一个增长方向。

更重要的是企业对人工智能计算的需求，当中很大一部分是对图片和视频的处理，GPU 不可或缺。英伟达、AMD 都加入了进来，面向自动驾驶公司、云端服务器销售。
