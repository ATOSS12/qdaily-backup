鹿晗与舒淇将扮演拯救地球的任务

一张新的海报在微博上发布，分别引起来自影迷与书迷的关注。

这海报的背景是一栋高楼大厦，玻璃帷幕上反射着东方明珠塔与其他高楼，天空漫布着炮火。这其实是由华视娱乐出品，腾华涛执导的新电影《上海堡垒》。故事改编自作家江南的同名小说。

之所以引发关注，除了来自原先书迷对改编作品的期待，更大一部分的原因是片商正式公布了男女主角人选：男主角鹿晗、女主角舒淇。

《上海堡垒》讲述 2042 年地球受到了外星势力的威胁，外星人企图夺取人类的能源。为了抵抗入侵，各国组成联合军队，但没想到各国的一级大城市接连失守，眼看上海堡垒面临入侵。此时，鹿晗所饰演的大学生汪洋，将追随着女指挥官林澜联手抵抗。

本片的出品方是华视娱乐，他们的特点向来就是以围绕热门 IP 作为发展项目的原则。比方说过去曾出品过的《致青春》以及《平凡的世界》。

在改编作品获得巨大成功之后，华视娱乐就更积极寻找适合的 IP，首选目标就从热门作家开始。

2014 年，华视娱乐的 CEO 王琛来到了成都，当时这里正举办“第九届中国作家榜文化盛典”，而华视娱乐就在现场宣布，五年内将完成 “顶级 IP 视效系列电影计划 ”，会斥资 50 亿，改编《龙族》、《斗罗大陆》、《上海堡垒》等作品，他们所选中的正是江南与唐家三少的作品。

早前王琛接受媒体采访时，就谈过选择 IP 的原则 “在我们的筛选体系里，只有好的 IP 才值得购买，所谓好的 IP 就是必须具有正确的价值观，并且有良好的粉丝基础和较强的稀缺性、可看性。”

无论是所谓的 ”好 IP ” 或是热门 IP 都是目前影视公司追逐的热点，但在一批 IP 热潮之后，有个很清楚的现实是，热门 IP 并不保证一定会大卖，越是热门的 IP ，背后都可能投入相对应更多的资源，这也就意味着在回收上有着更大的风险。

尤其像《上海堡垒》这样的科幻电影，在中国也不多，除了必须考虑影迷在场面或是效果上有着更高的期待之外，还得面临技术水准上的问题。

对此滕华涛似乎显得信心满满，有关制作上的技术问题，他说“中国电影市场发展潜力巨大，我们需要给观众回馈一些有足够工业制作水准的电影。”

目前《上海堡垒》尚未正式开拍， 预计 2019 年才会上映。
