特斯拉第三季度烧掉 14.2 亿，但没解决 Model 3 生产瓶颈_智能_好奇心日报

龚方毅2017-11-03 07:05:51

华尔街这回不信马斯克了。

第三季度财报发布后，特斯拉的市值两天跌掉 65 亿美元。

原因在于它第三季度巨亏 6.7 亿美元，并且在创纪录的烧掉 14.2 亿美元后，其关键车型 Model 3 仍面临严重的产能问题。

其实特斯拉第二季度财务表现也很糟糕，亏损加剧、下半年增加开支、而且还是债务融资，但外界因为 Model 3 看好它。

当时在财报披露前几天，伊隆·马斯克在 Model 3 交车仪式上表示，其产能会在 9 月底达到每周 375 辆、年底达到每周 5000 辆。以至于一些分析师由此已经开始期待特斯拉不远的将来实现盈利。

结果等到这次第三季度财报时，华尔街才知道特斯拉 7-9 月一共只生产了 260 辆 Model 3。这令它们失望，因而抛售其股票。

“虽然我们每周都能在 Model 3 产能瓶颈方面持续取得重大进展，但现在仍很难预料何时能真正突破该瓶颈。”马斯克在财报中说。他现在将每周生产 5000 辆 Model 3 的实现时间推迟到 2018 年 3 月底。

特斯拉将原因归咎于超级工厂造不出足够的电池，以及 Model 3 位于 Fremont 的工厂在焊接和组装工艺上遇到难点，使其慢于其他部件的生产速度、最终影响整个项目的进度。

Investing.com 的分析师 Clement Thibault 周三表示，特斯拉正垂直整合汽车制造业 —— 包括自建许多主要组件，如电池和座椅 —— 但通常这不适用于传统汽车业。

“传统汽车公司已经建立了分散的商业模式。 特斯拉相信它可以集中这个过程，但没有任何实际证据来支持它可以在每个项目以及每个时间周期内完成对应的任务。”Thibault 说。

上个月，特斯拉裁了数百员工，理由是业务能力不达标。马斯克对此在电话会议中表示，公司正在寻求聘用新员工，其技能更适合其生产目标。

工人们表示，特斯拉的生产系统可能导致与重复压力有关的伤害。其他员工说，他们没有经过适当的培训来处理一些危险化学品，并且经历了灼伤、皮疹和眼睛受伤。

特斯拉表示，解雇只是基于绩效考核、根据工会意见，绝不会贸然对雇员采取行动，也不会讨论个别情况。

结果昨天它又裁了 700 人。马斯克则在电话会议中说正考虑为 Model 3 引入更多机器人，以提供生产效率。

第三季度，特斯拉交付了 2.5 万台 Model X 和 Model S，今年计划交付 10 万辆。

然而，特斯拉现在决定在第四季度将两种车型的交付能力减少 10%，然后把这些产能拨给 Model 3。作为特斯拉最重要的一款大众化电动车，它直接决定了特斯拉是否能实现真正盈利。

马斯克宣称 Model 3 产能在 2018 年的某个时间点达到每周 1 万辆产能。但显然这一次，他没有能说服华尔街。
