「本周新片」十一观影指南，有几部可以乐一乐_娱乐_好奇心日报

王珊珊2017-09-30 16:48:23

开心麻花第三部有进步，成龙和王晶都难得可以一看

《羞羞的铁拳》

  • 导演: 宋阳 / 张迟昱
  • 主演: 艾伦 / 马丽 / 沈腾 / 田雨 / 宋阳
  • 片长: 100 分钟
  • 格式：2D/IMAX/中国巨幕

今年目前为止，出色的一部国产喜剧片。

开心麻花第三部根据同名话剧改编电影，不同导演，同一波演员。从讲电影故事的角度，这部有明显的进步：场景调度、节奏衔接都熟练自然，不是话剧而是一部电影；最重要的是相对于《夏》里意淫式的两性关系主题，《羞羞的铁拳》讲的励志和爱情，都是有现代感、且真正温暖的。

两条主线：一条是老梗——男女换身；另外一条就新鲜不少——职业拳击。这两条线索展开了充分的空间，让爱情、梦想、比赛（不过更准确的应该是“武打”）都有发挥之地。

这部电影也再次证明，当想法成熟、运用熟练，老俗套还是可以特别好笑——比如马丽（内里已经是艾伦）推开澡堂门，背景乐《好运来》疯狂响起来的时候。

《英伦对决》

  • 导演: 马丁·坎贝尔
  • 主演: 成龙 / 皮尔斯·布鲁斯南 / 刘涛
  • 片长: 116 分钟 
  • 格式：3D/IMAX3D/中国巨幕

成龙近年来的流水线动作片口碑不佳，去年的《绝地逃亡》豆瓣评分 5.5 分，前年的《天降雄狮》5.9 分。

到了今年这部《英伦对决》终于能被称作一次意外。电影踏实地一步步铺陈开故事格局，政治意味浓厚。成龙不再用杂耍式的动作喜剧，而是走真实感的剧情打斗，主题严肃走向认真，这都使得这个角色令人信服。

但是电影从几条线的人物动机塑造上，仍然缺乏完全的合理性，这是会感到遗憾的地方。

导演马丁·坎贝尔，他今年已经 74 岁，所导演的《黄金眼》也被认为是让詹姆斯·邦德系列重获新生的一部。

《追龙》

  • 导演: 王晶 / 关智耀
  • 主演: 甄子丹 / 刘德华 / 姜皓文 / 郑则仕 / 刘浩龙
  • 片长: 128 分钟
  • 格式：2D/中国巨幕

同样，如果按照王晶近年来的作品口碑度、值得信赖度，这部电影可能会超过你的期待值。

电影讲述了香港现代史上两大著名狠角色大毒枭跛豪（甄子丹饰）、五亿探长雷洛（刘德华饰）的传奇故事。 

电影仍然有不少俗套、常见的段落。但作为香港毒枭类型片，它有基础的类型框架、故事元素、美术功底，这些都堆积在一起，再加上刘德华、甄子丹等演员的表现，还不错。

《缝纫机乐队》

  • 导演: 大鹏
  • 主演: 大鹏 / 乔杉 / 古力娜扎 / 李鸿其 / 韩童生
  • 片长: 117 分钟
  • 格式：2D

相比《煎饼侠》，大鹏的第二部电影《缝纫机乐队》仍然延续了他以前的喜剧路数，但当他试图创造一个更实质的完整虚构故事，除了还有一些热闹的片段，电影在价值体系和人物主题上都存在着问题。

最核心的是，影片过于依靠“摇滚精神”这样一个太过具体的虚无主题概念。它究竟是不合时宜的旧时代，还是其实仍然属于有号召力的当下，实际上并没有解释清楚。

尤其是当结尾又回到《煎饼侠》式，对 20 世纪末流行偶像（上次是古惑仔，这次是 beyond）的致敬，“摇滚”就像是一个必须存在的符号。这导致电影里强调的理想都符号化而空虚，情绪高潮也就无从谈起。

《空天猎》

  • 导演: 李晨
  • 主演: 李晨 / 范冰冰 / 王千源 / 李佳航 / 赵达
  • 片长: 115分钟
  • 格式：2D

李晨的这部导演处女作，最大的问题还是假大空的台词、场面，以及尴尬的表演。没有特别令人感到逻辑崩盘的地方，但是从头到尾都好像是一部过时的电影。

比较有看头的，当然是飞机。
