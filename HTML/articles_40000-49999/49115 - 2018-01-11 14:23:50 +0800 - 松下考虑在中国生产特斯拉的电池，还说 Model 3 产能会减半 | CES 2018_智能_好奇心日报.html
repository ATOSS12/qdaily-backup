松下考虑在中国生产特斯拉的电池，还说 Model 3 产能会减半 | CES 2018_智能_好奇心日报

罗骢2018-01-11 14:23:50

特斯拉的产能达不到目标，松下的业绩也受影响

在今年的 CES 展会上，除了推出了新款 4K 电视产品 FZ950、新品相机 LUMIX GH5S 等数码产品之外，松下电器的电池业务也是关注的焦点。

日经中文网报道称，松下社长津贺一宏在接受采访的时候透露了未来电池生产计划，表示可能将在中国生产供特斯拉使用的电池。

作为特斯拉独家的汽车电池供应商，松下目前在日本和美国为特斯拉汽车生产电池。2016 年 1 月，松下与特斯拉联合投资 50 亿美元在内华达州兴建超级电池工厂（Gigafactory），成为全球最大的汽车电池工厂，日前已经投入生产。

津贺一宏这次表示，由于特斯拉已经计划在上海启动纯电动车的本地化生产。松下也在考虑推进对于中国工厂的投资力度，虽然尚未敲定在中国生产电池的详细计划，但松下已经在考虑中国生产特斯拉电池的可能性。

过去一年内有多家媒体报道特斯拉将在中国建厂生产特斯拉汽车。去年 10 月，特斯拉给《好奇心日报》的回应称，“特斯拉正与上海市政府探讨在该地区建设工厂的可能性。”

2017 年 5 月，松下就传出将在江苏兴建一座电池工厂的计划，新工厂将生产特斯拉使用的 18650 型三元锂离子电池，并且将与中国合作伙伴合资建厂。

在采访中，津贺一宏还透露了特斯拉将 2018 年第一季度的 Model 3 周产能目标从 5000 辆下调至 2500 辆。受此影响，松下也推迟原定的车载电池交付日期。

他表示虽然这会对业绩产生影响，但截至 2018 年 3 月的财报将涵盖下调的业绩，最终不会产生较大的影响。

上周三，特斯拉承认原定的 Model 3 生产目标将无法完成，比之前承诺的每周生产 5000 辆的产量目标计划减半，每周将只能生产 2500 台 Model 3 电动汽车。每周交付 5000 辆 Model 3 的目标被推迟到今年二季度末。

去年 11 月，特斯拉召开分析师电话会议的时候，表示生产电池的专门工厂千兆工厂 Gigafactory 的确遇到了电池组装的问题，而 Model 3 就使用千兆工厂生产的电池模组。

自 2017 年第三季度以来，特斯拉 Model 3 的量产计划就一直没有达标。随着最新的 Model 3 周量产计划减半的消息公布，特斯拉的股价也在当日下跌了约 2%。

作为特斯拉旗下价格主流的电动车，Model 3 产量的重要性不言而喻。中金公司发布报告评论称，在达到周产能 5000 辆之前，Model 3 会一直处于亏损或微利状态。如果 Model 3 的周产能不能在年底前达到 1 万台的话，随着 2018 年其他传统车企加快在电动车领域的推进速度，特斯拉的领先优势将会进一步失去。
