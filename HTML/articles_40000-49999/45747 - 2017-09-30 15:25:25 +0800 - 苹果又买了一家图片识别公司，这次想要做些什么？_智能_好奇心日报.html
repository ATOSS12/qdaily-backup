苹果又买了一家图片识别公司，这次想要做些什么？_智能_好奇心日报

罗骢2017-09-30 15:25:25

iOS 系统会更了解照片中的人脸信息

苹果很明显要继续改进自己的图片应用。根据科技媒体 TechCrunch 报道的消息，苹果收购了一家专门研究图像分析和面部识别技术的创业公司。

新近收购的这家名为 Regaind 的公司位于法国巴黎。苹果并没有透露收购这家公司总共花费了多少钱。跟往常一样，苹果没有明确说明收购的目的。苹果发言人称：“苹果经常会收购小型科技公司，我们通常不会谈论我们的目的或者计划。”

Regaind 开发的一种能分析照片的内容的计算机视觉 API 是苹果看上的技术。在他们的官网介绍上，理解图像的内容和评估照片背后的信息是他们主要研究的功能。

通过这种计算机视觉 API ， Regaind 可以能够分析照片的锐度、曝光、色彩和其他属性，并利用这些信息来分析出更多的信息。

例如，如果你用连拍模式快速拍摄一堆照片，Regaind 能自动找到其中最佳的照片，并将其用作照片库中的主要照片。还可以识别图库中完全重复的照片，并询问你是否合并或者删除。

利用这项技术，苹果可以改善系统照片应用中的 Memories 功能。iOS 上的照片应用会根据事件、位置、时间等信息来自动创建相册。使用 Regaind，iOS 系统将可以查找出视觉上相似的照片，并将最佳拍摄视为封面或者创建出包含最佳镜头的视频简介。

有趣的是，他们的计算机视觉 API 可以检测出照片中的人脸，以及照片中出现人的性别、年龄和情绪。这可能是苹果收购这家公司最主要的原因。

在 iPhone X 上，苹果引入了 Face ID 利用人脸识别来解锁设备。 TechCrunch 推测，苹果很可能会结合 Regaind 的技术来增强面部识别的功能。例如在 iPhone X 上区分出机主的照片，或者学习机主面部信息的变化。

苹果最近修改的隐私政策中提到了关于 Face ID 自主学习的功能，Face ID 可以自我学习进化，如果机主把平时蓄的络腮胡子刮了，机器会无法确定身份，觉得摄像头前的人可能是机主，这种情况下，Face ID 会要求输入密码并记录下机主没胡子的面部信息，之后就不用输入密码了。

从 iOS 10 开始，苹果就对自带的照片系统功能进行了大幅升级，带来了识图能力，例如输入“狗”之后，系统会自动筛选出照片图库中所有含有狗这类动物形状的照片，不过这个功能目前还比较简单，无法识别更多复杂的东西。

随着 Face ID 的推出，面部信息可能会带来更多功能上的变化，苹果在自己开发算法的同时，也开始花钱买技术，带来更多的图片功能。
