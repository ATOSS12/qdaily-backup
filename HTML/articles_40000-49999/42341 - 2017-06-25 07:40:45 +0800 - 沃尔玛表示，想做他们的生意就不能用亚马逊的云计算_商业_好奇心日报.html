沃尔玛表示，想做他们的生意就不能用亚马逊的云计算_商业_好奇心日报

李莉蓉2017-06-25 07:40:45

“每个人都知道，如果你用 AWS，沃尔玛就不会和你玩”

沃尔玛和亚马逊的竞争日益激烈，而且不仅体现在零售方面。

根据《华尔街日报》的报道，沃尔玛向相关的科技公司表示，如果想要做沃尔玛的生意，那他们就不能用亚马逊的云计算服务 AWS。而相关的公司不止是直接向沃尔玛提供服务的公司，还包括了和沃尔玛供应商合作的公司，比如和供应商合作的仓库云数据公司 Snowflake 就表示他们被告知要使用微软的云服务 Azure。

沃尔玛自己完全不使用 AWS，他们表示沃尔玛的数据大部分都储存在自己的服务器上，同时使用了 AWS 以外的云服务。沃尔玛的发言人 Dan Toporek 表示“我们更希望我们大部分敏感数据不是储存在竞争对手的平台上，这不是多让人意外的事情”，不过他也承认“一小部分”技术供应商的应用使用了 AWS。

按照《华尔街日报》的说法，Snowflake 并不是唯一一家被告知不能使用 AWS 服务的公司。和一家零售分析公司合作、为沃尔玛开发工具的软件开发公司 Lofty Labs 表示：“每个人都知道，如果你用 AWS，沃尔玛就不会和你玩”。

亚马逊自然不怎么高兴，他们的发言人表示沃尔玛错误地认为 AWS 在支持着亚马逊的零售业务，因此“持续尝试着胁迫”技术供应商，还说沃尔玛这样的策略将会损害生意，“很少能够获胜”。

沃尔玛向来担心用 AWS 服务将会令自己的业务受影响。2013 年他们收购了云计算公司 OneOps，去年年初，沃尔玛宣布将内部运营的云计算管理系统 OneOps 变成对公众开放的开源平台。通过 OneOps，开发者可以在不同云供应商之间切换的开源工具，不用受限于任何一家云供应商。

对亚马逊来说，AWS 是个很重要的生意。四月公布的第一季度财报显示，AWS 为亚马逊贡献了 8.9 亿美元，占到了八成。物流、员工等方面的投入令亚马逊的零售业务利润有限，而且可能还会持续到明年中期。

沃尔玛的行为将会给亚马逊的竞争对手们带去不少生意。微软表示沃尔玛是 Azure 最大的用户之一，不仅如此，微软全球商务部门副总裁 Judson Althoff 表示市场调研公司尼尔森就是“沃尔玛所直接带来的客户”。
