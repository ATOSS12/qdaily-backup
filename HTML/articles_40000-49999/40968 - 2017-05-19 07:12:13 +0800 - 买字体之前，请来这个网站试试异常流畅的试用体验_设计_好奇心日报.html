买字体之前，请来这个网站试试异常流畅的试用体验_设计_好奇心日报

胡晓琪2017-05-19 07:12:13

即便不买，也可以玩一玩。

一份好的平面设计作品，优秀的字体总是不可或缺的。

在商业案例中，尽管也有纯粹依靠图像表达的作品存在，但文字依然是传递信息的最主要途径，即便是有人说，平面设计就是要创造烘托字体、突出信息的氛围，也丝毫不会夸张。

然而，就算是相同的字体，选用不同的字距行距，运用不同的版面构成方法，也会产生不一样的效果。对于平面设计师而言，字体设计不光是不可忽视的重要环节，也是一个并不轻松的环节。

来自美国的两位设计师 Ben Kiel 和 Jesse Ragan，想把字体设计这份工作独立出来。他们在各自做了多年的字体设计之后，一起合开了一个名为 XYZ Type 的工作室，直接为需要字体设计的公司、工厂和个人提供服务。

为此，Ben 和 Jesse 制作了风格轻快、好玩的 XYZ Type 网站，人们可以直接通过网络购买 XYZ Type 的字体，整个过程就和你在网上买衣服、鞋子一样简单——你可以从 XYZ Type 提供的三种风格字体 Aglet Slab，Export 和 Cortado 中进行选择，然后直接在网页上进行试验。随便输入文字或数字，再选择字体的间距、颜色，就能立即看到字体的效果。如果你对当中的某种字体觉得满意的话，点击购买键直接把它拖入购物车就可以了。

XYZ Type 的这三种字体都还挺好看的，其中的 Cortado 算是较老的作品了，它线条流畅且优美，还曾获得过 2014 年 typographica 的“我们最爱的字体”奖。另外两种 Aglet Slab 和 Export，则是 Ben 和 Jesse 在 XYZ Type 成立之后的最新创作。

与此同时，它们还都不贵，比如 Aglet Slab，它又分斜体、常规体、加粗体等 12 种，每一种售价在 40、50 美元不等，要是打包购买所有的 Aglet Slab 的话，就只要 350 美元。

另外，XYZ Type 售卖的其实是使用权。你需要在购买的同时选择电脑、网页或者 app 这三种使用渠道，它会以相对应的 OpenType 或 TrueType 字体格式呈现。为了保护版权，它还会严格要求你输入一些限制条件，比如选定最多使用的电脑台数（上限 200 台）、网页每个月的最大页面浏览量（上限 1000 万），以及 app 的个数（上限 10 个）。不过，即使你选择的是给 200 台电脑使用，所花费的钱也还是和选 1 台电脑的一样多，XYZ Type 限制这些，只是为了更为严格地把控字体使用权的流向而已。

如果你通过购买 XYZ Type 现有的字体而喜欢上了他们的设计风格，你也可以直接找到这两位设计师，要求他们为你定制字体。
