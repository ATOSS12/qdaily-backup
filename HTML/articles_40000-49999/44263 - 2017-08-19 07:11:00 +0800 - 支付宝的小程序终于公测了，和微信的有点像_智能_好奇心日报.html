支付宝的小程序终于公测了，和微信的有点像_智能_好奇心日报

罗骢2017-08-19 07:11:00

和摩拜一样，ofo 是支付宝里最显眼的小程序

说了很久的支付宝小程序公测了。

年初微信正式发布小程序，三天后一份支付宝版本的小程序开发者工具页面截图就出现在网络上。

支付宝官方随后承认正在研发类似小程序的应用，但还在内部测试，会看市场反应在合适的时候推出。

此后，支付宝小程序再无消息。8 月 18 日，在内部测试了长达 7 个月之后，支付宝小程序才进入正式公测阶段。

在年初的截图里就显示，支付宝并没有试图和微信小程序进行差异化，无论是开发技术还是小程序这个名字，都和微信相一致。

从公测页面的开发文档来看，两家的小程序在文档代码里有着大量相似的结构。

支付宝小程序提供了 WiFi 、扫码、位置、媒体图片等常用 API 接口。从功能上讲并没有什么不一样的地方，

好奇心日报试用了支付宝小程序的 Demo，从小程序的产品体验到主体信息查询都和微信小程序的布局界面一样。

目前，支付宝小程序的入口有线下扫码、历史记录和用户分享三种。没有使用过的支付宝小程序只能通过扫码启动，或者通过好友分享获知小程序。在使用后，用户则可以在“应用”里的“最近”版块找到历史使用过的小程序。

之前还有消息显示，用户还可以自行选择把常用的小程序置于 App 界面首页方便快速启动。

小程序本质上是在微信已经开发好的框架里，利用微信原生的功能函数来快速实现其他功能的 HTML 应用。这种已经构架好的功能函数，如从相册选择照片，打开内置地图的功能都称为 API 接口。

API 和领先竞争对手相似并不少见，亚马逊在 2013 年推出 Kindle Fire HDX 平板，上面搭载了基于安卓系统修改的 Fire OS 4，但亚马逊对外宣称该系统是安卓的一个分叉。Fire OS 申明不提供 Google 原生的地图和云消息等 API 接口，但Fire OS 提供的诺基亚地图的 API 几乎就是 Google 原生地图 API 1.0 版本的克隆。

对于支付宝来说，使用和微信小程序相同的开发工具和技术并不是一件难以理解的事情，降低小程序的开发门槛，让大量成熟的微信小程序更加方便地移植到支付宝平台是快速扩大应用数量的方法。另一方面，相同的界面也让用户无需改变使用习惯。

但可能支付宝小程序的代码“过于”相似，丁香园前 CTO 冯大辉发了一条莫可名状的微博。

对于微信来说，小程序是否称得上是成功还要斟酌，但至少先发半年的微信小程序已经在功能上进行了多次优化和更新。

2016 年微信报告里，32% 的用户每天使用微信超过 2 小时，在这样强大的流量支撑下，小程序并没有成为我们日常使用的功能，原先设想的线下场景也很少见。

而仅仅在日常消费时候打开的支付宝，即便在短时间内依靠移植开发获得大量的小程序，也很难获得成功。
