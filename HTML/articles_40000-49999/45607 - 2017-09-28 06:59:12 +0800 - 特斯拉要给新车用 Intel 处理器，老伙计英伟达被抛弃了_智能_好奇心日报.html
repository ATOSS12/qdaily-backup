特斯拉要给新车用 Intel 处理器，老伙计英伟达被抛弃了_智能_好奇心日报

张智伟2017-09-28 06:59:12

Model 3 和将来新车型的信息娱乐系统，都将搭载 Intel 处理器。

彭博社援引不具名知情人的消息称，图形显示芯片厂商英伟达的大客户，特斯拉最近决定不再在 Model 3 及未来的新车上采用英伟达提供的处理器，转而向 Intel 采购芯片。

这些处理器芯片主要用来驱动 Model 3 的信息娱乐系统（infotainment）。

今年 7 月底，首批 30 台特斯拉 Model 3 交付时，我们已经对这款平价电动汽车的驾驶辅助和信息娱乐系统做过介绍。

跟 Model S 不同，Model 3 根本没有显示行驶速度和发动机转速的仪表盘，这些信息连同导航和各种娱乐应用全部被转移到了中控的那块 15 英寸液晶屏上。

屏幕尺寸虽然比 Model S 的 17 寸略小了一些， 但同样支持 Wi-Fi 和 LTE 数据网络连接。这些功能要求驱动屏幕的处理器，不仅要拥有不错的图形渲染能力，还要支持多模 LTE 网络和灵活设计的天线。

对于英伟达的车载计算平台 Drive CX 和 PX 而言，两大平台搭载的 Tegra SoC 并没有封装基带处理器。英伟达自从退出移动设备 SoC 市场之后，就不再对调制解调器（modem）投入研发资源。

这意味着采用英伟达自动驾驶平台的汽车公司，需要寻找到合适的外挂 4G 基带来保证高速网络连接，并在 5G 到来之前及早筹备。

在英伟达、高通、英特尔三家拥有自动驾驶软硬件方案的公司中，只有后两家具备成熟的通讯基带技术，而且都在测试自己 5G 网络调制解调器。

特斯拉在 Model 3 及未来新车中弃用英伟达的处理器，转向英特尔，很可能是一次研发团队内部技术选型的结果。但不能排除存在其它原因的可能性。

从公开的信息来看，特斯拉 Mode 3 进入量产阶段已经有近两个月，8 月的累积产量可能已经超过 200 台。

按照特斯拉的计划，9 月 Model 3 的产能要提高到 1500 台，12 月要达到月产 20000 台。
