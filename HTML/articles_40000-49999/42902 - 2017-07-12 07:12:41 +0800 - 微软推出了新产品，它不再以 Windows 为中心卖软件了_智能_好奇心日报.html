微软推出了新产品，它不再以 Windows 为中心卖软件了_智能_好奇心日报

徐弢2017-07-12 07:12:41

针对企业卖 Windows 对微软来说也不重要了。

昨天，在微软的合作伙伴大会 Microsoft Inspire 上，微软进一步公布了新的业务调整，推出了一项针对企业的新产品 Microsoft 365。

简单来说，新产品其实是把针对企业的 Windows 10 系统和 Office 365 打包卖，另外还包括了移动设备管理和身份识别系统 Enterprise Mobility+Security 的服务。

新产品 Microsoft 365 根据针对的企业客户规模大小，分为 Enterprise 和 Business 两个套装，针对中小企业的 Business 套装最多支持 300 人。其他方面，这两个版本所用的服务只有少许差异，例如前者用的是 Windows 10 Enterprise，Business 套装用的是 Windows 10。

Enterprise 套装从 8 月 1 日开始销售，售价需要企业客户根据具体产品方案定。Business 套装在 8 月 2 日推出公开预览版，预计是今年秋天正式上市，价格是单个员工每月 20 美元。

本质上，新产品其实改变了微软针对企业销售 Windows 10 系统的方式，进一步摈弃以 Windows 为中心的策略。以往，像惠普、戴尔等 OEM 厂商做企业生意的时候，把 PC 硬件和包括搭载在硬件上的 Windows 系统、Office 软件、管理系统等软件打包卖给企业客户。这些 OEM 厂商帮着微软卖 Windows 系统。

对于微软来说，它的客户每台电脑上搭载的 Windows 系统都要向微软交钱，也就是按照电脑台数在卖。2009 年，时任微软企业策略总经理的 Charles Songhurst 称，微软每份 Windows 收取的费用约 50 美元。这个价格保持了相当久的时间。

但现在微软的新产品 Microsoft 365，等同于把 Windows 10 系统纳入到 Office 365 销售体系中，以每月订阅费的方式进行销售。这是微软在更进一步摈弃以 Windows 为中心，转向在全平台销售生产力工具新进展。

把 Windows 10 搭载着 Office 365 一起卖给企业客户背后，其实是因为新的 PC 电脑卖不出去，但现有的企业用户仍然在用大量的 PC 电脑，他们仍然需要 Office 这些办公工具。根据微软公布的数据，目前有 1 亿 Office 365 商用客户。

不过，针对企业客户的 Windows 10 有了这种新的销售方式后，也带来了新的疑问。微软原本向 PC 厂商卖 Windows 系统，再按照份数收钱，现在微软绕过 PC 厂商直接向企业销售 Windows 系统，PC 厂商原本需要缴纳的费用是否需要继续缴纳？

另外一个可能存在的问题是，微软会不会将这种销售模式，带到面向普通消费者的 PC 电脑上？很可能带来的结果是，微软免去了 PC 厂商需要缴纳的费用，直接通过 Windows 10+Office 365 的个人订阅这种销售方式，向普通消费者收费。
