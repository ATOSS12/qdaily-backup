这么多年了，微软推出了一个方便连接 Windows PC 和手机的功能_智能_好奇心日报

张智伟2017-05-12 15:15:06

纳德拉公开说不会放弃手机业务，但 Windows 的每一步都在埋葬 Windows Phone。

微软在今年的 Build 2017 开发者大会上说，Windows 爱你的所有设备，包括 iPhone 和 Android 手机。

说白了就是要更方便的连接手机和 Windows 系统的 PC，无论你手机是 iPhone 还是 Android 的。

在 Windows 10 今秋的创造者更新（Fall Creators Update）中，微软加入了类似苹果公司为 Mac 和 iPhone、iPad 设计的 Handoff 功能。

具体到场景当中，比如你在 Android 手机或 iPhone 上面要完成的工作只做了一半，打开 Windows 电脑就能按照最新进度继续推进。

由于跨越了多个不同的操作系统，这个构想背后的工作量并不轻松。微软特意为此推出了一个名叫 Windows Graph 开发平台，它可以方便开发者为自己的应用软件开放和调用某些必需的 API 接口，从而实现类似 Handoff 的功能。

目前，已经用上 Windows Graph 的产品还不多，而且大部分是 Smartsheet、Zapier 等办公效率类软件。

跟苹果公司的 Handoff 相比，现在还不清楚在 Windows 10 上这个功能的实际体验怎么样。Windows Phone 全球份额无限趋近于 0 ，微软也把 Android 手机和 iPhone 用户看得很重要。

自从微软 CEO 纳德拉把挣不到钱的硬件业务划归到 Windows 部门以来，其产生的亏损一直由 Windows 来承担。这次开放对 Android 和 iOS 设备更底层的支持，本质上是进一步杀死 Windows Phone。

虽然纳德拉在公开场合还在说不会放弃手机业务，但谁知道呢？
