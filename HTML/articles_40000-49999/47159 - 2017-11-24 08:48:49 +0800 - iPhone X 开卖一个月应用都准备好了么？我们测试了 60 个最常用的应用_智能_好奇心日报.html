iPhone X 开卖一个月应用都准备好了么？我们测试了 60 个最常用的应用_智能_好奇心日报

罗骢2017-11-24 08:48:49

应用都还没有适配好 iPhone X 的全面屏

10 月 27 日，iPhone X 开始接受预定。四周之后，发售日预定的用户都已经收到了自己的新手机。

如《好奇心日报》早先的评测，这是 iPhone 家族里变化巨大的一代，我们已经熟悉的屏幕比例、Home 键和指纹识别都消失了。Face ID、新的屏幕形状和手势操作带来了截然不同的体验。

在带来新鲜感与乐趣的同时，iPhone X 的改变也需要第三方应用的配合。

这也是评测时我们遇到的主要问题。当时相当一部分应用在使用时上下黑边。当时已经可以全屏显示的应用，往往按钮位置也有问题。

现在四周过去了，应用开发者已经可以拿到真机开发。

我们体验了中国区 App Store 免费榜上前 50 个应用。我们还专门对 App Annie 手机游戏收入榜前十名的游戏应用进行了测试。

这其中有快手、腾讯视频、爱奇艺这类需要经常全屏播放视频的应用。全面屏的适配很大程度上影响着使用体验。

还有需要横屏体验的游戏应用。《王者荣耀》、《阴阳师》、《欢乐斗地主》，以及最近火热的射击游戏《终结者 2 审判日》等。

最后得到了几个结论：

几乎所有可以播放视频的应用都没处理好和 Home 条的关系

在全屏播放的时候，应用内屏幕下方始终会停留着白色或者灰色的虚拟 Home 条。而且，在横屏播放的时候这些应用都没有提供可以真正意义上使用全面屏的放大模式，全面屏的两侧会留出黑边。

在视频服务中，抖音、优酷、土豆、快手、西瓜视频、爱奇艺、火山小视频、B612 、Face U、美颜相机、哔哩哔哩弹幕视频，都出现了这些问题。

不只是在视频服务中，即便是主要功能已经适配很好的工具类应用，内置的播放器也都存在虚拟 Home 条适配问题。

即便在 iPhone X 上全屏，相当一部分应用在顶部缺口的适配上都存在细节问题

常见的是顶部刘海会遮挡到应用的内容。在 Face ID 导致的刘海出现之前，很多应用在顶部操作时候都会自动隐藏状态栏，来显示更多的东西。但现在屏幕正中间的刘海会自动遮挡住部分内容，这个问题在观看哔哩哔哩弹幕视频的时候格外突出。

更影响体验的是，应用内按钮可能会和系统状态栏相互重叠导致无法点选。令人尴尬的是，优酷全屏广告页面上，回到首页的按钮和系统状态栏相互重合，无法点选，最终只能结束进程才能回到首页。

左上角也有类似的情况，像是从一个应用跳转到新的应用中，左上角跳转回去的按钮会和应用本身的返回键相互重合。

另一个会出现在适配应用上的问题是图片拉伸。

由于 iPhone X 的屏幕比例为 19.5:9 ，比其他机型要更加细长。这导致了在大量进行适配的应用上，全屏图片都出现了不同程度的拉伸。在应用的开屏广告中尤为明显。类似进入阴阳师的载入界面上，背景右上角的字样被拉伸出了屏幕。

三大互联网公司自己的一堆应用里，腾讯的适配最好，百度最差

直到这周二，手机百度和百度地图才对 iPhone X 进行了适配。而最需要适配的视频应用，百度旗下的爱奇艺则比这两个应用还晚一天才对全面屏进行适配。

在适配方面，腾讯系的产品体验是最好的。腾讯视频是其中适配比较完善的一款应用。全屏播放腾讯视频的时候，底部虚拟 Home 条会自动隐藏，同时它也是唯一一个支持放大模式的应用。横屏播放状态下双击屏幕可以体验到完整的全面屏看视频的感觉。

但在微信中，底部的 UI 处理多少有点问题。打开摩拜的小程序，之前出现在最下方的点击开锁按钮会被 iPhone X 的虚拟 Home 条所遮挡。这个情况发生在多个应用上，虚拟 Home 键会遮挡住应用底部导航栏的文字内容。点击的时候可能发生点不中的情况。

在我们测试的过程中，微信还出现了一个bug，删除群成员的时候，最顶上的一名成员会被自动隐藏，需要下滑才能看见。这导致很难点中进行删除操作。

阿里的淘宝、天猫、UC 浏览器都在第一时间对 iPhone X 进行了适配，但是在体验上，淘宝在顶部区域出现了按钮重叠等细节问题，内置的播放器也会持续显示虚拟 Home 条。

而支付宝直到现在依然无法使用 Face ID 来进行支付。在设置解锁方式里，支付宝只能使用指纹和手势验证，正常来说，在 iPhone X 上原本用指纹的都可以开启 Face ID 刷脸，但这里的选项变成“本型号设备暂不支持指纹”

想要在 iPhone X 的支付宝上进行面部支付，只能使用支付宝通过前置摄像头进行的面部识别验证。但这个在 1 千块的智能机上都能使用的功能，无论是速度、精确度和适用性都完全无法和 Face ID 相媲美。

游戏中，热门新作都进行了适配，而老牌游戏普遍没有

最近火热的射击类手游中，《小米枪战》、《穿越火线》和《终结者 2 审判日》等游戏对 iPhone X 都进行了适配。但在老牌知名游戏中，只有《王者荣耀》和《阴阳师》很快进行了适配。腾讯另一款《欢乐斗地主》、暴雪的《炉石传说》和《开心消消乐》等游戏都没有适配。

有趣的是，由于 iPhone X 的屏幕比其他机型都要长，有玩家此前询问 iPhone X 是否可以获得更多王者荣耀中的战场视野。随后，王者荣耀官方回应称，已经在屏幕两侧加上渐变黑色处理，确保和其他机型的显示画面一样。这样的处理也解决了 Face ID 刘海部分的显示情况，但并不符合苹果之前提出的适配规定，就是不要使用黑边来遮盖 Face ID。

但实际游戏中，在 iPhone X 上玩王者荣耀依然会遇到一些问题，在发动类似闪现、回城等技能的时候，由于很容易误接触到虚拟 Home 条导致游戏被切回桌面。这个适配后带来的细节问题已经大量出现在王者荣耀的玩家论坛上。

  • 完全没有适配的应用依然很多

完全没有适配的应用中包括一些适配需要并不高的银行、运营商等应用：

唯品会、火山小视频、中国移动、内涵段子、B612、小红书、工商银行等。

而拼多多这类用户量很大的应用并没有对 iPhone X 进行适配也令人有点意外。

苹果已经有了一套规范，但大部分开发者还没来得及调整设计

在官方指导的适配文档里，对于有 Face ID 影响的状态栏和底部虚拟 Home 条的处理都做了详细的规范，这些规范可以处理目前所有的应用适配问题。

在 iPhone X 上，状态栏的高度是高于之前所有的 iPhone 版本的。这导致了之前在 UI 开发中可以用默认高度数值 64 来设置的状态栏加导航栏调整为必须使用动态布局 ，否则顶部的 UI 就会出现重叠的现象。

而有些应用之前花费心思来隐藏状态栏也变得不可用。显示在状态栏处的内容会被刘海所切割，在抖音、哔哩哔哩中都可以看到这样的问题。

另一方面，大部分适配的应用在处理底部的时候都是直接用和底部导航栏颜色一样的元素填充底部圆角区域的内容，导致了大量留白的出现。这是因为苹果在官方设计文档中建议这样做，确保不会对虚拟 Home 条进行影响。

而影响视频播放体验的虚拟 Home 条，苹果也在文档中做了正确的示范。在播放视频时，开发者可以开启虚拟 Home 条「自动隐藏」功能从而获得沉浸式体验。开启后，视频播放时虚拟 Home 条将自动消失；单击屏幕后就又会出现。

虽然苹果给出的文档对与可能出现的适配问题都做了规范和提示，但开发者的时间不是很够。9 月发布会后的测试版虚拟机并不完整，当时看不到控制中心、没有桌面背景、在 Mac 上拖放卡顿严重。

而且今年这么大的变化，只在虚拟机上测试是不够的，真机上会碰到一些虚拟机上看不出来的问题。

此外这还只是基础的适配问题，只是确保用起来没问题而已。更长的屏幕对用户视线、交互都有影响，我们没有看到应用为更长的屏幕重新设计用户界面、基本也只有几个海外应用推出黑色主题迎合显示黑色更省电的 AMOLED 屏幕。
