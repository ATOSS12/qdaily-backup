C 罗 Instagram 粉丝过亿，但像他那么风光的运动员真不多 | 好奇心小数据_娱乐_好奇心日报

周哲浩2017-05-05 18:49:56

竞技体育中的“马太效应”相比其他产业要厉害许多。

本周刚刚在欧冠半决赛上演帽子戏法，帮助皇马以 3:0 的比分战胜同城对手马竞后，皇马的当家球星 C 罗迎来了一项新的里程碑。不过这个里程碑和绿茵场没有什么太大的关系，而是在 Instagram 上——他的粉丝数超过了 1 亿。

这样一来，他成为了第一个在 Instagram 上收获亿数量级别的粉丝的运动员。而在整个 Instagram 上，粉丝数量高过 C 罗的只有四位音乐人，分别是赛琳娜·戈麦斯 (Selena Gomez) 、爱莉安娜·格兰德 (Arianna Grande) 、泰勒·斯威夫特 (Taylor Swift) 和碧昂斯 (Beyoncé) 。其中，霉霉和碧昂斯的粉丝数不过比 C 罗多了 100 万。如果 C 罗能够维持目前这样的粉丝增长速度的话，那么他很有可能在这个月就完成对她们的超越。

不过 C 罗的风光无限或许并不能代表整个足球中运动员们的情况。竞技体育中的马太效应，即强者愈强，弱者愈弱的情况相比其他产业要明显得多。

下图罗列了在 Instagram 上的粉丝数最多的足球运动员和音乐人各七人。很明显就能看出，从排名第二的内马尔开始算起，足球运动员的关注度就有明显的下滑。而到了排名第七的伊布，粉丝数就只有 2400 万，这个数字远低于同样在音乐人中排名第七的麦莉·赛勒斯的 6500 万。

如何量化这种马太效应？我们可以将经济学中基尼系数的概念和计算方法应用在这里。基尼系数通过将实际收入分配和完全平均分配的理想情况做比较，得出一个 0 到 1 之间的数据，数字越大说明收入分配越不平衡。

根据这样一种算法，这七位足球运动员在 Instagram 粉丝数上的贫富差距值达到了 0.283，而七位音乐人的同一数据则为 0.1。当然，这里选取的数据量非常小，但这些位于整个金字塔塔尖的运动员和音乐人还是能够说明，相比起音乐行业，体育行业的竞争更为残酷，赢者通吃的法则在体育行业中体现得更为明显。

除了 Instagram 的粉丝数量，类似的情况也体现在运动员和音乐人的收入差异上。基于《福布斯》给出的数据，下图汇总了年收入最高的足球运动员和音乐人各十名。同样根据上述算法，足球运动员的贫富差距为 0.274，而音乐人的贫富差距则为 0.167 。

马太效应在体育行业中体现得更加明显，这本不是一件稀奇的事情。俗话说，“文无第一，武无第二”，在一个一定会决出胜负的行业中，自然是胜者才能获得最多的关注和喜爱。正如哈佛商学院教授安妮塔·埃尔伯斯在《爆款》一书中写道的那样：“决定超级明星收入水准的主要因素并非依据某些绝对的指标，而是明星自身与他人的相对表现——即本人和对手的胜负较量结果。这一特征或许在体育界表现得最为明显。”

这样的例子自然并不少见。费德勒在过去十年间一直是收入最高的网球运动员。这得益于他从 2003 年开始建立起来的一个几乎接近于统治性的王朝—— 4 年 11 座大满贯奖杯。这让他迅速成为最炙手可热的网球运动员，除了来自于赛事奖金的收入，赞助商也纷纷与他签订高额的赞助合同。费德勒网坛第一人的身份，甚至能够一直让他在状态下滑、战绩不佳以后，都依然是最受关注的网球运动员。

在顶级运动员的光环下，一些技术同样出众的运动员就自然会受到压制。美国橄榄球运动员马特·卡塞尔在 2009 年的时候和堪萨斯城酋长队签下了 6 年 6300 万美元的合同，这一数字比他此前在新英格兰爱国者队的合约要高得多。

之所以卡塞尔能够突然拿到这样高的合约是因为在 2008 赛季，新英格兰爱国者队的当家球星、曾获得过 3 次超级碗 MVP 的汤姆·布雷迪因伤缺阵，卡塞尔获得了替补出场的机会，并在当赛季表现出色。可以说，要不是布雷迪受伤，卡塞尔很可能继续籍籍无名下去。

回到足球的例子来说，俱乐部们愿意为顶级球员一掷千金，与他们自身能够为球队带来的价值不无关系。虽然足球是一项团体运动，但例如 C 罗、梅西、内马尔这样的顶级实力的球员还是能够在球场上凭借个人能力发挥出决定性的作用——就比如开头提到的 C 罗包办了球队的所有进球。

除了体育行业本身的性质以外，近来一些俱乐部的运作方式也在加剧这一趋势。其中最典型的例子自然就是皇家马德里了。21 世纪初，时任皇马主席弗洛伦蒂诺推行“银河战舰”策略。2001 年的齐达内、2002 年的罗纳尔多、2003 年的贝克汉姆、2004 年的欧文、2005 年的罗比尼奥，一年引进一名巨星，使得俱乐部比赛的门票、电视转播权、商业赞助和周边产品的销售都开始价格飞涨，最终使得皇马成为全球收入最高的足球俱乐部。

2006/07 赛季，皇马新主席卡尔德隆上任，决意改变弗洛伦蒂诺的策略，把大牌球员送走。然而，这不仅没能提升皇马战绩，球队的经营水平也出现了一定的下滑。最终，卡尔德隆在三年后下台，弗洛伦蒂诺重新执掌皇马，并以 1.25 亿美元的价格招揽 C 罗，这种巨星策略再次让皇马回到巅峰。

俱乐部的业绩依靠于巨星，所以俱乐部又会把得到的收入花到最顶尖的球员身上。顶级球员的价值在这个过程中进一步得到放大，就像当今足坛，除了梅西和 C 罗，能够在商业价值上和两人匹敌的球星真的不太多。
