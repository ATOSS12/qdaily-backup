特斯拉 Model 3 量产问题还没解决，但据说在好转_智能_好奇心日报

马若飞2018-01-06 07:46:14

特斯拉说问题部分解决了，但资本市场可能要失去耐心了。

特斯拉发布公告称，2017 年第四季度，公司交付了 1550 台 Model 3 轿车。2017 年第三季度以来，特斯拉 Model 3 的量产计划就一直没有达标。特斯拉原来计划到 2018 年初，Model 3 可以达到每个月 5000 台的产能，但现在只有原来计划的十分之一。

不过按照特斯拉的说法，情况已经有所好转。

特斯拉说，Model 3 产能瓶颈有进展，在 12 月底的时候生产率已经有提高：12 月的最后 7 个工作日，工厂一共下线了 793 辆 Model 3。在最后几天，已经可以达到每周 1000 辆的理论数字。特斯拉在销量报告里说，2017 年第四季度结束时，还有 800 台 Model 3 正在交付，未计算在产量数字中。

特斯拉说，原计划每周 5000 辆的产能将在 2018 第二季度实现，比原计划的第一季度推迟三个月。

去年 11 月，特斯拉召开分析师电话会议的时候，承认生产电池的专门工厂千兆工厂 Gigafactory 遇到了电池组装的问题，而 Model 3 就使用千兆工厂生产的电池模组。

除了 Model 3，特斯拉的其他车 2017 年卖得更多。2017 年第四季度，Model S 交付 15200 辆，Model X 交付 13120 辆。2017 全年特斯拉一共交付了 10.13 万辆Model S 和 Model X，同比 2016 年增加 33%。

这说明一件事，特斯拉在弗里蒙特的总装厂流程顺利，Model 3 的困境应该不在这里。

Model 3 的状况比较紧迫。2017 年的美国市场上，通用汽车推出平价电动车 Bolt 已经累积了很可观的销量：2017 年全年，Bolt 交付了超过 2.3 万辆，12 月交付了超过 3200 辆。2016 年 11 月 和 12 月，Bolt 的交付量大约 500 台，它没有遇到特斯拉类似的产能问题。

而 Bolt 是特斯拉 Model 3 最主要的竞争对手。

2017 年，美国市场上卖得最多的电动车是特斯拉 Model S，第二名是雪佛兰 Bolt，第三名是特斯拉 Model X。虽然特斯拉是卖得最好的电动车已经持续了两年，但 Model S 和 Model X 昂贵价格带来的销量是没法支持这家公司盈利的。

Model 3 的交付延迟可能已经让投资者们失去了部分信心：摩根大通最新的研报说，特斯拉 Model 3 最初的目标是易生产和高销量，但特斯拉已经在生产上烧掉了 10 多亿美元，靠 Model 3 赚钱的可能性太低。摩根大通大幅下调了对特斯拉的预期，建议卖出特斯拉的股票，预期股价是 185 美元，而特斯拉目前的股价在 310 美元左右。

而投资者的信心将影响到特斯拉未来融资的难度，这家公司过去借了很多钱，现在还没有盈利，未来依然需要借钱来维持。
