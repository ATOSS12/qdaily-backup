Face ID 隐私问题引起争议，美国参议员要求库克做解释_智能_好奇心日报

罗骢2017-09-15 16:41:59

Face ID 最多的段子是睡觉时候刷别人脸来支付，这不会真的发生

面部识别技术 Face ID 取代了 iPhone 沿用 4 年的指纹识别，成为了被库克描述成“未来的手机” iPhone X 上的解锁工具。

苹果是由足够的实力和影响力来推动一个成熟技术向另外一个新技术转变的手机厂商。但 Face ID 面部识别技术真的已经成熟到可以被大众接受的程度吗？

iPhone X 的顶部集成了包括红外镜头、泛光感应元件、点阵投影器和 TrueDepth 摄像头系统等 8 个组件来实现 Face ID 的功能，以及具备神经网络引擎的 A11 仿生处理器一起处理用户面部特征。

为了不让用户担心识别精度和保证隐私，苹果在发布会上着重强调了使用了 10 亿张用户数据来训练 Face ID，其中还和好莱坞模型团队进行了人脸模型对比验证，以保证 Face ID 所识别的人脸不能被照片之类的东西骗过。

此外，苹果也像对 Touch ID 指纹识别一样，将识别人脸的数据保存在本地，不上传到苹果的服务器。

但这些并不能打消公众对于 Face ID 隐私保密性的担忧，美国参议院议员，隐私、科技和法律司法委员会主席弗兰肯（Franken）在昨天直接给苹果 CEO 库克 Tim Cook 写了一份信件，要求苹果公司提供更多关于 Face ID 隐私和安全的细节。

Franken 在信件中对库克说：“关于 Face ID 是否会影响 iPhone 用户的隐私和安全，是否能在不同人群中表现同样出色，仍然存在很多疑问。为了让使用您产品的数百万美国用户了解详情，请您提供更多有关内部如何处理这些问题的信息，以及为了保护用户隐私采取的一些措施。”

第一个问题是用户面部数据保存问题，这是自从指纹识别技术引入手机就一直会出现的质疑，iPhone X 上是否会存储 Face ID 收集处理过的数据，如果这些数据能够从设备中提取出来，苹果公司是否会将其存在离线服务器上？

另一个问题是这几天在网络上热议的强制解锁问题：苹果做了什么保护措施，来防止他人强行使用 iPhone X 刷主人脸。

苹果在发布会上宣称工程师和几千人合作生成“ 10 亿张图像”来训练人脸识别神经网络也遭到质疑。

Franken 希望苹果可以更加详细地披露这 10 亿数据具体是如何制作出来，其中是否对黑人、亚洲人等不同种族、性别，年级进行了均衡的处理。如何防止 Face ID 由于数据上的不足导致在不同群体上产生偏差？

信中提到的其他问题还包括要求提供关于Face ID是否“始终开启”、是否一直处于搜索人脸的状态，以及系统是否会捕捉并存储尝试解锁人照片信息的细节。

最后，是美国科技公司和政府在这几年始终争议的焦点问题。面对政府和执法机构的解锁请求，苹果公司会如何来应对？

过去几年里，苹果一直不愿意把这些涉及用户隐私的信息交给执法机构，并且不断在系统上增强硬件安全技术保护。在去年圣博娜迪诺恐怖袭击案中，法院判决苹果应该配合 FBI 解锁凶犯的 iPhone 5C，但苹果态度坚决地拒绝了法院命令，从而引发了一场用户隐私和社会安全的争论。

早在 2013 年，这位参议员 Franken 就曾因为指纹识别技术的隐私问题写信给库克，寻求解答。

苹果公司也在随后简短地针对 Franken 的问题进行了回复。和上次一样， Franken 要求库克在 10 月 13 日前回复这封信，但不是强制要求。

随着 iPhone X 的发布， Face ID 也有了更多具体的信息。

负责苹果软件的高级副总裁 Craig Federighi 在网上回复留言透露，当用户直接按住手机两侧的按钮，iPhone X 将暂时禁用 Face ID，从而转为使用密码解锁。他解释说这将保护用户在紧急时刻不会触发 Face ID 。

针对带着墨镜如何解锁的问题，Craig Federighi 表示红外线摄像头可以透过大部分的墨镜镜片，从而让 Face ID 识别到用户眼睛。你不需要摘下墨镜就可以解锁。

而对于他在 iPhone X 发布直播过程中第一次演示 Face ID 失败的质疑， Federighi 也解释说，这并不是 Face ID 失败错误，而是由于演示机在之前被其他员工重新启动，导致 Face ID 第一次使用时候需要输入密码。

事后的回放显示，现场的提示的确为“ your passcode is required to enable Face ID”（你需要用密码来启用面容ID）。这和目前重启 Touch ID 后系统要求密码输入才能进入的提示一样。
