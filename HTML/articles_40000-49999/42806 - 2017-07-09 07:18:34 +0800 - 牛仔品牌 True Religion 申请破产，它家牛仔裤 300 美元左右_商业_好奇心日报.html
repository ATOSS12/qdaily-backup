牛仔品牌 True Religion 申请破产，它家牛仔裤 300 美元左右_商业_好奇心日报

温欣语2017-07-09 07:18:34

除了零售业惨淡这一大趋势外，公司自身的设计和营销也没能即时做出调整。

美国牛仔裤品牌 True Religion 本周申请破产，这是最新一个受电商影响而关闭的南加州服饰品牌。

True Religion 所属私募基金 TowerBrook Capital Partners 和贷方，其中包括高盛集团等达成交易，贷方会填补公司 3.5 亿美元的债务，并且获得重组后公司 90% 的股权。

在公司进行重组期间，True Religion 仍然开放营业，但会把重心转向线上购物，同时会和房东重新谈判租赁合约。

“我们正在采取重要的措施来降低债务，让 True Religion 这一标志性的品牌重燃生命力，并且为公司未来的增长和成功重新定位，” 其中包括将把更多精力投入到线上购物，True Religion的 CEO John Ermatinger 在声明中说。

True Religion 的总部在南加州的曼哈顿海滩，公司有 1900 名员工，在 140 家 True Religion和 Last Stitch 店里售卖牛仔裤和其他服饰。在商场和一些小的零售店里都能看到品牌的实体店。去年为了减少成本，公司已经关闭了 20 家店。

True Religion 成立于 2002 年，曾因为售卖一系列的中高端牛仔裤而出名。公司现在网站上的牛仔裤价格仍然保持在 199 美元以上，最贵的有 319 美元。2007 到 2012 年间，公司的收入几乎翻了 3 倍，2013 年公司收入达到 4.9 亿美元。

不过，也是在 2013 年，True Religion “开始感受到顾客从传统的零售商转向线上购物，公司销量开始下滑，” True Religion 的 CFO Dalibor Snyder 说。

线上购物的趋势不仅没有消失，这些年还不停加速冲击着公司的生意。快时尚品牌例如 ZARA 和 H&M 都让 True Religion 逐渐失去竞争力。

True Religion 不是唯一一家受此冲击的服饰品牌，总部在南加州的 American Apparel、 Pacific Sunwear of California、 Nasty Gal 和 Wet Seal 等都在过去两年内寻求破产申请。

从全美来看，那些长期依赖商场渠道的品牌都过得不好。Ann Taylor、Loft 等品牌的母公司 Ascena 零售集团计划关掉 250 家店。今年年初，打折鞋店零售商 Payless ShoeSource 申请破产，服饰品牌 The Limited 关闭 250 家店。

当然，不能全用大趋势解释自己的失误，True Religion 公司自身也存在缺陷。曾经的潮流牛仔裤制造商再也不潮了，连公司自己也承认“设计的新产品并不能和顾客产生共鸣，”Snyder 在声明中说。

截止今年 1 月 28 日，公司收入为 3.695 亿，但亏损达到 7850 万美元。

加州时尚交易协会主席 Ilse Metchek 在接受洛杉矶时报采访时表示，中高端牛仔裤的市场仍然有，只是 True Religion 并没有抓住机会。

“他们的零售店没什么激动人心的地方，他们也没有在品牌营销方面让其成为一个值得期许的品牌——顾客愿意追求这一品牌”，即使价格并不便宜。

“你不能再把一层一层的牛仔裤堆在桌子上，期望这样能让你的消费者兴奋，”Metchek 说，“这里必须得有看头，有没有一个让人兴奋的点？这一点能定义你的品牌？”
