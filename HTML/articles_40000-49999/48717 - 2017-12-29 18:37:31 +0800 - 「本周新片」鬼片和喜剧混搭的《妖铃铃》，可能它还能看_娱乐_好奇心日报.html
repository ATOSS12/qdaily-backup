「本周新片」鬼片和喜剧混搭的《妖铃铃》，可能它还能看_娱乐_好奇心日报

魏倩2017-12-29 18:37:31

另外两部可以不要去看了，安心跨年吧。

《妖铃铃》

  • 导演：吴君如
  • 编剧：周运海 / 查慕春 / 王贻兴
  • 主演：吴君如 / 沈腾 / 岳云鹏 / 张译 / Papi 酱
  • 片长：88 分钟
  • 格式： 2D

影片故事发生在一个名叫“萌贵坊”的危楼里，为了让驻扎在此的四户钉子户搬迁，开发商不惜请人扮鬼。住户们合伙请来法师玲姐驱鬼，双方扮鬼斗法，相互惊吓，开发商和钉子户之间的矛盾也逐渐升级。

定位在香港，《妖铃铃》相当不错地结合了旧式港片的多种元素。四户钉子户居住的危楼，让人想起《功夫》中的猪龙寨，得益于拥挤狭窄的布景结构，影片在单一空间中呈现出相对丰富的戏剧效果，这同样来自《新难兄难弟》等港式群居空间影片。再加上人物设置中《古惑仔》、《无间道》等元素的偶尔闪现，观影过程惊喜不少。

上世纪八九十年代，鬼片和喜剧是香港电影最流行的两种类型，当时也有过不少《开心鬼》等两种类型结合的佳作。《妖铃铃》继承了这种又惊又笑的影片特点，因此全片 88 分钟，大部分时段观众都处在两种高峰情绪当中。

在“建国后不能成精”的禁令之下，如何讲好鬼故事一直是国产惊悚片的问题。影片前半段对都市怪谈、僵尸片等鬼故事常见元素的利用都恰到好处，笑点设置也算及格。等到丧尸桥段出现，影片的惊悚怪异感达到高峰，虽然部分内容节奏略感拖沓，但大面积的丧尸画面在国产电影中出现，依然算是个小小的胜利。

不过，《妖铃铃》本质上还是一部面向内地市场的影片。作为喜剧演员吴君如的首部导演作品，除了方中信和吴镇宇，其他主要演员都来自内地，岳云鹏和沈腾的段子也颇具内地特色。

作为喜剧演员吴君如的首部导演作品，全片中唯一拖沓的点可能是挖掘角色内心世界的感人片段。由于法师铃姐的人物形象过于丰满，群戏中，几位钉子户的内心剖白和惨痛经历略显不合时宜。好在它并未影响全剧情绪节奏，马上用密集的欢笑和恐惧掩盖了喜剧作品最危险的尬场。

最后提示，结尾的几个彩蛋还不错。

《二代妖精之今生有幸》

  • 导演：肖洋
  • 编剧：肖洋 / 郭异雯
  • 主演：冯绍峰 / 刘亦菲 / 李光洁 / 郭京飞 
  • 片长：110 分钟
  • 格式： 3D

同样是讲述人妖相恋，《二代妖精》用行动证明，《美人鱼》居然是一部不错的电影。

和后者类似，以“二代妖精”为名，影片想要制造的冲突焦点在于现代都市中的妖怪生活。片中，妖怪管理局局长向男主角介绍，经过多年繁衍，妖怪已经渗入人类社会的各个方面，它们可能就是你隔壁的邻居、报纸上的大明星，或是街边偶遇的女孩。

在影片前半小时，这种类似《黑衣人》的设定确实起到了一定效果。郭京飞饰演的妖管局队长一面在房产中介工作，一面负责妖界治安，其身份反差具有喜感。

可惜的是，《二代妖精》的主线剧情始终游离在这种反差之外。年幼时，小男孩袁帅救了一只动物园中的白狐。长大后，穷困潦倒的他与白狐再次相遇，对方告诉他自己是一只来人间报恩的狐狸精，报恩的方式十分老套——以身相许。

除了二人初遇地点为动物园而非深山，表白场所是游乐场而非林中木屋、共同乘坐的交通工具是地铁而非马车之外，影片故事和传统的聊斋其实区别不大。身处上海的两人与现代都市价值观的格格不入，在反映男主心声的主题曲中充分暴露，满床狐毛中，“狐狸精与我洞房，条件和人不太一样”的歌词让本该感人的爱情故事变得格外尴尬。

另外，由于节奏控制不良，捉妖、夜总会、逃亡、横店片场和决战等剧情几乎可以拆成几个零碎片段，片尾两次情节高潮也没有形成对应关系，对于对剧情逻辑有追求的观众而言，长达 110 分钟的观影过程是个考验。

最后，作为喜剧片，对“丑恶社会现象“的反讽虽不高明，但实际观影中笑点还算密集。如果是刘亦菲或是冯绍峰的影迷，或是宽容一点，将它视作段子带奇幻故事，尚能勉强一看。

《解忧杂货店》

  • 导演：韩杰
  • 编剧：东野圭吾
  • 主演：王俊凯 / 迪丽热巴 / 董子健 / 李鸿其 / 董李无忧
  • 片长：109 分钟
  • 格式： 2D

很难相信，这部《解忧杂货店》的导演是 2011 年拍出《Hello! 树先生》的韩杰。

《解忧杂货店》原作的故事在中国算是耳熟能详。那个能帮人实现梦想的杂货店，穿梭于过去未来之间的投信口，一度成为人间温情的象征。

和东野圭吾的其他作品不同，全书实际上由几个完全不同的故事串联而成。这为影片改编带来了不小的难度，如何使影片具有完整性，又能凸显原作中的细腻和温情，是日本改编版也没有完全解决的问题。

但到了韩杰这里，我们几乎看不到任何这方面的努力。全片如同春节时央视播出的公益片广告，从主题到表现形式都没有实际关联。几个短片又平庸的极为平均，讲述着“追梦”、“家人”等陈词滥调的鸡汤主题。除了故事割裂，正如所有公益片广告一样，几位客串的明星也并没有展现出任何演技，他们出现的价值可能只在，让观众席上开始玩手机的人们抬一下头，说一句，哦，是成龙。
