特斯拉上海工厂动工，初期规划年产量 25 万辆_商业_好奇心日报

谢金萍2019-01-08 14:23:32

它将生产入门款 Model 3 以及尚未发布的小型跨界车 Model Y，且只供应大中华区。

特斯拉在中国上海的工厂于 1 月 7 日动工。作为特斯拉第一家海外工厂以及可能是其第二大工厂（具体厂房面积未知），它将生产入门款 Model 3 以及尚未发布的小型跨界车 Model Y，同时兼顾电池制造。从续航里程上看，入门款 Model 3 指的是续航约 354 公里的版本，而高性能版本则为 499 公里——该版本以及 Model S 和 Model X 仍由美国工厂生产。

“今年 Model 3 将在这里小批量生产，明年大批量生产，”出席工厂动工仪式的马斯克在现场说道。特斯拉尔后对外发出的宣传资料显示，该工厂将在年底前生产 Model 3 和 Y，一期生产规模 25 万辆/年。且从这里生产的汽车，只供应大中华区。

《华尔街日报》报道称，特斯拉希望从新能源行业的创业公司转型成为全球主流汽车制造商，中国市场是至关重要的一环，上海工厂也成了关键所在。根据特斯拉的规划，上海工厂产能最终达 50 万辆/年，这与其位于加州弗里蒙特的工厂——为其当前最大工厂——年产能目标几乎一致。

上海咨询公司 Automobility 的首席执行长 Bill Russo 称，特斯拉上海工厂是一场赌博。一方面该工厂是中国第一家外商独资汽车工厂，所有资本都落在了特斯拉身上——总投资达 500 亿元，这对此前面临着高达 685.7 亿元（10 亿美元）债务的特斯拉来说将有更大的盈利压力，它在 2018 年第三季度好不容易才突破产能实现三年来首次盈利。另一方面特斯拉面临着一个不确定的中国市场。

受关税调整的影响，特斯拉在中国的销量、销售价格出现了较大的起伏。2018 年一季度特斯拉在中国销量增长 110% 后，三季度下滑 37%，10 月时进一步下滑至 70%。为应对中国销量下滑，特斯拉 11 月 22 日称将在中国销售的特斯拉 Model S、Model X 价格下调 12%-26%，是目前降幅最大的一次。作为特斯拉主要的收入市场来源，2017 年特斯拉中国的营收占其总营收的 17%。

降价通常都是为了刺激消费需求，但对特斯拉来说并非完全有利。近期特斯拉在美国也宣布所有型号的汽车价格下调 2000 美元，这引来了分析师对特斯拉盈利进一步情况的质疑。据路透社计算，在实现盈利的 2018 年三季度当季，特斯拉每交付一辆汽车的税前利润约为 3200 美元。分析师估计，按照特斯拉目前的交付速度，2019 年降价 2000 美元意味着减少 7 亿美元的收入。而特斯拉股价在公布产能和降价后开盘下跌 9.4%，是最近 3 个月以来较大的跌幅。

另一方面，中国汽车市场整体情况令人忧心忡忡。尽管 2018 年电动车销量首次超过了 100 万辆——在政府补贴和其他优惠政策的推动下，但同期整体汽车市场销售持续下滑，将迎来 20 年来首次年度销量下滑。
