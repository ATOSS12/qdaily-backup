亚马逊再推视频购物，淘宝直播也推出了独立应用_商业_好奇心日报

张丹2019-02-11 15:49:39

都在争夺用户时间和注意力

亚马逊也开始推视频直播购物了。

近日亚马逊上线了直播和视频播放功能 Amazon Live ，要在平台上播放亚马逊品牌商家或自制商品相关的视频节目，消费者在观看时可以通过点击下方的商品链接了解产品详情，并进行购物。

目前这一功能已经同时在网站和手机应用中上线，中国大陆和香港地区目前还未上线。在亚马逊手机应用首页中，Amazon Live 已经以独立的导航栏形式出现，网站首页还未上线独立的入口，内容均由商家自己上传。

除了上线视频直播功能外，亚马逊还上线独立的直播和视频购物应用 Amazon Live Creator。就跟微博和淘宝的直播功能一样，在该应用里，商家可以通过该应用实现与观众互动，用户则可以边看直播边买。一位从早期测试阶段就开始使用该直播功能的卡片游戏制作商表示，直播功能让商品详情页每日访问量增长了 5 倍，也明显增加了销售额。

这不是亚马逊第一次尝试视频类购物，两年前还上线过一个名为 Style Code Live 的视频购物功能，主要是邀请当时热门的电视节目主持人通过介绍以及与嘉宾互动的方式推销产品，产品品类主要集中在时尚和美容类产品，但该功能仅存活了 3 个月的时间就下线了。

相比较“Style Code Live”与传统电视购物几乎一致的内容表现形式，Amazon Live Creator 上的内容表现形式更轻松，涉及的商品类目也更多。现在已经上线的内容形式有主持类、产品展示类、产品实操类等，商品类目则涵盖了智能家居、游戏、玩具、美妆护肤、厨房用品等。

淘宝也在春节前上线了独立的直播应用——淘宝直播，这其实是将原本内置在淘宝应用内的直播单独拿出来做了个应用。淘宝直播负责人赵圆圆表示，在上线至今的 3 年左右时间里，观看直播内容的淘宝移动用户超过千万，但淘宝未公布商家订单转化量等具体数据。

据 QuestMobile 发布的 2018 年移动应用报告，去年短视频应用是使用时长方面同比增速最快的应用分类，达 33.1%，增速是排在第二名的即时通讯类的近一倍，综合电商应用的使用时长增速则仅为 5.1%。

短视频内容确实可以让消费者更直观的了解商品，也比文字内容更由吸引力，但如何让用户养成在另一个独立应用里买东西的习惯，是这些电商平台的直播应用所面临的挑战。
