以及，今天是韩国前总统卢武铉自杀去世 10 周年。

今天是 2019 年 5 月 23 日，这一年的第 143 天。

1995 年的今天，Sun Microsystems 公司推出了一款新的编程语言：Java。这种编程语言具有“一次编写，到处运行”的特性，理想状况下，只需完成桌面端的开发，程序就能编译成标准的字节码，在任何装有 Java 虚拟机（JVM）的设备上运行——比如后来的智能手机。

Java 的雏形是 Sun Microsystems 公司为家用电器设计的 Oak 语言。由于当时的市场对智能化家电需求不高，这个项目被暂时冻结了起来。随着 1990 年代互联网的发展，尤其是 1993 年第一款网页浏览器 Mosaic 的出现，公司重新发现了 Oak 的应用前景，而将 Oak 改造为 Java 的任务落在了计算机科学家高斯林（James Gosling）身上。

在 Java 诞生的时代，主力编程语言是 Fortran、COBOL 和 C/C++。C 语言广泛应用于商用编程，但它当时还不够“便携”，要想同时运行于不同平台还比较困难。对于需要跨平台运行的商业应用来说，Java 的出现无疑是一个好消息。

发布后不久，Java 被 IBM 公司采用，成了一种主流语言——更多开发者开始学习、使用和改进，不断加快它的运行速度。不过，Java 最初宣称的“一次编写，到处运行”并没有完全实现，而是收获了“一次编写，到处调试”的调侃。

2009 年，甲骨文公司（Oracle）收购了 Sun Microsystems，同时接手了 Java 后续版本的开发。今年 3 月，甲骨文推出了最新的 Java SE 12。

从最初的网页开发工具，到后来的蓝光播放器、Android 系统、大数据处理，Java 的应用场景不断延伸，也反映了整个计算机工业的发展。和 24 年前相比，今天的编程语言要丰富得多，但 Java 依然被普遍用于大型企业的应用开发。

在延安文艺座谈会上的讲话

1942 年的今天，毛泽东在延安文艺座谈会上作了结论发言。这个发言和 5 月 2 日的引言共同组成了《在延安文艺座谈会上的讲话》，一份探讨“文艺工作和一般革命工作的关系”的重要文本。

毛泽东在讲话中引用列宁的名言，认为文艺应当”为千千万万劳动人民服务“，必须站在无产阶级的立场上，满足“最广大群众”的需要。这一主张的背后，是他对文艺性质的一个定义：

一切文化或文学艺术都是属于一定的阶级，属于一定的政治路线的。为艺术的艺术，超阶级的艺术，和政治并行或互相独立的艺术，实际上是不存在的。无产阶级的文学艺术是无产阶级整个革命事业的一部分，如同列宁所说，是整个革命机器中的“齿轮和螺丝钉”。

将文艺锚定在阶级、政治之上后，毛泽东进而提出了文艺批评的政治标准与艺术标准的统一。他强调，“一切危害人民群众的黑暗势力必须暴露之，一切人民群众的革命斗争必须歌颂之，这就是革命文艺家的基本任务。”他也警告，“缺乏艺术性的艺术品，无论政治上怎样进步，也是没有力量的。”

在抗日战争、延安整风等背景下，毛泽东的讲话对解放区的文艺工作者产生了重大影响。而在新中国成立后，它也被作为文艺政策的一个纲领性文件。

德国基本法

1949 年 5 月 23 日，《德意志联邦共和国基本法》在西德境内正式生效。这部宪法性文件在 1990 年两德统一后成了德国的正式宪法。

《德意志联邦共和国基本法》在很大程度上体现了对魏玛宪法的反思。基本法采用了一种罕见的体例，将“基本权利”作为第一章，作为对纳粹体制的否定。它也取消了魏玛宪法中的紧急立法权——将近 20 年前，希特勒和纳粹党正是凭借国会纵火案后出台的紧急法令，埋葬了昙花一现的魏玛共和国。

逝者：卢武铉

2009 年的今天凌晨，韩国前总统卢武铉在离家不远的烽火山“猫头鹰岩”跳崖身亡，时年 62 岁。卢武铉的遗书并未披露自杀的具体原因，但由于当时卢武铉正在接受李明博政府的受贿调查，许多民众将他视为政治清算的牺牲品。

出身寒门、职高毕业、自学通过司法考试的卢武铉有“草根总统”之称。1981 年，卢武铉出任釜林事件的辩护律师，开始活跃于韩国的政治舞台（这段经历后来被拍成了宋康昊主演的电影《辩护人》）。2002 年，卢武铉代表统一民主党赢得大选，成为韩国总统。

担任总统的 5 年内，卢武铉继承了前任金大中的“阳光政策”，对朝鲜持友善姿态。在美、日、中、俄四个大国之间，他推行“均衡外交”。例如，他一方面要求收回美军在韩国的战时指挥权，以强化韩国的国防自主性（没有成功），一方面又完成了韩美自由贸易协定的谈判。内政方面，他倡导“清廉政治”，提出建设一个高效的政府，以及结束韩国的政经勾结问题。

2008 年，朴源次案爆发，调查人员发现了朴源次向韩国总统府行贿数百万美元的证据。已经卸职的卢武铉在道歉书中承认，自己的夫人通过青瓦台行政人员收了朴源次的钱用于还债，但他也表示对此并不知情。涉腐丑闻对卢武铉是一个巨大的打击。就在夫人权良淑被传唤次日，卢武铉自杀。一周之内，数百万人前往他的灵堂吊唁。
