今天是 2019 年 4 月8 日，这一年的第 98 天。

米洛的维纳斯

根据并无实证的说法，1820 年的今天，一位名叫 Yorgos Kentrotas 的农民在希腊米洛斯岛的古城废墟里发现了一尊大理石雕像。雕像刻画的据称是希腊神话中的爱神阿芙洛狄忒，人们因这一主题及其发现地而称之为“米洛的维纳斯”（Venus de Milos）。

“米洛的维纳斯”创作于公元前 2 世纪，由 Alexandros of Antioch 打造，显示了古希腊人像雕塑的进一步发展。陈列在卢浮宫的雕像缺失了双臂，按照复原图，爱神的左手原本托着帕里斯送出的金苹果。除了神话传说，这尊雕塑也反映了古希腊人对裸体的新看法——爱神握住下身衣物的姿态带有更强的挑逗性，这种性感在更古老的雕塑中相对罕见。

将近 200 年后，“米洛的维纳斯”依然是世界上现存最著名的古希腊雕塑，每年都会吸引无数观众前往卢浮宫一睹芳容。

时报广场

如果你去过纽约曼哈顿，应该不会错过位于百老汇大道与第七大道交汇处的时报广场（Times Square）。1904 年的今天，当时还叫朗埃克广场（Longacre Square）的它改成了现在的名字，原因是广场上的一位新客人：《纽约时报》（The New York Times）。

当年 1 月，雄心勃勃的报社老板 Adolph S. Ochs 决定把总部搬到这里。搬迁计划将于次年才付诸实施，但 Ochs 已经提前和市政部门打好招呼，在广场区域新建一座地铁站，并用报纸的名字来命名地铁站和整个区域。4 月 9 日，《纽约时报》刊出了头条：《城市新中心得名时报广场》。

如今，《纽约时报》的总部搬到了第八大道的纽约时报大楼，但“时报广场”的名字保留了下来，继续作为流行文化与商业世界的一个象征。

发现超导现象

1911 年的今天，荷兰物理学家昂内斯（Heike Kamerlingh Onnes）在实验室中用液氦将金属汞冷却到了 4.19 K。此时，他忽然发现汞的电阻几乎为零，换言之，电流可以几乎不受阻碍地通过汞。

昂内斯意识到了这个发现的重要性，他很快发表了实验结果，并在后续论文中将其描述为“超导现象”（superconductivity）。

许多物理学家都对低温状态（甚至是绝对零度）下的金属导电性抱有兴趣，但直到昂内斯于 1908 年首次液化氦气、刷新人造低温纪录，相关研究才成为可能。1913 年，昂内斯因液化氮气和发现超导现象获得诺贝尔物理学奖。

一般而言，超导现象是指一些材料在临界温度下电阻为零的现象。超导材料的物理特性已经带来了广泛用途，核磁共振成像、高能加速器等技术都拜其所赐。未来，它还可能助力量子计算和更高效、环保的电能传输。

过去一个多世纪，超导相关的研究长盛不衰，5 次获得诺贝尔奖。科学家们尤其感兴趣的是“高温超导体”，这种材料如果真的存在，将极大提升超导现象的实用性。在这方面，朱经武、赵忠贤等人于 1987 年取得了突破性成果。目前，科学家们还在寻找临界温度更高的超导材料。

Windows XP 退役

2014 年的今天，微软公司（Microsoft）正式停止对 Windows XP 系统的支持服务。这意味着，该操作系统的用户将不再收到微软提供的软件更新和技术支持。微软在官网上开辟了一个专页，警告继续使用 Windows XP 可能面临的安全、合规性等风险，鼓励用户迁移至新的 Windows 系统。

这一天距离 Windows XP 发布已超过 12 年。微软后几代的操作系统中，Windows 7 发布于 2009 年，Windows 8 发布于 2012 年，Windows 10 则将于 2015 年发布。

Windows XP 是微软首个面向消费者、使用 Windows NT 架构的操作系统，在用户界面和稳定性上都有重要提升。2007 年 1 月，Windows XP 的市场占有率曾达到 76% 的历史最高点。Windows XP 的巨大成功使之成为盗版软件的主要目标，也让微软一度在操作系统市场取得几乎垄断的地位，受到监管部门的密切关注。

庞大的用户基数和使用惯性，让微软淘汰 Windows XP 的进程格外缓慢。到 2014 年 8 月，已退役的 Windows XP 仍有将近三成的市场占有率。在国内，甚至有安全软件厂商代替微软，向坚持使用 Windows XP 的用户发布更新。目前，Windows XP 在全世界大部分地区的市场份额在 4% 以下。根据百度统计，Windows XP 在中国的市场份额也已从 2015 年的超过 50% 下降至个位数。
