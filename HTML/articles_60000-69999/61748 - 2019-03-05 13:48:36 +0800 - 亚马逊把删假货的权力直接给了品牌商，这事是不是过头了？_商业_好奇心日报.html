亚马逊把删假货的权力直接给了品牌商，这事是不是过头了？_商业_好奇心日报

徐弢2019-03-05 13:48:36

亚马逊在打击假货有了一些新努力，但不知道是不是个好的尝试

亚马逊在打击假货有了一些新努力：给品牌更大的权限移除平台上的假货，不再需要等待亚马逊内部团队的审核。

2 月 28 日，亚马逊推出了一项新的知识产权保护计划 Project Zero，其名字代表着“集合亚马逊与品牌的力量，将假货减少至零”。

该计划的核心，是一项针对品牌的自助式假货移除工具。在控制后台，参与该计划的品牌可以搜索旗下产品的关键词、图片，随后对搜索结果中认定为假货的商品进行标记，移除标记的商品或者商家。

在这次新计划下，亚马逊主要负责前期管理工具的培训，以及后续品牌措施的监督。

亚马逊称，品牌必须保持较高的准确率，以保持他们在 Project Zero 计划下的权益。该公司还将要求，参与 Project Zero 的品牌需要接受必要的培训，并将持续对品牌的行为进行监控，防止该工具被滥用。

即便亚马逊仍将管控品牌的行为，与亚马逊以往的知识产权保护措施相比，在这次的 Project Zero 计划下，品牌获取了更大的权限。

过去，品牌在发现假货后，需要向亚马逊发出申请。亚马逊的内部团队对此进行评估后，再决定是否进行移除、对消费者进行退款。

通常，亚马逊内部团队针对这类打击假货商品的诉求，其反馈机制需要 4 个小时。 亚马逊曾给出的数字是，针对参与知识产权保护计划 Brand Registry 的商家，亚马逊内部团队平均在 4 小时内对打击假货的诉求进行调查。

参与 Brand Registry 计划的商家，需要向亚马逊提交商标、产品图片等信息。亚马逊称，超过 4 万品牌已经参与了 Brand Registry 计划。

这种品牌反馈、平台审核知识产权的做法，在互联网公司间是个较为常规的做法。美国电商网站 eBay、视频网站 YouTube 等公司也都是类似的做法。

但这次，亚马逊的 Project Zero 计划采用了类似于先斩后奏式的机制，似乎是向品牌更大程度地示好。

不过，大品牌抄袭小品牌的案例也时有发生，这类事件中界定抄袭的难易、品牌体量大小，可能都会影响对抄袭的判断。有可能产生的情况是，大品牌利用 Project Zero 挤压了小品牌的生存空间。

在 Project Zero 计划下，亚马逊还推出了 2 个新功能。其中一个是“自动保护机制”，亚马逊利用品牌提交的 logo、商标等信息，使用机器学习加快对平台上商品的审查速度。

另一个类似的功能是，亚马逊为品牌提供一串产品序列号。品牌可以将这串亚马逊的产品序列号印制在单个产品或者包装上，确保该产品在进入亚马逊库房时确保产品不是假货。这是一个付费功能，亚马逊针对不同数量规模的商品收取每件 0.01-0.05 美元的费用。

这次 Project Zero 计划可以说是 Brand Registry 计划的升级版。Project Zero 计划目前还在内测阶段，由亚马逊面向美国公司发出邀请。但亚马逊要求，申请参与 Project Zero 计划必须参与 Brand Registry 计划。

亚马逊称，在过去数个月内，该公司已经邀请大约 15 个品牌参与 Project Zero 计划进行测试，包括包袋品牌 Vera Bradley、宠物除毛器品牌 ChomChom Roller 等。

与品牌提交更多信息获得更好保护相比，亚马逊允许第三方商家以相当低的成本开店、销售商品，假货很容易出现。这些商家只需要简单的几个信息，就可以将商品放在亚马逊平台上销售，包括公司名称、邮箱、地址、电话、信用卡、身份证、银行账户。2018 年，亚马逊将近一半的商品销售来自于第三方商家。

今年 2 月初，亚马逊在发布年度财务报告时，第一次在报告中提到了平台上售卖的假货给亚马逊带来的风险。这可能是大品牌过去几年持续抗议亚马逊上假货的一部分结果。
