因电子系统和配件质量问题，《消费者报告》不再推荐 Model 3_商业_好奇心日报

罗骢2019-02-22 13:33:10

特斯拉股价一天之内下跌 3.7%

2 月 22 日，美国消费者联盟创立的知名测评杂志《消费者报告》在官网发文，将此前推荐购买的特斯拉 Model 3 电动汽车调整为不推荐购买。

《消费者报告》对特定车型的推荐基于几个因素，包括车主的反馈、碰撞测试性能以及《消费者报告》汽车团队进行的测试和评估。而基于最新的特斯拉 Model 3 车主反馈调查，《消费者报告》认为 Model 3 在车漆、车窗质量、电子系统等方面都存在问题。

根据超过 500 名 Model 3 车主反馈，特斯拉的电子系统会出现间歇性故障，例如无人操作时中控屏幕会自己启动一些按钮。这个故障会导致音乐自动播放、音量调到最大值或者自动打开地图等。

还有一些车主在回复调查时抱怨车辆玻璃出现裂缝。《消费者报告》自己的测试 Model 3 也出现了相同的问题。在今年 1 月份，《消费者报告》的测试车辆因为室外寒冷的低温导致后车窗出现一个大裂缝。

《消费者报告》汽车测试高级主管 Jake Fisher 表示：“ Model 3 很多问题都是电子产品问题。例如，导航/信息娱乐屏幕存在一些问题，我们还看到车漆、玻璃等其他问题。”

这些使用上的质量问题通常被《消费者报告》归类到可靠性得分中。新的调查显示：Model 3 的可靠性得分已经低于平均水平。Fisher 表示使用上的可靠性一直是特斯拉电动车的短板。

特斯拉中高端车型 Model S 在 2018 年 10 月《消费者报告》不再被推荐购买，原因是空气悬挂问题。SUV 车型 Model X 则主要是因为后座鹰翼门设计，从未获得《消费者报告》推荐。

《消费者报告》对 Model 3 评价在过去发生两次转变。2018 年 5 月之前，Model 3 被《消费者报告》评为不推荐购买，理由是车辆刹车距离过长，远超其他测试车辆， 甚至比福特 F-150 全尺寸皮卡停车距离还多出 2 米。

之后，特斯拉依靠远程软件更新将刹车距离减少了 6 米。《消费者报告》才将 Model 3 调整为推荐购买。但当时 Fisher 在接受《今日美国》采访时还是强调，“值得推荐购买的 Model 3还有其他例如座椅不舒适、车内噪音过大等问题。”

这一次《消费者报告》的评级下调也证实了许多分析师对特斯拉 Model 3 质量下滑的担忧。去年，特斯拉为达成 Model 3 的生产目标，大幅增加了产量。甚至在加利福尼亚州弗里蒙特的装配厂外竖立一个永久性帐篷，增加了一条 Model 3 装配线。这使市场对车辆质量问题产生质疑。

在 Model 3 的评级下调后，特斯拉股价一度跌幅达到 3.7%。特斯拉发言人回应《消费者报告》称：“我们为 Model 3 设定了极高的标准。杂志的调查从去年 7 月到 9 月进行的，但我们现在已经做出了重大改进，绝大部分问题已经通过设计和制造技术改进得到解决。我们的退货政策也允许任何对汽车不满意的客户获得全额退款。”

虽然许多特斯拉车主对他们的汽车的可靠性不满意，但《消费者报告》称这些车主对他们的电动车整体感到满意。

就在两周前，《消费者报告》根据超过 50 万车主的调查结果给出了 10 个最令人满意的车型，特斯拉 Model 3 被评选为最令车主满意汽车。92％ 的特斯拉 Model 3 车主表示，如果重新买车， 他们依然会选择 Model 3。Fisher 在这次评级下调的文章中也指出 Model 3 的驾驶体验很好，主要归功于电动机的出色操控性和即时推力。

作为特斯拉面向主流市场的车型，Model 3 始终受到车主的关注。目前，特斯拉正在推进中国市场 Model 3 的销售。

2 月 11 日，首批用于正式交付的特斯拉 Model 3 的货轮“Glovis Symphony”抵达天津港。另外两艘载有 Model 3 的货轮 “Morning Cindy” 和 “Emerald Ace” 也在一个星期后分别到达上海港和天津港。三艘货轮装载的 Model 3 都将于今年 3 月交付给中国市场中预购的车主。

上周，特斯拉公布了 2018 年财报，卖车收入为 176.32 亿美元，同比增长 106.59%，但整体还处于亏损状态。同时在中国的销售收入下滑了 15%，这跟去年中国提高美国汽车进口税至 40%有关，为了应对高关税，特斯拉车价提价，随后又不得不主动降价。

特斯拉在财报中强调，今年将重点发展中国和欧洲等美国之外的市场。为了扩大 Model 3 在中国的需求，特斯拉春节前还将价值 4.63 万元的 Model 3 “Autopilot” 自动辅助驾驶变为限时免费安装。并称今年上海工厂将完工，并于今年年底或明年初投产。

 Model 3 的评级下调可能会影响汽车销量。作为一个品牌，特斯拉在《消费者报告》排名 33 个品牌中下跌 11 名，至第 13 位。特斯拉与克莱斯勒并列在今年汽车品牌排名上降幅最大。
