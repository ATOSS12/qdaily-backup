特斯拉推出 3.5 万美元 Model 3，要将销售渠道全部转为线上_商业_好奇心日报

罗骢2019-03-04 06:55:39

马斯克还表示：特斯拉第一季度可能无法盈利

2 月 28 日，特斯拉在官网公开信中表示：推出 3.5 万美元售价的 Model 3 和 3.7 万美元版本的 Model 3 Standard Range Plus，同时，特斯拉将逐渐关闭全球范围内的线下体验店，转向线上卖车。

此次推出的 3.5 万美元版本 Model 3 续航里程为 220 英里（约合 354 公里）。3.7 万美元版本的 Model 3 Standard Range Plus 续航则多出 9%，续航里程为 240 英里（约合 386 公里）。

但从官网的配置来看，3.5 万美元版本的可选项则比 3 年前发布会上宣传的少了很多，车身颜色只有黑色，包括更多车漆，轮毂配置等都变成选装项，需加钱才可以配置。Model 3 统一配置，主要目的还是降低制造成本，压低售价。自动化系统管理的柔性生产线上，选配项目减少意味着 Model 3 加工设备的利用率可以提高，产线运行也可以更灵活。

2016 年 4 月 1 日，特斯拉创始人马斯克发布 Model 3，当时宣传的基础售价就是 3.5 万美元。让此前昂贵的特斯拉电动车成为普通消费者可以购买的车型。

但提前支付定金下单的消费者迟迟没有等到 3.5 万美元的 Model 3 出货，马斯克表示过早生产 3.5 万美元售价版本的 Model 3 没有利润。在此次基础版本推出之前， Model 3 的起售价为 4.2 万美元，在北美市场和豪华车奥迪 Q5 接近。

推出新车的同时，特斯拉还宣布逐渐关闭线下体验店，理由是节省成本。特斯拉称这将使得电动车平均车价降低 6%，也使得 3.5 万美元的 Model 3 可以尽早推出。目前，特斯拉在北美市场可以在官网和手机上购买特斯拉电动车，手机购车功能将在全球市场陆续推出。

传统汽车生意中，车辆的生产和销售都是分离的。汽车公司造出来的车先卖给 4S 店，车厂没有库存压力，分销商还能快速打开渠道。4S 店从车厂买车后，再慢慢卖给顾客，要承担巨额资金压力，但通过分销可以获取后续服务的利润。

这种模式还受到美国法律的保护。由 17000 多家经销商组成的 NADA 推动制定了《特许经销商法案》，其中规定了不允许汽车厂商开设直营店。在很多小地方，汽车经销商是当地最大的组织联盟，纳税可以达到 20%，且带来了很多投资。

特斯拉并未选择和经销商合作，而是从一开始选择体验店和网络直销的模式。

但由于《特许经销商法案》的存在，特斯拉一度在全美 48 个州无法设立体验店卖车，之后特斯拉虽然在美国大部分地区开设了体验店，但其中有很大比例的新车是通过经销商销售，这提升了特斯拉的销售成本。

“美国关店主要是因为受到强大的经销商联盟的影响，一些州政府对特斯拉直营店做出限制，但中国不存在这一问题。”一位特斯拉内部人士向《第一财经》表示。

公开数据显示，截至去年底，特斯拉在全球共有 378 家商店和服务中心。特斯拉并未透露此次关店的具体数量以及会节省多少成本，只表示可以降低 6% 的平均车价，使得 3.5 万美元的 Model 3 可以尽早推出。

目前已经确定将关闭门店的州包括康涅狄格州和新墨西哥州。马斯克表示：“我们不得不采取关店措施，会导致一些人员的削减，这是毫无疑问的。我也希望有其他办法，但是很不幸的是，我们不得不削减销售方面的人员，这是绕不开的。”

特斯拉表示，未来特斯拉仅存的少数门店将主要起到展览展示和信息服务的功能，而不会成为销售中心。在公开信中，特斯拉的服务中心数量是否会调整并未作出声明。

在中国，特斯拉的体验店通常开在高端商场内部，例如北京芳草地，上海新天地等位置。租金成本相对较高。一位特斯拉直营店的店员向《好奇心日报》表示，“中国市场的直营店暂时不会受到影响，因为特斯拉在中国市场还处于宣传阶段，完全在线购车也尚未被中国市场接受，但之后将大力推广线上购车。”

关闭线下门店的确会降低运营成本，但同时对特斯拉的售后服务体验也会产生不小影响。

目前，特斯拉在上海地区只有 3 家服务中心，在线下单后，车主需要前往服务中心提车。这一定程度影响用户体验。特斯拉目前还测试新的交付系统，通过封闭式拖车从工厂直接运送到用户家中。

对于特斯拉来说，资金短缺是目前面临最大的问题。负债情况显示，特斯拉有一笔 9.2 亿美元的可转债在 3 月 1 日到期，由于特斯拉股价没有达到可转债要求的 360 美元，这部分债务无法转成股权支付，特斯拉必须以现金的形式连本带息支付这笔债务。

但特斯拉的最新财报显示：特斯拉持有现金流只有 7.18 亿美元。马斯克此前表示特斯拉并不需要融资，就可以偿还债务，维持公司经营。但目前来看，还很难说。

马斯克周四在电话会议中表示，预计今年第一季度特斯拉盈利的可能性不大，主要因为第一季度向欧洲和中国出口遇到了很多前所未有的挑战，他预计特斯拉第二季度将能实现盈利。特斯拉股价当天下跌 8%。
