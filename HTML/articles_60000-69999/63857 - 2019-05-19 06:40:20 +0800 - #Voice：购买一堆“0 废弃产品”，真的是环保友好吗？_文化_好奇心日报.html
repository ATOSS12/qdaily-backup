#Voice：购买一堆“0 废弃产品”，真的是环保友好吗？_文化_好奇心日报

顾天鹂2019-05-19 06:40:20

来自消费主义的洗脑：买买买可以解决一切，包括环保

Reddit 上一名 r/ZeroWaste 板块的用户对整天霸屏的“看看我新买的 0 废弃 XX” 感到了厌烦。她在忍受了数周后发帖：你们网购的很多 0 废弃产品都要消耗大量的不可循环包装，运输过程中还产生了很多碳排，还有好多东西都可以在任何地方的二手店买到，购买行为本身就是问题的一部分。

竹炭制品，秸秆制品，金属吸管，纸质购物袋……环保友好产品的市值在快速增长。在 2018 年，可循环水瓶的市值达到了 80 亿美元，环保材质（竹制和有机棉等不含杀虫剂和化学药剂的材料）市场预计在 2025 年价值 930 亿美元，绿色包装（可循环或可降解的自然材质）将在 2021 年达到 2150 亿美元的市值。如今，即使是在 Kickstarter 上搜索“可重复使用的吸管”，也能跳出几十种结果；Etsy 上的 0 废弃自制新手包更是五花八门。

迎合消费者对环境污染的焦虑，大小公司们适时转向绿色环保，贴心地为人们提供“不浪费资源”的另一条路径，它恰恰也很容易达成，只要你把旧物品都换成绿色的、可循环的替代物，购买打着环保标签的产品就好了。虽然这其中显然忽略了 3R 中的 2R：减少（Reduce）和重复使用（Reuse）——它最大的效应可能是心理上的安慰，似乎你为环保做出了一点微小的贡献。

当然，你也可以理解为什么“重复利用”相较之下更少被强调，因为它赚不到钱，但是不断卖出环保产品却是好生意。

Anna Marie Boneau 是一位 0 废弃网红博主，她很少使用可循环产品，因为她限制了自己的购买。在她看来，“循环”应该是不得已使用的最后一招，而“少购买”就是第一道防线，包括那些被定义为环境友好的商品。她担忧 “0 废弃”很快会变成另一种消费主义生活方式。

我们已经被消费主义社会训练出一种习惯——哦，我遇到了这个问题，那么我要买什么才能解决它呢？其实你明明可以环顾四周，找到更多能用上的资源。

Anna Marie Boneau

Boneau 认为，购物似乎是改变生活方式的捷径，买一个 0 废弃新手包、一夜之间告别浪费坏习惯，听起来也很有诱惑力，但这不是唯一的答案。比如在买竹制餐具前，也许人们该打开抽屉找找有没有旧餐具可继续使用。环保主义者要如何生活，其实更多关乎心态、意识和更周虑的选择。

就连一家专门出售 0 废弃产品的商店 Eco Collective 都宁愿顾客少买东西。“我们一直在谈论理性消费，在这里，我们从来不鼓励过度消费，哪怕这意味着销量减少”。这家店还会鼓励顾客去二手店购买玻璃罐之类的常见物品，并教他们用旧床单和旧 T 恤制作手提包。“很多环保教育都是在店里进行的，我们告诉大家的第一件事就是，别把 0 废弃和拒绝塑料当成一个概念。如果你已经有了一堆塑料衣架，你就不该去重买一堆木制的。当然，如果你有需要，请做出负责的选择。”

不过也有一种观点认为，用不着对极力想改变生活方式的环保主义新人太过苛刻。只要他们购买的产品确实有效助其长期保持 0 废弃习惯，那也没什么不好。
