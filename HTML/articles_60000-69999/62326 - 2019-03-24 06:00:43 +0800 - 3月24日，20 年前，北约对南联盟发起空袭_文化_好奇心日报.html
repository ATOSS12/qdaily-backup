3月24日，20 年前，北约对南联盟发起空袭_文化_好奇心日报

蔡一能2019-03-24 06:00:43

同样在这一天发生的“瓦尔迪兹”号漏油事故，则成为石油业管制、海洋保护的前车之鉴，但人们并没有牢记教训。

今天是 2019 年 3 月 24 日，这一年的第 83 天。

1999 年的今天，以美国为首的北约军队轰炸了南斯拉夫联盟共和国。这场空袭一共持续了 78 天，最终为贯穿 1990 年代的南斯拉夫内战画上了句号。

二战结束后，南斯拉夫建立起一个多民族的社会主义国家。随着建国领袖铁托于 1980 年去世，以及苏联和整个东欧的改革进程，南斯拉夫境内各民族的分离主义倾向日益严重，最终演变为 1990 年代加盟共和国的独立潮，阿尔巴尼亚人占多数的科索沃就是其中之一。

版图的剧烈变迁，让原南斯拉夫境内各民族、各加盟共和国冲突不断。塞尔维亚人主导的南联盟政府坚称享有科索沃的主权，北约则认定南联盟对科索沃境内的阿尔巴尼亚人实施了种族清洗，决定军事介入。当年 6 月，迫于北约的军事压力，加之俄罗斯的斡旋，南联盟同意妥协，从科索沃撤军，由联合国维和部队监督科索沃的重建进程。

科索沃战争结束后，南联盟事实上失去了对科索沃的治权。2008 年，科索沃宣布独立，但尚未获得国际社会的一致承认，亦未加入联合国。南联盟领导人米洛舍维奇后来受到国际刑事法庭的审判，病逝于海牙羁留中心。

北约对南联盟的空袭伴随着巨大争议。一方面，它基本结束了科索沃境内的种族危机；另一方面，它也开创了北约在未获联合国授权的情况下、对并非敌对国的主权国家使用武力的先例。对中国驻南联盟大使馆的“误炸”，造成 3 名中国记者丧生，深刻影响了中国人对西方的看法。

发现结核杆菌

1882 年的今天，德国微生物学家罗伯特·科赫（Robert Koch）宣布结核病是一种由结核杆菌引起的传染性疾病。这个发现颠覆了人们对结核病的理解。

19 世纪，结核病是欧洲流行程度和病死率最高的疾病之一，但人们仍对它缺乏了解，普遍将其归咎于遗传原因。19 世纪中期以后，科赫、巴斯德等微生物学家发展出病原体致病理论，成功解释了多种疾病的发生原理。结核杆菌的发现再次证明了细菌学这门新科学的价值。

1905 年，科赫凭借“对结核病的相关研究和发现”获得诺贝尔生理学或医学奖。不过，他提出的结核病疗法被证明是一项失败的研究。直到 1921 年卡介苗问世、1943 年链霉素被成功分离，人类才在和结核病的较量中逐渐占据上风。

“猫王”入伍

1958 年的今天，流行音乐巨星埃尔维斯·普雷斯利（Elvis Presley，“猫王”）以平民身份应征入伍，前往德克萨斯州的营区报到。当年 10 月，普雷斯利前往德国，在美国陆军第 3 装甲师服役。

2 年的军营生活中，普雷斯利的人生遇到了重大变故。入伍当年，他的母亲去世，普雷斯利获准返乡，不久就踏上了前往德国的旅程。另一方面，他在德国认识了一名空军上尉的养女、14 岁的普莉希拉（Priscilla Presley），两人于 1967 年结婚，育有 1 个女儿。

入伍也是普雷斯利演艺生涯的一座分水岭。重新投入音乐界后，他的佳作数量明显减少，但还是斩获一座格莱美奖；同时，他将更多精力放在了好莱坞电影上。

“瓦尔迪兹”号漏油事件

1989 年的今天，美国埃克森石油公司的“瓦尔迪兹”号油轮在阿拉斯加州威廉王子湾触礁，导致 1100 万加仑的原油泄漏，被认为是当时世界上最严重的环境污染事件。

漏油事件污染了约 2080 公里的海岸线。有统计称，事件最终导致约 25 万只海鸟、3000 只海獭、300 头斑海豹、250 头秃鹰和 22 头虎鲸死亡，阿拉斯加的渔业也受到严重影响。

2004 年，阿拉斯加州法庭判决埃克森石油公司支付 45 亿美元的巨额赔偿，创下此类案件的纪录。联邦政府方面采取了更积极的措施。1990 年，时任美国总统老布什宣布完全冻结美国近海的石油开采。然而，近海石油开采禁令在小布什执政时期解除，奥巴马政府也扩大了近海油气田开发。

2010 年，英国石油公司在墨西哥湾的钻井平台发生泄漏，重演了 1989 年的那场环境灾难。

Mac OS X 发布

2001 年的今天，苹果公司发布了新款图形化操作系统 Mac OS X v10.0，内部代号为“猎豹”（Cheetah）。系统名称中的 “X” 为罗马数字中的 10，意味着这是 Mac OS 系列的第 10 个版本。然而，Mac OS X 实际上开启了一个独立发展的新系列，苹果个人电脑操作系统自此开始以 “Mac OS X + 数字版本” 来命名。

早在 1990 年代，苹果公司就尝试为麦金塔电脑（Macintosh）电脑开发新一代的操作系统，但所有尝试均告失败。最终，苹果请回了联合创始人乔布斯，以及他离开苹果后创立的 NeXT 公司，后者为 Mac OS X 提供了 OPENSTEP 的 API 架构。

对普通用户来说，Mac OS X 的变化是显而易见的：Aqua GUI 带来了柔软的边缘和半透明颜色，让操作系统的外观更具有亲和力。批评者则认为新的 GUI 显得不够专业，拖慢了系统速度。
