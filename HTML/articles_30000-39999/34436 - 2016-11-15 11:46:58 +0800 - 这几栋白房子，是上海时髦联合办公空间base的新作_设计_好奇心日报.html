这几栋白房子，是上海时髦联合办公空间base的新作_设计_好奇心日报

高雅2016-11-15 11:46:58

效率和体量，可能会成为这个白房子团队面临的新问题。

上周四晚的上海安福路上有一场电影之夜活动，啤酒、电影、日本小食，稀松平常的周末氛围。有趣的是，活动地点外面的整栋大楼被包了起来。一个名为 base 的团队正在重新整修这栋大楼。很快，这栋大楼会变成 base 一系列标志性白色建筑中的一组。

黑白外观是 base 所有改造项目的特点，它能让 base 的改造迅速和周围建筑区别开来。 同时，“黑白是一个让空间显得干净最简单的方法。”首席设计师 Borja 解释道。

base 是翌成创意旗下的品牌。我们曾报道过翌成创意通过投资运营、改造上海老房子做了一系列生意，旗下品牌 base 团队负责房地产开发设计和商业运营，包括办公空间 base OFFICE、服务式公寓 base LIVING、联合办公空间 base CO，并配有完善的物业管理。

除了旧项目的整修，base 今年在张江和宋园新开了两个项目。和以往不同的是，这次的改造体量更大。

由四栋大楼组成的宋园项目，前身是职业技术学院，位于黄金城道，这次将 base OFFICE、 base LIVING、base CO 整合在了一起，在数量和复杂程度上都很比以前的项目更具有挑战性。首席设计师 Borja Trujillo Rodríguez 在接到这个大项目的时候，首先考虑的问题便是建筑与建筑之间的连接性，他用小型的木制庭院将楼宇连接。而不同的入口又将动线分离开来，确保空间人流的有序流通。

社交区域是 base 改造项目很看重的一部分，基本每一栋楼都设有大露台。宋园 base LIVING一个 170 平米的复式房型，大的木质阳台占据了二层一大半的空间，这在市城区很少见。 这样的户型一共有三套，价格在 35000/月。“因为我们的客户很多都是外籍人士，他们有这样的需求，你可以在阳台上和朋友一起 barbecue，办 Party”， base 的生活管家解释道。base 的所有设计都围绕着这些目标客户的生活方式。

宋园项目毗邻古羊路酒吧餐饮一条街，这里聚集了很多日本人。新的 base Living 已经有几位日本客户即将搬入。base 称，针对大公司 1 号楼的办公室大部分也已经租掉了。

位于张江集电港的项目，原本是一栋旧式的酒店大楼，从 2015 年年底了解项目、签订框架协议到今年十月第一位德国客人入住，用了不到一年。半透明、透明的玻璃材质是张江项目的特色，让人想起科幻电影《机械姬》里的场景。深化设计师 Goran Kordić 解释，玻璃是一种常见的、简单的材质，这种半透明的空间，可以保证足够的光线。张江项目的客户经理对《好奇心日报》说，“如果客户担心隐私性，我们也会根据他们的需求再进行改造，这并不会消耗太多成本。”

张江集电港周围 IT 企业较多，需求偏向小而精，base 首次增加了 SOHO 的概念。室内设计有很强的空间错落感，木质材料“包裹”起来的卧室建在略高的平台上，并用推拉门区隔。“这在一个空间内制造了两种不同的氛围，客厅／办公区的空间更大，卧室的天花板更低一点，会让人有一种被保护、被庇护的感觉”，而木质的材料，“会让人觉得有种镇静、自然的感觉”。

“这也和功能性有关，”Goran 对《好奇心日报》说，“张江的户型普遍偏小，提升的区域下设计成了储藏柜，可以充分利用空间。”

值得一提的是，此次张江的项目购买方是盛煦地产，一家专业经营物业投资及资产管理公司，而在今年年初，在华平投资的连线下，盛煦和翌成两家公司合并了。

看起来，翌成正在迅速发展。它位于工艺美术创业园区的办公室也扩展到了两层，项目更新速度较之前更快，面积也更大了。不过，新项目目前入驻率并不高。base 的工作人员解释说，这是因为“体量比较大”的原因。

效率和体量，可能会成为这个白房子团队的新问题。另一个问题在于跨区域的业务扩张。翌成的存量房改造计划扩展到了北京，base OFFICE 近日落户北京朝阳区，两层的办公空间改造自一个老百货公司。同样，这个办公空间拥有黑白分明的外观，裸露的水泥柱、原木、玻璃、钢梁。
