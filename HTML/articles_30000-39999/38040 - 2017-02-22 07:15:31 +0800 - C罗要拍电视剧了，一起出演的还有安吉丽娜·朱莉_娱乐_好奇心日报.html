C罗要拍电视剧了，一起出演的还有安吉丽娜·朱莉_娱乐_好奇心日报

宣海伦2017-02-22 07:15:31

讲述的是一个叙利亚家庭在内战中逃离家园的故事

一向活跃于文体界和商界的 C 罗最近又有新动向，将参与土耳其电视剧《生命之桥》的拍摄，同时确认出演的还有安吉丽娜·朱莉和黎巴嫩歌手南希·阿吉莱姆，后者是阿拉伯世界和中东地区的超级巨星。

导演 Eyup Dirlik 目前已经确认了这一消息，电影将在四月第一周开拍。电视剧题材和叙利亚内战导致的难民潮相关，讲述一个叙利亚家庭在内战中逃离家园的故事。难民问题也是 C 罗和安吉丽娜·朱莉长期关注的公益领域之一，两人此前都曾多次参与相关的慈善活动。根据外媒报道，这部电视剧由叙利亚的邻国土耳其投资拍摄。

除此此外，C 罗本人在土耳其也拥有自己的生意和投资。根据《太阳报》报道， C 罗曾帮助土耳其足协主席 Yildirim Demiroren 经营其购物中心，这部电视剧的拍摄对他在中东的生意也会有所帮助。

尽管这是 C 罗首次出演电视剧，但对于在商业广告中身经百战的葡萄牙人来说应该是驾轻就熟了。去年欧洲杯前夕，C 罗为耐克出演了有史以来最长的一支品牌广告片《灵魂互换》——时长 5 分 57 秒，充分展现了一番演技。

目前几位明星在剧中的角色和戏份比例还无从知晓，不过考虑到西甲密集的赛程，很难想象葡萄牙人会为电视剧抽出太多时间。但即使只是露个脸，这几位流量担当也能为这部难民题材的电视剧带来不少关注了。根据数字媒体工具 Hookit 的统计，C 罗在 2016 年在社交网络上共发布了 1703 条消息，总计生成了 22.5 亿余次社交互动。

此外，还有一条流传已久的八卦也让 C 罗和安吉丽娜·朱莉可能出现的对手戏充满看点—— C 罗的 11 位正牌前女友或多或少都和安吉丽娜·朱莉神似。
