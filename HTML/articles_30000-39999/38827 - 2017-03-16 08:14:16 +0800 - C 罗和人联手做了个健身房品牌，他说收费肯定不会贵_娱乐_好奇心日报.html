C 罗和人联手做了个健身房品牌，他说收费肯定不会贵_娱乐_好奇心日报

宣海伦2017-03-16 08:14:16

比起奇奇怪怪的瘦脸仪，还是投资健身房适合 C 罗呀

C 罗终于进入了自己最得心应手的领域，与美国健身品牌 Crunch 合作创建了新的连锁健身房 CR7CrunchFitness，第一家门店开在马德里。

Crunch 是创立于纽约的健身俱乐部连锁加盟公司，主打高性价比的低价策略，一个月的会员费最低只要 9.95 美元，在美国 32 个城市有超过 145 家门店。CR7CrunchFitness 品牌预计会成为 Crunch 旗下的高端线，但目前还不清楚具体定价情况。

C 罗在开业仪式上说：“这家健身房的理念很特别，在这里锻炼会非常有趣。这里收费低廉，没有年龄或工作上的限制，任何人都可以来，我认为这能激励更多的人来健身房锻炼。”此外，他还否认了外界关于他疯狂健身的传言，“（每天做 3000 个仰卧起坐）这不是真的，通常我每周训练 3-4 次，每周加起来都不能保证做到 1000 个。”

C 罗的个人官方账号也在微博上发布了这一消息，用的是其代言的努比亚黑金传奇手机?

克里斯蒂亚诺·罗纳尔多素以强悍的身体素质著称，2016 法国欧洲杯半决赛上 C 罗在比赛中上演惊天一跳，接应队友格雷罗头球破门。根据赛后数据显示，这一跳的高度达到 2.88 米，比 NBA 球员的平均水平还要高。

为了制造这一跳，C 罗需要在起跳过程中制造 5 重力单位的力，相当于 5 头非洲猎豹起跳时制造的力。这说明 C 罗肌肉中的快肌纤维达到了很高的程度，帮助其获得更快起跳速度。

再加上 C 罗有在进球后展示自己的傲人身材的习惯，这也让他成为众多健身器材、健身食品争相追逐的代言对象。此前他曾为日本 MTG 代言旗下腹部健身仪 SIXPAD。

找一个发展成熟的行业公司合作，然后使用自己的名字创建新品牌，是 C 罗布局自己商业帝国的惯用手段，C 罗已经用自己的名字创建了内衣、鞋类、连锁酒店等品牌。去年 7 月，C 罗在西班牙的旅游胜地伊比萨岛开出了第二家 “CR7” 酒店，第一家位于他的家乡丰沙尔，合作方是佩斯塔纳酒店集团。
