C 罗和皇马续约至 2021 年，这会是他的最后一站吗？_娱乐_好奇心日报

宣海伦2016-11-07 18:21:37

如无意外，这将是他在皇马得到的最后一份合约

皇马官方宣布与当家球星 C 罗续约至 2021 年，考虑到届时 C 罗将年满 36 岁，签下这份五年长约意味着葡萄牙人希望终老伯纳乌的决心。

续约仪式将在当地时间 11 月 7 日下午 13 点 30 分（也就是北京时间今晚 20：30）伯纳乌球场的主席办公室进行，更多有关合同的细节尚未公开。考虑到一周前皇马刚刚为大将贝尔提供的 1.08 亿英镑六年合同，附带价值 9 亿英镑的解约金条款，C 罗的新合约价值应该只高不低。

C 罗在 2009 年以创纪录的 8000 万英镑自曼联转会皇家马德里，在这里他到达了职业生涯的巅峰，为球队获得两座欧冠冠军、一座西甲冠军、两座西班牙国王杯冠军、两座欧洲超级杯冠军和一座世俱杯冠军。

如果仅仅从荣誉层面看，现年 31 岁的 C 罗在过去一年看起来依然“呼风唤雨”。率领葡萄牙夺得欧洲杯冠军、入选了 2016 年度的金球奖 30 人、入选世界足球先生 23 人名单，并且在两项评选中都有很大的呼声获奖。

但欧冠、欧洲杯密集赛事的投入，膝盖反复的伤病，让葡萄牙人在皇马的战术体系中发生了变化，开始和队友分享火力权。新赛季续约后的贝尔的年薪已经超过 C 罗，成为队内年薪最高的球员。

C 罗新赛季为出场 12 次，共献出 7 球 5 助攻，目前在联赛射手榜上排第七位。这对于绝大多数效力于五大联赛的前锋来说，都是相当不错的数据，但到了 C 罗或梅西这样的球员身上的时候，人们就开始讨论：C 罗的状态为什么下滑了？C 罗还能在皇马踢多久？

C 罗此前曾不止一次地表示过，自己希望可以保持竞技状态，一直踢到 40 岁，也曾表示过愿意在皇马结束职业生涯。但鉴于皇马一贯的治队作风，这事儿并不容易办到，C 罗之前的两任皇马队长卡西利亚斯和劳尔都在职业末期离开，选择转会到其他俱乐部以获得足够的出场时间。

在足球世界，一名 31 岁能与俱乐部签下一份五年长约（并且很可能是世界上最昂贵的合同），是一件不常发生的事情，即使他是 C 罗。

首先，FIFA 对皇马的转会禁令（直到 2018 年 1 月才能引进球员）使葡萄牙人在续约谈判中处于绝对优势，皇马清楚地知道短时间内仍然无人可替代 C 罗的作用，无论是在竞技场还是在商业层面，无数赞助商和球迷都会对其续约额手相庆。

此外，C 罗此前的合约还剩余一年半左右，让这位当家球星作为自由球员免费离开显然是皇马最不愿看到的事情。无论 C 罗未来是前往其他欧洲联赛、或者美国大联盟继续职业生涯，涉及的都将是一份天价合同。

考虑到续约后将支付给 C 罗天价年薪（C 罗很有可能再次超过贝尔，成为队内第一高薪的球员），皇马在数年后卖掉 C 罗成为一件很可以想象的事情，这和葡萄牙人的健康状况实在是有很大关系。
