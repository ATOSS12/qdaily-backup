苹果收购了生产力工具 Workflow，这是做什么用的？_智能_好奇心日报

周韶宏2017-03-27 12:30:17

教你怎么用编程思维使用生产力工具。

  • Workflow 是个“用应用编写应用”的工具，就像生产线那样完成重复任务。
  • 你可以自己编写流程，让不同应用的功能在一个应用里完成指定动作。
  • Workflow 的团队已经被苹果收购，目前在 App Store 由付费调整成免费。

苹果最近收购的一家公司，产品是一个让人开脑洞的生产力工具。

你的手机里有多少应用？常用的那几个，每一天也许都会完成特定的重复动作。比如你有搜集壁纸的习惯，每天都会在固定的几个网站下载图片；或者把保存在笔记本里的文章发送到邮箱，之后再删除原笔记。

Workflow（只有 iPhone 版）的理念在于，在一个应用里调用其他应用的指定功能，设定好运转的流程，它就会自动完成某样任务，为你节省时间。比如搜集壁纸，在 Workflow 里点击一次，下载、保存、分享这几个动作就能一次完成，还可以重复，甚至完全自动化。

最关键也是脑洞最大的“自定义流程”，该怎么设置？

想要创建一个 Workflow 也就是工作流，最基本的界面有两个：Actions 和 Workflow。前者相当于各个模块，是流程运转的素材；后者就是放置模块的流水线。Workflow 调用了很多 iOS 原生应用的功能，比如添加日历、联系人、拨号、发送邮件……也有第三方软件比如印象笔记。

把这些素材拖拽到 Workflow 的界面，按照流程摆放就会形成一个工作流。点击最上方的开始按钮就可以运行。到这里你应该会发现，Workflow 是一种模块化编程，但你不需要写下任何代码。

举一个简单的例子，你希望把最近拍摄的若干张照片变成 GIF，并且自动上传到 Facebook 主页。Workflow 可以实现。把拍摄照片、制作 GIF、快速预览、分享这几个的模块拖拽到编辑面板，顺序排放，一个工作流就完成了。

使用 Workflow 本身是一个循序渐进、不断发明的过程，一开始会有些学习成本，娴熟之后甚至能实现许多意想不到、也更复杂的“流水线”。

如果脑洞还没打开，Workflow 也提供诸多已经设置编写好的工作流，你可以在 Gallery 里挑选使用，观察和模仿这些工作流也是一个学习的过程。多个工作流也能变成一个工作场景，比如“起床”，你可以设置刷牙时间自动提醒、自动播放音乐、咖啡因摄取记录、然后让 Siri 为你朗读今日要闻……

所有的工作流都会以标签的方式摆放在应用主屏，它们实际上等于一个个应用了。它们也适合放在 Widget 里快速启动。

iOS 系统本身开放的 API 并不如 Android 多，但 Workflow 神奇的地方就在于用有限的权限，组合成无数种可能。被苹果收购之后，Workflow 将获得更多的权限，让它的玩法更多。

Workflow 原本售价 18 元，现在已经变成免费应用。
