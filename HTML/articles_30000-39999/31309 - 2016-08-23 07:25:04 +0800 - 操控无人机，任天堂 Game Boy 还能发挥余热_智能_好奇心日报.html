操控无人机，任天堂 Game Boy 还能发挥余热_智能_好奇心日报

徐弢2016-08-23 07:25:04

曾经是世界上卖得最好的游戏机，现在还可以用来作无人机的遥控器。

任天堂 Game Boy 无疑是相当成功的游戏机，靠着 4 个机型在 14 年时间卖出了 1.2 亿台，到 2004 年是当时销量最高的游戏机。

当然，现在看来这款游戏机性能太差，但仍然有不少爱好者继续改装这款游戏机来做一些新的东西，例如配合键盘制作旋律等。现在，还有人拿来做无人机的遥控器了。

这个网络账号名叫 Gautier Hattenberger 的用户在家里找到了以前玩的游戏机 Game Boy，很快计划改装用来操控无人机。他在网上论坛 Paparazzi UAV Blog 贴出了改装方法，并放出了一段操控 Parrot 的无人机 ARDrone2 的视频。

在演示视频中，他可以通过 Game Boy 游戏机上的方向键和 A、B 键来调整无人机左右前进和上下。

在改装上，这位游戏玩家主要做的工作，是将 Game Boy 上的按键指令能被支持 Wi-Fi 操控的无人机识别。为此，他用游戏机连接线、Arduino 开发板和 FTDI 芯片，将按键指令传输到另一端的 USB 接口上，再连接到配置了 Paparazzi 软件的电脑，后者作为无人机的基站，跟配置了 GPS 的无人机进行连接。

当然，这中间还是需要编写部分代码，将其中涉及的 Arduino 开发板等利用起来，Gautier Hattenberger 已经将代码部分放在了代码托管服务 Github 上。

改装 Game Boy 游戏机来播放音乐、增强硬件性能已经有很多教程了，用来做无人机的遥控设备，倒还是比较新鲜的事情。
