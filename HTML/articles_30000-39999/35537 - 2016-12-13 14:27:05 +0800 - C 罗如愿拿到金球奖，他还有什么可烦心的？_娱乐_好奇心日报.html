C 罗如愿拿到金球奖，他还有什么可烦心的？_娱乐_好奇心日报

宣海伦2016-12-13 14:27:05

即便状态下滑，他仍然是最有商业价值的足球运动员

拿下欧洲杯和欧冠冠军，C 罗的第四座金球奖奖杯可谓众望所归。即将过去的 2016 年对 C 罗来说无疑是完美的一年，即使你会发现，人们在评价他的杰出成就时，更多地会使用“无可比拟的领袖精神”，而不是其令人惊叹的进球数据。

C 罗在欧洲杯决赛开始不多久便意外负伤，此后带伤在场边鼓舞队友的那一幕被拿出来反复描说；C 罗自己在致谢辞中也反复强调“团队”，直言“如果皇马或者葡萄牙没能夺得冠军，那我也很难获得金球奖，对我而言团队是重要的。”

如果单单只看数据，梅西在今年俱乐部和国家队的 61 次出场中完成了 58 球、32 次助攻，而 C 罗则是 55 次出场，51 个进球、17 次助攻。

现年 31 岁的 C 罗依然锐不可当，但他作为球员的角色正在悄悄发生变化，他依然是队长无可争议的领袖、精神支柱，但翻看数据，你会发现 C 罗也开始为队友“做饼”了。C 罗在欧洲杯期间完成 3 球 3 助攻，不仅完成锋线任务，也是队内的助攻王，为队友创造了多次绝佳机会。

除了身体状态在未来几年可能出现下滑，C 罗前不久还卷入税案纠纷，德国媒体爆料 C 罗通过在维京群岛设立的一家公司转移了高达 1.5 亿欧元的收入以逃避税款。

几天前，其经纪公司 Gestifute 代表 C 罗发布官方声明，承认 C 罗海外资产超过 2 亿欧元，但没有逃税举动。根据 Gestifute 方面提供的材料显示，C 罗在西班牙的资产为 23506231 欧元，他在海外的资产超过了 2 亿欧元，具体数字是 203793952 欧元。

考虑到西班牙对外籍人士高昂的税收标准，顶级运动员的纳税问题始终是西甲逃不开的话题。根据西班牙纳税相关法律，政府需外籍就业人群大额合法纳税，而且对于外籍人员纳税标准随年限均有递增。因此，大多西甲高薪外援选择 4-5 年后离开或被挂牌出售，其主要原因就是俱乐部无力支付球员年薪以外的 43% 税款。

西甲联赛是目前最强的足球联赛，也拥有世界上最多的顶级球员。西甲球员连续八年包揽金球奖自不必说，毕竟我们正处于梅西和 C 罗的时代，今年金球奖排名前 9 的球员里，西甲包揽了其中的 7 个名额，分别是 C 罗、梅西、格列兹曼、苏亚雷斯、内马尔、贝尔和佩佩，也体现出西甲联赛的强势。
