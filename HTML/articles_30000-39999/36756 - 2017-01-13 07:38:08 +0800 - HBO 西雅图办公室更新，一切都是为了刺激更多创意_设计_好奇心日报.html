HBO 西雅图办公室更新，一切都是为了刺激更多创意_设计_好奇心日报

胡晓琪2017-01-13 07:38:08

要同时应对多种不同的工作模式，一个简单、灵活的办公空间就成了必须

由于业务发展的不错，最近， HBO Code Labs 租用了西雅图 Hill7 的最顶部的三层楼，打算扩建为它的西雅图办公室。

这个新的办公室由加州的设计工作室 Rapt Studio 来操刀室内设计。Rapt Studio 是个颇具创造力的工作室，我们曾经写过它设计的新型照相馆 Blink。除此之外，Rapt Studio 还为 Eventbrite、Adobe、Ancestry 这样的科技公司做过设计。

在这个总面积达 120000 平方英尺（约 11148 平方米）的空间内，HBO Code Labs 的程序员、设计师、工程师都在这里办公，因而它必须足够灵活，可以支持不同的工作模式。

尽管说 HBO 的总部位于纽约，不过近些年来，西雅图这个办公室对于 HBO 来说，显得越来越重要了。

HBO 在 2012 年的时候，于西雅图创办了 HBO Code Labs。随后，这个被 HBO 当作是数字产品灵感诞生地的实验室，就推出了通过有线电视订阅的付费视频点播服务 HBO GO，以及每月 14.99 美元可以畅看的流媒体视频服务 HBO Now。由于给公司带来了不错的收益——截止至 2015 年，HBO Now 的付费订阅用户已超过 80 万，HBO Code Lab 被寄予了 HBO 这个老牌电视网在顺应数字化时代潮流中的厚望。

HBO Code Labs 在积极地扩建自己的创意团队，他们还想要创造出更多像是 HBO Now 这样成功的产品。因而，原本的小办公室显得不太够用了，HBO Code Labs 才需要一个更大，也更好的地方，来激发出他们的创造力。

“由于他们的工作性质，空间的灵活性和适应性是很重要的”，Rapt Studio 说道。

为了满足不同工作模式的人对于办公空间的差异化需求，Rapt Studio 并没有在这里做过多的装饰，而是通过各种家具的灵活配置，让偌大的办公空间最终呈现出了一个简单、自然的状态：它配备了简单的架空地板系统，为方便日后设备配置的改变和扩充；它不设固定的桌椅，让员工可以自行调整；它还提供了舒适的沙发供员工休息。

办公室的接待层位于第九楼，在接待台的背后，除了 HBO 的背景板之外，还有大面积的绿植，让整块区域显得自然而友好。

与此同时，这一楼还有一个配备多种电子设备的区域，用来展示 HBO 的新产品。在接待台的附近，还有一个舒适的放映室，到访 HBO 的人，可以去那儿调看 HBO 的剧集。

HBO 的室内部分通过一个中央楼梯相连。在第二层的楼梯附近，有一个可以喝咖啡的休息区域，一组焦糖色的沙发，让这里变得温暖起来。

在工作区域内，还可以任意选择喜欢的办公方式：一些较高的桌子是为了让员工们可以随意地站着谈话，比起正襟危坐，这样的休闲的方式更容易激发他们的灵感；在沿着走廊一侧的墙壁上，配备了白板，员工们可以在这里进行“头脑风暴”。
