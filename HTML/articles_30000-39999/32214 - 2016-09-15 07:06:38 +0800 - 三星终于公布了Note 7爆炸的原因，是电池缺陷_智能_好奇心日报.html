三星终于公布了Note 7爆炸的原因，是电池缺陷_智能_好奇心日报

张智伟2016-09-15 07:06:38

锂离子电池比锂聚合物电池制造成本更低。

三星上个月发布的旗舰手机 Note 7，充电时电池过热燃烧，甚至爆炸的新闻仍在发酵。

针对这一事故，三星电子手机业务负责人高东镇（Koh Dong-jin）表示，主要是由于 Note 7 采用了存在缺陷的电池所致。

从供应链的消息看，Note 7 电池由三星集团旗下的 SamsungSDI 供应 70%，总部位于深圳的新能源科技（ATL）供应 30%。

目前，全球发生爆炸事故的 35 部 Note 7 全部采用了来自 SamsungSDI 的锂电池。由于电池电芯内部的一个设计缺陷，当用户给 Note 7 充电时会遇到严重发热甚至爆炸的情况。

值得庆幸的是，国行版本的 Note 7 采用了 ATL 提供的电芯，不存在安全问题。不过三星中国被国家质检总局约谈后，还是对已经发货的 1858 台国行 Note 7 做了召回换机处理。

根据知名拆机网站 iFixit 提供的消息，在中国以外市场发售的 Note 7 使用的都是锂离子电池（lithium-ion battery），而不是更为稳妥安全的锂聚合物电池（lithium-polymer battery）。

这两者的区别并不复杂，锂离子电池中的电解质材料主要为液态，而锂聚合物电池中的电解质主要是干状或胶状的固态物质。在能量密度上，两种锂电池同档位的产品差别也不大，但是锂离子电池的生产成本要更低一些。

为了增强续航能力，三星 Note 7 比上一代的 Note 5 电池容量增加了 500mAh，还通过 USB Type-C 接口支持了更快速的充电。这就给液态的锂离子电池带来了安全风险，更何况三星自己生产的电池还有设计缺陷。

继美国联邦航空管理局（FAA）和多家航空公司将 Note 7 列入禁飞名单之后，国内的海航和首航也在本周二针对三星 Note 7 增加了安全风险提示。

虽然比 iPhone 7 发布还要早几个星期，但 Note 7 还没来得及跟新 iPhone 正面较量就接连爆出安全事故，多少有点可惜。

三星目前正在全球范围内召回已经售出的 250 万台 Note 7，同时向 ATL 大幅增加了电池订单。未来所有的 Note 7 都将采用这家供应商提供的电芯。
