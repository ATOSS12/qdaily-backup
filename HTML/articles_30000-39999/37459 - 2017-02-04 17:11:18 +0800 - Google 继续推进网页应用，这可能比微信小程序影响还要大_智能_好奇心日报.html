Google 继续推进网页应用，这可能比微信小程序影响还要大_智能_好奇心日报

张智伟2017-02-04 17:11:18

手机上的网页应用不再只是添加个桌面快捷图标那么简单了。

在接下来的几周里，习惯在 Android 手机上使用 Beta 测试版 Chrome 浏览器的用户，将会获得一项新功能：通过 Chrome 添加到主屏幕的图标将不再只是个快捷方式，而是使用体验接近本地 App 的应用。

开源的 Android 则会在系统层提供 Progressive Web App 所必需的基础文件存储、更新和通知推送等功能支持。也就是说，即便在断网状态下，通过 Chrome 添加的这些应用也可以保证基本功能正常使用。

Progressive Web App 在 2015 年正式推出，按照 Google 的计划，将来的正式版 Chrome 浏览器也将支持这个功能。

去年 5 月的 Google I/O 大会上，负责 Android 的团队还推出了类似的增强版网页应用，Instant Apps。不用下载安装某些电商应用，也能在几步之内完成购买。

对于开发者来说，Progressive Web App 其实是一种新的产品形态，相当于网页应用和本地软件的混合体。

用户不需要下载体积硕大的软件安装包，也不用等待安装过程，点开就能用，用完就走。在后台运行时，对系统计算资源的占用很多情况下也没有传统的本地软件高。

从理念上看，Google 主导的 Progressive Web App 和微信小程序非常相似，只不过要比小程序早了近两年，潜在影响力也更大。

现在的智能手机全部通过应用商店下载和更新软件，Progressive Web App 和微信小程序的产品形态需要确立新的衡量标准，无法再以下载量评判推广效果和产品的商业价值。

随之而来的挑战是双向的，营销及推广人员需要快速行动和决策，而用户则要改变原来本地软件的使用习惯。两件事情都不容易。
