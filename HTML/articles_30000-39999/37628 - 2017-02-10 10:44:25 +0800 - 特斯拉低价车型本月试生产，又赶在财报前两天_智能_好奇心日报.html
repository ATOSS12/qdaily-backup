特斯拉低价车型本月试生产，又赶在财报前两天_智能_好奇心日报

龚方毅2017-02-10 10:44:25

试生产和量产还有相当距离。

特斯拉本周三表示，它们会在这个月关闭其位于加州的装配厂一周时间，为批量生产公司低价车型 Model 3 做准备。

它们将利用这次计划内的短暂停产间隙，为 Model 3 增加油漆车间和其它常规维护设备。这些准备工作完成后特斯拉将于本月 20 号试产 Model 3。

特斯拉发言人称此次停产不会对公司第一季度产能和车辆交付产生重大影响，它们已经增加生产日补足产能。但他拒绝对生产计划置评。  

Model 3 是特斯拉于去年 3 月 31 日发布的一款车。它的外形设计和特斯拉 Model S 类似，但尺寸小一些。

除了车身尺寸比 Model S 小，Model 3 价格也更便宜。入门版 Model 3 售价 3.5 万美元，大约是 Model S 的一半。后者在美国的入门版售价为 7.13 万美元。

目前尚不清楚 Model 3 中国区售价。但如果参考 Model S 约 68.9 万元的国内售价，Model 3 可能卖 33.8 万元。

Model 3 将使用铝合钢材制造，比 Model S 更加轻量化，这使得 Model 3 续航里程可以达到 215 英里，还略高于 Model S 入门款的 210 英里续航里程。

高级驾驶辅助功能方面，自动驾驶功能、选装的四驱电机也在 Model 3 配置列表里，它还将具有与 Model X 类似的全景挡风玻璃与玻璃车顶。

截止到去年五月，特斯拉大约已经获得 37 万台 Model 3 订单。

按计划，Model 3 将于今年 7 月批量生产。试产和量产之间只隔五个月，意味着这次试产基本就是小批量生产。

在汽车工业中，量产不是一下子就按照最大产能来启动的，而是先有小量的试生产来拉动车厂的生产线以及供应链配合。这样整个供应链和生产线上相关的所有环节才会逐渐地建立起来生产能力。

对特斯拉以及他们车上的所有零部件的供应商来说，生产线以及工人都需要在试产过程中不断练习，才能够最终熟练地按照作业指导书进行生产操作。

在公布财报的时间节点调整造车计划，成了特斯拉近年的习惯。大约从 2015 年第三季度开始，特斯拉都在每个财报截止前的那个季度交付数量激增，然后下个月陡跌。

只是特斯拉现在连试生产计划都跟着财报节奏了。

分析师和汽车行业供应商也表示特斯拉的时间表有些激进，他们指出特斯拉之前发生过产能不足的情况。去年第二季度特斯拉一共交付 14370 辆轿车，低于原计划的 17000 辆。

巴克莱分析师布莱恩·约翰逊认为特斯拉无法在今年交付 Model 3，摩根士丹利分析师亚当·乔纳斯相对乐观地认为 Model 3 交付时间会延至今年晚些时候。

路透社的信源还称，截止到上周特斯拉仍在对 Model 3 的设计进行调整。

如果是对外形设计做调整，那在一般新车开发流程里不多见。车辆外观造型设计一般会在比较早的时候定下来不再改动。因为设计改变之后，车厂需要对车辆重新进行包括耐久试验在内的测试， 否则不能将没有试验的新设计直接用于量产。

即便特斯拉的开发流程比一般车厂更紧凑、可以赶时间完成重新测试工作，但设计问题越晚发现、修改耗费的资源越多。

而且这么做会导致供应商重新对模具进行调整，最终可能会关系到特斯拉能否如期量产、交付车辆。
