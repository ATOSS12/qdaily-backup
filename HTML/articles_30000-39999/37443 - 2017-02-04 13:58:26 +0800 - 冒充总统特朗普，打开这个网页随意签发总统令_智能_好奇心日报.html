冒充总统特朗普，打开这个网页随意签发总统令_智能_好奇心日报

张智伟2017-02-04 13:58:26

所见即所得，尤其适合群聊引战时充当挡箭牌。

  • 特朗普总统令生成器是一款以假乱真的 web 网页应用，在手机或电脑上直接打开页面就能填文字进去。
  • 它支持多种语言，能调整字号大小，编辑完需要手动截图再分享。
  • 拟真度很高，特朗普标志性的“地震波签名”也被保留下来。

新一任美国总统唐纳德·特朗普就职不到一个月，签署的十几份总统令已经引来了不少麻烦。

最近，国外有个程序员业余写了个免费的 web 应用，以特朗普亲手展示总统令的照片为背景，允许用户修改纸面上的字体内容。

这款 web 应用虽然只有一个功能——生成总统令，但它能实现的效果却是多种多样的。

你可以用文字肆意嘲讽网络上的某些不良风气，也可以模仿特朗普推特账号的语言风格，做一个地道高级黑；还能根据平日里的聊天内容发起一个引战的话题。

相比于 Android 上的类似应用，这款生成器的优点是支持多种语言文字，所见即所得，还能调整字号大小。

一个值得注意的细节是，总统令的落款仍然保留了特朗普的“地震波签名”，比同类 Android 应用拟真度更高，讽刺效果也会更强烈。

关于特朗普非同寻常的签名，笔迹专家 Marc J. Seifer 研究发现，特朗普写完自己的全名大约需要 6-7 秒，而前任总统巴拉克·奥巴马只要 2-3 秒就能完成。

在字形上，这位专家还发现特朗普的笔迹里有不少向上、向下的箭头，“显示出他非常具有攻击性，喜欢非黑即白地看待事物，而且不喜欢妥协”。

这就为网友们拿总统令嘲讽特朗普提供了理由。

在社交网站推特上，已经有不少人用这款总统令生成器大开玩笑了。

如果你也有槽要吐，不妨也在手机或电脑上试试。这款 web 应用托管在 Github 上，可能需要科学上网。

稍微有点麻烦的是，编辑好之后要手动截图，然后才能分享出来。
