物联网语言是什么不重要，这个标志设计很萌倒是真的_设计_好奇心日报

徐佳辰2017-01-16 06:58:41

为一门语言，设计一个形象。

2017 年的 CES 大会上，专注于物联网无线通讯技术的 Zigbee 联盟发布了全新的物联网语言系统 Dotdot。这是一种新的无线通讯语言，可以让智能家电设设备在不同的网络条件下共同协同工作。换言之，在不久的将来，你不用再担心只能在 wifi 环境下工作的冰箱无法和只能在蓝牙环境下工作的电灯合作这件事情了。它们有了一门专门的语言，可以进行独立沟通。

Zigbee 联盟这次和品牌设计公司 Wolff Olins 合作，为 Dotdot 打造了一整套完整的视觉系统，与技术一起在 CES 上亮相。

这个标志造型很简单。你甚至可以用手机把它打出来，它长这样：

:||

完整版的更像是一张挺呆萌的脸。

Zigbee 对这个新的标志介绍说：“此次我们和 Wolff Olins 旧金山办公室合作，希望能设计一套同时面对商家和消费者的视觉系统。而最后的结果甚至在技术层面上有了新的突破——这也许是第一个能够很简单地用 CSS 语言绘制出的 Logo。”

这也是这个标识的特别之处：它可以不以一个传统的 AI 或 PDF 文件形式存在，而是一段挂在 Github 上的字串符，可以用 CSS 语言把 logo 渲染出来。同时，它还可以以今年 Google 和 Monotype 共同开发的谷歌通用字体项目中的两个开源字符来输出。

单独的标志设计不足以为 Dotdot 带来完整的形象，Wolff Olins 还制作了一个小短片，形象地介绍了这门新语言的功能。
