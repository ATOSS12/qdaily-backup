墨尔本新开的啤酒花园餐厅，融合了工业风和复古风_设计_好奇心日报

胡晓琪2016-09-21 16:32:37

开放性和私密性，都在这个 Garden State Hotel 里。

澳大利亚建筑设计事务所 techne，把墨尔本的一座老纺织厂，改造成了一个占地 2000 平方米，可以容纳 840 名顾客的啤酒花园餐厅。

新建成的这家餐厅虽然名为 Garden State Hotel，它并不提供任何宿泊服务。The Garden State 这个名字来自维多利亚时期，餐厅想借由这个名字，来表达一种怀旧情怀。

Garden State Hotel 保留了原来纺织厂的锯齿状屋顶，把这一块区域改造成了错落有致的开放型中央花园。同时，从玻璃屋顶上安装了一些高低不同的气球灯，它们落在树桠之间，为整个空间也带来了一丝灵动。

tachne 事务所的负责人认为，“保留原有的屋顶形状，通过丰富内部空间的层次，并更新了空间设计的风格，能明显感受到这种新旧风格间的碰撞”。

一楼也是整个餐厅的主要部分，它有 670 平方米，其中，一半空间在室外，有着锯齿状的玻璃屋顶，另外一半则位于建筑内部。因此，室外的部分可以用作烧烤和聚会，室内是主要的用餐空间。

临街的那部分开放空间里，有着透明的玻璃屋顶，一个鸡尾酒吧台，一个舞池，以及一块数字屏幕，这里最多可以容纳 250 位客人，整体装修成了一种简约的工业风格。

从中央花园可以到达的地下一层，有一个好听的名字，叫做 Rose Garden。它以玫瑰色为主色调，被装修成了略显奢华的复古风。

在吧台和室内的餐桌上，复古造型的花瓶中插着玫瑰花束作为装饰。吧台旁边的墙壁上还有一副由当地手工艺人绘制的玫瑰涂鸦。

在一楼的花园上面的这一层，叫做 Balcony Dinning Room，它是一处较为隐秘的空间，也是整个 Garden State Hotel 唯一一个仅接受预订的地方。

这里只能容纳 20 个人就餐，提供 5 至 8 种季节性菜单，有预订的客人可以通过直达电梯到达这里。除了主要的就餐区域和厨房之外，它还配备一个私人的洗手间，小酒吧，以及一个允许吸烟、可以看到楼下的啤酒花园的小露台。

Garden State Hotel 的最顶层，则是一个名叫 The Observatory 的空间。它也有一个小酒吧和直达电梯，这里可以容纳 120 个客人，也是整个餐厅视野最好的一层。在这里，你可以通过窗子，观赏到墨尔本市区的繁华街景。
