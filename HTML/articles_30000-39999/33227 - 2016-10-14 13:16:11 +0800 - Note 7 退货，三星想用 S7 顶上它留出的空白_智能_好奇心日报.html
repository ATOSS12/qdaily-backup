Note 7 退货，三星想用 S7 顶上它留出的空白_智能_好奇心日报

张智伟2016-10-14 13:16:11

这家公司想用 S7 和 S7 edge 努力挽留高端用户。

今天凌晨，三星公布了全球范围内的 Note 7 退换货细则。

其中，国行版本的 Note 7 机主如果选择退货，除了全额退款之外，用户还能拿到 200 元补贴；如果选择换购今年 2 月发布的高端机型 Galaxy S7 或 S7 edge，在退货款之外，还可获得 600 元补贴，而换成其它的三星手机就只能拿到 300 块。

针对非国行 Note 7，三星也支持全款退回，而且不要求归还包装盒、耳机配件以及相关赠品。

在 Note 7 爆燃事故最为严重的美国，三星推出了类似的召回政策。换购 S7 或 S7 edge 能得到 100 美元，而退货就只能拿到 25 美元，并且需要发票配件齐全。

很明显，三星在 Note 7 电池风波之后，想通过补贴挽留那些并不缺钱的高端手机用户。

然而，这个努力很可能不会有太大成效。

来自品牌营销服务机构 Branding Brand 对 1000 名三星手机用户的一次调查显示，有 40% 的受访者明确表示将不会再购买三星手机。

其中有 30% 会去买 iPhone，62% 的人会选择其它品牌的 Android 手机，还有 8% 会选择一周多前发布的 Google Pixel。

这对三星而言，并不是个好消息。

更糟糕的是，经销商和运营商渠道在 Note 7 停售召回之后，除了 iPhone 7 并没有太多选择。

《好奇心日报》查询了美国四大运营商，以及欧洲最重要五个国家英国、法国、德国、意大利和西班牙排名前几位的运营商网站，发现大多数运营商首页在主推 iPhone 7 之外，又重新把三星 S7 edge 放在了显眼位置展示。

美国的 Verizon 和英国的 EE，因为跟 Google 有合作，重点推荐的是月初发布的 Pixel 手机。此外，在德国、意大利和比利时的运营商网站上，索尼 9 月份的新旗舰 Xperia XZ 和华为的 P9 & P9 Plus 也在重要的广告位上。

苹果、Google、索尼、华为等厂商都可能从三星召回 Note 7 事件中获益。

面对此种情况，三星唯一能做的就是尽量止损。

在广告投入方面，三星迅速撤掉了此前为 Note 7 准备的大量广告资源，包括网络视频广告、墙体印刷广告等。

目前，三星在 YouTube 和优酷网的官方账号已经将所有 Note 7 视频广告清除。10 月 11 日晚间，王府井地铁站内的 Note 7 墙体广告也在一夜之间被全部撤下。

百度搜索 Note 7，现在还能看到京东、天猫、中国联通在百度投放的关键词广告，但在这些网站上已经买不到这款手机了。

供应链方面，三星已经彻底停止 Note 7 的元部件供应和生产，转而为明年的 Galaxy S8 做准备。按照三星的原定计划，S8 会在明年 3 月份发布。

而根据韩国经济媒体 ETODAY 的报道，三星内部决定将发布时间提前至 2 月。

除此之外，三星还有可能为 S7 edge 准备了一款珊瑚蓝的版本。其颜色工艺和珊瑚蓝版 Note 7 非常相近，但在 Note 7 上市初期，珊瑚蓝版很难买到。

Note 7 召回，让三星错过了中国的双十一，也错过了西方国家的圣诞购物季。缺席这两个手机厂商大力促销的买买买节日，三星下半年两个季度的财报必然会受到影响。

可惜的是，除了 S7 和 S7 edge，三星也确实拿不出像样的高端手机产品了。靠着珊瑚蓝配色，三星可能要度过历史上最难熬的几个月。
