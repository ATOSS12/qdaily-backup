今年这么多电影，谁家的海报最想让你一探究竟？_娱乐_好奇心日报

陈仪萱2016-12-16 07:38:26

初印象不好，还怎么撩人

了解一部电影，我们往往是从预告片或是一张海报开始的。

而海报，作为一种静态艺术，直接体现出了电影的内容、气质、审美与风格。年底，各家媒体都评选出了心目中的最佳海报，如著名电影媒体 indieWIRE 、烂番茄网站、The Playlist 网站、Screen Crush 网站、电影评论网站 MUBI 等。（戳链接可见完整榜单）

从类型上来看，这些上榜的影片涵盖了多种题材，包括喜剧、传记、惊悚还有纪录片；有颁奖季的大热影片《月光男孩》、《爱乐之城》，有商业大片《星际迷航 3》、《银河护卫队 2》，也有小众影片《绿色房间》、《你瞧，网络世界的幻想》等。

有些上榜的影片可能对你来说有些陌生，不过它们的海报，通过不同艺术风格的展现，或许能让你产生一探究竟的念头。

《好奇心日报》整理了这些海报的一些特征，你可以看看，为什么这些海报被评为了年度最佳。

电影剧情的巧妙嵌入

《月光男孩》

导演: 巴里·詹金斯

主演: 崔凡特·罗兹 / 加奈儿·梦奈 / 娜奥米·哈里斯 / 安德烈·霍兰 / 马赫沙拉·阿里

影片讲述的是一名男子在孩童、少年、青年时期，他长大、相爱、发现自己的性取向，经受外界非议和内心挣扎后，找到了自我的故事。这部影片成了今年颁奖季的大热影片，而它的海报也被各家媒体纳入了最佳海报榜单中。

Screen Crush 对这张海报的评价是“《月光男孩》至少有一部分票房应该归功于这张迷人的海报”，因为它从一定程度上代表了这部电影。海报主体是由饰演主角孩童、少年、青年的三位演员的脸拼凑而成的，以不同的颜色（绿色、紫色、蓝色）来表现男主在不同阶段的心境与经历，MUBI 评价说，“这张海报仿佛在凝视你的灵魂。”

《小姐》

导演: 朴赞郁

主演: 金敏喜 / 金泰梨 / 河正宇 / 赵震雄

影片根据英国作家莎拉·沃特斯的小说《指匠情挑》改编，讲述的是 1930 年日本殖民统治朝鲜时期，继承巨额财产的贵族小姐（金敏喜饰演）、觊觎其财产的骗子伯爵（河正宇饰演）、受雇于伯爵接近小姐的侍女（金泰梨饰演）以及小姐的监护人（赵震雄饰演）之间围绕金钱和爱情发生的故事。

相对于突出人物的主海报，国家版的手绘海报表达更为含蓄，颇具东洋气息，是来自 Empire Design 的作品。整个金箔色背景下，在翠柏丛中穿插着片名字体和五个经典场景，整个精致而神秘的画面展现了电影本身的华丽与阴谋。耐人寻味的是，韩国版的手绘海报跟美版有些不同，不仅替代了姨母自杀和激情戏两个限制级场景，树上也多了樱花。

《克里斯汀》

导演: Antonio Campos

主演: 丽贝卡·豪尔 / 迈克尔·C·豪尔 / 蒂莫西·西蒙斯

影片讲述的是上世纪 70 年代电视记者 Christine Chubbuck 的生平。在 1974 年，在美国佛罗里达的  WXLT 电视台发生了一个恐怖的悲剧，原本是一个谈论自杀的节目，而节目主持人Christine Chubbuck，一位忧郁症患者，在直播时选择自杀结束了自己年轻的生命。

海报是来自波兰海报设计师 Brandon Schaefer 的设计，你可能听说过 The Poster Boys（海报男孩），这个组合是由两位海报设计师 Brandon Schaefer 和 Sam Smith 组成，他们经常会在网上不定时分享好看的海报设计，同时在 iTunes上创立同名播客，聊聊海报设计有关的故事。这张海报的设计被 Screen Crush 列为年度海报第一，认为这张海报不仅很好地说明了影片内容，也可以当作是美国流行文化的一种概述与表达。你可以戳这个链接看看这位设计师的其他作品。

强烈的感官冲击

《一个国家的诞生》

导演: 内特·派克

主演: 内特·派克 / 阿米·汉莫 / 佩内洛普·安·米勒

《一个国家的诞生》讲述的是 1831 年，黑人牧师奈特在美国弗吉尼亚洲，率领黑奴解放非裔美国人的自由运动，最终成为引发内战爆发的导火线。

海报整体构图形成了美国国旗的样子，红色象征着血腥暴力，展现了黑人起义的壮观场面，海报好看，不过电影评分挺低。

《屏住呼吸》

导演: 费德·阿尔瓦雷兹

主演: 史蒂芬·朗 / 简·拉维 / 迪兰·明奈特 / 丹尼尔·祖瓦图 / 艾玛·博科维奇

《屏住呼吸》被评为第 22 届美国评论家选择电影奖最佳恐怖片，影片讲的是三位小青年潜入一位盲眼的退伍老兵家中行窃，却发现屋主不但身手了得，还在地下室囚禁了一名女子。在一名同伴被击杀后，剩下两人不得不奋力逃亡。从海报上看，整个背景都是黑色的，而一个女人捂着嘴，意味着看到了惊人而可怕的事实，营造出了前有狼后有虎的氛围。

《绿色房间》

导演: 杰瑞米·索尔尼尔

主演: 安东·叶利钦 / 伊莫琴·普茨 / 阿莉雅·肖卡特

影片背景设置在美国西北部一个荒僻人烟稀少的地区，几个玩朋克摇滚的年轻人在无意中撞见一起恐怖暴力事件后，被关进与世隔离的房间里，和打算杀他们灭口的新纳粹分子展开搏斗。

小众 Cult 片的气质、不安的混乱以及暴力惊险的感官刺激，从这张海报都可以瞥见。而令这些影评媒体更为欣赏的一点是，这款海报还致敬了著名朋克乐队 The Clash《London Calling》的专辑封面，不过把 Paul Simonon 手里挥舞的吉他变成了一把长刀。

《黑暗宇宙》

导演: 安德烈·祖拉斯基

主演: 萨宾·阿泽玛 / 让-弗朗索瓦·巴尔梅 / Jonathan Genet

影片根据波兰作家维托尔德·格姆布洛维茨的同名小说改编，讲述两个从祖国逃难的年轻人乔纳森和约翰，住进了波兰小镇的旅馆，意外卷入了镇上诡异离奇的事件之中。

海报由 Maida's design 设计，不仅整个画面黑白红对比醒目，还有一些引人注目的小细节——比如手工刻字的标题以及吊死在树上的小鸟，巧妙地提示了影片的情节。

复古风，同样不失创意与新潮

《银河护卫队 2》

导演: 詹姆斯·古恩

主演: 克里斯·普拉特 / 佐伊·索尔达娜 / 戴夫·巴蒂斯塔 / 范·迪塞尔 / 布莱德利·库珀

这张海报看起来似乎更像是一张复古质感的时尚大片或是音乐专辑封面。在这张黑白画面上，大家熟悉的星爵、卡魔拉、德拉克斯、火箭浣熊、勇度和涅布拉均全员回归现身——别忘了依偎在星爵脚边的树人格鲁特。

《德·帕尔玛》

导演: 诺亚·鲍姆巴赫 / 杰克·帕特洛

主演: 布莱恩·德·帕尔玛

一部关于“美国的希区科克” ——德·帕尔玛导演的纪录片。海报是来自 80 年代经典电影海报艺术家之一 Steven Chorney 的作品。在海报上，导演德·帕尔玛透过百叶窗瞥视他生命中最棒的一次拍摄，或许这个场景比起电影本身更加让人印象深刻。

《初恋这首情歌》

导演: 约翰·卡尼

主演: 福迪亚·瓦尔什-匹罗 / 露西·宝通 / 杰克·莱诺

小清新文艺片，讲的是在 80 年代的爱尔兰，14 岁男孩康纳在学校倍受同学欺负，只好从音乐中寻找慰籍，后来爱上了比他高一级的女孩拉菲娜，同时欣喜地发现她也是个音乐迷。

海报介于手绘与胶片之间，既展现出了 80 年代的复古感，也契合了初恋的画风。

海报，同样可以是静态艺术品

《爱乐之城》

导演: 达米安·沙泽勒

主演: 艾玛·斯通 / 瑞恩·高斯林 / 芬·维特洛克 / J·K·西蒙斯 / 水野索诺娅

已经拿奖拿到手软的《爱乐之城》，海报也获得了青睐，The Playlist 将其排在第二位，选了两版海报。一版是整体设计以黑胶唱片封面为模板的复古海报，置身于蓝色色调中的瑞恩·高斯林与艾玛·斯通显得十分忧郁；另一版整体设计是用代表忧郁的蓝色色条构成了如同钢琴琴键的画面。男女主角相拥亲吻的剪影和城市的夜景融进了一片蓝色之中。

《霓虹恶魔》

导演: 尼古拉斯·温丁·雷弗恩

主演: 艾丽·范宁 / 阿比·丽 / 贝拉·希思科特

影片讲的是漂亮姑娘杰茜只身独闯洛杉矶模特界的故事。比起影片内容，海报漂亮更多，用迷幻风格的紫色来显示表现光怪陆离的时尚模特界，女主角在灯光照射下显得性感魅惑。

《Buster's Mal Heart》

导演: Sarah Adina Smith

主演: 拉米·马雷克 / DJ·考尔斯 / 凯特·林恩·希尔

讲述了一个山里人在一家空旅馆里度过冬天，他一直在做着一个梦，梦里的男人在海上失踪了，结果他发现自己就是那个梦里的男人。

海报不禁让人联想到《布达佩斯大饭店》，整体设计十分干净利落，不过细看你会发现一些诡谲之处，比如数字颠倒的时钟、倒映着的脸。

你瞧，这些海报能让你立刻会心一笑

《被挠》

导演: 大卫·法里尔 / 迪伦·里夫

主演: 大卫·法里尔 / 迪伦·里夫 / 大卫·斯塔尔 / 哈尔·卡普 / 戴维·达马托

影片是部同性纪录片，讲的是一个同志记者想报名参加挠痒大赛，却被告知该运动仅限直男。倔性上来的他展开调查，结果发现挠痒帝国背后隐藏着惊人的真相，作为一部纪录片，《被挠》拍出了一种悬疑追凶的感觉。

海报本身简单，切题，引人注目，大概这种感觉，有点像去年《蚁人》那张无需多言的海报。

《雪崩行动》

导演: 马特·约翰逊

主演: 马特·约翰逊 / Owen Williams / Josh Boles / Jared Raab / Andrew Appelle 

《雪崩行动》讲述的是四位 CIA 探员以拍摄纪录片的名义，欲揭穿 NASA 阿波罗计划（有不少人认为阿姆斯特朗根本没有登上月球），却意外将自己一步步推向凶险的边际。

在海报中， 一个人正在用镊子夹住地球，暗示着电影的主题——关于阿波罗登月计划是否是人为伪造的争论。

《你瞧，网络世界的幻想》

导演: 沃纳·赫尔佐格

主演: 凯文·米特尼克 / Lucianne Walkowicz

作为“新德国电影”四大天王，赫尔佐格在这部纪录片里用没什么感情的德国腔英语探讨了我们自以为很熟悉的网络世界。他采访了包括特斯拉和 SpaceX 总裁埃隆·马斯克、发明 IP 协议的美国计算机科学家罗伯特·卡恩、被 FBI 通缉的世界级头号黑客凯文·米特尼克等网络科技界著名人物。

海报提出了一个赫尔佐格想向观众提出的一个问题：“如果有一天网络消失了，脆弱的人类文明会不会跟着毁灭？”

欲言又止，也很撩人

《龙虾》

导演: 欧格斯·兰斯莫斯

主演: 科林·法瑞尔 / 蕾切尔·薇兹 / 约翰·C·赖利

已经是不少媒体 2015 年度的最佳海报之一，今年依旧出现在了 indieWIRE 、烂番茄的榜单上，看来影评网站对《龙虾》这张海报也是爱得深沉。

作为一部讲述“单身是会变成狗的”的电影，海报对单身族的杀伤力也不弱。海报来自 Vasilis Marmatakis 的设计（他也设计过《龙虾》导演前两部片子《Dogtooth》、《Alps》的海报）。这张海报实际上把影片想要表达的“孤独”、“被需要”可视化了，或许主演科林·法瑞尔怀抱着的就是一只无形的大型龙虾。（男主角在电影中称没找到伴侣后想变成一只龙虾）

《擦枪走火》

导演: 本·维特利

主演: 布丽·拉尔森 / 基里安·墨菲 / 阿米·汉莫 / 迈克尔·史麦利 / 沙尔托·科普雷

故事发生在 1978 年的波士顿，剧情围绕一座军火交易的仓库展开，交易出错，局势也变得越来越糟。

The Playlist 将这张海报排在了第一位，海报中从四面八方伸出的手握枪支的手，含蓄而又想让人一探究竟：开枪的人是谁？枪子儿对着又是谁？他们之间有什么矛盾？一张海报便描述了错综复杂的剧情。

《疾速特攻》的海报跟《擦枪走火》有点相像，被 Screen Crush 纳入了最佳海报榜单，不过相比之下，《擦枪走火》的艺术表达感更强一些。

简洁，也是一种强有力的表达形式

《第一夫人》

导演: 帕布罗·拉雷恩

主演: 娜塔莉·波特曼 / 彼得·萨斯加德 / 格蕾塔·葛韦格

影片讲述的是美国前总统约翰·肯尼迪在 1963 年 11 月 22 日遇刺身亡的前四天时间里，发生在前第一夫人杰奎琳·肯尼迪身上的故事。整张海报的全红色调十分扎眼，背景与娜塔莉·波特曼的一身红衣融为一体，更加凸显了人物的面部表情和手部动作，也暗示了人物的情绪涌动。

《某种女人》

导演: 凯莉·雷查德

主演: 克里斯汀·斯图尔特 / 米歇尔·威廉姆斯 / 劳拉·邓恩 / 杰瑞德·哈里斯

影片改编自著名畅销女作家梅丽·麦洛（Maile Meloy）短篇小说，讲述的是四个女人在荒凉严峻的美国西北小镇努力打拼的故事。海报是素描画风，其实暗含主题——我们看到了是被加工过的别人的形象，而事实可能并非如此。

《科洛弗道 10 号》

导演: 丹·特拉亨伯格

主演: 约翰·古德曼 / 玛丽·伊丽莎白·温斯特德 / 小约翰·加拉赫

该片根据约翰·坎贝尔和马特·司徒肯所写的小说《地下室》改编，讲述了一个女孩因车祸失去知觉，醒来后发觉自己被囚禁在一个地下室里的故事。

这部影片从消息首次公开到正式上映中间只隔了两个月，此前没有任何宣传，神秘感十足。海报同样很神秘而简洁，从影片名中的“L”延伸出了白色线条，成了挖穿地底的通道，海报还埋了一个包袱：“Monster come in many forms”——所以会是从地底吗？

《摩天大楼》

导演: 本·维特利

主演: 汤姆·希德勒斯顿 / 杰瑞米·艾恩斯 / 西耶娜·米勒

影片讲述的是在一所与世隔绝的高层公寓中住户之间的阶级斗争与暴力事件。海报中的汽车倒映的小小人影和扭曲的摩天大楼暗示了电影情节，设计得十分巧妙。

不过它的另外一款海报你可能更熟悉一些，是抖森的美好肉体。

嗯。就不放出来了。
