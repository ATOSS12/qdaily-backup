耐克跑步应用大幅度更新，却引来各种差评_智能_好奇心日报

方园2016-08-30 07:21:31

“再大的更新，也不能以数据丢失为代价。”

自从 2010 年推出以来，Nike+ Running 一直是款颇受好评的跑步应用。不过最近，耐克对其进行了大幅度升级，并更名为 Nike+ Run Club 之后，却引起了用户的疯狂吐槽。

截至昨晚，最新的 5.0.2 版本的 Nike+ Run Club 在 App Store 共 153 条评价，平均得分仅仅为一星半。大多数人都给出了有理有据的负面反馈。我们阅读了其中大部分的评论，总结下来有以下几个主要的槽点。

  • 新版 UI 设计，包括主色调变动过大
  • 语音播报由英文改成了中文
  • 初始单位默认为英制，且一度无法更改
  • 原先的微博、QQ 账号登录入口消失
  • 经常性闪退
  • 跑步里程数据记录错误
  • 晨跑/夜跑的标签混乱
  • 进行中的训练计划消失
  • 过去跑步中的心情记录、跑鞋数据丢失（跑鞋数据已恢复）
  • 奖章系统和挑战模式下线

这其中，一部分是应用大幅度更新后不可避免的用户适应问题，比如应用图标由原先标志性的红色改为了绿色，让人感觉“以为谁帮我下载了一个新软件”。

不过这多少也可以预见。这次的界面改版很明显是在和几天前推出的 Nike Training Club（NTC）保持一致。有意思的是，App Store 中新版本的 NTC 下却几乎是清一色的好评，为数不多提及 UI 设计的评论也都是“界面简洁漂亮”、“看着舒服”。

真正让用户感到生气的，应该是新版应用糟糕的使用体验，和对于过往数据的处理。从默认的英制单位，到微博链接的取消，都可以看出，新版本几乎完全没有考虑到中国市场的本土化。

更加“全球化”的差评来自数据方面。无论是在 Facebook 还是在微博，正在进行的训练计划的丢失都是引起最大争议的一项。从耐克官方在社交媒体上的反应上来看，这似乎是在预计中的。他们认真地在 Facebook 页面下回复说：

“我们收到了你的反馈。这里是包含有你之前训练计划数据的一份 PDF 文件。”

跟帖网友对此的回应是：“哇，一份 PDF 文件！感觉就像是回到了 1998 年！感谢耐克让我重返 16 岁！”

从耐克官方的宣传来看，这次他们的升级的本意之一是想增加 NRC App 的社交功能，以此来为跑者提供更多动力。比如新增的类似 Instagram 的 # 话题标签（Hashtag）功能被重点突出，以取代原有的奖章和挑战模式。但这样的改动，就如同微信步数排行榜一夜之间变成了全国人民自拍晒图大会一样，很难令人接受。

讽刺的是，NRC App 上效仿社交媒体的“动态”部分，反倒是成了用户分享糟糕体验的一个平台。有网友在上面提出了一个问题：

“请问各位，有什么其他推荐的慢跑 app？”
