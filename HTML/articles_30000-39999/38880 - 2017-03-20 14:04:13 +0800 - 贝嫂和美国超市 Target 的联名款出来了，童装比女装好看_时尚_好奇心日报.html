贝嫂和美国超市 Target 的联名款出来了，童装比女装好看_时尚_好奇心日报

张田小2017-03-20 14:04:13

贝嫂可能计划做童装。

贝嫂去年10 月就喊着要和标榜设计的美国超市 Target 合作联名款，现在， Victoria Beckham X Target 就要开卖了。上周，贝嫂在自己的 Instagram 上晒了晒这个系列的真面目。

Victoria Beckham X Target 一共有 200 个款式，包括常规女装，还有大码女装和童装，其中一些童装还能和女装搭配成母女装销售。这个系列女装的售价在 20—70 美元之间，童装在6 美元—30 美元之间。

为了宣传这个联名款，贝嫂不仅给这个系列拍了硬照，还拍摄了第一支视频广告。大概也是考虑到 Target 商超的定位，为了唤起广泛的共鸣，广告还特地选用了贝嫂“辣妹时期”的作品 Turn it Up 做背景音乐。

另外，贝嫂在谈到这个系列的童装设计时也搬出了女儿 Harper 小七的故事。“我知道小七喜欢穿什么，小朋友都喜欢舒适。我把所有做母亲学到的经验都放到了这个联名设计中。比如，她很喜欢《爱丽丝仙境梦游》的故事。”她说。

自 2008 年推出了自己的同名时装品牌 Victoria Beckham 后，贝嫂也开始玩起了联名系列。2014 年，她和 Nails Inc.公司合作，推出了 “Victoria Beckham x Nails Inc.”限量版指甲油系列，紧接着是她和雅诗兰黛联手推的两个联名款。

事实上，她的时装生意经营得也不错。

2014 年，贝嫂在伦敦开了直营店后效果显著。当年， Victoria Beckham 有 3400 万英镑的营业额， 2013 年的收入也不差，有 3000 万英镑。虽然从账面上看，Victoria Beckham 不太漂亮，贝嫂还要靠着小贝的公司 Beckham Brand Holdings Limited 的姊妹公司 DB ventures Limited 的注资来保证不亏损。但事实上贝嫂是花钱太多，把赚到的钱都用来招人、扩店和装修了。其中花销最大的一笔是在伦敦中心的梅菲尔开的三层旗舰店，每年店铺租金是 65 万英镑，加上装修等费用一共 280 万英镑。

这也是为什么今年 2 月，贝嫂和她的合伙人 Simon Fuller 也在为品牌寻找注资，顺便搭建国际渠道。

而贝嫂这次和 Target 的合作则是部分考虑到了 Target 的覆盖面。

“我可以通过 Target 接触到那么多女性消费者。这倒不完全是说我们的这个联名系列价格比较亲民，而是因为 Target 在美国到处都有店。这些都是我过去没有办法接触到的消费者。”贝嫂说，“我也总在说想通过我的设计让女人变得更强大，让她们展现出自己最好的一面，那么就不应该只面对那些买得起我同名品牌的消费者来做设计。”

说白了，贝嫂就是想借着 Target 的店铺去接触一些新的消费者。而 Target 的首席商品官 Mark Tritton 则表示，贝嫂会愿意和他们合作也是想要了解一下大众市场，特别是童装市场。

“一直以来我都想尝试一下童装。”贝嫂说，“这说不定就是我下一个要开发的产品线。谁知道呢。”

4 月 9 日， Victoria Beckham X Target 将会在 Target 的店内售卖，Victoria Beckham 的电商上也会售卖这个系列的精选款。在一些 Target 的店里，“亲子装”将放在一起售卖。“对于 Victoria Beckham 这个品牌来说，也会经历很多零售方面的革新。” Mark Tritton 说。
