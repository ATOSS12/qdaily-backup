咨询机构预估，今年 Windows 系统设备出货量会被苹果超过_智能_好奇心日报

朱若淼2017-01-10 07:01:15

这还是没有计算 Android 设备数据的情况下，未来三年它的增长都不太乐观

美国咨询公司 Gartner 发布的最新报告预测，到今年年底采用苹果操作系统的设备销量将首次超过采用微软操作系统的设备。

未来三年内，苹果的优势还将继续扩大。

尽管今年受到 iPhone 销量下滑的影响，iOS 和 mac OS 去年的出货量为 2.48 亿台，较上一年下滑了 10%。不过 Gartner 预计今年这个数字会有所反弹，iOS 和 mac OS 合并份额将增长 8%，预计总销量会达到 2.68 亿台。

未来两年，苹果设备的销量也将持续上涨，预计在 2018 年和 2019 年分别达到 2.76 亿台和 2.85 亿台。

Windows系统的设备销量情况正好相反。

2016 年 Windows 系统设备的销量约 2.6 亿台，领先于苹果。但近几年它与苹果设备销量之间的差距已经逐渐缩小了。

Gartner 预计今年 Windows 设备的出货量将减少，约为 2.52 亿台。其中大部分是 PC 设备，同比下滑 3%。而此后两年，它的增长也已经达到天花板。Gartner 预计 Windows 设备的出货量在 2018 年和 2019 年分别为 2.53 亿台和 2.57 亿台。

“全球设备市场的销售增长都停滞了。”Gartner 的分析师 Ranjit Atwal 在上周的声明中称。智能手机的增长主要来自于亚太地区。他还提到称，“PC 市场的销售刚刚触底。”

去年第三季度，Garner 发布的报告中 PC 在全球的销量持续 8 个季度下滑，连苹果也不例外。

但受此影响最大的是微软。尤其在 Windows 手机没有赢得市场的情况下 Windows 设备的销量几乎完全依赖于 PC 的出货量。未来这部分市场的增长已经放慢了。

尽管 Gartner 仍乐观的预估未来 PC 的销量还有提升的空间，因为这个市场将迎来一批更新换代潮。但也没有乐观到哪里去。

去年 10 月它曾预计到今年 PC 销量将恢复增长，但最新的报告中这个时间被推迟到了明年。而它预测今年 Windows 设备的出货量数字比 2015 年还要低。

此外在 Gartner 这份报告中并没有统计另一大操作系统 Android 设备的出货量数据。

而 Android 系统的智能手机早就超过了苹果和微软占据了整个智能手机的大部分市场份额。

而在今年全球卖出的 23 亿台设备中， 运行 Windows 操作系统的设备市场份额仅占 11.2%。

在全球智能手机、电脑销售增长都在放缓的环境下，微软遇到的问题要更严重些。
