还记得你在电脑上玩的那个纸牌接龙吗？现在手机上也有了_游戏_好奇心日报

徐弢2016-11-24 16:44:42

微软自己开发的手游，想从自家的 Windows 平台扩展到主流平台上去。

你是否还记得 Windows 自带的纸牌休闲游戏？微软现在把它从自家 Windows 搬到了两大移动平台上，iOS 和 Android。国内用户可以下载。

这款名为纸牌合集（Microsoft Solitaire Collection）的游戏，包含 5 种不同的纸牌游戏。玩法跟你从小熟悉的没有差别，只要左右移动纸牌进行整理同花色，或者是两张牌相加得 13，都是相对简单的。

游戏支持微软账号登陆，还有内购可以增加道具。目前下载的用户会获得 1 个月的付费服务，可以使用道具。

Windows 系统上的纸牌游戏已经有 25 年历史了，靠着 Windows 系统的普及，微软的休闲游戏很快普及。根据官方数据，过去 4 年里，有 1.19 亿用户在支持 Windows 8 和 10 系统的 PC 和移动设备上玩过这款游戏。

但支持 Android 和 iPhone 手机还是第一次。不过，移植看上去做得不是很好，中国区 App Store 上架的版本都还没有支持中文。

按照微软官方的说法，这款游戏支持在云端备份游戏进度，他们希望用户可以在各种不同的设备上都可以玩到这款游戏。这种拓展方式，看上去就跟微软对于 Office 套件的策略比较相似，不再把自家服务绑在 Windows 平台，而是进入竞争对手 Android 和 iOS 平台，获取尽可能多的用户。

这跟微软在 2014 年的做法不一样。为了在移动端获得更多用户，微软在 2014 年大量收购生产力工具，成为了 iOS 和 Android 平台上重量级的开发者。

现在看来，微软会把更多的 Windows 上的服务推向更主流的平台上，包括游戏。

在这背后，一部分原因可能跟微软调整手机系统有关。过去 1 年来，微软努力号召 iOS、Android 应用开发者帮忙开发 Windows，还做了几个移植工具，开发者只需要修改部分代码，并切换微软的服务，就可以做出 Windows 手机应用。

但这部分的努力相继宣告失败，开发者缺少为用户量少的 Windows 手机平台开发应用的动力，这会导致他们进入到“用户少－开发慢”的恶性循环。因而，即便开发 Windows 的手机应用后，更新和维护仍会慢不少。
