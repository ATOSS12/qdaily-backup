装上它，你可以在地铁里学写代码

  • Encode 是个学习写代码的小工具；
  • 它把学代码这项看起来很复杂的工作拆解到一个个具体的代码作业里让你完成；
  • 如果你想用它学习更多的编程技巧需要在应用内购买课程。

Encode （iOS）的教学方式跟 codecademy 类似，都是通过让你根据编辑器里的提示学习写代码。

它们唯一的区别就在于 codecademy 需要在电脑上操作，Encode 只需要下载一个 App 就好了。

它提供了多种编程语言课程。你可以用它了解到 JavaScript 及 HTML 等编程语言。每种编程语言都被分拆成一个个编程任务。

比如，如果你想知道一个网页是怎么写的，可以进入 web 栏学习 HTML 和 CSS 的编程语言。

它在每个部分都将代码的涵义放到那个部分的任务中，你需要根据页面顶部的提示使用正确的标签写出相应的代码。写完代码点击 Run 后，它就会根据你写的代码运行，如果代码写错了，它是不会让你进入下一个部分的。

由于是手机端，不能像一些代码编辑器一样在你输入代码标签开头后自动弹出余下的部分，但它贴心地在编辑框顶部设计了一些标签符号，让你用手机写起代码来不会太费事。

尽管比起 PC 端的编程学习工具，它是简陋了些。但如果你恰好想了解编程，Encode 倒是一个可以让你随时练习的小工具。
