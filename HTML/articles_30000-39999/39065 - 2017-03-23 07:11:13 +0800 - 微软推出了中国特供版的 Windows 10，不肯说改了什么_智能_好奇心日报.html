微软推出了中国特供版的 Windows 10，不肯说改了什么_智能_好奇心日报

谢若含2017-03-23 07:11:13

被除名后，微软希望重新进入到中国政府的采购清单当中。

根据《华尔街日报》的报道，目前微软已经完成了为中国政府订制的特别版 Windows 10。以此让微软的操作系统产品重新进入到中国政府的采购清单当中。

这个订制系统由中国电子科技集团与微软合作开发而成，他们共同成立了一家合资企业，前者占股 51%，是中央直接管理的十大军工集团之一，主要从事重大通信与电子设备、软件和关键元器件等的研制生产。

微软大中华区董事长兼 CEO 贺乐赋（Ralph Haupter）在接受财新网记者的采访时称，与普通 Windows 10 相比，专用版或会去除一些针对消费者的应用和功能，以增强产品的易管理性和安全性，满足中国政府的特殊需要。

但他也表示，这款产品的核心始终是一款 Windows 产品，以确保这个系统可以使用生态系统内的现有应用。

但微软并未透露他们做出了哪些具体的调整。

微软开发特供版的 Windows 系统这一事件需要追溯到 2013 年，斯诺登的棱镜门事件曝光的相关机密文件显示，微软曾向美国联邦调查局（FBI）和国家安全局（NSA）提供 Skype 视频通话、Outlook 和 Hotmail 的用户数据，帮助政府监测用户信息。

2014 年，为了更新系统，微软开始停止为 Windows XP 系统提供技术支持。当时大部分中国的政府机构和国企采用的是 XP 系统，停止更新容易遭受到安全威胁。

当年，中国中央国家机关采购中心在一则《关于进行信息类协议供货强制节能产品补充招标的通知》中规定：所有计算机类产品不允许安装 Windows 8 操作系统。

但对于微软来说，与中国政府的订单合作很重要。从 2003 年开始，微软不断加大在华投资，政府采购规模一度高达 1500 亿元。它的中国业务非常依赖政府订单。

中国政府专用版 Windows 10 是微软首次为政府开发的定制版操作系统。除了微软之外，其它的美国科技公司，比如高通、英特尔同样和中国政府有着类似的合作。

不过，微软这一次为中国政府开发的“特供版” Windows 10 系统能不能被买账是另一回事。目前，中国政府和大型国企内部开始推行 Linux 系统，同时也在开发独立的操作系统。

是不是所有电脑都在用 Linux，是不是没人装盗版 Windows 和 Office 就不知道了。但微软也不会去起诉吧。
