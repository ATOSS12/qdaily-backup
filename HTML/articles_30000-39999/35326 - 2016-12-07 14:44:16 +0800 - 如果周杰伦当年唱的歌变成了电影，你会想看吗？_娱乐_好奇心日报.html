如果周杰伦当年唱的歌变成了电影，你会想看吗？_娱乐_好奇心日报

陈仪萱2016-12-07 14:44:16

这次他和方文山带来了 10 个音乐 IP 计划。

12 月 5 日，周杰伦与方文山分别以“最强创意顾问”和“首席文化官”的身份加盟爱尚传媒，并公布了《蒲公英的约定》、《三年二班》、《亲爱的那不是爱情》、《倒带》、《说好的幸福呢》、《爱在西元前》、《听见下雨的声音》、《布拉格广场》、《热带雨林》、《茉莉雨》十部音乐 IP 计划。

这是周杰伦与方文山自出道合作以来，首次加盟内地传媒上市公司。像之前高晓松、何炅、李湘和韩红加盟影视公司或出任高管都给背后的公司积攒了大量关注。这次周杰伦与方文山加盟的爱尚传媒同样是看重两人的明星效应与 IP 资源。

爱尚传媒的主要领域是影视电商，2016 年 1 月正式在新三板挂牌上市，主要在明星+网红电商、影视制作、影游联动等方面展开布局，目前已经与 MRJ 品牌（周杰伦原创品牌）以及明星电商星品库 J 概念进行关联控股整合，跟优酷有两个项目在合作，一是“边看边买”的影视电商项目《明星战衣》；另一个是方文山执导的网剧《蒲公英的约定》，这两个项目都属于合一集团年度直播战略“五大头部内容”。

而两人与爱尚传媒的合作，主要集中在音乐 IP 影视化这一方面。方文山将主导由他填词制作的经典歌曲音乐 IP 的开发与运作，并透露将音乐 IP 进行开发的动因，“ MV 是一种形式，把歌曲影像化会是什么样子呢？我觉得这是一种新的尝试。”而周杰伦则称，自己这几年已经写了 5 个电影剧本，未来三年将与爱尚传媒合作将其拍成电影，但表示自己不会亲自执导。

这十个音乐 IP 的具体开发计划尚未确定，方文山透露将会执导其中的两部左右（《说好的幸福呢》、《蒲公英的约定》），其中《蒲公英的约定》目前确定将被拍摄成网剧，这将是一部互动性质的网剧，会有 AB 两种剧情走向。而方文山也表示不排除购买其他的音乐 IP ，“林夕的作品很好，有传唱度，陈奕迅的《十年》也很好听，我们也不排除买这些音乐 IP 。”

其实，音乐 IP 的影视化并不鲜见，但表现参差不齐。去年高晓松的《同桌的你》获得 4.5 亿票房，何炅《栀子花开》也取得 3.8 亿票房，而像《睡在我上铺的兄弟》电影则票房口碑双双失落。此前合一影业也曾瞄上音乐 IP 改编电影，在国家广电总局的立项公示里有《你的背包》（陈奕迅）、《她来听我的演唱会》（张学友）和《小情歌》（苏打绿）。

音乐 IP 与小说、漫画 IP 的改编不同，它所能依托的内容基础十分单薄，主要是依靠歌曲传唱度、明星效应以及群体的共同记忆来吸引受众，而且并非所有的歌曲都是适合被 IP 影视化，要获得市场认可依旧需要一个站得住的故事。
