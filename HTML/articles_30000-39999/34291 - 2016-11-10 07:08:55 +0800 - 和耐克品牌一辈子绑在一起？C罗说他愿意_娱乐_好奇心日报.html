和耐克品牌一辈子绑在一起？C罗说他愿意_娱乐_好奇心日报

宣海伦2016-11-10 07:08:55

如果耐克要在足球领域签下一份终身合约，C罗确实是不二人选

几天前皇马与 C 罗签下一份五年长约，耐克就宣布与葡萄牙人续约。C 罗在视频中将这份合约形容为 “for a life”，外界猜测耐克为其提供的是一份终身合约，类似于 NBA 巨星勒布朗-詹姆斯得到 6 亿美元终身制合同。

耐克方面没有披露合同的具体期限和额度。但根据《阿斯报》报道，这份合同总价值将达到 10 亿美元。

这份合约预计将为 C 罗带来 2400 万欧元的年收入，而在 2016 年福布斯体育明星年收入的榜单上，C 罗已经凭借 8800 万美元登顶成为收入最高运动员。

尽管“梅西和 C 罗谁更强”的争论可能永远不会有答案，但从商业价值上看，C 罗绝对是耐克在足球领域的不二人选。

看一眼耐克官方给 C 罗的评价：“C 罗收获了又一个令人难以置信的成功年份，为俱乐部和国家队拿下了双冠，他无疑是这项运动的标志性人物。C 罗超越了足球本身，是社交网络上最受欢迎的运动员，也是世界上最著名的运动员。”

诚如上文所说，C 罗确实是当时最强之一，也很受欢迎。先来看一眼他的不完全版荣誉簿?

除此以外，无论是在球场还是场外，C 罗的存在感很强。他懂得如何运用自己的名气，不管是自己的生日，还是和女友之间的悄悄话，都可以作为自己服饰品牌的营销话题。

C 罗在社交网络上表现非常活跃（包括给耐克打广告），他是第一个 Instagram 粉丝量突破 2 亿的运动员，一条消息的商业价值在 23 万欧元左右；同时他还非常注意经营自己的个人形象，在场外对球迷也是极尽耐心，对签名合影来者不拒，与球迷互动频繁，比如射门时把球踢到了球门后的小球迷身上，赛后亲自去看望了小球迷，还送他全队签名球衣……这些暖心故事在社交媒体上流传甚广。

31 岁的 C 罗无论在哪方面都臻于成熟，这显然是品牌在签下一份终身长约时所看重的。

C 罗本身的成功经历也和耐克的企业精神相契合。以勤奋努力著称的 C 罗，对自身经年累月的控制力达到了恐怖的地步，其体脂率达到了 7%，据说是职业球员中最低的。无怪乎足球圈有两大流传的错觉：C 罗没天赋，梅西不努力。实在是前者的后天努力、后者的天赋太过惊人的缘故。

C 罗本人对这次续约评价道：“这是我整个职业生涯中最好的一份合同。耐克很聪明，他们知道挑选最好的。我很高兴，因为我是他们挑中的最佳之一。”

而对于耐克来说，在其最强势的足球资源曼联被阿迪达斯抢走之后，耐克急于巩固在这一领域的顶级资源，比如说此前他们不惜花大价钱提前买断切尔西和阿迪达斯的合同。

在皇马宣布和头号巨星续约到 2021 年后，这意味着葡萄牙人在伯纳乌至少会得到一个体面的告别，耐克此时的续约也就显得更加顺理成章。
