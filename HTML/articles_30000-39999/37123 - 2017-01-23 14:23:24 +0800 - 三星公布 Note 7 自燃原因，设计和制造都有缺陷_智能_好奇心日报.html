三星公布 Note 7 自燃原因，设计和制造都有缺陷_智能_好奇心日报

张智伟2017-01-23 14:23:24

三星说这不能全怪电池供应商，不会提起诉讼。

1 月 23 日上午，三星如约公布了去年 Note 7 手机发生燃损事故的原因，是内部的聚合物锂电池在设计和制造上存在缺陷，导致了短路和发热燃烧。

去年 8 月发售以来，三星一共卖出了 306 万台 Note 7，截至今天已经召回了 96%。

在过去几个月里，三星对召回的 20 万台手机和 3 万块电池进行了多重对比测试和使用场景模拟还原。同时，还对已经发生燃损的 Note 7 进行了拆解分析。

Note 7 的电池供应商有两家，一个是三星集团旗下的三星 SDI，另一家是总部位于深圳的 ATL 新能源科技，供应比例据估计约为 7:3。

研究结果显示，采用两种锂电池的三星 Note 7 都发生了短路导致的燃损现象，只不过两种电池的故障发生方式不太一样。

三星将两家的电池分别称为 A 电池和 B 电池，没有明确说明 A、B 分别来自哪家供应商。

具体而言，A 电池的燃损来自右上角的电池负极板受到压迫，导致短路。

而第二批召回的 B 电池出现了铜箔融化现象，原因是超声波焊接产生的毛刺压迫电池，导致电池内部发生短路，部分电池甚至没有使用绝缘胶带。

在第三方机构的调查中，三星找到了美国的 UL、Exponent 和德国 TÜV Rheinland 三家公司各自做独立调查和分析。结果跟三星自己的研究基本相同，两种锂电池出现了不同机理的燃损情况。

此外，TÜV Rheinland 还针对电池的公路运送、仓储等环节做了抽样调查。从东莞运往越南河内工厂的 500 块电池中，抽取了 150 个样品进行分析，并没有发现运输过程对电池品质的影响。

由此基本可以确定，三星 Note 7 事故的根源在电池，但这并非全都是电池供应商的责任。

三星电子移动业务负责人高东真承认，发售前没能对 Note 7 做充分的安全性测试，导致了多起燃损事故的发生。三星不会对供应商提起诉讼。

为了确保安全性，三星推出了 8 项电池安全检查流程，其中包括耐久性测试、X 光检查、充放电测试、增强使用测试等。

电池直接关系着机身厚度和手机坚持的时间。而电池技术的进展远远赶不上芯片和应用的发展速度。为了更薄、密度更高，各家厂商都在设计时尽可能将电池的能力逼到极限。Note 7 是最严重的，但规模更小的电池问题也时有发生。

有了 Note 7 的教训，之后手机厂商对电池大概会小心一点吧。
