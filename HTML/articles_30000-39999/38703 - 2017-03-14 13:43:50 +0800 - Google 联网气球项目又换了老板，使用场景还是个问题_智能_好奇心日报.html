Google 联网气球项目又换了老板，使用场景还是个问题_智能_好奇心日报

张智伟2017-03-14 13:43:50

Project Loon 的商业化遥遥无期。

上任才六个月，Google X 实验室孵化的联网气球项目 Project Loon 负责人 Tom Moore 最近决定交出领导权。

接替他的是 Project Loon 的另一位技术骨干，Alastair Westgarth。加入 Google X 之前，他曾是高性能天线解决方案提供商 Quintel 的 CEO。

如此短暂而仓促的人事交接，在 Google X 这个关注前沿科技的实验性部门并不罕见。

Project Loon 的构想说起来并不复杂。它使用漂浮在平流层中的气球向地面提供高速无线网络接入服务。

由于这些气球的飞行高度距离地表通常有 18-25 千米，必须携带面积足够大的太阳能电池板、无线信号收发器、调制解调器等设备，才能向地面的手机用户提供稳定的网络接入。

Project Loon 的气球 2013 年在新西兰做第一次升空测试时，使用的是 ISM（工业、科学和医疗专用）频段，号称能提供跟移动 3G 相当的网络传输速率。

后来又切换到 4G-LTE 频段，理论上，这能为地面上特定范围内的用户提供更快速的网络连接。

今年 2 月，Project Loon 用上了时髦的机器学习技术，可以让有限的几只气球组成阵列，在平流层中做相对运动，从而让整个编组悬停在地表某个特定区域上方。

这样一来就有效减少了覆盖某一区域网络所需要的气球数量。

然而，Project Loon 的负责人还没有想清楚应该怎样把这一系列技术商业化。

从最初的设想来看，Project Loon 可以为自然灾害救援或灾区重建提供临时的网络服务，比搭建临时地面基站效率要高得多。

不过，Project Loon 显然还没发展到随取随用的地步。
