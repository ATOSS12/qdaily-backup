《保卫萝卜》推出儿童图书，把生意做到游戏之外_游戏_好奇心日报

唐舒畅2015-02-13 16:33:19

《保卫萝卜》给我们提供了一个思路：与其去高价买一款 IP，还不如好好培养自己的原创 IP。

刚刚过去的 2014 年被称为一个游戏的 IP （Intellectual Property，知识产权）元年。各种热门动漫、网络小说的 IP 纷纷被炒出天价。国内最大游戏开发商之一的畅游 ，以 1.5 亿买下《秦时明月》的版权，创造了行业之最。

这么烧钱的事情也不是人人都愿意干，飞鱼科技和旗下的塔防类休闲手游《保卫萝卜》最近就推出了儿童图书。在一个月里，这套图书已经销售超过 80 万册。从手机游戏走向实体书，在国内并不多见。在国内，手机游戏往往作为一个 IP 的下游存在着。比如最近被美国《商业周刊》报道的《小时代》手游，都是现有其他领域的成品在转向手游领域。

这个例子或许说明，如果你有个好的游戏，也许经营它比收购别的所谓的 IP 更重要。根据飞鱼科技的最新半年报显示，2012 年发行的《保卫萝卜》及其后续的《保卫萝卜 2》下载量接近两亿次，在 2014 年上半年依旧有着 100 万的每月付费用户，每个付费用户平均收益也有 5.2 元。

这次《保卫萝卜》首先选择了儿童图书市场，飞鱼科技的高级公关总监沈新华承认《植物大战僵尸》图书系列给了他们一定启发。《植物大战僵尸》系列 18 个月内发售超过 1000 万册的成绩，证明了将儿童文学和手机游戏结合起来的可能性。《保卫萝卜》和《植物大战僵尸》一样，两者的角色都偏卡通动漫风格，里面的角色都比较可爱、容易吸引女性和孩子。这次在推出图书的时候，飞鱼科技在故事书之外特地配套亲子互动、游戏教学等相关图书，也是瞄准了游戏的用户群体。《保卫萝卜》下一步动作是在考虑推出儿童动画。除此之外，飞鱼科技旗下的另外的一款 IP 《囧西游》改编的网络反讽剧也即将登场。

除了飞鱼科技，巨人和腾讯也正在对自己旗下的各类游戏 IP 进行强化，比如巨人的《征途口袋版》以及腾讯的“全民”和“天天”系列。不过至今，这些大型公司对游戏 IP 的重视，更多还是在游戏领域本身，比如《征途》从网络游戏改编为手机游戏。不过一旦要真的开始 IP 的布局，走向非游戏领域，也是迟早的事情。
