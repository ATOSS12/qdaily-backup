今日应用：总被抄袭的Opera浏览器如今有了进阶版_智能_好奇心日报

唐云路2015-01-30 16:57:25

如今的浏览器市场已经不是 20 年前那个硬件条件有限的时候了，新来的“熟人”还会有几分机会？

  • Vivaldi 是 Opera 联合创始人和前 CEO Jon von Tetzchner 在 2013 年底开始的新项目。
  • 目前，Vivaldi 推出了第一个技术预览版本，支持 Windows、Mac OS X 、Ubuntu、 Linux 等操作系统。这款浏览器基于 Chromium 内核开发，这让它看起来在很多地方都和 Chrome 很像。在扩展程序方面，Vivaldi 支持使用 Chrome 扩展程序，可以在  Chrome 应用商店里下载。
  • 点击网页左下角的按钮，可以调出左侧边栏，包括书签、邮件、联系人、下载列表和笔记几项。现在邮件和联系人功能还无法使用，从官网的说明来看，开发团队正在这方面下功夫。

在官网上，Vivaldi 浏览器是这样讲述它的故事的：

“1994 年，两名程序员开始开发一种网页浏览器，目的是打造一个在有限的硬件条件下快速运行的浏览器，Opera 浏览器由此而生。直到 2015 年，Opera 浏览器改变了自己的方向……于是，打造一个新的浏览器就成为了自然而然的事。”

2011 年，Jon von Tetzchner 离开 Opera，2013 年，Vivaldi 开始在 Jon von Tetzchner 资助下进行开发。虽然，这看起来不像是一个推出新浏览器的好时候。联合创始人之一 Tatsuki 在接受 The Next Web 时表示，时下的主流浏览器都在提供类似的服务，而不是在简单的基础上保持差异化的产品。

尽管目前功能还不够完善，但是从一些细节上还是能看出，这是为“愿意研究浏览器的用户”设计的浏览器。从浏览器预置的书签内容，一定程度上也能看出这一点。

快速打开常用界面（Speed Dial）的功能，如今已经在很多浏览器上都能见到，而它最初出现，还是在早期的 Opera 上。在 Chrome 浏览器把卡片式网页推广出去之后，连 IE 都改成了多个标签页集成在一个窗口里的呈现方式。在 Vivaldi 这里，除了标签栏会随着网页内容配色而变化配色方案，用户还可以将几个网页拖曳到一起，在需要同时打开众多网页的时候，就可以把网页按需要归类。

笔记功能也相当实用，可以在把网址添加到收藏夹的同时，随手截图并添加笔记，防止日后整理收藏夹的时候，忘记它当时为什么重要。

两天试用下来，Vivaldi 给我的感觉是精致和美，但是一时半会儿之间我还不会考虑更换常用的浏览器。毕竟，从技术预览到现实里真正好用的产品，Vivaldi 还需要一段时间。问题在于，Opera 创始之初的那份情怀，够不够支撑用户足够长的等待。
如果你曾经在功能机时代，在诺基亚上用过 Opera Mini，也许不久之后，会愿意试一试这个。
