Surface 3 发布了，Windows RT 却死了_智能_好奇心日报

王杰夫2015-04-02 01:52:27

不如换个名字，我们重头来过。——微软

如果你是软粉，且渴望一款逼格高、方便出街、价格合适的移动办公设备，那么微软刚刚发布的 Surface 3 平板电脑就应该可以满足的你的这些需求。作为一款迟了一年且有诈尸嫌疑的科技产品，微软还在指望它什么。

虽然 Surface 3 从坟堆里爬了出来，可却没有带上它的老搭档 Windows RT，这一次它搭载了完整版的 Windows 8.1 操作系统，这也正式宣告 Windows RT 终于入土为安了。

可能有些人至今还弄不清普通 Windows 和 Windows RT 的区别。简单来说，前者和你平常用的 Windows 系统完全一样，而后者除了名字像 Windows 之外，其它跟 Windows 没啥关系。

自从微软自己做 Surface 系列电脑以来，高端产品 Surface Pro 搭载的一直是完整的 Windows，低端产品 Surface 则采用“官方赝品” Windows RT 系统。

结果可想而知，买了便宜货的不明真相用户发现他们的电脑连 exe 文件都运行不了，愤而退货；而懂点行的用户或是直接加钱上 Surface Pro，或是转而买 iPad，低端 Sruface 平板销量惨不忍睹。

这一次，终于舍弃了 Windows RT 的 Surface 3 低端平板竞争力看起来强上不少，Surface Pro 价格太高，iPad 太过娱乐，当前市场缺少一款廉价的生产力平板。去年 Surface Pro 3 在全球平板电脑市场低迷的前提下依然取得了强劲的增长，其主打移动办公的市场定位功不可没。

这一次 Surface 3 裸机定价 499 美元，与 iPad Air 2 定价相同。虽然，iPad Air 2 的性能更加强大，但受制于 iOS 系统的能力，这些好看的硬件指标除了让它运行的更加流畅外并不能有效提升工作效率。而 Surface 3 除了性能不足以支撑视频图像处理、大型游戏等工作，处理日常工作任务还是绰绰有余的。

不过，虽然这一次 Surface 3 的价格降到普通学生可以接受的范围内，但相关配件的价格可一点也不低。配套的蓝牙键盘 130 美元，手写笔 50 美元，一套下来总价接近 700 美元了。而且因为这一代 Surface 3 尺寸有变，原先的键盘不再适用，对于有意购买的用户来说，裸机应该才是最好的选择。

在苹果传闻中的生产力平板 iPad Pro 消息越来越少，主要精力放在 Apple Watch 上时，微软选择发布 Surface 3 来扩展它在移动办公市场的疆域，至少时机选的不错。
