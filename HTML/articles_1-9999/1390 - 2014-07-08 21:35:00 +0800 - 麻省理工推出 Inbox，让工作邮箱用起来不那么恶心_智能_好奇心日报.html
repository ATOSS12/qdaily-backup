麻省理工推出 Inbox，让工作邮箱用起来不那么恶心_智能_好奇心日报

马宁忆2014-07-08 21:35:00

近日，由 MIT 和 Dropbox 前校友、前雇员发起的创业公司 Inbox 正式成立，而它的目标是打造新一代邮件平台。这个新一代平台的原理和刚刚问世的 Gmail API 类似，就是帮助开发者解决那些运作已久的协议 （如：IMAP）所带来的麻烦。

习惯 Gmail、QQ 邮箱的方便设置以后，你还能忍受公司的企业邮箱么？至少我一直没能自己在 Sparrow 上配置我们的企业邮箱。填了一堆 IMAP、SMTP、SSL、TLS 的选项，最后邮件发送还是会出问题。

近日，由 MIT 和 Dropbox 前校友、前雇员发起的创业公司 Inbox 正式成立，而它的目标是打造新一代邮件平台。这个新一代平台的原理和刚刚问世的 Gmail API 类似，就是帮助开发者解决那些运作已久的协议 （如：IMAP）所带来的麻烦。换句话说，Inbox 其实是为开发者打包解决了一系列冗繁的流程，从而使他们能够更加方便和高效的进行开发。同时，Inbox 相对 Gmail API 更具优势，因为它不仅限于 Gmail 邮箱，雅虎、公司的微软 Exchange 等邮箱也同样适配。

“当我在 MIT 读书的时候，我就把邮件工具作为我的一篇论文课题，并且分析了为它开发附加功能的难度。”Inbox 的联合创始人、Dropbox 和 Nest 的前雇员 Michael Grinich 说道，“而这其中最麻烦的，就是那些陈旧又冗杂的协议。而这也就是 Inbox 想帮开发者解决的问题。”

虽然， Inbox 所建立的这个新一代邮箱平台并不能直接使广大用户收益。但随着开发程序的简化，更多的邮箱优化功能势必会向我们涌来。

不便利的邮件服务实在太糟心，就坐等优化的邮箱服务了！
