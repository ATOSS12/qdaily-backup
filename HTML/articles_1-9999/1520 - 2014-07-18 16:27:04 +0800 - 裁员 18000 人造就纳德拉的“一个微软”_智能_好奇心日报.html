裁员 18000 人造就纳德拉的“一个微软”_智能_好奇心日报

黄俊杰 2014-07-18 16:27:04

微软迎来 34 年历史上最大规模裁员，它的背后是权力天平的又一次转移。

7 月 17 日晚上 7 点，正式加入微软刚 3 个月的 2.5 万名原诺基亚员工收到一封来自新老板萨提亚·纳德拉（Satya Nadella）的邮件。邮件试图通过堆砌“协力与战略整合”（Synergies and Strategic Alignment）之类的拗口表述来缓和信息。但它要表达的意思再简单不过：微软要裁 18000 人，其中 12500 人来自诺基亚。

这是微软 34 年历史上最大规模的裁员，占微软公司雇员总数的 14%。裁员将在未来一年内完成，绝大部分被裁人员将在未来半年内收到通知。

在 18000 人失去工作的背后，是微软权力天平的又一次转移。

“我们在减少部分业务职位的同时，也在其它一些战略领域新增职位。”纳德拉在裁员邮件里这么写道。的确，裁员比例如此之高不是因为一年净利 200 亿美元的微软无力负担，而是因为这些人不再是微软的战略重心。

不到一周前，纳德拉在关于公司转型的公开信中写道：“我们的核心战略是，在移动为先、云为先的世界里，成为一家生产力和平台公司。我们仍将再造生产力，以便帮助地球上的每个人和每个组织获得更多的收获。”

不是生产力重心的不仅仅是刚并入微软的诺基亚手机业务，还有微软唯一一个能让消费者排队购买的产品——Xbox 游戏机。原本计划拍摄自制剧、利用 Xbox 入侵影视内容的 Xbox Entertainment Studio 也在昨晚宣布关闭。

就在裁员消息公布前一天，纳德拉在接受《财富》采访时说：“我们可以在核心业务以外做一些事。我希望公司能继续为 Xbox 感到骄傲、为它所需的保障，但与此同时，不将它与我们的核心业务混淆在一起。”

鲍尔默时代，你不可能从微软高层口中听到这样的句话。事实上，每当外界质疑微软失去创新能力或者消费业务乏力，Xbox 和 Kinect 都是鲍尔默用来反驳的例证。

但纳德拉已经不需要这些例子了。他在晋升 CEO 前负责微软云计算与企业业务，这是整个微软在过去 5 年里唯一高速增长的业务部门。

2013 年微软各业务全年营收（单位 10 亿美元，图片来自 Statista）

2013 年，微软通过云计算与企业业务获得 202.8 亿美元，超过了微软长期的支柱业务 Windows，仅次于 Office——那也是纳德拉曾经负责的业务。

最大的失败者——随诺基亚回归的埃洛普

在战略中心转移中，失势最多的无疑是掌管微软硬件设备业务的斯蒂芬·埃洛普（Stephen Elop）。这位刚刚带着诺基亚手机业务回到微软的原诺基亚 CEO 一下便失去了半数下属、部门内最受消费者认可的 Xbox 被划入“非核心”。

不过最糟糕的还是埃洛普一手准备的 Nokia X 也被砍掉。

在与裁员消息同步发给硬件设备业务员工的邮件中，埃洛普写道：“我们计划将未来的一些 Nokia X 手机设计和产品转向 Windows Phone，以提供更多低价 Lumia 产品。”

“转向”这个词的言外之意是那些新手机原本并不是为 Windows Phone 所准备。这显然不在埃洛普的计划之中。

不到 5 个月前，在巴塞罗那举办的移动世界大会上，上千媒体记者和参观者将诺基亚展馆围得水泄不通，为的就是一睹已经宣布要被微软收购的诺基亚发布 Nokia X。

Nokia X 是诺基亚的 Android 手机，将所有 Google 网络服务与应用替换成微软和诺基亚自有产品。也是埃洛普在 Windows Phone 产品长期亏损的情况下所做的尝试。

2014 年 6 月，微软发布第二代 Android 智能手机 Nokia X2（图片来自诺基亚）

对于埃洛普来说，推出 Android 手机就等于承认说此前数年专注于微软 Windows Phone 的策略是失败的。事实上，发布会的提问环节就有人提问说：“你之前说诺基亚的 Symbian 平台在燃烧，现在 Windows Phone 是不是也成了燃烧的平台？”我身边的与会者听到这话几乎都露出不坏好意的笑容。

但埃洛普还是决定推出 Nokia X。Windows Phone 业务一直没有起色，市场占有率维持在 4% 以下，而 Android 系统和诺基亚品牌的组合能够帮助微软服务进入低端市场。对于即将执掌微软硬件业务的埃洛普来说，卖出更多搭载微软服务的手机无疑能帮助他在公司内部获取更多话语权。

当时已经宣布要退休的鲍尔默看上去也支持这个决定——收购宣布后，鲍尔默到访北京期间曾接触过 AOL 的开发团队（Asha On Linux，诺基亚 Android 系统的代号）。

时任诺基亚中国区总经理白若霄（Erik Bertman）在发布会后的采访中对我说，Nokia X 将会是一系列产品。事实上也确实如此，收购完成 2 个月后，已经是微软一个部门的诺基亚还发布了 Nokia X2 系列智能手机，对上一代产品中一些为人诟病的设计进行了调整。

但微软的 Android 实验到此为止了。不仅 Nokia X 被砍掉，就连负责该项目的北京研发中心也将受到裁员的直接影响。这个在埃洛普执掌诺基亚期间大力投入的全球研发中心未来将只“承担辅助研发”。

对 Google 来说这无疑是个好消息。亚马逊、微软推出的 Android 手机使用了不同于 Google 的基础服务，特别是地图和应用商店，是在分裂 Android 生态圈。为此 Google 在一个月前的开发者大会上启动了 Android One 计划，自己将整套软件和硬件设计提供给低端机型生产商，以对抗 Nokia X 等产品。

诺基亚原本在拉美和东南亚的低端手机市场依然保有相当的占有率。纳德拉砍掉 Nokia X 彻底消除了 Android 在这些地区可能遭遇的分裂问题。

类似的大变动在 2009 年也发生过，那一年同时赶上金融危机缩减成本以及公司架构调整。在 2009 年的调整中，Windows 被作为单独的业务拆出来，权力的天平很快滑向它的负责人史蒂文·辛诺夫斯基（Steven Sinofsky）。

辛诺夫斯基一方面提出一个名为 One Strategy 的组织管理哲学，通过削减中层管理人员以精简管理层级；另一方面利用 Windows 业务的强势地位打压公司内的竞争对手。Windows Live Mesh 以及之后一款的看上去颇有希望的双屏平板 Courier 均在辛诺夫斯基的压力下终结。

随着 Windows 8 的失败，辛诺夫斯基在 2012 年离开微软，而 Windows 在微软的地位也不如往年。

Windows 和硬件产品一并失势

今年 3 月，纳德拉上任没几天，为 iPad 充分优化的触控版 Office 办公软件面市。Office 业务营销总经理朱莉亚·怀特（Julia White）亲自拿着一款 iPad Air 上台演示。Office for iPad 面市头一周便被下载 1200 万次。

另一方面，为 Windows 8 平板的触屏优化的 Office 至今没有现身——微软将 iPad 的优先级放在了 Windows 平板之前。

尽管纳德拉的公开信用词拗口而模糊，但他对业务优先级的判读相当明晰。

Office for iPad 扮演着 Office 365 云端办公软件的入口——用户可以免费用它浏览文档、但必须付费订阅 Office 365 服务方能解锁编辑功能。单从 Office 365 本身考虑，iPad 无疑比销量低迷的 Surface 或其它 Windows 平板更能吸引到用户。

类似的例子还有 Windows Azure 云计算服务 3 月更名为 Microsoft Azure。Azure 是微软应对亚马逊的云计算服务，和 Office 365 同为微软增长最快的服务。在上一财季的营收会议上，微软 COO Kevin Turner 说 Azure 当季收入比一年前增长了 150%。但 Windows Azure 的名字对产品的推广造成了阻碍，参与该业务在中国拓展的工作人员曾对我抱怨说自己每次都得对创业公司解释说它支持 Linux，不一定要用 Windows。

这是一件非常小的事，但在 Windows 主导一切的年代是不可想象的。

在纳德拉的领导下，微软的云计算与企业服务不再是为 Windows 操作系统提供差异化竞争力的辅助产品，而是成为了真正的主力。Office 365 看上去没有太像样的竞争对手、最近又一次调整价格冲击尚未在商用市场站稳脚跟的 Google 办公服务；Azure 在不断增加功能、投入价格战以追赶业内老大亚马逊 AWS。

看上去微软依然会赚很多钱，裁员也有助于缩减成本，问题在于，削减消费产品投入、越来越偏向生产力工具和企业服务的微软还是那个“让每家都有一台电脑”的技术先驱么？

不过相比每次演讲都把微软全部产品数一遍的鲍尔默，纳德拉能够坚定地做出选择这一点本身已经是进步。

去年夏天鲍尔默宣布名为“一个微软”的重大重组计划，试图提升内部协作效率、不再各自为战。不过他的重组计划里，消费、企业、桌面、移动、云计算、游戏……什么都不能丢、一切都要兼顾。谁也弄不清鲍尔默的“一个微软”到底是怎样的微软。

而纳德拉想要的微软却很清晰——一个以生产力工具为核心的公司。目前微软超过半数的营收来自商业客户的软件授权，这部分所占比例比一年前更高。此次裁员削弱设备业务将会进一步让业务向云计算和企业业务倾斜。

不论对错，现在纳德拉至少知道自己想要什么样的微软。
