从Tizen 到 Windows，哪个系统也救不了三星_智能_好奇心日报

唐云路2015-01-12 21:43:52

尽管过去的一年不太容易，三星倒是一直没放弃努力。

三星最近一次推出 Windows Phone 还是在 2014 年的春天推出了 ATIV SE。Windows 8.1 系统上市后，三星也没有紧跟着推出相应的设备。当然，这都是 2014 年的事情了。

2014 年，三星过得不太容易，搭载 Android 系统的 Galaxy 系列手机表现不佳，三星智能手机的出货量突然停止了增长，而曾经被寄予希望的的自有操作系统 Tizen 手机也一直推后发布时间。在刚刚结束的美国消费电子展上，三星表示会在电视而不是移动设备上主推 Tizen 平台，种种迹象表明三星已经基本放弃了自己的手机操作系统。

现在，三星又把目光转向了 Windows Phone，以此削减对 Google 和 Android 系统的依赖。据韩国 Korean Times 援引三星内部人士消息称，三星正在测试 Windows 8.1 系统在自家设备上的稳定性。这意味着，我们可能在接下来的时间里，见到一大批搭载 Windows 系统的三星手机。

这位内部人员表示，三星希望通过生产 Windows Phone，占领低端和中端市场。

可是这看起来并不是一个真正能将三星从被颠覆的困境中解救出来的方案。三星缺乏的是一个完全受其控制的软件平台，在自家的应用生态系统上也同样薄弱无力，为了减少对 Google 的依赖，从 Android 手机转向 Windows 手机，无非就是希望在 Windows 平台复制 Galaxy 系列旗舰手机曾经的辉煌。

先不论 Windows Phone 现在低得可怜的市场占有率，对于三星来说，该解决的问题仍然在那里：从 Android 平台转向 Windows 平台，三星仍然缺乏对应用系统和软件平台的控制力。即使三星像三年前进入智能手机市场时那样，如卖 Android 手机一般成功地将 Windows Phone 卖火了，仍然避免不了在过去的一年里走过的老路：仍然没有抓住运营商的渠道，仍然会有新的厂商加入，仍然难以解决低价竞争的困境。

在一个别人家的平台上，永远会有新的竞争者跑得比你更快，到那时，不知道三星还能不能撑到再转战新的平台的那一天。

当然，那些都还是遥远的假设。至于最近能否看到 Windows 三星手机大量上市，还取决与三星和微软何时能够在专利使用费的诉讼上达成一致。如果顺利的话，可以想象，最早在今年秋天，我们就会在营业厅、地铁站以及公交站牌看见铺天盖地的三星手机广告，和以往唯一不同的是，那些手机的界面可能将由 Windows 风格的彩色方块组成。
