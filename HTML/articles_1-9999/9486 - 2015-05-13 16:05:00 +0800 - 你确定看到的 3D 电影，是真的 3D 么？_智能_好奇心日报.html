你确定看到的 3D 电影，是真的 3D 么？_智能_好奇心日报

王嘉俊2015-05-13 16:05:00

看到假 3D 电影，很可能只有字幕有 3D 效果。

这儿有个网站，让你知道什么是“真”3D 电影的名单。

真假 3D 一直是电影界有趣的事情。自《阿凡达》上映以来，市面上掀起了 3D 电影热。观众们热衷观看 3D 电影，院线也能获得丰厚利润，毕竟 3D 电影票几乎是普通影票的两倍左右。

但要拍好真 3D 电影并不容易。3D 电影拍摄的原理是，通过两台摄像机模拟人的眼睛，拍左眼和右眼画面，然后通过计算机把拍摄的图像合成实现 3D 效果。相比 2D 拍摄，它要用到昂贵的双镜头摄影机和配备大量镜头，而由于摄影机沉重对片场操作的要求也更高。3D 电影拍摄的存储容量会翻倍，数据处理会远多于 2D 电影，这都让 3D 电影的拍摄成本非常高。

以几部经典著作为例子，《阿凡达》制作成本高达 3 亿美元，《愤怒的小鸟》制作成本也将近 1 亿美元。

但观众往往不会区分真 3D 和假 3D 电影，如果采用 2D 电影转 3D 的方式，成本会大大下降，而票房有时也不错。

在这方面做的最好的当属卡梅隆导演的 3D 版本《泰坦尼克号》。

卡梅隆把 1997 年版本的《泰坦尼克号》进行了 2D 到 3D 的转换。2D 转 3D 和人眼观察物体的原理有关：当有一定距离的两个点观察同一个目标时，如果知道两点之间的距离，以及两点看目标的夹角，就能知道目标和两点的距离。人类进化使得可以本能去感受物体的远近，而计算机也能根据这个原理，来模拟人类的立体观感。


所以一个简单的 2D 转 3D 过程是把图像进行平移。人类瞳孔的平均距离是 6.4 厘米，计算机会根据景物应有的景深数据，把两幅图像进行平移，就能制造出粗糙的 3D 体验图像。当然具体操作会复杂很多，但大体是给图像加上景深数据，以此改变用户观看体验。

这么做的成本要远低于 3D 电影拍摄。3D 版本的《泰坦尼克号》只花了 1800 万美元和 60 周时间。相比之下，《阿凡达》花费了 3 亿美元的制作成本，前后耗时 14 年。 而票房往往还不错。《泰坦尼克号》3D 版本在中国的票房超过了 10 亿元，很简单就能知道对经典电影的 3D 转换，有潜力获得高额回报。

所以不少电影都开始做 2D 到 3D 的转换了。《一代宗师》、《功夫》、《2012》、《冰河世纪》等电影都推出了 3D 版本，试图基于以前打下的名声，以低成本的方式继续挖掘利润。

这在中国尤其行得通。《泰坦尼克号》3D 版本大部分票房来自中国，除了观众对 3D 电影的固执喜爱外，另一个原因是在中国看真 3D 电影和 2D 转 3D 电影经常会傻傻分不清楚。

这是院线屏幕设备的原因。国内常用的 3D 设备，从次到优主要有杜比单机 3D、杜比双机 3D、RealD3D 和 IMAX。其中杜比 3D 系列普及率最高，大部分的院线都采用杜比设备。

这其中还有很多院线配套措施不完善，灯泡亮度严重不足，画面会显得很暗。设备的落后直接让真 3D 和假 3D 电影经常处于差不多的低质量水平，虽然观影体验有所差别，但丝毫没有体现出真 3D 电影应有的优势。

考虑到 3D 票价更高，很多假 3D 拍摄的片子其实你也看不到 2D 版本，因为 3D 版本票价更高。

如果你对真 3D 是真爱，可以在这个网站查询。
