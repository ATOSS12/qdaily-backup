Google 手机项目 Project Tango ，明年面世_智能_好奇心日报

马宁忆2014-06-27 19:38:38

Project Tango 是 Google 旗下一款具有实验性质的手机项目，由于有望复制真实世界的 3D 模型被认为是未来手机。最近，而 Google 在 I/O 大会上表示，它将与 LG 合作在明年推出消费级别的 Project Tango 。 这是走向未来的节奏么？

上周，亚马逊发布了它的首款智能手机 Fire 手机。虽然这款手机并不如外界所猜测和预期的那样拥有酷炫的裸眼 3D 技术，但它还是能够被称为一台特别差异化的智能手机。

除了Fire 手机，Google 的 Project Tango 手机也是这么一台跟 3D 发生着关系的手机了，但是和 Fire 手机的 3D 理念基本不同。它所呈现的 3D ，是当人们拿着它四处走动的时候，可以自动扫描周围并即时绘制 3D 图像，并以此复刻一个真实世界的 3D 模型，因此它是一款寄希望于帮助人们了解关于空间和移动的未来手机。

这个颇有些实验性质的手机项目原属于摩托罗拉的 ATAP 部门，之后归 Google 所有。在Google 将摩托罗拉出售给联想时，ATAP 部门是小部分被保留的部门。今年 3 月 14 日，Google 将 Tango 的 200 多台原型机发放给开发者。4 月，它公布了和 NASA 的合作计划，相机将被整合进 NASA 的机器人 SPHERES 送往太空。而最新的消息是，Google 在 I/O 大会上公布，自己将和 LG 合作，在明年推出消费级别的 Project Tango 相关设备。

“目前，我们正在和 LG 一起努力，希望能够在明年推出 Project Tango 手机。它早期版本的开发工具将和 Unity 和 Unreal 游戏引擎合作，任何会使用它们的人都可以来为 Tango 开发应用。”Google 说道。
