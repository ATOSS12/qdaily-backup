药盒里的数学家 Memo Box 用算法提醒你吃药_智能_好奇心日报

王嘉俊2014-11-05 21:45:50

在提醒你吃药的时候，Memo Box 其实在做数学题

来自剑桥的两个学生最近在 Kickstarter 发起了一个众筹项目 Memo Box，用贝叶斯定理提醒你吃药。

这听起来很唬人，但使用起来很简单。你不需要在软件里记下自己吃药的时间，只需要把药盒和它的应用通过蓝牙连接起来，药盒就会接管一切：

当你忘带药盒出门，或者忘记吃药，Memo Box 会提醒你；

当你不知道要吃什么，或者重复吃药，Memo Box 还会提醒你。

Memo Box 就像一个贴心的小管家照看生病的你，让你能够快快康复。

但如果你自己按时吃了药，它就是一个安静的小药盒，不会像闹钟一样定点来烦你。这种安静来自深邃的思考，在每天跟随你、观察你的时候，它其实在做数学题。

我们上面提到了贝叶斯定理，这个定理由英国数学家贝叶斯 ( Thomas Bayes 1702-1763 ) 发展，用来描述两个条件概率之间的关系。不过你不必了解定理的详情，简单来说，这不过是估测 A 事件发生之后，B 事件发生的概率。

落实到 Memo Box 上便是，它会收集你过去吃药的时间数据，以及它在提醒吃药后你的反馈，然后建立一个模型。这样，用户就不用老是惦记着设定吃药的时间，Memo Box 会在最合适的时候提醒你。

如果你总是不理会它的提醒，它可能会和你的爱人投诉 （设置 Memo Box 关联到你的爱人，这样他就能知道你吃药的情况），让他们来处理你吃药的问题。这样，如果我们想知道爱人、父母和孩子是否按时吃药，就可以通过 Memo Box 来实现。

现在 Memo Box 处于众筹早期，标准版定价是 45 美元，众筹截止到下个月的 18 号。
