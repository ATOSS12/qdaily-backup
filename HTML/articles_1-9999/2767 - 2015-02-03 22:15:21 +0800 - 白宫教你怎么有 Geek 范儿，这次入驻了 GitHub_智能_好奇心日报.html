白宫教你怎么有 Geek 范儿，这次入驻了 GitHub_智能_好奇心日报

唐云路2015-02-03 22:15:21

连年度预算都放到 GitHub 上面，白宫在数字领域已经挺拼了。

既 2014 年 10 月入驻社交平台 Medium 之后，白宫又在数字领域插了一脚。这一次，奥巴马将年度预算放到了数据开源平台 GitHub 上。

GitHub 是一个共享虚拟主机服务，它已经成为最流行的开源项目托管平台。人们可以在上面寻找自己喜欢的项目，为其贡献修修补补之类的代码，或是干脆参与到项目的运作中去。日前，美国军方也将安全代码放到了 GitHub 上面，来获得非政府用户的反馈，加强代码的安全性。

这次，白宫将预算放到 GitHub 上，使 2016 年的美国年度预算成为了“第一个开源预算”，白宫将预算涉及的数据以“机器可读的方式”公开，这样任何人都可以将这些数据制作成可视化的图表或产品。

当然，这并不是奥巴马治下的白宫第一次接触 GitHub，早在 2013 年，它们就曾经在 GitHub公布了一项政策草案——公开数据草案（Open Data Policy）。此外，白宫的多个 App 产品的代码也托管在 GitHub 平台上。不过，像这样将预算政策开源，对于白宫来说还是第一次。

在此之前，白宫在数字领域的策略，给人的印象还停留在活跃的社交网络账号上。回顾一下白宫近年来在数字领域的策略，虽然还比不上迅速占领社交网络的某些极端组织，依靠 Facebook 赢得大选的奥巴马政府其实一直都不算落伍。

在 Medium 上开账号，写博客抢粉丝

2014 年 10 月，美国白宫宣布入驻新的社交平台 Medium。如它们在首页致辞中所说的，白宫将在 Medium 上与公众分享科技、创新、教育、经济等各方面的话题。开通账号第一天，美国总统奥巴马就在 Medium 上发表了第一篇博客文章 Why I’m Betting on You to Help Shape the New American Economy. 如今，白宫在 Medium 上的账号已经拥有了 5.5 万关注者。

Medium 是一个个人出版社交平台，由 Twitter 联合创始人 Evan Williams 与 Twitter 前员工 Biz Stone 共同创立，于 2012 年底上线。

和在 Tumblr、Facebook、Twitter 上开设账号一样，这是白宫在社交网络上做出的又一尝试。

竞选始于网络

从 2007 年宣布竞选美国总统时起，奥巴马的竞选团队就活跃在网络上。

2008年，奥巴马的团队发出了 1300 万封邮件寻求竞选支持，有 400 万捐款者通过网上电子渠道捐款，其支持者网站My.BarackObama.com 拥有 200 万会员。几大社交媒体 Facebook、YouTube、MySpace 和 Twitter 上都活跃着奥巴马团队的拉票界面。



Twitter 回应绝不手软

2012 年的共和党全国代表大会上，导演 Clint Eastwood 以“看不见的奥巴马”为题，对着舞台上的一把空椅子指责奥巴马未守承诺。对此，奥巴马团队的反应相当迅速，很快在 Twitter 上发布了奥巴马坐在总统专座上的照片，配文"This seat is taken"。这条推文成为当年最具影响力的推文之一，白宫数字团队的反应速度居功不小。


上 Reddit 答“天涯网友”问

除了在 Twitter 上经常发发照片和新闻，偶尔还要集中与网友互动。

2012 年，奥巴马在问答网站 Reddit 上参加了 AMA(什么都能问 ，Ask Me Anything)频道的在线访谈，Reddit 用户给出了超过 22000 条问题和评论，原计划 30 分钟的访谈最终被延长到 45 分钟。访谈的高人气不仅导致 Reddit 网站因流量太大而崩溃，在访谈结束后的 24 小时之内，有 520 万人围观了访谈内容。至今为止，这个页面上保留有 23954 条评论。AMA 并不是名人专属的访谈区，上过访谈的人既有伍迪·哈里森、鸟叔这样的明星，也有科学家、企业家甚至众筹创业者，还有在某些方面拥有一技之长的人，总的说来， 这就是个所有人问所有人的地方。

Reddit 这个吸引了众多年轻用户的群体对于政治宣传来说再合适不过。这一访谈活动无疑为奥巴马吸引了不少“千禧一代”的票数。

此外， 2013 年 和 2014 年的 2 月，奥巴马还曾在 Hangouts 环聊上与 Google 用户视频聊天，谈论的话题非常广泛，从网络环境到自家女儿的职业前景都有包括。此外，奥巴马亦曾将 Tumblr 创始人 David Karp 邀请至白宫，参加一个关于教育的在线 Q&A，活动中讨论的问题都是从 Tumblr 中选取出来的，为了吸引 Tumblr 较为年轻的用户群体，奥巴马在 Tumblr 上的首次亮相就是一张与 David Carp 合作的 GIF 动态图，白宫也在 Tumblr 上开了官方账号。

白宫也玩线下活动

白宫的 Facebook 账号拥有 268 万粉丝，与 Google+ 账号表现相当，其Twitter 账号更为活跃，拥有超过 534 万关注。在图片分享方面，白宫在 Pinterest 和 Instagram 上也开设了账号，分享关于总统、第一家庭以及官方活动的照片。

白宫不仅在各家社交媒体上开设了账号，而且常常组织粉丝来白宫参加线下活动。下面这条推文配图拍摄于今年 2 月，三位正在自拍的粉丝正是通过 Twitter 获得邀请，到白宫参加总统与第一夫人欢迎法国总统奥朗德来访的仪式。

从硅谷找 CTO

今年 9 月，白宫宣布任命 Google X 高管 Megan Smith 为美国白宫首席技术官（CTO）。这一职位设立于 2009 年，在 Google 人进驻白宫之前，这一职位由 Todd Park 担任。Park 在任期间，前 Google 工程师 Michael Dickerson 等来自硅谷的多名员工都被白宫纳入旗下。不过，Park 负责的医保网站 HealthCare.gov 与奥巴马的医保政策一样倍受争论。以至于有分析师表示，虽然美国坐拥硅谷，但是却没有招到合适的技术管理高层。

比起硅谷的互联网巨头，白宫的技术能力显然还有提升空间。

看白宫此前在社交网络上的表现，频频在Tumblr、Facebook、Google+ 等社交媒体也是频频提到对“千禧一代（ Millennials）”的期望。“千禧一代”指 1980-2010 年出生的人，常见于营销、研究或者报道等较为正式的场合，习惯性的用这个词暴露了它和社交网络依然有距离。

不过，学会将数据公开到 GitHub 上，倒是让人对白宫的 Geek 范儿有了些许期待。就如同两年前白宫将政策草案放在 GitHub 时，在 GitHub 的 Ben Balter 说的那样：改变才刚刚开始。在数字领域刷存在感，也不是仅仅靠社交网络上发几张总统的照片。
