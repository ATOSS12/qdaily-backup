Analog Memory Desk 模拟存储工作桌_设计_好奇心日报

2015-03-12 15:46:57

你可能也曾经有过想要灵感突现想要随手记下点儿什么却发现四处找不到纸的尴尬经历，等到找到铺开，灵感可能早就跑了。另外，有时候我们用便签记下一些即时性强东西，等到明天不再需要他们，便签便扔的满桌子都是。Kirsten Camara 为了解决这个问题推出了一款叫做 Analog Memory 的桌子，这个桌子的底下有两个轮轴，1000 米长的纸跨过桌面被轮轴固定在桌上，桌面的纸张面积足够你写一阵子，当这些信息过期后，可以被转到桌底下，如果需要被翻出来，只需要转动另一边的轮轴即可。等到全部的纸张用完，便可以整卷回收再利用。点击这里可以看到这个设计师的个人网站。
