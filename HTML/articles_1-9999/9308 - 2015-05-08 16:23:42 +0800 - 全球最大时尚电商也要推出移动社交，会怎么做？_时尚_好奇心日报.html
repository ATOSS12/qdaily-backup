全球最大时尚电商也要推出移动社交，会怎么做？_时尚_好奇心日报

张田小 2015-05-08 16:23:42

如果你拍一张夕阳，软件也许会向你推荐一款应景的 Burberry 裙装。

正在与 Yoox SpA  (YOOX.MI) 合并成为全球最大奢侈品电商的 Net-a-Porter 宣布将推出其最新的移动社交平台 The Net Set 。

不过现阶段 The Net Set 还没完全开放，只能通过邀请注册。

早在 2013 年 9 月，Net-a-Porter 就已推出用于分享和购物的 The Netbook 。The Net Set 副总裁 Sarah Watson 指出 The Netbook 实际上是 The Net Set 的测试版本，自 2014 年底，她就联合现任 The Net Set 创意总监的 Alexandra Hoffnung 着手于 The Net Set 的开发。

The Net Set 实际是在为顾客搭建奢侈品购买交流的社区平台。Net-a-Porter 在周三发表声明称，建立账户后，顾客能关注到品牌、朋友、其他时髦用户的风格，也能买到超过 350 种线上最走红的设计系列。有了 The Net Set ，Net-a-Porter 第一次能和顾客有直接交流。

The Net Set 引入第三方图片识别科技的产物。如果你用 The Net Set 拍下一张你朋友的 T 恤的照片，软件将会帮你在网上找到这件衣服，或者找到一件相似品。另一个功能是，如果你拍一张夕阳，软件也许会向你推荐一款应景的 Burberry 的裙装。

“社交媒体的兴起，时尚博主的走红让全球爱美人士能互相欣赏彼此的穿搭和风格，” Net-a-Porter 的创始人 Natalie Massenet 说，“2015 年的 The Net Set 移动应用就相当于 2000 年的 Net-a-Porter.com 桌面网站”。适用于 iPhone、iPad 和 Apple Watch 的 The Net Set 将于 5 月 13 日登陆 iTunes App Store ，Android 安卓版本也在开发日程中。

Net-a-Porter 此举表明 Net-a-Porter 想进一步利用每月 600 万的独立 IP 访问量，在 Twitter，Facebook 和 Instagram 等 9 个全球主流社交应用上超过 410 万的粉丝，来进一步拓宽影响力。

The Net Set 的出现表明 Net-a-Porter 也加入社交电商市场。其他公司也在考虑如何将粉丝力量变成购买力。去年，RewardStyle 成立 LikeToKnow.it ，如果你在Instagram 对心仪的产品点击“喜欢”， LikeToKnow.it 会让你的邮箱收到相关的购买链接。 Vouge 和 Matches 都在用 LikeToKnow.it 。4 月获得 2500 万 C 轮融资的二手服装交易应用 Poshmark 就是个 eBay 的社交版。

但社交电商们面临的一个问题是如何让用户买他们看上的商品。英国网络研究公司Baymard Institute 研究表明，70% 的潜在网购用户会最终放弃购物。
