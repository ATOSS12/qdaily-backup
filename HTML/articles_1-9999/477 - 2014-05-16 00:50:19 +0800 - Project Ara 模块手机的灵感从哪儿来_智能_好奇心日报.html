Project Ara 模块手机的灵感从哪儿来_智能_好奇心日报

马宁忆2014-05-16 00:50:19

任何好的设计都来源于一个好的灵感。NDD （NewDealDesign）的主持设计师 Gadi Amit 说，Google 实验性手机 Project Ara 的灵感来自于蜂箱。

任何好的设计都来源于一个好的灵感。那这个灵感是从哪儿来的呢？如果你有一双发现美的眼睛，可能哪哪儿都能迸发灵感。所以当你看到一个蜂箱的时候，你看到了什么？ NDD （NewDealDesign）的主持设计师 Gadi Amit 告诉我们，他从中看到了 Google Project Ara 模块手机。

Project Ara 是由美国国防部先进研究项目局（DARPA）的Paul Eremenko 和 Google 的ATAP 部门一同研发的新型手机。它由许多大小不一的手机模块组成，支持用户像组装乐高积木般的随意拼接它们。因此，Project Ara 的理想状态是，在主板不变的情况下，人们可以在有需要的时候调换手机的各种功能模块，达到“不用换手机却胜似换手机”的效果。作为 Google 在将 Motorola 出售给联想时保留下的少数部门之一， ATAP 部门旗下的 Project Ara 和另一款实验性相机 Project Tango正一同探索性的展示出 Google 对再次占领智能手机领域的野心。

人类有许多来源于大自然的优秀灵感， Project Ara 也不例外。最早，它的灵感来源于分叉状的树枝。不过，在树叉设计出炉之后，第三方生产商表示自己无法生产出能填补树枝周围空白处的模件，也就是他们无法以原本做手机的方式来实现这个想法。于是，设计公司 NDD 和 ATAP 部门 意识到，是时候制作一款完全不同的手机了。

“我觉得这款手机研发中最重要的时刻就是我们谈起结构的时候。我想这个灵感应该是源于我来自一个建筑师家族的这个背景吧” Amit 说道。

于是，手机的草图开始尝试趋向于绘制一个城市的鸟瞰布局图，也就是用道路和不动产楼房来分出一块块区域从而呈现出结构。设计团队想通过借鉴这种精准的结构图来进行 Project Ara 的设计。不过，他们很快意识到，这样太机械和呆板了，而他们的设计应该有更多的激情。此外，模块之间的边界也应该比现在更“柔软”更“浅”。于是，它们想到了分格过渡并没有十分僵硬的蜂箱，这个同样来自于大自然的灵感。

不久前，这款预计于 2015 年上市的实现性手机与耳机品牌 Sennheiser 协议合作，后者将为 Project Ara 提供音频服务，同时弥补自己作为一家主流音频服务商从未将技术真正运用到手机的缺陷。而在 4 月15 - 16 日，Project Ara 刚刚在旧金山召开了开发者大会，引发了大批开发者的追捧。

虽然这个模块手机想改变的不仅是手机产业，更是整个科技产业的未来，但在短时间内，这个有些实验性的手机可能也并没那么容易被所有人接受。那就多开发出一些好玩的功能模块吧，至少这会让人更有兴趣来对它探个究竟。
