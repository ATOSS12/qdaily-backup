Windows 10 来了，但微软想靠它重当霸主还得期待奇迹_智能_好奇心日报

李硕明2015-01-22 17:33:16

更友好、免费升级，但晚了 6 年

微软在今天凌晨正式发布面向消费者的 Windows 10 预览版，花了两个小时介绍下一代 Windows 操作系统的各项功能。

“我们希望能让人们从需要 Windows 转变为选择 Windows 并且喜爱 Windows。”微软 CEO 萨提亚·纳德拉在发布会上表示。在他看来，今天不论对微软还是 Windows 来说都是个重大的日子。

Windows 10 确实重要，苹果和 Google 早已经改变了用户对于操作系统交互方式的预期。2014 年纳德拉给员工的公开信中写道，如果把手机、平板、PC 等计算设备都算起来，Windows 只占到所有份额的 14%。

微软在 2012 年推出的 Windows 8 遭到了一片恶评，用户甚至买了新电脑之后装回 Windows 7——就像著名的失败产品 Windows Vista 一样。在之前的几年中 Windows 8 的占有率一直就在很低的水平上。直到去年的 11 月，Windows 8 的市场占有率才达到 16.8%。

Windows 8 过低的市场份额和糟糕用户评价，也让各家 PC 厂商有点失去耐心。惠普、宏基、联想等传统合作伙伴都转向 Google ，开发了数款 Chromebook 和 Android 平板。

Windows 在手机上的情况更惨，就算凭借诺基亚的 Lumia 系列一度拿到了第三大手机操作系统的位置。不过 Windows Phone 的全球份额最高峰也只有可怜的不到 4%，现在已经不到 3%。而在最大的智能手机市场中国，Windows Phone 已经不到 1%。

对于微软来说，Windows 10 有两个任务：

  1. 改变 Windows 8 给桌面用户留下的阴影，让 Windows 系统成为微软作为生产力工具厂商的优势而非阻力；
  2. 在移动市场赢得第二次机会。

那么 Windows 10 打算如何同时赢回这两类用户的心？Windows 10 中的一系列新功能就是微软给出的答案：

Spartan 浏览器

改变形象就从改名字开始。恶名昭彰的 IE 浏览器不再是 Windows 的默认上网工具。代替它的新浏览器被称为 Spartan（斯巴达人）计划，取自微软在 Xbox 上的主力游戏《光晕》（Halo）。不过微软并没有让 Spartan 完全取代 IE，而是选择了两者共存的方式。Spartan 负责代替原本的 Modern 版 IE，而桌面版的 IE 仍然会继续存在。

Spartan 有一个全新的设计，标签卡在地址栏上方，看上去更像 Chrome 的设计。点击左上角的按钮，你就可以让标签卡的界面上升，看到打开的标签网页的预览。

阅读模式、Onenote 笔记批注、Cortana 是 Spartan 的三个核心功能。

按照微软的设想，当你在网上搜索某个内容或者餐厅、登陆 Facebook 查看和家人朋友的日程安排时候，Cortana 会给出提示“我已经知道路线导航和营业时间”和你的日程安排。

OneNote 是 Spartan 整合的另一个功能。微软鼓励厂商在平板中支持手写笔，现在你可以直接在 Spartan 打开的网页上用笔做记录，你同样也可以用键盘和鼠标在网页上作记录。所有批注会被保存在微软的云端空间，你也可以在手机上看之前的批注。用户可以选择这些批注笔记是保存到 Onenote 还是分享给好友。

Spartan 的阅读模式可以让你更专注的阅读网页内容，点击地址栏右侧的阅读模式按钮，就可以进入阅读模式。微软之前已经在 Windows 8.1 和 Windows Phone 8.1 中加入了阅读模式。你可以在阅读模式中，利用整合的 Onenote 批注笔记。

Cortana 人工智能语音助手

微软终于把 Cortana 整合进了 Windows 10。虽然在之前已经泄露的版本号为 9910 的 Windows 10 技术预览版中，Cortana 已经基本成形，不过当时 Cortana 还只能执行一些相当简单的任务。

在全新的 Windows 10 预览版中，开始按钮的旁边有一个搜索框，用户可以点击搜索框或者直接说 Hey Cortana，就可以启动 Cortana。Cortana 的白色界面会在搜索栏上面弹出，可以显示当前用户的日程，所在地的天气，Notebook。 

在 Windows 10 中，微软将 Cortana 和 Windows 10 更深入地整合在了一起。Cortana 也支持搜索文档，包括 Onedrive 的内容。你可以在新的地图应用中，激活 Cortana 寻找你想要去的地点。Cortana 也可以记住你的车停在哪里，这一功能之前要通过下载 HERE Drive 应用实现，现在微软将它整合进入了 Cortana。

现在 Cortana 在 Windows 10 中几乎是无处不在。

开始菜单

和 Windows 8 在 PC 屏幕上放几个硕大的方块不同，Windows 10 的磁贴为鼠标操控和大屏幕优化，现场展示中，同屏最多可以呈现 96 个磁贴。

不是所有人都喜欢全屏的方格。Windows 10 的开始菜单提供了定制尺寸，你可以自己决定开始菜单需要多大。在这个屏幕尺寸繁多的世界，这是个好点子。而针对平板的开始界面则与 Windows 8 相同。这里有一张清晰的官方对比图：

触屏版 Office 来到 Windows 

继微软宣布了 iPhone、iPad、Android 版本的 Office 应用之后，Office 终于来到了 Windows 10 平台。

Windows 10 版本的 Office 将会是一款通用应用，微软表示全新的 Office 将会成为 Windows 10 的一部分，包括 Word、PowerPoint、Excel、Onenote，同时面向手机、平板、PC。

Windows 10 版本 Office 采用和之前 iPad 和 Android 上相同的用户界面，整体设计要比之前 Windows Phone 的 Office 软件更好。在手机上，微软通过二级菜单的方式实现传统的 Ribbin UI。

微软还在 Windows 10 中更新了之前的邮件和日历应用，两款应用都经过了重新设计。不过微软并没有直接将两者合并，而是分别被命名为 Outlook Mail 和 Outlook Calendar。

Xbox 游戏平台

微软在新的 Windows 10 预览版中加入了全新设计的 Xbox 应用。这款新的 Xbox 应用可以显示你的 Xbox 游戏，你和好友的成就，来自好友的邀请，动态信息。

全新的 Xbox 应用可以让你录制 30 秒的游戏视频，按下 Windows+G 的组合就可以录制视频了。录制好的视频会直接分享到你的 Xbox Live 上。

通过 Windows 10 中的 Xbox 应用，你可以和朋友跨平台玩游戏。Windows 10将带来更多多人游戏，通过Xbox跨平台解锁更多 Xbox Live 成就。Windows 10 将支持将 Xbox One 游戏串流至任何 Windows 10 PC 或平板，也可以和一台由 Windows 10 驱动的 Xbox 联机一起玩游戏。

微软同时发布 DirectX 12 游戏 API。DirectX 12 将是 Windows 10 独占。微软称 DirectX 12 可以让开发者更有效地控制计算资源，让游戏性能提升 50%、同时改善了游戏在手机上的耗电。

手机上的 Windows 10

Windows 10 是一个统一核心的全平台操作系统，将会横跨手机、平板、PC、Xbox。微软也对 Windows 10 的应用进行了更新，这些应用可以同时运行在手机和平板上。

遗憾的是 Windows 10 在手机上所推出的新功能大多是 iPhone 和 Android 手机早已实现的。面世 5 年后，Windows 智能手机还在这些基本功能上追赶着竞争对手。

Skype：现在 Skype 已经和消息应用整合在了一起，类似苹果的 iMessage 用户可以在一个应用中选择是通过短信还是 Skype 给朋友发送消息。你可以通过不同的颜色进行判断现在是在用哪一款应用聊天。

通知中心：在 Windows 10 中，微软为通知中心加入了快捷操作中。类似 Android 的通知中心和苹果的控制中心，你可以在通知中心里快速选择设备的操作选项。手机端也不再有 只能显示 4 个快速操作选项的数量限制。

磁贴屏幕背景：从微软现场的演示来看，Windows 10 手机版已经可以用自定义图片作为屏幕背景，而非之前只能选择的黑白两种颜色。

新的设置界面：Windows 10 手机版带来了全新的设置界面，重新设计了图标，并对选项重新进行了分类。相比 Windows Phone 8.1 的设置界面，新版 Windows 10 的设置界面要更加清晰。

可以直接回复的通知：Windows 10 手机版现在可以让你在收到通知时，直接进行快速回复，不用再回到应用中。

两款硬件

本次发布会上，微软还推出了两款硬件产品：

Surface Hub

如果你之前对微软推出的 Perceptive Pixel（巨屏设备）还有印象的话，那么你对 Surface Hub 也不会感到太陌生。

Surface Hub 是一款 84 英寸的 4K 智能电视，内置电脑，多点、多笔触摸屏，内置先进的传感器，双摄像头、麦克风阵列和扬声器。

微软认为 Surface Hub 将会在会议、白板、视频、笔记，办公场景发挥重要作用。

HoloLens 全息投影

让人出乎意料的是，微软在发布会上带来了自家的全息头戴式设备——Microsoft HoloLens。

和那些主打虚拟现实或者现实增强的设备不同，Microsoft HoloLens 是一款可以进行全息投影的设备，这意味着你身边的人也能看到交互界面：

在微软的宣传视频中，这款 Microsoft HoloLens 可以将你的日程安排，物品信息都显示出来。甚至拨打 Skype 电话也是全息投影，如果你在向别人求教某个技术问题，Microsoft HoloLens 可以直接将别人的建议变成指示方向的箭头。

发布日期和定价

新的 Windows 10 预览版将会在下周推出，你可以通过之前的 Windows Insider 下载更新，微软也会发布新的预览版 ISO 镜像安装文件。正式版的日期尚未公布，微软只说将在今年晚些时候发布。

微软大概是真的急需用 Windows 10 夺回市场，这次让 Windows 10 免费升级了，至少是在一年内，原有的 Windows 7、Windows 8/8.1 用户可以直接免费升级到 Windows 10。

迟来 6 年的 Windows 10，能拯救微软吗？

2009 年 7 月，微软推出 Windows 7，一扫 Windows Vista 留下的阴影，很快成为全世界占有率最高的操作系统。但就在微软为 Windows 7 而骄傲的同时，iPhone 和 Android 已经快速崛起。

2014 年第 2 季度每卖出 100 部智能手机，有超过 84 部是 Android 手机、11 部 iPhone。Windows Phone 已经不到 3 部。 

微软知道自己错过了进入移动时代的机会，而 2012 年搬来救火的 Windows 8，却让微软遭遇了失败。除了遭到用户的抛弃，Windows 8 也让 OEM 厂商抛弃了微软。曾经的队友纷纷离开，与 Google 合作推出 Android 平板和 Chromebook，比如联想、惠普、戴尔、宏基。

而 Mac 也在 Windows 8 失败的这段时间内快速增长，苹果凭借 Mac 的增长在 2014 年成为了全球第五大个人电脑厂商。

Windows 10 才是 Windows 7 之后，微软应该拿出来回应苹果和 Google 的作品。现在微软晚了 6 年。 

在微软传统优势的企业市场，Windows 10 和 Office 的组合看上去不错。但是微软想要的不只是这么多，它还要在手机市场和赢得一席之地。

但 Windows 10 在手机上带来的种种新功能大多是 iPhone 和 Android 早已有的功能。我们没能看到微软带来任何非 Windows Phone 不选的理由。今天，Windows Phone 的市场占有率已经不到 3%，已经很少有成熟的公司选择在 Windows Phone 上率先发布自己的应用，已经推出的应用往往也长时间没有升级。

在今天的移动市场，Windows 10 想成功，乐观和几个“新”功能已经不够，微软得期待奇迹的出现。
