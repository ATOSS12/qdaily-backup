今日应用：不习惯浏览器书签？Google给你新选择_商业_好奇心日报

徐弢 崔绮雯2014-11-04 16:17:22

有了 Chrome 新的 Bookmarks Manager 插件，再也不用担心收藏夹又丑又乱了。

  • Bookmark Manager 是一款 Chrome 浏览器新推出的小插件，可将添加了书签的网页汇总起来，不会像以前一样容易胡乱防止到导航栏，现在可以放心给待阅链接加“星标”了；
  • 整理书签心情都会变好，因为 Bookmark Manager 把书签汇总页面视觉化了，有图片、标题、摘要外链更像真实的“书签”，一目了然，整理效率也变高了；
  • 如果上述的理由还不够方便，小插件还支持“搜索书签”功能，不仅标签的标题和网址，还能搜到网页里面的关键字；

每天都能够在互联网上看到许多新鲜有趣的内容就想快速保存起来，但要么忘记打开，要么就是不知道究竟跑到了哪个标签文件夹的角落里，甚至有可能乱七八糟地躺在浏览器的导航栏上，体验略糟糕。

应运而生的就是所谓的“待阅”应用，例如 Pocket、Instapaper 的浏览器插件及其手机应用。不过他们可能要遇到一位新对手了，就是 Google 新推出的 Bookmark Manager，它是基于 Chrome 浏览器的一款“待阅”扩展插件，安装之后，点击导航栏上的“星星”图表即可进入汇总的书签页面，设计简洁漂亮。

以往的 Bookmarks 汇总页面真是让人看腻了：文件夹图表，点进去之后是一串长长的“标题+链接”表单……所以在看到新的 Bookmarks Manager 页面时感到很舒服，目前每条链接会附带图片、标题、摘要，浏览一眼就能知道大概，这对于有海量书签需要挑选资料的人们来说十分重要。

另外，点按地址栏旁边的收藏按钮时，弹出的收藏框也相当精美，还可以选择代表这个书签的小图片。真的不会告诉你，为了看这个弹框我收藏了几篇文章。

有趣的是，Google 将自身的搜索特长放入了 Bookmark Manager 中。新插件的说明中宣称新能够遍及网址和标题，标签内的网页内容也能够被搜索到，经过我们的测试，中英文都可用。不过。目前这个功能槽点满满——只支持安装插件后添加的新书签，之前的书签依然不可搜索！这对于书签重度用户来说非常不便，谁会重新将之前收藏的书签取消收藏再添加一遍吗？

总而言之，这个书签汇总页面十分适合经常在电脑浏览器面前工作和刷新闻的人们，但如果说 Bookmarks Manager 要成为目前市面上待阅应用的强力竞争者还相当有难度，因为如 Pocket、Instapaper 在手机端阅读体验良好的应用才是他们的核心竞争力，而打开 Chrome 手机版浏览器之后，我们失望地发现书签管理页依然没有随着 Bookmarks Manager 的出现而改变。
