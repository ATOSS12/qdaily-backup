微软要送走 IE 了，新浏览器将以 Microsoft 开头_智能_好奇心日报

李硕明2015-03-18 23:15:52

“用 IE 毁一生”这句话也许以后会成为暴露你年龄的新方式

根据 The Verge 的消息，微软打算在今天夏天发布的 Windows 10 上正式用 Project Spartan 浏览器取代 IE，而正式的浏览器名字，可能会带有“微软”字样。

微软市场营销主管 Chris Capossela 在 Convergence 会议上确认了这个消息。Project Spartan 还只是一个代号，他们正在考虑给 Spartan 浏览器设计新的名称。至于 IE 浏览器，它仍将会继续存在于 Windows 10 中，微软表示这么做主要是考虑到企业用户的需要，为了保证网络的兼容性而已。

Chris Capossela 表示这款代号为 Project Spartan 的浏览器将会是微软接下来在 Windows 10 中主推的浏览器。实际上，在最近泄露的 Windows 10 Build 10036 版本中，IE 已经被微软划归到附件中去了。对于微软来说，他们希望能够用新的 Spartan 浏览器和过去的 IE 划清界线，改变用户对 IE 传统的“卡、慢、丑”的形象。

微软在过去也考虑过通过换名的方式，摆脱 IE 的臭名声，不过最终没有成功。关于 Spartan 浏览器的新名字，微软还认真地在内部做了一个调查。它们发现如果在新浏览器的名字中加入“Microsoft”的字样，就会吸引英国的 Chrome 用户转投到微软这边。

其实微软也早该放弃 IE 了，至少从数据上来看是这样的。而微软甚至因为 IE 在移动互联网时代，错失了发展机会。从 2014 年 12 月开始算，IE 只占到了移动互联网流量的 13.42%，而 Chrome 的数据是 42%。在移动互联网时代，IE 和其他竞争对手的差距几乎已经是无法补救的了，微软只能尝试通过更换浏览器这样的方式了。

IE 你好，IE 再见。
