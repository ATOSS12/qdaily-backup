苹果开放测试暴露了自己是 Windows 用户，你怎么看？_智能_好奇心日报

唐云路2014-09-28 21:11:25

人们紧紧跟随每一版 iOS 系统的更新。可是苹果自己却不太愿意升级生产线上的 Windows 系统。

截止 8 月底的数据显示，全世界联网的 PC 中，仍有 23.8% 的电脑运行着 XP 操作系统。尽管大部分的用户，已经无法获得安全漏洞的修补。仍在坚守 XP 系统的“软粉”并不孤独，苹果和你们在一起。

苹果开放了测试中心给媒体参观，本意是让大家看看它们的测试有多严格。但除了测试以外，人们还注意到，苹果的质量控制实验室里还在用 Windows XP 系统。

Windows XP 系统并不仅仅被用于苹果的质量控制环节，今年 7 月，苹果 CEO 蒂姆·库克在 Twitter 发表了一张参观 Mac Pro 生产线的图片：


工人前方的超薄 iMac 电脑上运行的并不是苹果自家的 OS X，从蓝绿相间的标签栏和任务栏，不难辨认出是 Windows XP 的经典界面。

严格意义上说来，XP 系统已经不再安全。苹果仍在生产、测试环节使用安全性不太能够得到保障的操作系统，可能因为这些测试机器和装配机器一般由专门的厂商提供，使用 Windows 系统应该是出于配套软件的需求。


今年 4 月 18 日，微软宣布将不再为 XP 操作系统提供免费的安全补丁、更新和其他服务。一些大批量使用 XP 服务却来不及升级系统的公司和组织，可以选择与微软签订一年的“延期服务“，比如英国政府就为此支付了约 920 万美元的“延期服务费用”。苹果是否与微软签订了此类协议还属未知。

而根据只有 XP 系统还支持的 IE 6 浏览器的使用数据显示，仍在使用 XP 系统的用户，超过三分之二来自中国。需要注意的是，这个数据仅能反映联网的电脑的 XP 份额，并不包括那些大量在生产线上服务着的，使用公司内网或是干脆不联网的 XP 前辈。
