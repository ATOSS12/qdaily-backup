13 岁的 Windows XP 会活过 Windows 10 吗？

当 Windows 8 、Windows 8.1 更新时，有多少人又默默地装回了老系统？从数字看还真不少。

尽管去年 4 月，微软已经停止更新 Windows XP，但根据  Net Applications 统计数据，全球市场操作系统市场份额中 Windows XP 占到 16.94% ，是第二大的市场。市场占有率最高的是 Windows 7 ，达到 58.04 ％，同样不是最新的 Windows 8.1。而这款最新的系统只占到 10.55％。另外，3 月刚刚发布的 Windows 10 预览版有 0.09％ 的用户。

还有这么多人用 Windows XP，作为微软的芯片供应商，英特尔也被坑了。 3 月英特尔调低了下个季度的收入预期，原因之一就是微软新系统吸引力不足，还有很多人守着旧电脑和 Windows XP 的组合。本来 PC 机的市场需求就被大屏手机和平板电脑挤占了一部分，新系统还不好用，消费者越发不想买了。

这部分 Windows XP 的坚实用户主要来自中国。Windows XP 在微软官方停止更新之后，国内不少做杀毒软件的公司跑出来为它撑腰：XP 的安全，我包了。于是，根据百度统计，你看到中国操作系统的市场份额是这样的。

Windows XP 还存在于近 50% 的电脑上，而大部分来自企业用户。某地方税务系统的工作人员告诉《好奇心日报》，“单位还是 XP 系统多，只有重要岗位才会配 Win 7 系统。” 这可能是普遍情况。

看来 Windows 10 能从 Windows 7 直接免费升级的推广手段，到了中国可能需要换一换。

这个夏天 Windows 10 就要来了。而终年 13 岁的 Windows XP 不知道还要诈尸多久？
