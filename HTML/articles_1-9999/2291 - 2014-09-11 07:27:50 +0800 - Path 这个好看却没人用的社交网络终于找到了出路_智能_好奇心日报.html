Path 这个好看却没人用的社交网络终于找到了出路_智能_好奇心日报

高宇馨2014-09-11 07:27:50

两年前如日中天的私密社交网络 Path 经历了痛苦的挣扎转型，今天传出的苹果收购对它是个尽头，对苹果是个开始。

Path 的故事听起来很传奇。如果没有在今天传出要被苹果收购的消息，也许会是一个比较悲伤的故事。

这是一个通过限制好友数量来营造小圈子社交环境的社交网络应用，Path 在 2.0 时代迎来它的巅峰，作为一家创业公司开发的小众应用，它在连续几个月里一直居于苹果商店下载量榜单的前五位，独特的设计在当时被当成是业界标杆。30 岁身材瘦削喜爱穿着厚呢子大衣的 CEO Dave Morin 因此成了硅谷科技产业的明星。就连 Nike 这样的大牌，也把自己的手环（Fuel Band）的运动信息接入 Path。

Dave Morin 与歌手 Dr.Dre 在昨天苹果发布会上

Path 一直将自己定位为一个私密社交网络，将“生活的瞬间与真正认识你的人分享”。最初对好友人数的限制是 50 人，到 Path 2.0 版放宽到 150 人。它擅长把已经成型的应用做得更加“小而美”。比如 1.0 版本像是令人眼前一亮的 Facebook，2.0 版本加入了“时间轴”，3.0版本加入表情商店，现在 4.0 版本专注于私信和语音信息，最新的版本还可以使用“附近”找餐厅。 

由于 Path 对准了一个细分的市场，又在易用性和设计上领先，在最初的一段时间，国内出现大量的跟风抄袭，比如时光流、点滴、登登、美刻、七彩等应用，仅苹果商店上与 Path 类似的应用就达到 10 款以上。

但是进入 2013 年， Path 忽然销声匿迹。国外专门做应用下载量分析的 App Annie 的数据显示，在社交网络应用里，它在美国市场下载量排名为 177，一段时间甚至掉出了 App Annie 上最受欢迎的前 1000 个 iOS 应用。 

Path 的根本问题或许还是因为大多数人不需要太多社交网络，在海外 Facebook、Twitter、Instagram 分别占据了日常、新闻和图片分享，国内则是微信、微博。而且庞大的 Facebook 和微信也都有针对小范围分享的设计。

当一个社交网络做不到主流，用户在上面找不到朋友，它就很难存活下去。或许这也是 Path 今年试图转向通信工具的原因。

今天传出苹果收购 Path 具体价格还没谈妥，但是根据消息源的透露已经进行到只差签字一步的程度。

苹果收购 Path 应该是为了优化社交功能，尽管 iOS 8 和 OSX 10.10 上也有了出色的 Messages 应用，但是苹果尝试做的社交网络一直都很失败，比如 2010 年推出的 Ping。另外，收购 Path 苹果也能够借 Path 在 Andrioid 平台上的资源，进行跨平台的操作。
