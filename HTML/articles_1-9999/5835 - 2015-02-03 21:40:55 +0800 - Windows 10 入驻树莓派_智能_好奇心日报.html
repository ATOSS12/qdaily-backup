Windows 10 入驻树莓派_智能_好奇心日报

王嘉俊2015-02-03 21:40:55

Windows 10 免费用？

树莓派今天发布了二代版本 Raspberry Pi 2，不仅性能增强了，而且可以免费用 Windows 10，但价格不变还是 35 美元。

树莓派是一款 Mini 电脑，它只有卡片式大小，但五脏俱全有很强的延展性。你可以基于树莓派编写程序，也可以通过外接硬件把它变成好玩的产品。下面是用树莓派做出的有趣样品。

各种版本的树莓派

微博机器人：通过外接温度传感器，定时发送温度监控结果到微博。
远程开门器：开门只需发送 HTTP 请求，就可以自动打开。
飞行器：可以用手机来控制航拍。
坦克：有了飞机不能少了坦克，理论上说可以装上小手枪吧。
超级计算机：类似乐高积木的方式拼凑而成，拥有 1TB 存储空间和 64 个处理器，造价 24000元以下。
树莓派+Kindle：用 Kindle 做显示屏，用树莓派做处理器，虽然没太多用处看着就很酷呢。
这些产品大多是用上一代版本做出来的，这次发布的 Raspberry Pi 2 处理能力要好上不少。它搭载了主频 900MHz 的四核 Cortex-A7 架构处理器，内置 1GB LPDDR2 内存，这样它的 CPU 单线程性能提高 1.5 倍，Sunspider 跑分提高 4 倍。比较直观的感受是上一代已经可以流畅运行《雷神之锤 III 竞技场》了，Raspberry Pi 2 肯定会有更强大的应用场景。

它还提供很多外接口，包括两个全尺寸的 USB 接口、一个 Micro USB 接口、一个 HDMI 接口、一个 3.5mm 音频接口和一个 RJ45 以太网接口。这样你可以通过接口联结其他的硬件，做一些有视频、音频、照相、网络功能的硬件产品。

树莓派一直以来就以促进学生计算机教育作为宗旨，由英国慈善组织“Raspberry Pi 基金会”创办。在 2012 年 3 月正式发布之后，树莓派取得了非常大的成功，不仅在学校而且也在整个社会掀起了计算机编程和开源硬件的潮流。过去树莓派只能使用基于 Linux 的系统，这对很多非计算机专业的学生来说有较大难度。Windows 10 的加入让树莓派变得更容易使用，这可能会让树莓派更加流行。

考虑到过去一  Pi 难求的情况，如果感兴趣的话最好抓紧购买。不过现在只能买到老版的系统，要用上 Windows 10 版本的 Raspberry Pi 2，还需要一些时间。
