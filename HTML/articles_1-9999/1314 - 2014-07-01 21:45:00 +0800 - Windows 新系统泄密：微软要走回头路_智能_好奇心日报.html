Windows 新系统泄密：微软要走回头路_智能_好奇心日报

唐云路2014-07-01 21:45:00

预计将于 2015 年春问世的 Windows Threshold 有望回到传统桌面，为键盘／鼠标和触屏用户分别设计操作界面，以挽回留守 Win 7、抵制 Win 8 的用户。

微软的下一代操作系统要来了，这个代号为 Threshold 的系统或许将吸引那些被 Windows 8 吓住的 Windows 7 的用户。

Windows 8 是微软近年来在用户界面上最大的变化，许多用户在升级 Windows 8 后非常不习惯没有开始菜单的界面，过度依赖手势的设计使得很多用户刚开始的时候甚至不知道如何关机，而在没有触摸屏的笔记本电脑上被强迫使用滚动磁帖界面实在是太糟糕的体验。许多用户表示将继续使用他们所熟悉的 Win 7 系统。而中国政府采购中心因安全问题宣布禁止政府 PC 使用 Win 8 系统也令这一状况雪上加霜。

据资深微软新闻记者 Mary Jo Foley 称，微软在 Threshold 的研发中将继续 Win 8.1 更新的脉络，放弃 Win 8 系统试图用一套操作界面贯通 PC、平板和智能手机界面的的思路，针对触控和键盘/鼠标用户的不同需求分别打造使用系统。或者可以说，Threshold 系统的最大任务就是纠正 Win 8 用一套通用系统强奸非触屏用户的错误，挽回那些习惯用键盘/鼠标办公的用户。

对使用台式电脑或是传统笔记本的用户，Threshold 将默认优先显示传统的 Win 32位界面，而对Surface 或是 Lenovo Yoga 之类的触控、键盘二合一设备，是 Metro 界面优先还是传统界面优先取决于设备是否连接了键盘。新的 Threshold Mobile 将为 Windows Phone 和平板电脑提供与传统 PC 完全不同的界面。

今年早些时候，微软宣布开始菜单将在新系统，而不是 Windows 8.2 更新中回归。新的开始菜单可以由用户自定义，用户可以选择显示 Metro 风格的应用或是将它们一个不留。

如果 Threshold 就如消息所称在明年春天问世，Windows 8  就将成为微软史上最短命的操作系统。在此之前， Threshold 的预览版本有望于今年秋季出现在公众面前。而在此之前 Windows 8.2 的更新将以低调的形式在近期推送给用户。

在告别 XP 以后，Windows 7 就成了微软的用户们最爱的操作系统，就像 Vista  出现以后，人们才发现对 XP 系统的真爱一般，微软似乎很擅长用牺牲自家新系统的方式激发用户对上一代操作系统的怀旧。这次可能走的回头路，对微软来说反而是进步，毕竟面对越来越多的选择，用户不仅可以选择抵制 Windows 8，留守 Windows 7 ，也可以选择彻底放弃 Windows，转投更好用的操作系统。
