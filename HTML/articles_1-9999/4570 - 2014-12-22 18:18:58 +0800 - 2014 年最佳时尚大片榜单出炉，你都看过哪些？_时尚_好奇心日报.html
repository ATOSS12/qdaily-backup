2014 年最佳时尚大片榜单出炉，你都看过哪些？_时尚_好奇心日报

吴亭蓉2014-12-22 18:18:58

从奢侈品大牌到设计师品牌都表现不俗，还有明星素颜照上榜。

在即将过去的 2014 年里，时尚品牌们除了忙活推出新品和奔波于几大时装周，为春秋两个系列推出广告片也是头等大事。让我们回顾这一年的两季广告，看看都有哪些值得年终再看一次的广告片。

2014 春季系列广告

Miu Miu 2014 春季大片找来了几位年轻明星，有《为奴十二年》里的 Lupita Nyong，这位获得了奥斯卡金像奖最佳女配角的黑人是目前好莱坞最亮眼的新星。还有 16 岁的美国青年女星 Elle Fanning、奥尔森三姐妹里最小的 Elizabeth Olsen 和澳大利亚女星 Bella Heathcote。而这种选择对于一个针对年轻消费者的奢侈品品牌来说再适合不过了。经过这次的广告后，也看出 Miuccia Prada 十分关注对新星的挖掘。

Alexander Wang 2014 春季大片由 Steven Klein 执镜，广告片里的 Anna Ewer 在学校的洗手间里尽现不羁和叛逆，而整个画面呈现出怀旧和冰冷感。而在这个月刚发布的 Alexander Wang 丹宁系列牛仔裤的广告片里，Steven Klein 也用同样的画面质感，但大尺度地展现了 Anna Ewer 的性感。

Balenciaga 2014 春季大片同样由 Steven Klein 执镜，拍出了 Daria Werbowy 如雕塑一般气场十足又高贵冷艳的广告片。

Marc Jacobs 2014 春季大片由摄影师 David Sims 用灰暗的沙滩场景打造出了世界边缘感。但片中的感觉和造型与 Miley Cyrus 一贯的风格并不同，不过却能让人过目不忘。

Mulberry 2014 春季大片 Tim Walker 用明亮和梦幻的风格打造出春意昂扬感。片中 Cara Delevingne 穿着素雅地出现在下午茶会上。

Alexander McQueen 2014 春季大片里 Steven Klein 镜头下的 Kate Moss 一头亮橙色的头发，和自己相同模样的娃娃一起出现。整个广告片呈现出一副暗黑和未来感。

Prada 2014 春季大片里模特们在 Steven Meisel 镜头下穿着各式的 Prada 最新服饰整齐地排列着，就犹如一副多彩的班级合照，

Celine 2014 春季大片也是多彩的视觉冲击，但和 Prada 的呈现方式不同，Celine 广告片更像是一副油画，而广告里的 Daria Werbowy 和 Julia Nobis 衣服上斑斓的色块就像不同的颜料。

2014 秋季系列广告

Dolce & Gabbana 2014 秋季大片由 Domenico Dolce 执镜，整个广告片里呈现出丛林里梦幻的童话故事的感觉，而镜头内出现的模特也多得数不过来。

Chanel 2014 秋季大片里老佛爷将时下最潮流的 Normcore 和运动风结合，他将广告片现场搬到了拳击场，而片中的 Cara Delevingne 和 Binx Walton 更是酷炫冷峻。

Givenchy 2014 秋季大片也是一次性在镜头里塞下几个超模，Mert 和 Marcus 将镜头放置在高位，从高处拍下不同姿态的模特们。广告片里无论是妆容、室内陈设还是画面质感都呈现出哥特风格。

Balenciaga 2014 秋季大片也依然出彩，延续了春季大片的冷艳感，Steven Klein 将大片调至黑白也运用了电脑技术将片中的 Gisele Bundchen 发型变为富有力量感的利索超短发。

Moschino 2014 秋季大片里 Steven Meisel 用黑白色调和模特们的端庄造型把 Jeremy Scott 快餐单品打造出了有格调的高雅感。

Lavin 2014 秋季大片里找来了 Edie Campbell 一家人，几代人一同出现在同一个广告片里，Tim Walker 用轻快的音乐和明亮的画面给广告片打来一种轻松的温馨感。

Sonia Rykiel 2014 秋季大片里 Gisele Bundchen 以素颜出境，Juergen Teller 用这种方式呈现出广告片的自然随性感。不过，光是明星素颜这一噱头，就足以吸引不少关注度了。
