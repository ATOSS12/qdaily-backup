"To Kill a Mockingbird" Sequel "Go Set a Watchman"

自完成《杀死一只知更鸟》之后，Harper Lee 似乎就在这个世界上消失了，以至于当 88 岁的她的新书 Go Set a Watchman 出来的时候，很多媒体都把这当成了「突发新闻」。不过即便如此，她也不愿意为了这本书公开露面。

以《杀死一只知更鸟》中一幕场景为灵感的插画

这本新书显然是《杀死一只知更鸟》的续篇，预计在今年七月出版，起印 200 万册，同时会有电子版。新书其实比《杀死一只知更鸟》开始得更早。《杀死一只知更鸟》为她赢得了普利策奖，还被改编为电影。

同样有新书露面的是作家米兰·昆德拉，这也是他在过去十多年内出版的唯一一本英文小说。
