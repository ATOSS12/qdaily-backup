一切都会过去的，这些包装都会降解的_设计_好奇心日报

胡莹2014-09-11 19:25:54

瑞典设计工作室 Tomorrow Machine 日前发布了“ This Too Shall Pass ”（一切都会过去）系列食品包装设计，使用天然可降解的材质作为食物包装，不仅对生态环境毫无危害，还可以辨识食物的新鲜程度。

吃完东西后，总是会留下许多食品包装，只能作为垃圾处理，交给大自然去慢慢“消化”，这过程中造成的环境污染难以避免。为此，瑞典设计工作室 Tomorrow Machine 提出了一个创新的食品包装概念，不仅对生态毫无危害，还可以帮助用户辨识食物的新鲜程度。

让食物的外包装与内里的食物以共生的方式共享保存期限，这样的设计是不是很大胆？对，Tomorrow Machine 日前发布的“ This Too Shall Pass ”（一切都会过去）系列食品包装就做到了这一点。

“ This Too Shall Pass ”主要使用天然可降解的材质作为食物包装，简约的外形搭配以明亮的色彩，实用、美观。

Tomorrow Machine 的创始人之一 Anna Glansén 告诉《好奇心日报》，这款食物包装的设计灵感来自于仿生学，她们想利用大自然本身来解决环保的问题，于是联想到了水果的各种果皮以及鸡蛋壳。

Anna Glansén 说：“这种即时食品包装在运输过程中可以压缩起来用以节省空间，它是一种以生物作为基本原料的食品包装，或是可以 100% 生物降解的材料。当用户把热水倒进这种包装时，它会产生热反应，从压缩包装变为一个碗状的包装。这是最新一代的可持续包装设计，所使用的材料都是最环保的。”

当然，除了环保之外，“一切都会过去”还会告诉你食物的新鲜程度，这种与食物共生的外包装材料使得消费者不用打开包装，通过外观就可以判断食物的新鲜程度，这样就不会有食品的保质期到了还不知道的烦恼了。

在启动“一切都会过去”项目时，Tomorrow Machine 的核心问题是找到合适的包装材料。Anna Glansén 告诉《好奇心日报》，为了找到一款合适的材料，大量测试是必须的──看看不同的材料在一起会起什么反应。

目前，“一切都会过去”系列共有 3 款不同的设计。

“橄榄油”包装

“橄榄油”包装是将涂上蜂蜡的焦糖作为包装材质，因为蜂蜡可以有效防止液体流出，而加热到 100° 与 170° 的焦糖包装会呈现不同的颜色，用以区分不同品种的油。最最重要的是，焦糖本身不会和油相融，使用时只要像敲鸡蛋一般那样，就可以将里面的油轻松取出。

Anna Glansén 表示：“焦糖是包装油的理想材料，为了防止焦糖包装与空气中的水分发生反应，我们为它覆盖了一层薄薄的蜂蜡。而在包装被打开的那一刻，焦糖将不再受到蜂蜡的保护，慢慢地接触到水和空气，就会自动开始降解，直至几天后消失不见。”

“覆盆子冰沙”包装

“覆盆子冰沙”包装所用的材质则是洋菜胶。这是一款饮料包装，用来盛放果汁、牛奶或者冰沙等饮品，洋菜胶具有半透明的特点，可以透过它看到内部液体的颜色。Tomorrow Machine 经过测试发现，在一般室温环境下，存放一个月左右的洋菜胶会渐渐缩小，也有助于提醒消费者要赶快饮用它。谈到这款包装时，Anna Glansén 对《好奇心日报》说：“这款包装的寿命很短，不管是包装新鲜果汁还是牛奶都需要冷藏，在运输过程中还应避免其暴露在阳光下。”

“印度香米”包装

“印度香米”包装被设计为一座小山的形状，渐变的颜色给人以富士山的感觉。外包装同样选择了蜂蜡作为原料，质感就像是一层薄薄的果皮，使用时也只需像剥掉橘子皮一样将包装剥掉即可。Anna Glansén 表示：“谷物包装通常都很薄很软，所以制作起来也很不容易。”此外，她还提到，这款包装也同样适用于面粉与糖。

对于想要减少家庭垃圾的人来说，“ This Too Shall Pass ”回归食品本身，用最简单自然的方式将其包装设计。Anna Glansén 在接受《好奇心日报》采访时表示：“这系列食品包装目前还不会大规模应用于食品包装行业，但我们相信不远的将来它有成为主流包装设计的可能性，因为技术上已经可以完全实现。”
