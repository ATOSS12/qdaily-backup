微软反攻手机市场，乐观真是这公司的可再生资源_智能_好奇心日报

Nick Wingfield2015-01-19 23:35:47

微软想用 Windows 10 重振自己的手机业务

本文由《纽约时报》授权《好奇心日报》发布，即使我们允许了也不许转载。

乐观在微软是一种可再生资源。

2011 年，为了在智能手机领域赶上 Google 和苹果，微软终于迈出了迟来的一步，宣布与芬兰手机制造商诺基亚建立合作关系。诺基亚和微软高管在联合公开声明里说：“我们将颠覆其他移动生态系统。”

去年，在销量萎靡不振的情况下，微软以 70 亿美元收购了诺基亚的手机业务，计划以此加强硬件和软件之间的融合，进而推动销量增长。

尽管如此，销售并没有就此好转。2014 年第三季度，微软手机在全球市场的份额不到 3% 。大多数分析人士已不再抱任何希望，长期以来，他们一直质疑微软的手机战略。也许最为严重的是，许多手机应用开发商极少或者根本没有关注 Windows Phone，而他们的支持或者忽视往往决定着移动技术的成败。

这暗示着乐观主义的出现。

上个月，萨帝亚•纳德拉（Satya Nadella）第一次以微软首席执行官的身份出席了股东大会。

周三，微软计划在西雅图总部召开发布会，公布新一代个人电脑操作系统 Windows 10 。微软表示，新系统会帮助赢回手机开发商，因为他们能更容易地将电脑应用程序移植到移动设备上。微软也希望新系统能提高应用程序的实用性，以推动智能手机的销售。

这一举动是萨帝亚•纳德拉为拯救微软日益衰落的智能手机业务，迄今为止所做的最大努力。大约一年前，他开始担任微软首席执行官。到目前为止，纳德拉已经让微软的移动产品整体的实用性变得更强了。去年，他发布了适用于苹果 iOS 和 Google 安卓设备的办公软件，同时他还为几乎所有设备的开发人员改进了开发工具。

这些策略推动微软股票升到了 15 年来的最高值。本•汤普森（BenThompson）说，他“长时间以来第一次对微软的策略感到满意。” 汤普森目前是一名独立技术分析师，曾经帮助管理过微软和 Windows 开发商之间的关系。

不过他说，微软在提升其移动平台重要性上的尝试仍然面临着挑战。“微软无法做到这一点，这要取决于开发商。从这个角度来看，我认为希望不大，” 汤普森说。

在本周召开发布会前，微软拒绝安排任何高管的采访。但显然不代表微软在放弃努力。

“我们正在稳步推进，我们的愿景是让用户可以在所有设备上都能体验到 Windows 系统，”微软发言人弗兰克•肖在一份声明中说，“在 Windows 10 系统下，我们将实现手机、平板和电脑上 Windows 平台的进一步统一，让开发商有更大的发挥空间。”

微软进一步表示，其应用商店上的应用已经超过了 56 万个，大部分都是在智能手机上使用的，较上年同期增长了 60%。（苹果和 Google 表示，它们的应用商店上有超过 100 万个应用程序。）同时，微软还说，它提供了人们想要在智能手机上使用的应用程序中的 85%－95% 。Netflix 、Facebook 、Spotify 和其他第一梯队的应用都能在 Windows Phone 上使用。

然而，微软公布的这些数据中并没有包含其他一些流行的应用程序。如果你搜索 Lyft 、HBO Go 或者 Snapchat，你不会找到对应的 Windows Phone 版本，你也不可能找到慢跑爱好者颇为青睐的手机应用 RunKeeper 。

“挑战在于安卓和 iOS 系统革新的速度太快了。我认为，要不是在这两个主要移动平台上耗费了太多功夫，我们也可能会考虑其他平台。” Redfin 的首席执行官格伦•凯尔曼（Glenn Kelman）说。Redfin 是一家互联网房产中介商，开发了苹果和安卓手机上的应用程序，但是目前还没有 Windows Phone 版。

应用不是人们买智能手机的唯一原因，但却在其中起了很大的作用。而设备制造商和无线运营商对软件的支持不足，则是他们避免选择某些型号手机的原因。移动业务发展迅速的中国设备制造商华为也说，由于销售不畅，他们未来将停止生产 Windows Phone 。

三星制造过一些运行 Windows Phone 系统的手机，但微软和它的关系同样也遇到了麻烦，一切都因为微软收购了诺基亚。在去年底三星和微软之间的一场诉讼中，三星指称，对诺基亚的收购在微软的利益之间引起了冲突，从而不再利于三星推广 Windows Phone 。

“除非微软不再和它自己的生态系统自相竞争，否则 Windows Phone 这个平台不会有任何发展。”独立移动产业分析师奇坦·沙马（Chetan Sharma）说。

微软在某些国家有着很大的优势。根据市场调研公司凯度移动消费者指数（Kantar Worldpanel ComTech）统计，在意大利，Windows Phone 在去年 11 月的智能手机销量中占到了 12.7%，而 iPhone 的份额是 17% 。但根据凯度的数据，美国的 Windows Phone 销量只占总量的 3%，而 iPhone 则占到了 47%。

低迷的销量数据，意味着 Windows Phone 很难被纳入创业公司的应用开发计划，因为它们的资源特别有限。为 iPhone 和 iPad 制作了受人欢迎的天气应用的Dark Sky，它的联合创始人亚当·格罗斯曼（Adam Grossman）在邮件里说：“这种状况肯定让微软很恼火，而且它也有理由恼火。”

微软一直以来都承认需要扩展它的应用选择范围。它也已经提出要资助优秀的开发商开发 Windows Phone 应用，资助方式包括了在一些情况下，把编程工作外包出去。透露这一信息的是一位前微软高管，由于内容涉密，他要求匿名。

但即便这样也不够，一位业界顶级的手游开发商说，微软在一年前放弃了让他的公司支持 Windows Phone 的努力。“我们需要一个实实在在的市场，并且要有全球范围内比较大的装机量，才能让为 Windows Phone 做开发这件事显得值得。”这位高管不愿公开自己的名字，以维护自己和微软的关系。

最受欢迎的手机游戏之一 Minecraft 在 12 月推出了 Windows Phone 版，而它在其他平台上已经运行了好几年。更何况，这是在微软斥资 25 亿美元买下该游戏的开发商以后才发生的事情。

另外，一些 Windows Phone 上的重要应用要么未经打磨，要么更新得不如其他平台勤。Facebook 旗下的照片分享应用 Instagram 一年多以前才推出 Windows Phone 版，尽管比其他平台晚了好多，但它还是没有录制视频的功能，而这是 Instagram 将近两年前推出的功能。

“我觉得就目前的情况来看，新的手机操作系统已经没有机会了，”风险投资公司 Andreessen Horowitz 的合伙人本尼迪克特·伊凡斯（Benedict Evans）说。

但这并没有挡住立陶宛开发商 Game Insight 的 CEO 阿那托利·罗波托夫（Anatoly Ropotov）的脚步，他所在的公司为 Windows Phone 和 Windows PC 两个平台开发游戏。罗波托夫在邮件里说，Windows Phone 会帮助他的公司接触到东南亚和其他一些新兴市场中的消费者。

“许多应用发行商正在意识到，在这个平台上有钱赚，”他说。

劝其他开发商改变态度，似乎是场艰苦的斗争。但有一个领域可以帮助 Windows Phone 翻盘，那就是企业应用。这是微软的传统强项，因为 Windows 依然统治着企业用户。云存储公司 Box 负责产品和平台的副总裁 Chris Yeh 说，无论微软在硅谷是否过时，他的公司都会为 PC 端和手机端开发 Windows 10 的应用。

“他们在公众认知、特别是移动业务的公众认知上吃了大亏，”他说。“但只要你离开个人消费领域、进入任何一家美国中西部或者美国以外的公司，你就会发现，拥有数十万员工的公司用的都是 Windows 的桌面操作系统。这就是现实。”
