兰蔻小黑瓶修复基因？遗传学家笑了_商业_好奇心日报

陈思吟2014-07-08 21:35:00

联邦贸易委员会（ FTC ）指控欧莱雅广告涉嫌欺诈。但欧莱雅集团坚称自家产品安全、有效，并强调如果你觉得没有效果那一定是用的时间还不够。

这年头护肤品牌的广告越来越有点神神叨叨了。“你值得拥有”的巴黎欧莱雅绝对是个中翘楚。最登峰造极的要属它们为 Youth Code 系列护肤品设计的广告文案：

“十年基因研究，独创 PRO-GEN 科技，激活肌底，一滴修护，肤质改善，定格青春！”，末了还不忘加注一句：“产品效果均通过了临床验证”。

广告播出之后，遗传学家们首先笑了——化妆品公司竟然比他们还抢先一步研究出了基因的秘密。然后，大家都笑了——美国联邦贸易委员会（ FTC ）指控欧莱雅广告涉嫌欺诈，要求欧莱雅不得在兰蔻 Genifiqu 系列（就是“小黑瓶”）与巴黎欧莱雅 Youth Code 系列广告中再次宣传所谓的“影响基因”功效。但在随后发布的声明中，欧莱雅集团坚称自家产品安全、有效，并强调如果你觉得没有效果那一定是用的时间还不够。

卷入风波的巴黎欧莱雅 Youth Code 精华肌底液在国内售价 350 元/ 50 ml ， 50ml 兰蔻 Genifiqu 精华肌底液售价则达到了 1080 元。后者更已成为欧莱雅集团下最具知名度的单品之一。

欧莱雅援引了两篇学术论文试图证明产品的科学性，但有科学家指出欧莱雅试图通过胶带剥离、紫外线照射与清洁肌肤等方式来验证基因是否被激活，但两篇研究都无法证明乳液能够激活基因，或者能令肌肤保持年轻状态。美国联邦贸易委员会也不认同欧莱雅的声明，指出“兰蔻 Genifiqu 系列在广告中宣称产品经过临床验证，仅仅使用一周， 82% 的女性肤质大为改善， 85% 的女性肌肤变得透亮， 91% 的女性肌肤变得柔软。这样的广告语充满错误且具有误导性。”

这已经不是欧莱雅集团第一次被指控广告欺诈，以下这串列表或许可以让你更好理解护肤品广告和它实际功效之间的差距：

1.  2007 年 7 月，英国广告标准管理局禁播了欧莱雅睫毛膏广告，广告中宣称使用睫毛膏后睫毛能增长 60% ，但后来欧莱雅承认在广告中为模特佩戴了假睫毛。

2.  2008 年 8 月，英国广告标准管理局禁播欧莱雅除皱日霜与纤体产品，广告说辞被认为涉嫌虚假夸大产品功效。

3.  2010 年 12 月，瑞典市场法院指控欧莱雅旗下品牌兰蔻与薇姿的除皱产品在广告中误导消费者。

4.  2011 年 8 月，英国广告标准管理局禁止播出兰蔻与美宝莲的两支粉底广告，英国广告标准管理局认为两支通过后期技术美化了模特的肤质，可能会误导消费者。

5.  2013 年 12 月，兰蔻微整形系列在华突然下架，被认为与广告中不实宣传立体塑颜效果有关。

6.  2014 年 3 月，意大利反垄断监管机构认为巴黎欧莱雅 REVITALIFT 系列涉嫌虚假宣传，应当删除产品效果与美容手术一致的内容。

或许有一天护肤品真的能够达到美容手术般的效果或是可以改变基因，但至少不是现在。
