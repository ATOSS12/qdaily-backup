Airware：一个想要做无人机操作系统的初创公司_智能_好奇心日报

赖江锟2015-04-23 16:06:55

用遥控器、手机应用控制飞行器？弱爆了。

无人飞行器不仅可以成为酷毙的玩具，还能在专业领域替代人工，例如高空拍摄、能源勘探、地质监测等。但当那些公司买回无人飞行器后可能会傻眼，怎么用好它？全程派人手动操控听起来效率低下。近日，美国创业公司 Airware 发布了一套商用的无人飞行器的系统 Aerial Information Platform (AIP)，让公司们正确地、系统化地使用无人飞行器。

在 AIP 的五个组成部分中，Flight Core 盒子是整个系统的大脑，它运行有 Airware 的自动飞行软件。将 Flight Core 安装到飞行器上，就能实现对飞行的控制与数据的搜集等功能。

Flight Core

在地面上，有两套不同的系统 (仅支持 Windows 系统的电脑或平板) 可以用来指挥飞行任务。

基本指挥系统 Ground Control Station 能让你建立飞行计划、设置地理围栏、以及突发情况应对措施等。高阶的 Configuration Manager 系统可以让你创建更大、更复杂的飞行设置，对 Ground Control Station 中的选项进行组合和添加，然后将最终的指令发送给 Ground Control Station。

飞行器采集到的数据会被传送到云服务 Airware Cloud 中，可以在你的公司内部进行分享。云服务还会帮忙检测飞行路线是否符合规定，公司可以放心地计划。

此外，为开发者准备的 App Core 是一个 API，允许他们为 Airware 系统开发应用。

目前，Airware 已有两个合作伙伴，通信公司 Infinigy 和通用电气 (GE)。Infinigy 希望在通信塔检修时，用无人飞行器替代人工会更加安全，在 2013 年曾有 13 人死于检修过程。GE 虽然没有说明他们将如何使用 Airware 的系统，但已经与工具厂商展开合作，利用无人飞行器检查线路和设施。

想要面向商用市场的 Airware 系统，单个飞行器需缴纳 2500 美元的年费。虽然不是免费，但并不影响 Airware 吸纳新用户，因为 2500 美元会远低于雇佣人工，而在目前的市场上，造硬件的公司多，做软件的少。
