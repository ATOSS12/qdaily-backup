# 什么是共享房产计划？我可不愿和房东用一把牙刷。

别担心，房东是不会半夜爬到你床上的。共享房产就是允许你事先买入房价的一股，这一股的金额可以根据你的经济状况来协商，在房价的 25%－75% 之间，接受贷款，剩余的费用用房租来还。

# 既然还是要付房租，我何必要买房呢？

这样的话，每个月你支付的房租就可以当作你对此处房产未来的投资了。

# 谁会跟我共享产权？

房屋协会，他们是非盈利的房东，主要提供一些廉租房。共享房产的目的就是为了让更多来自不同背景的工薪阶层能够拥有一套自己的房产。

# 共享产权计划现在有什么新的优惠消息吗？

位于伦敦西南部的房屋协会泰晤士谷（ Thames Valley Housing ）最近推出了升级版。你可以根据自己的能力，在第一次购买之后小幅度提高持有房产的比例。在一般的共享产权计划中，如果你想增加持股比例的话，每次最少额外购进 10% ，并且是以那个月的房屋市价为标准，而不是你买第一股时的房价，这中间的溢价会是很惊人的。泰晤士谷委托剑桥大学做了一份调查显示，在过去的共享产权计划中，只有 3%－5% 的人选择日后增加持股，因而他们认为这个升级的版本会提高人们的积极性。

# 这听上去不错，具体怎么操作？

参与升级版共享产权计划的用户，第一次购买股份时与房东签订的协议中允许他们每年可以增加 1% 的持股比例，而这个 1% 的价格参照的是初始房价而不是房屋市价。
