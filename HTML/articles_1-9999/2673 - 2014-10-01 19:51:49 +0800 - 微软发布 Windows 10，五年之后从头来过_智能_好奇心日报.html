李硕明2014-10-01 19:51:49

除了奇怪的名字，Windows 10 还带来了什么？

今天凌晨，微软在旧金山召开了下一代 Windows 操作系统的发布会。出人意料的是，大家等来的不是 Windows 9 或者 Windows TH 之类此前泄露的名字，而是 Windows 10。

对于新名字，微软的解释是 Windows 10 是一个重大飞跃。用微软 Windows 业务负责人 Terry Myerson 在发布会的话上说“等你看到完整的产品，就会觉得这是最恰当的名字。”

那么 Windows 10 究竟带来了什么？

回到桌面时代

在今年  Net Applications 发布的 8 月数据报告中显示，Windows 8/8.1 两者加起来的份额 13.37% ，而 Windows 7 和 Windows XP 份额分别达到了 51.21% , 23.89%。

微软将键盘/鼠标操作和触控体验强行统一在一套用户界面下的尝试无疑是失败的。此次 Windows 10 的一个重点便是纠正 Windows 8 的错误。

  • 开始菜单回归

全新的开始菜单左侧部分和之前的 Windows 7 版本大同小异。整体效果更加扁平化，左侧是应用列表，左侧顶部是登录账号信息和关机菜单，左侧底部是搜索框，右侧是类似开始屏幕的应用磁贴。

所不同的地方在于开始菜单的右侧。你可以将应用拖到这里变成磁贴，整体效果类似于之前的 Windows 8 的开始屏幕。

你可以在屏幕底部的任务栏设置中选择具体形态，究竟是选择开始菜单还是满屏色块的开始屏幕，这次决定的权利终于到了用户的手里。

  • Modern 应用窗口化运行

Windows 8 的全屏应用对于桌面用户来说实在是不太友好，在大多数桌面用户仍然使用非触屏的笔记本电脑时，强制全屏显示应用降低了用户的使用效率。虽然 Windows 8.1 中对此作出了一些改进，可以把 Modern 应用固定到任务栏并且可以在任务栏启动变得更像桌面应用，但是仍然没有解决实质问题。

在 Windows 10 中微软已经让这些应用可以窗口化运行了，看起来和一个桌面应用几乎没什么差别。

  • 两套用户界面

那么在触屏笔记本电脑、平板和手机上呢？开始菜单并不适合。

Windows 10 将会同时覆盖这些设备，但它会专门提供一个用户界面，看上去非常接近 Windows 8 的开始屏幕。

基本上，Windows 10 是同时提供了两个交互界面，以适应不同设备。

新功能

除了解决之前系统中存在的老问题，微软这次也为 Windows 10 带来了一些新功能。

  • 多任务管理

Windows 10 新增的多任务管理界面 Task View 功能类似苹果 OS X 操作系统上的 Expose 界面，一键将所有应用窗口的缩略图集合在一起。

虚拟桌面整体效果和 OS X 面类似，单击任务栏左侧第二个按钮就可以打开虚拟桌面，用户也可以向左滑出进入虚拟桌面，同时管理界面也包括当前正在运行的应用。

此外 Windows 10 也完善了虚拟桌面支持，可以更容易地在多个桌面之间拖放窗口。

  • 更好的企业支持

Office 和云计算的主要客户都是企业用户。此次 Windows 10 发布会的重点也被放在企业支持。Windows 10 将会隔离个人信息与企业数据，方便公司管理。

发布会上微软甚至演示了命令行，解决了困扰开发者多年的 Ctrl+V 无法直接在命令行中粘贴文字的问题。

  • 统一应用

Windows 8 和 Windows Phone 智能手机操作系统的一个为人诟病之处是它们的应用并不通用。即便是平板设备，下载的应用也与手机不同，不像 iPad 和 iPhone。

而开发者也需要针对不同 Windows 设备开发不同的应用，并维持多个版本。面对 Windows Phone 个位数百分比的份额，开发者更没动力再去开发一个单独的应用。

微软承诺说 Windows 10 将统一手机、平板到键盘 PC 的应用——同一个商店、点一次下载。

但诸如同一个应用如何在适配不同设备的同时保持一致体验之类的细节问题，需要等到明年 4 月的微软 BUILD 开发者大会才能揭晓。

接下来呢？

Windows 10 发布会提供的信息颇为有限。微软没有公布 Windows 10 在手机上的用户界面，没有告知新系统售价，也没有发布早先传言的消息通知系统、桌面版 Cortana 语音助手、Wi-Fi 感知等新功能。

按照计划，微软将在 2015 年下半年正式推出 Windows 10。不过本周开始，你就可以去申请提前参与 Windows 10 的技术预览测试。目前的测试版仅限于桌面版本，功能也不完全。用微软的话说，这是为知道 DLL、BIOS 究竟是什么的人而准备。

从目前提供的信息来看，微软推翻了 Windows 8 的尝试，选择在 Windows 7 的交互逻辑上加入一些 Windows 8 的元素，比如干净、扁平的设计和动态磁贴。

从 2009 到 2014 的 5 年时间里，iPhone 从一个小众玩物变成一个周末卖出 1000 万部的热门商品、开辟了上百万应用的生态系统和新的商业模式，OS X 也在今年发布的 10.10 版本上与 iOS 建立了深度整合。

Google 的 Android 从一款手机到今天最大的移动操作系统，覆盖超过十亿用户。

与此同时，微软花了 5 年时间，兜了一圈，从 Windows 7 走到了一个更好的 Windows 7。

哦，不对，它的版本号赶上了 OS X。
