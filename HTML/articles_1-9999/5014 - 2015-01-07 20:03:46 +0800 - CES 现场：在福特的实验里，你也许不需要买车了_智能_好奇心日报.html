CES 现场：在福特的实验里，你也许不需要买车了_智能_好奇心日报

唐云路2015-01-07 20:03:46

汽车公司已经不再满足于卖卖车而已

福特 CEO Mark Fields 在拉斯维加斯消费电子展（CES）上宣布了接下来的 Smart Mobility 计划。这项计划包含在全球 25 个城市开展关于新型驾驶方式的实验，其中大部分与汽车共享有关。汽车共享、识别停车位、动态社会班车等实验项目，看起来都不太像是一家以汽车销售为主的公司会做的事。

Mark Fields 在发布会上表示，福特“希望成为一个产品和移动性并重的公司，并改变这个世界移动的方式”，许多汽车厂商都在试图提供比制造汽车更广泛的交通解决方案。

或许可以这样说，从推出 Sync 智能车载系统的时候起，福特就没有准备停留在“卖车的”公司。在本届 CES 上，福特推出了基于黑莓 QNX 系统重新打造的 Sync 平台，与此同时，福特正在研发的还有无人驾驶汽车和一系列与驾驶相关的自动化应用。 Mark Fields 的路线图不仅包括智能汽车，还有智能道路以及智能城市。

比如在福特总公司所在地密歇根，福特正在进行一项 Big Data Drive 的测试，接受测试的汽车将被装上一个“黑盒子”用来实时收集行驶数据，包括行驶路线、交通状况等。这些数据也在福特的开源平台上实时共享。

在伦敦，福特将部属类似租车服务的车辆测试，人们可以按使用时长付费，租用福特提供的测试车辆。

而在纽约测试的则是“动态社会班车”的模式，人们可以通过应用预约班车或者公共汽车接载，与公共汽车不同，这些“动态班车”可能是由私人驾驶，乘客在预约的应用里也可以对他们进行评价，这看起来就像是巴士版的“拼车”，这与此前微软在西雅图地区推出的私人教练（private coach services）服务也有些相似。

在伦敦和印度，福特正在开展令多人共享一辆车的测试，这也就意味着，并不是每个人都需要购买一部车。奔驰与 Car2Go 在此前已经推出过类似的项目。

在美国密歇根州的迪尔伯恩市，福特在测试一项汽车交换项目，用户可以通过 Car Swap 寻找空闲的、符合需要的车。在你家的车不能满足临时需求的时候，你可以通过这个系统借用别人的车。

这些测试并不是福特的首创，其中的一些已经不少公司已经做到相当的规模，典型的就是 2014 年动作频频的 Uber。对于这些测试的结果，Mark Fields 在发布会上表示并不指望每一个测试都能够指向一个创新解决方案，“我们只知道，总有一些会有用。”

不过对于收入基本来自卖车的福特来说，这些实验真的能够进化成减少汽车使用的流行服务么？
