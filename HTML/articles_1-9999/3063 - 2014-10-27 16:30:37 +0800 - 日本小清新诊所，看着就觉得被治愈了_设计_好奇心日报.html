日本小清新诊所，看着就觉得被治愈了_设计_好奇心日报

胡莹2014-10-27 16:30:37

日本设计工作室 id 日前打造了一家满眼薄荷绿的诊所设计，给人放松而柔软的感觉，等待治疗的心情也轻松许多呢。

你去过的绝大多数诊所长什么样？你心目中理想的诊所又会是什么样？先来看看日本设计工作室 id 最新完工的一个诊所设计作品——住吉堂针灸院（ SUMIYOSHIDO Kampo Lounge ），单单是这颜色是不是已给人瞬间被治愈的感觉？

住吉堂针灸院位于日本爱知县半田市，是一家专门面向愿意接受中草药与传统中医针灸疗法的病人的诊所，但它的店内布置却不似我们刻板印象中的中医馆。

原本满墙的放置各类中草药的抽屉和诊所墙面都被刷成了清新的薄荷绿。设计师认为：“那些一格一格的小抽屉储存的不仅仅是中草药，我们认为这些抽屉格盛满的更是中草药的一种历史文化积淀。”

基于此，id 工作室在这些抽屉格的设计上颇费心思，在一排排整齐堆叠的抽屉格之外，它们还专门设计了用来摆放中草药样品的小格子，便于医生与病患查看药品的口感与气味。暴露在储存空间之外的中药样品罐，也如装饰物一般使整个搁架变得立体了起来。

为了赋予整个内部空间一个相对统一和谐的表达方式，id 工作室在灯具、桌子以及沙发的设计上均做了统一处理，使之与抽屉格保持一致的调性，薄荷绿的着色是为了表达中药绝大部分来源于天然的植物的意思。这种柔和的颜色与轻盈的灯光、沉重的家具共处一室时，创造出一种对比强烈却又异常和谐静谧的氛围。

与外部空间不同的是，内部较为私密的用来针灸的空间，被设计师定义为纯净的白色，厚厚的褶皱窗帘意图让病人进入到这个空间内能够完全放松下来。

外部门诊区域的外立面被设计为大片的落地窗与透明的玻璃门形式，自然光线得以轻松涌入的同时，也成为了一个大型的展示窗口。此外，设计师还对入口处的等候区做了特别的处理，保证了病人在就诊前能够有充足的休息空间，也使得各个空间的功能更为专一，避免了就诊病人被打扰的烦恼。

了解 id 工作室的朋友或许会记得他们去年的作品东京 Ki Cafe 店面设计，“ Ki ”在日语中是“树”的意思，因此 id 工作室就索性把它变成了一座小森林，原本的钢质桌腿往上延伸并长出“枝桠”，漆成黑色后，在整体白色的背景下就像一片树丛，造出清新且有灵气的空间。再将视线转回到住吉堂针灸院的设计，不由得感慨 id 工作室还真是打算将小清新的风格进行到底呢。
