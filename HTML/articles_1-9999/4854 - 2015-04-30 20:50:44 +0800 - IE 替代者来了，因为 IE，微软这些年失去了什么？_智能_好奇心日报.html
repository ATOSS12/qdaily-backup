IE 替代者来了，因为 IE，微软这些年失去了什么？_智能_好奇心日报

崔绮雯 唐怡园2015-04-30 20:50:44

新浏览器的名字叫 Microsoft Edge

IE 终于要彻底地走了。微软新浏览器叫做 Microsoft Edge，此前它的内部代号是斯巴达人计划，来自 Xbox 游戏 Halo。

微软 Build 2015 开发者大会上，操作系统团队的副总裁 Joe Belfiore 称微软要让消费与创意结合起来。

Edge 里内嵌了 Cortana 智能语音助手。此外，为了更好的平板体验，你可以直接在网页上涂涂画画、做标注。

为适应新环境，Edge 用了新的同名渲染引擎。作为一个从头开始的浏览器，它将直接兼容 Chrome 和 Firefox 上的丰富插件。

尽管 Edge 是 IE 的接替者，但鉴于许多企业还保留着多年前开发的，只能用于 IE 的在线应用，微软允许新用户在升级时保留旧的 IE 浏览器。过时的 IE 至今依然在给微软添麻烦。

19 年前诞生的 IE 是微软开拓互联网服务的标志，但之后因为 IE 相关的决策，微软也错过了不只一次技术变革。

究竟 IE 为微软带来过什么，它的中途停滞和懈怠，又让微软失去了怎样的互联网及移动互联网机会？

IE 的兴起以及网景公司的浏览器大战

1995 年 12 月是微软公司的一个转折点。比尔·盖茨（Bill Gates）宣布微软互联网服务进军，召集了大批工程师停下手中的工作，投入 IE 浏览器的研发。

当时，互联网刚刚兴起，网景公司（Netscape）凭借自家的浏览器“网景导航者”在一年之内挂牌上市，网景浏览器产品也卖出了几百万份。还在做操作系统软件的微软，已经在互联网服务上已经慢了一步。在尝试收购网景公司失败之后，比尔·盖茨决定认真地研发一个浏览器 Internet Explorer。为了争夺市场份额，微软使出了狠招，让 IE 浏览器和 Windows 操作系统捆绑销售，这相当于将浏览器价格直接拉到零。

1997 年，尽管当时 IE 的市场占有率仅为 18%，网景占据了 72%，但当微软Windows 95 以及将要推出的 Windows 98 预装 IE，以及 IE 升级到较稳定的 4.0 版本之后，网景公司逐步失去价格和软件实力的优势，不断降价。1998 年，IE 的市场份额就已经超过了网景浏览器，成为市面上使用最多的网页浏览器，而光靠卖浏览器软件而生存的网景公司，则被美国在线（AOL）收购，从此隐没。

IE 6.0 是最广泛使用，也是“空窗”最久的产品

在打败网景之后，IE 基本上成为了最受欢迎的互联网浏览器，微软也成功把控住了互联网入口。1999 年，微软正式推出了 IE 5.0 版本的浏览器。两年之后的 2001 年，IE 6.0 和操作系统 Windows XP 一起推出。但在其后的 5 年之间是 IE 的“空窗期”，微软并没有改进这款浏览器并让其发挥更重要的作用，只是在原有版本上进行了一些弹窗广告屏蔽等修复优化性质的更新，真正将 IE 变成一个微软 Windows 操作系统里的一个内置功能。

从数据上看，2005 年是 IE 最为风光的一年：IE 4.0-6.0 版本的浏览器一共占据了 90% 的市场份额，垄断了整个网页浏览器市场。直到 2006 年 Windows Vista 发布，IE 才更新了 7.0 版本，到现在更新到 IE 11.0 版本。但讽刺的是，后续微软无论怎样升级，“统治”时间最长的 IE 6.0 依然是目前被最广泛使用的浏览器版本。也正是这 5 年，让 Google Chrome、Firefox 等浏览器的发展有了可乘之机。

那长达 5 年的空窗期，微软参与了一系列互联网服务竞争，但只有一个特别成功的案例：

1. 与 Google 竞争，推出 3 个搜索引擎，最后都成了 Bing

为了和 Google 的搜索引擎竞争互联网入口，微软曾经先后推出过 3 个搜索引擎。

2005 年 1 月，微软完成了自己第一个搜索引擎的开发，产品名为 MSN Search，网址为“search.msn.com”。当时，MSN Search 的搜索相关性匹配非常糟糕，难以凭借产品追上 Google 的步伐。

2006 年 3 月，微软又推出了一个全新的搜索引擎，名为“Windows Live Search”。时隔不久，微软在搜索中抛弃了 Windows 的品牌，所以这个搜索引擎后来改名为“Live Search”。

尽管数量众多，但微软却未能通过内置搜索引擎等复制当时对网景公司一役的成功。其中一个原因是，Google 当时当时已经和多家 PC 制造商谈拢了预装 Google 的工具条的合作方式，这里面就带有 Google 的搜索等方面的小工具服务。

2009 年，微软把三个搜索引擎都融合成了 Bing。此时 Google 已经推出比 IE 好用太多的 Chrome 浏览器，默认使用 Google 搜索。目前，Bing 的市场份额还不到 20%（依然在蚕食雅虎搜索的份额），但时隔多年，依然无法撼动 Google 的地位。

2. 和索尼、任天堂抢游戏市场，推出 Xbox Live

2001 年微软公司推出的 Xbox 游戏机，开始进入一直由索尼公司和任天堂两家公司主导的游戏机市场。而 Xbox Live 则是 Xbox 系列产品的在线对战平台。这个平台还可以供玩家进行沟通交流，以及付费下载电影和电视节目地方。目前，这个平台算是微软互联网服务当中算是较为成功的一个，目前在全球有 4600 万用户。尽管他们的负责人，也称“Xbox 创新之父”的 Boyd Multerer 在昨天已经确认离开团队了。

3.  Window Live 系列整合，名字改来改去也没有活下来

这也许是最让人摸不着头脑的整合和改名。2001 年 10 月 23 日，微软宣布 MSN Messenger Service 改名为 MSN Messenger。2005 年 12 月 23 日，Windows Live Messenger 8.0 beta 1 正式发布，MSN Messenger 正式改名为 Windows Live Messenger。这个一度流行起来的 IM 工具，包括 Windows Live 这个品牌都在 2012 年之后取消或者服务关闭。

其他大公司当时都在做什么？

反观之，微软最重要的两个竞争对手 Google 和苹果都在这个时间段占领了各自的高地：

Google：摸索出适用至今的广告产品，推出更广泛的互联网服务

2001-2006 这 5 年恰好是 Google 发展的重要时机，他们不仅仅是一个搜索引擎而已，和微软多种尝试无果不同，Google 成为了全球最大的互联网服务提供商。我们把 Google 当时做过的重要尝试也列出来了：

  • 2003 年， 发布了 Google AdSense 广告工具，让用户可以在自己所有的网站上嵌入 Google 广告；
  • 2004 年 4 月，愚人节推出 Gmail，当日收获 4.25 亿个用户，Gmail 账号也成为 Google 未来多个网络服务的通用账号；
  • 2005 年，Google Maps 诞生。6 月，移动端的 Google Mobile Web Search 推出，Google 搜索引擎从网页端走向移动端；
  • 2006 年，更广泛的互联网服务推出：Google 日历（Calendar）、Google 翻译（Translate）、Google Checkout（为移动支付产品 Google Wallet 铺路的产品），网页端文件应用 Doc&Spreadsheets（现在 Google Docs 产品的原型）。
  • 2006 年 10 月，收购 YouTube，这个视频网站为 Google 带来了持续接近 10 年的海量流量和用户。

苹果：iTunes 接入数字消费

在微软和 Google 正打得火热之际，苹果安然享用 iTunes 和 iPod 带来的巨大成功。尤其是 iTunes 以音乐商店起始的服务由于使用及其方便，彻底改变了互联网音乐盗版下载风向，改变了音乐行业对互联网的成见。到 2006 年 2 月，iTunes 已经累计售出了 10 亿首音乐，成为全球最大的网络音乐分发商之一。

iTunes 培养的数字消费、以及与设备相连接的下载模式最终被用于 iPhone 和 App Store，带来了比 PC 更大的移动市场。

微软早该放弃 IE 了，至少数据也这么说

到现在为止，微软要放弃“IE”品牌这个决定也许不错。一个新的 Spartan 品牌可以洗脱本来 IE 给人以“二十世纪初的 PC 端网页浏览器”的印象，而数据也证实了 IE 浏览器的江河日下：根据 Net Applications 的数据，IE 浏览器市场占有率已经从高峰期的 90% 下降到了 58%，在移动端，IE 的占有率更加惨，只有 2%。

（数据来自 Net Applications）
