Windows 10 的新功能：自动安装游戏糖果传奇_游戏_好奇心日报

赵长存2015-05-16 16:35:00

你能接受系统预装游戏这种事吗？

打开国产 Android 手机的应用商店，几乎都会见到醒目的一键下载所有软件、所有游戏的醒目提示。你会为这种引导式安装不快么？现在微软要做得更过分一些。

微软近日在官方博客上宣布，热门游戏糖果传奇 （ Candy Crush Saga） 将登陆未来的 Windows 10 及其手机版上。不仅如此，这款游戏还会在用户升级和初次安装新版系统时自动进行安装。看来这一最新版的 Windows 系统还没有正式发行，微软已经迫不及待的想强加给它更多的卖点。

Candy Crush Saga 是由英国游戏公司 King 开发的一款宝石消除类的策略游戏。除了玩法特殊，它还通过 Facebook 和腾讯移动游戏平台建立起了广泛的口碑。甚至在 2014 年，成为了 Google Play——Android 的官方市场中下载量最高的游戏。而在 App Store 上这款游戏也常年保持在智力游戏排行榜的前 3 位。不过直到 2014 年末，Windows Phone 的用户才能在自己的手机上看到它。

多年来，Windows Store 中应用增长一直缓慢。微软也试图通过向开发者发放奖励，试图给 Windows 系统增加更多的应用和游戏。但看上去收获甚微。到了 2014 年末微软的应用商店中 Windows Phone 的应用数量仅为 30 余万，苹果的 iOS 系统和 Google 的 Android 系统却早已拥有超过了百万的应用。而且优秀的作品对进入 Windows Phone 系统表现的并不积极，比如糖果传奇就是如此。

此前对诺基亚的收购也没能给 Windows Phone 带来多大的帮助。在这个软件吞噬世界的时代，用户还是会看中平台对软件的支持能力。

为了给 Windows 10 带来更多应用，微软早先也宣布要帮助 iOS 和 Android 的开发者，快速将现有应用移植到 Windows 平台。此次微软和 King 的合作恐怕也是为了吸引开发者加入。不过为了开发者去惹用户，真的好么？
