Google 如何召集精英“黑客”，且“不作恶”_智能_好奇心日报

崔绮雯2014-07-17 16:25:00

Google 最近的新项目叫做 Project Zero，就是召集了世界一流的黑客们来找 Bug！

Google 最新曝光的项目 Project Zero 告诉我们，怎样召集精英“黑客”且“不作恶”。
根据《连线》的报道，Project Zero 的目的，是赶在那些利用各大热门软件漏洞作恶的人们之前找到漏洞，并且将他们消灭。
“人们使用互联网时不应该有这样的担忧：害怕黑客甚至是政府利用软件漏洞感染个人电脑并且监视你的举动。” Google 网络安全团队的负责人 Chris Evan 表示，Project Zero 将会给集合精英的黑客们寻找漏洞，发现后，就会把 Bug 告知应用开发商，让他们有 2-3 个月时间进行修复，这个过程会在保密状态。修复之后，Google 就会在 Project Zero 的项目页面上公开此漏洞。
Project Zero 的其中一名成员是年轻的 90 后黑客 George Hotz。他在 2007 年成为第一个攻破 iPhone AT&T 网络的年轻人，却被苹果无视，后来他还找到了索尼的 PlayStation 3 的漏洞，却被索尼告上法庭。
而当 George Hotz 找到了 Chrome OS 的漏洞时，Google 却给他 15 万美元的奖励，感谢他帮忙找到漏洞，并且希望他帮助修复漏洞。不久后，Google 邀请 George Hotz 加入公司的精英黑客小组 Project Zero。
无论 Project Zero 将来是否会履行他们所说的职责，但无可置疑的是，Project Zero已经变成了 Google 的最好的公关项目，他们也抓住机会公开招募精英黑客人才。

“我们相信，大部分安全研究人员都热爱这项事业，我们会提供一个新的场所，让你们公开并且专注地从事你所爱的事业。”
