微软又给产品改名了，来看“改名大户”的辉煌历史_智能_好奇心日报

李硕明2014-09-15 16:53:05

改改改，为什么产品名字都要变来变去？微软说他们有很多要说的。

根据 GeekOnGadgets 拿到的一份内部文件显示，微软计划让 Lumia 成为旗下的旗舰品牌。文档提到微软将在市场宣传材料中淡化 Windows Phone 和 Nokia ，统一用 Windows 和 Microsoft 代替。

实际上这个消息对于微软来说并不意外。在最近的 Lumia 930 One Windows 宣传视频中只是用 Windows 替代 Windows Phone ，而更为明显的是 HTC One M8 for Windows 的名字中并没有出现 Windows Phone。

实际上这早就不是微软头一次给旗下产品改名了。微软对于更换产品名称似乎有着特别的兴趣，于是我们汇总了一些 Windows 改名的黑历史材料：

改名是为了重启服务

2001 年 10 月 23 日，微软宣布 MSN Messenger Service 改名为 MSN Messenger。2005 年 12 月 23 日，Windows Live Messenger 8.0 beta 1 正式发布，标志着原有的 MSN Messenger 正式改名为 Windows Live Messenger。

但就在 2012 年，微软直接取消了 Windows Live 这个品牌，原有的 Windows Live 产品再次全体改名，团队的新名称为 Winodws Serivce 团队。对于 Windows Live 品牌取消的原因，微软企业副总裁 Chris Jones 承认 Windows Live 并没有达到微软对于联机体验的期望。

在 2012 年 9 月 1 日，微软对部分 Windows Essenitials 软件进行了改名，Windows Live Mail 将更名为 Windows Mail Desktop（最早 Mail 软件是从 Mail Desktop 改名而来的）；Windows Live Writer 将更名为 Windows Writer。就连 Windows Live ID 也被改名为 Microsoft ID、

类似的情况也发生在微软的其它服务上，比如 Hotmail 以 Outlook.com 的名字重新启动。在 Reddit 有问必答中，IE 团队就曾经透露计划通过改名摆脱 IE 的臭名声。

有时候改名也是不得已

不过有时候产品改名也不是微软自己愿意的，比如 Skydrive 改成了 Onedrive、Metro UI 改成了 Microsoft Design UI，主要原因是侵权。

2013 年 7 月，英国法院裁定微软旗下的 Skydrive 侵犯了英国 Sky 广播的商标 Sky，结果导致微软在全球范围内用 Onedrive 替换 Skydrive 。

Metro UI 也是类似的命运。微软在 Windows 8 发布时采用的设计语言 Metro UI，因为和德国公司麦德龙（ Metro AG）的商标过于相似，不得不放弃 Metro UI 这个名称。

但更多是内部斗争的体现

更多的改名是微软内部权力倾斜的结果。比如 SkyDrive 的前身最早属于叫 Live Mesh，是微软首席架构师 Ray Ozzie 推动的云计算业务的一部分。在 2009 年的大规模改组和裁员中，Live Mesh 所在业务部被并入 Windows 与 Windows Live 部门，这个部门完全由 Windows 所主导，Live Mesh 改名为 Windows Live Mesh，Ray Ozzie 离职。

不过风水轮流转，今年随着微软云计算业务的崛起、Windows 8 的惨败，Windows 不再是微软最重要的部门，原本名为 Windows Azure 的微软云计算业务直接更名为 Microsoft Azure。

改名这件事不是只有微软一家会做，但是要说科技巨头中把旗下产品改名最频繁的，恐怕也只有微软了。
