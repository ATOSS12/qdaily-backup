特斯拉发布新车 Model 3，但要 2017 年才能开_智能_好奇心日报

马若飞 2016-04-01 21:34:52

吸引预定一辆面都没见过的车？特斯拉做到了

特斯拉酝酿已久的低价车型 Model 3 今天正式发布。

发布会上，CEO 伊隆·马斯克在回顾了公司过去的三款车型之后，正式展示了新车 Model 3。而在发布会之前，已经有部分没有见过车的粉丝就已经掏钱预定了。

Model 3 的整体风格和 Model S 类似，都是 fastback 风格的四门轿车，但尺寸小一些。前脸设计与 Model S/X 都有较大不同，车头完全封闭，连过去假装传统汽车进气格栅的部分也没有，初看的感觉略显怪异。

Model 3 将使用铝合钢材制造，比 Model S 更加轻量化。自动驾驶功能，选装的四驱电机也同时具备。Model 3 也将具有与 Model X 类似的全景挡风玻璃与玻璃车顶。

马斯克称，新车的百公里加速仅需 6 秒，续航里程不小于 215 英里，约等于 346 公里。新车将于 2017 年底交货，售价 35000 美元，今天开始预定。

然后，发布会就在现场观众的欢呼声中结束了。就像马斯克之前在 twitter 上暗示的，今天的发布会只是一部分内容。更多细节或许会在 Model 3 接近量产时再宣布。

不得不说，特斯拉“期货”这一套营销方式相当吸引眼球。之前发布的 Model X 车型也是在很长之间之后才交货，如今的新车 Model 3 展示的细节更少，相信 Model 3 现在还处于较为早期的开发阶段。

结合特斯拉过去的定价规则，Model 3 在中国的售价将在 32 万人民币左右。因为国内现在对电动汽车有政策偏重，特斯拉原本的吸引力也有了进一步加强，比如牌照和限行方面的优惠。

不过，因为 Model 3 的交货日子还早，这期间如果电动汽车政策有变，或者油电混合动力或者燃料电池汽车取得更大的进步，电动汽车的“技术光环”如果不能持续，特斯拉的“酷”的优势也就不复存在。

这种危机对于国内的电动汽车厂商也是同样存在的。某些国内厂商粗制滥造的电动汽车借着政策的东风大谈所谓“弯道超车”，技术更先进的混合动力反而得不到补贴。

不过纯电动汽车的补贴未来将大幅收紧，再加上特斯拉这样产品力更强的“明星”正面竞争，国内这些不思进取的电动汽车厂商的好日子要到头了。
