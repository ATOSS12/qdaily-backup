特斯拉的低价新车明天才来，但这里有一份剧透_智能_好奇心日报

唐怡园 2016-03-31 23:40:18

多亏特斯拉 CEO 伊隆·马斯克自己在 Twitter 上实力推广。

苹果升级了 iPhone 5s，iPhone SE 正在拉拢一批 Android 手机用户。而曾经被称作汽车界苹果的特斯拉也要推出“迷你 Model S”了。

北京时间 4 月 1 日上午 11 点半，特斯拉会在位于加州的一家特斯拉体验店里发布新车 Model 3。

今天，位于悉尼和墨尔本的特斯拉体验店门前已经排起了接近百人的长队。澳大利亚是 Model 3 首批开放预订的国家。预订用户可以获得 1500 美元优惠，但税后 Model 3 在澳洲可能卖到 6 万美元。

这些连夜排队的特斯拉车迷显然更想抢先看到车。不过，直到现在，Model 3 谍照不是 PS 的，就是蒙着黑布的。


和往常一样，特斯拉汽车的重要消息都是其 CEO 伊隆·马斯克在 Twitter 上或者各种采访中自己透露的。关于 Model 3 ，他都说了些什么？以及还有哪些其他剧透？

Model 3 长什么样？

“会比 Model S 小 20%。”、“外形一定很酷，但没那么怪，肯定是辆汽车。” 2014 年 6 月 Model 3 当时还叫 Model E 。被福特起诉侵权之后，换了名字，却一直活在马斯克对特斯拉未来的描摹中。

Model S 可以算是中型轿跑，车长为 4.98 米。按照该参数计算，Model 3 会是一辆车长在 4 米左右的小型车，也就是紧凑型轿车。马斯克在公开采访中谈到 Model 3 时，也将其车内空间与 5 座三厢车奥迪 A4 并论。

至于外形，接近特斯拉工程和设计团队的人向科技媒体 Electreck 透露，为了减少对电池的耗损，Model 3 的风阻力系数想要接近 0.2。这是介于跑车和赛车之间的水平。一般赛车风阻力系数为 0.15 。分别以轮胎覆盖设计和流线型车身线条著名的大众 XL 1 和通用 EV1 当作参考。

特斯拉汽车首席设计师 Franz von Holzhausen 却在接受《连线》采访时表示，“如果你想把一款车推向全球消费大众，它就不能让消费者感到困扰，甚至是被吓到。总之，造型经典很重要。” 看来和官方海报暗示的一样，Model 3 可能会和 Model S 长得差不多。

这辆车性价比怎么样？

“3.5 万美元，这是价格。”去年 9 月，马斯克就告诉你了。然而，最近 Model S 可能也要涨价了，涨幅在 5 万元左右。如果新车价格确定，其国内税后价在 20 万人民币左右，价格和将在同一时间发布的比亚迪秦纯电动版 EV 300 接近。

马斯克承诺 Model 3 单次充电续航里程约为 321 千米（ 200 英里）。2017 年才能量产的通用纯电动车 Bolt 采用了 64.6 千瓦时的锂电池，达到了同样的续航里程。为了在竞争对手中脱颖而出，Cairn 能源研究咨询的分析 Sam Jaffe 认为，特斯拉 Model 3 会搭载 65 千瓦时锂电池，续航里程达到 365 千米（225 英里）。

2500 美元的自动驾驶功能 AutoPilot 、 6000 美元的 Ludicrous 加速模式，还有同样需要花钱升级的电池包，也会出现 Model 3 的配置清单可选项中么？特斯拉 CTO JB Straubel 说得比较含糊，“我们为 Model 3 设计了新的汽车底盘，新的电池结构，新的发动机技术。”言下之意，如果 Model 3 有这些功能，也都是不一样的版本。

Model 3 没有创始人系列签名限量版。说实话，一款主打高性价的汽车，也没必要学价格在百万左右的豪车系列 Model X 和 Model S。

不只有 Model 3？

发布会前夕，马斯克依然活跃在 Twitter 上。他说，明天的发布会只是 Model 3 的第一部分。第二部分是什么？网友@Ross Bishop 就问了，“我看不到完整的 Model 3 了么？” 马斯克回应，你可以看到整车，重要组成部分却需要日后升级。

我们很难知道“重要组成部分”是什么？但之前有传言称，Model Y 不是说笑的，而可能作为 Model 3 的变体一起出现在发布会上。但会不会多一款车，以及究竟怎么变？就要等明天的发布会了。
