特斯拉还要出新车，回归敞篷跑车_智能_好奇心日报

马若飞 2016-05-19 03:40:31

Model 3 之后的又一辆“期货”吗？

Model 3 还没影呢，特斯拉就说要出新车了，这次是一辆跑车。

和之前的报道差不多，特斯拉汽车北欧地区主管 Peter Bardenfleth-Hansen 确认，特斯拉会在 2019 年推出下一代 Tesla Roadster 敞篷跑车。

特斯拉的第一辆产品就是敞篷跑车。2008 年，刚诞生不久的特斯拉在 Lotus Elise 双门跑车的基础上研发了 Tesla Roadster 双座电动跑车，理论续航里程为 320km。

从 2008 年面世到 2012 年停产，一共卖出了 2250 辆左右。当时的售价是 109000 美元起。Tesla Roadster 也是世界上第一款量产的电动跑车。

关于特斯拉的新跑车，Peter Bardenfleth-Hansen 表示它会更大、更快，当然也不再是像第一代一样在别人家车的基础上修改，而是基于特斯拉 Model 3 设计制造。

第一代 Roadster 的性能为最大 270 匹马力，400 牛·米 的扭矩。这个数字和现在的 Model S/X 没法相比，Model S P90D 车型能提供 762 匹马力和 713 牛·米的可怕性能。特斯拉新跑车的性能一定比现在的车型更强。

特斯拉 Model S 和 X 都有一个单独的“狂暴模式”，英文叫“Ludicrous”，直译就是“滑稽”模式。去年，伊隆·马斯克曾说以后还会有更夸张的加速模式，名字同样来自恶搞《星球大战》系列的喜剧电影《太空炮弹》（Spaceball），名字或许会是“Maximum Plaid”。
