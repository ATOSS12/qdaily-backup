特斯拉 Model 3 已经订了 27 万台，出货能跟得上？_智能_好奇心日报

马若飞 2016-04-05 22:04:31

特斯拉至今所有的汽车产量还不及新车订单的一半。

经常有批评者抨击小米等厂商卖“期货”手机是“耍猴”，可有另外一群消费者甘愿买特斯拉的“期货”。特斯拉 Model 3 电动轿车开始预售 3 天后，预订量已经超过了 27 万 6 千辆。

姑且按照美国 1000 美元定金来计算，Model 3 就已经给特斯拉带来了超过 2.7 亿美元的现金，订单总额则超过了百亿美元。

在特斯拉之前，汽车基本上都是一锤子买卖。若非遇到严重缺陷，销售之后的用户体验基本上和厂商关系不大。但特斯拉用科技公司的思路让汽车变成了类似消费电子的产品。

和特斯拉自建超级充电网络相比，无线升级汽车系统都不算是大手笔了。伊龙·马斯克在 Model 3 的发布会上说，2018 年特斯拉计划在全球建造 7200 个超级充电站。

这些超级充电站是特斯拉的一项豪赌：用配套设施打消人们对电动汽车的续航焦虑，但首先要花掉一大笔钱。

汽车行业从来都是门槛极高巨无霸级别的产业。初来乍到的特斯拉产品惊艳，却面临着产能的问题。特斯拉目前在卖的 Model X 就是如此。

特斯拉昨天宣布，今年第一季度一共交付了 14820 辆车，其中 Model S 12420 辆，Model X 2400辆，比一月份预期的 16000 辆要少。特斯拉表示，是 Model X 复杂的技术拖累了产能，但依然强调了 Model X 今年 8 万到 9 万辆的预期交付数量。

特斯拉从 2008 年建立到今日的 7 年间，共生产了大约 12 万辆汽车。

虽然产能一直在爬坡，但横亘在公司面前的是五年内高达 27 万辆的 Model 3 订单。这对特斯拉来说肯定是痛并快乐着，未来的两年内，公司必须找到办法提升产能，以避免 2020 年用户依然拿不到 Model 3 愤而退款的尴尬。

按照特斯拉的说法，Model S 和 X 消耗掉了全球的锂离子电池的产能，于是他们建造了以电池产能“千兆”为名的特斯拉超级电池工厂（Gigafactory）。

另一方面，特斯拉现在位于加州的总装工厂依然处在产能爬坡的阶段。这家工厂在 2010 年之前属于丰田和通用，原来的产能能达到每年 50 万辆，但特斯拉现在的产量让人对 Model 3 的交付预期充满了担心。
