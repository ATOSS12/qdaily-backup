家居杂志做展览，就应该用这些场景给人更多灵感_设计_好奇心日报

胡莹 2016-02-16 21:33:55

你可能会觉得，这些场景与我们的实际生活存在差距，但这种陈列方式营造出来的氛围能够带给我们的启发，是家居用品、装饰品在材质、颜色、功能等方面的搭配。

Lotta Agaton 是来自瑞典的顶尖室内装饰造型师，十多年她与 Hanna Nova Beatrice 创办了家居杂志《居住》（Residence），后来又离开杂志成为了一名自由职业者，也曾加入过瑞典版的 ELLE Interior 杂志。

很多人说，Lotta Agaton 在室内装饰方面的才能是与生俱来的，似乎她随手挑的东西都很百搭，经她手装饰的空间自然、温馨，简直温暖到心坎里。

最近，Lotta Agaton 联合《居住》杂志在斯德哥尔摩家具展上策划了一场与家居生活相关的展览，选用瑞典本土设计团队 Note Design Studio 的产品在 ArkDes 建筑博物馆内搭建了一系列家居场景，细细琢磨你还能发现 Lotta Agaton 装饰手法与 Note 团队的设计风格颇为契合。

她为 7 个方形盒子空间赋予了不同色彩的主题，“我们对 Note Design Studio 在色彩方面的运用很熟悉，这一次的展览也希望借由色彩语言传递他们作品的意义。”

Note Design Studio 是《居住》杂志 2015 「年度设计师」赢家，用评委的话说，这个成长于斯德歌尔摩的团队很擅于用设计与消费者沟通，告诉其设计在居家生活中的重要性。

“在今天这个时代，不管是设计师还是设计公司，在推出新家具产品时，几乎不可能不为其设计一个卖相好的搭配场景或是样片，” Lotta Agaton 表示。“搭配场景变得越来越重要，甚至已经成为设计师创作过程的一部分，单件家具是不完整的，除非将其置于整体家居场景中才会有效果。”

Lotta Agaton 的这一想法与英国设计师 Jasper Morrison 的观点有几分相似，他在今年的斯德哥尔摩家具展上告诉 Dezeen ，“有太多家具品牌不明白好的图片好的造型的重要性，也不知道找一个家居造型师对于提升产品形象的好处。”

Lotta Agaton 认为，家居产品的消费方式近年来变化很大，社交网络上的作为已经影响到品牌如何销售。

比如为了展示 Note 为 Zero 品牌设计的 Silo 橙色版本吊灯，Lotta Agaton 将其放置在一个灰褐色基调的餐厅场景中，用简洁的木制餐桌搭配 Note 最新推出的 Arch 搁架，既醒目又融洽。

又如这个纯白色的卧室场景，明显单品则是 Plateau 储存支架，以及为 Örsjö Belysning 品牌设计的 Vinge 台灯。

而在黑色主题的电视房，一把为瑞典品牌 Fogia 设计的 Sling 休闲椅颇为抢眼，搭配 Bouroullec 兄弟去年为三星设计的衬线电视，更是提升了整体空间的格调。

同样是为 Fogia 设计的 Rise 沙发，则出现在蓝色主题的起居室内。

而为西班牙品牌 Sancal 设计的 Tonella 椅子则是褐色主题空间的焦点。

你可能会觉得，这些场景与我们的实际生活毕竟存在差距，但这种陈列方式营造出来的氛围能够带给我们的启发，是家居用品、装饰品在材质、颜色、功能等方面的搭配。
