猫眼电影从美团独立，估值 83 亿

19:40 更新

今天下午，猫眼公司宣布完成分拆，并引入新投资者进行战略重组。新投资者是光线传媒，光线将以股权交换、现金等方式，获得猫眼电影 57.4%的股权，新美大持股比例降到 32.6%。

根据估算，猫眼电影估值在 83 亿左右。此前，微影时代和淘宝电影（现已改名为“淘票票”）均已宣布完成融资，微影时代估值为 20 亿美元，淘宝电影估值为 137 亿元。根据双方协议，新美大可以向光线传媒提名一名董事。

苹果想要买下媒体公司时代华纳？《金融时报》援引知情人消息称，苹果正在考虑收购媒体公司时代华纳。但谈判还停留在早期，苹果 CEO 蒂姆·库克还未参与其中。不过，去年，苹果互联网软件和服务高级副总裁 Eddy Cue 已经和时代华纳企业战略主管 Olaf Olafsson 聊过一次了。

甲骨文和 Google 的官司打了两年，最后 Google 赢了。本周四，法院二审判定 Java API 不具有著作权，Google 暂时摆脱了甲骨文的纠缠，顺便重提了 Andorid 开发的开放性。甲骨文表示还会继续上诉。最初，为了让 Android 系统兼容 Java，Google 大概用了 1.1 万行代码，包括 37 个 Java API。甲骨文收购 Java 创造者 Sun 公司之后不久，对 Google 提起诉讼。这次也不是结束，甲骨文早已表示将会继续上诉。

在法国，麦当劳可能要交更多的税。最近，法国总统奥朗德发起的一项提案：跨国公司要缴纳更多的税款，麦当劳因此陷入了财务审查，它被怀疑故意算少了在法国的营收，以此来避税。根据法国本土一家杂志 L’Expansion 预测，麦当劳预计会交 3 亿欧元（约 22 亿人民币）的税和罚款。

2017 年，星巴克要在上海开第一家烘焙咖啡工坊。周四下午，星巴克宣布将在上海即将建成的兴业太古汇开一家烘焙咖啡工坊（Reserve Roastery & Tasting Room）。这是星巴克头一次将烘焙咖啡工坊开到海外。2700 平方米的规划总建筑面积，也将使得上海新店超过今年 4 月刚宣布建成的纽约门店，成为全球最大的星巴克咖啡馆。

滴滴出行要把内置地图换成高德的？凤凰科技拿到知情人的消息称，滴滴出行和高德成立联合工作组，在推进地图切换和对接等问题。昨天，阿里巴巴发布的 2015 年财报中，阿里巴巴和蚂蚁金服承诺将分别认购滴滴出行 2 亿美元优先股。此前，腾讯地图团队一直在和滴滴优化服务。
