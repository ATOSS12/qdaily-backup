为了卖东西给企业，微软又要支持竞争对手了_智能_好奇心日报

唐怡园 2016-03-08 21:24:38

Windows 在微软内部的地位真是越来越低

据《纽约时报》报道，本周一，微软宣布自己的数据库服务器软件（Microsoft SQL Server）将支持开源操作系统 Linux 。

以往，公司的服务器只有装了 Windows 系统，才能使用微软的数据库服务器和相关管理软件。现在，微软去支持竞争对手了。

市场调研机构 Gartner 数据显示，数据库管理软件市场上，占有 43％ 市场份额的甲骨文，是全球最大的数据软件公司。其数据库服务器支持的正是 Linux 系统。微软则以 21.4 ％的市场份额位居第二。

2014 年 7 月，纳德拉在关于公司转型的公开信中就曾明确表示过：“我们的核心战略是，在移动为先、云为先的世界里，成为一家生产力和平台公司。”

这意味着，Windows 操作系统不再是最重要的部分。事实上，随着 Windows 手机的失败、个人市场 Mac 的崛起和企业服务器市场 Linux 的普及，Windows 也不再是软件销量的保证。

就拿服务器这门生意说，2011 年到 2014 年 3 年时间里，Linux 系统服务器出货量增长了 120 万台，销量达到 360 万台。 Window 系统服务器出货量则从 650 万台跌到了 630 万台。

前者增长势头猛，后者却出现下滑。那如果只支持 Windows，反倒会影响服务器和软件的销量。

同样的情况也出现在微软的其它生意上。

于是，你看到苹果 2015 年秋季发布会上，微软负责 Office 部门的副总裁 Kirk Konigsbauer 亲自上台演示了如何在 iPad  Pro 上使用 Office 办公软件。事实上，早在 2014 年 11 月，Office for iPad 已经推出。

为了吸引更多的开发者使用，微软的云计算服务 Microsoft Azure 和开发者工具 Visual Studio 也都支持 Linux。

跨平台，已经成为微软销售 Office 办公软件和云计算生意的新策略。现在，服务器生意也是一样。
