《星战》背后的特效公司找了个中国股东，不光是因为钱_娱乐_好奇心日报

韩方航 俞斯译2016-01-13 03:17:22

在 Base FX 参与过的项目中有《星球大战》、《捉妖记》、《美国队长 2》、《变形金刚 4》等。

《星球大战：原力觉醒》和《捉妖记》分别是去年北美和内地的票房冠军，进入 2016 年后，这个“冠军”效应还在继续：Base FX，这家参与了这两部大片特效制作的中国特效公司，拿到了一笔新的投资。

出钱的是这两年在电影行业颇为活跃的华人文化产业基金（CMC）。Base FX 公司   CEO Chris Bremble 告诉《好奇心日报》，这是 BaseFX  成立 10 年来第一次引入外部投资者。

Base FX 这家公司对于大部分影迷来说可能有些陌生，与工业光魔这些老牌的特效公司相比，成立于 2006 年的 Base FX 算是行业里的后起之秀。从 2010 年开始，Base FX 凭借《太平洋战争》（2010）、《大西洋帝国》（2011）、《黑帆》（2014）几部美剧连续获得艾美奖的杰出视觉效果奖，算是为自己在好莱坞打响了名声。

这些成功的作品也帮助 Base FX 吸引到了行业里的老大哥工业光魔的注意，Base FX 与工业光魔以及卢卡斯影业在 2012 年签订了战略合作协议。这份协议不仅帮助 Base FX 参与到了更多好莱坞大片的后期制作当中，同时也成为了促成这一次投资的重要砝码。

Bremble 告诉《好奇心日报》，华人文化和 Base FX 之间的谈判从 2014 年上海电影节之后就开始了，“整个过程推进得很快，2015 年初就达成了初步协议。”随后，谈判的重点就放在了 Base FX 和工业光魔、卢卡斯影业、以及迪士尼的关系上－－简单说，华人文化希望这项合作可以延长，保证 Base FX 在未来几年的发展前景。在 Base FX 和工业光魔的战略合作延长了 5 年之后，这笔投资协议也就正式达成了。

投资 Base FX，华人文化看中的特效制作在如今的电影行业，尤其是内地电影行业越来越受到重视的趋势，而 Base FX 是为数不多在内地市场（《白银帝国》、《金陵十三钗》、《捉妖记》等）与好莱坞都“吃得开”的公司，自然成为了最理想的投资标的之一。

“在《捉妖记》成功后，已经有超过 30 家投资机构与我们接触，我们会在数月后和这些机构商谈 B 轮融资的事宜。不过此时此刻，我们很高兴自己能够进入华人文化的大家庭。”Bremble 告诉《好奇心日报》。

对于 Base FX 这样的特效公司来说，钱或许还是第二位的，最重要的是能不断接到高质量的电影项目，不断有客户找上门来。而华人文化此前投资的那些电影公司们，正好可以满足这一需求，这或许也是 Base FX 最终选择华人文化做为投资方的原因。

在投资 Base FX 之前，华人文化分别和梦工厂动画组建了合资公司东方梦工厂，发布了自有品牌微鲸电视。过去一年还和 IMAX 成立了一支影视投资基金、和华纳兄弟一起组建了旗舰影业。显然，这些公司对于电影特效的需求正在变得越来越大，投资 Base FX 同时也可以用于自己旗下公司的项目的后期制作。

在拿到了来自于华人文化的投资之后，Bremble 希望 Base FX 可以扩大产能，同时开发更多的原创内容。“Base FX 也在开发自己的影视作品，有了这一次的投资，Base 就能够把技术用在自己的内容之上。” Bremble 说。

在 Base FX 和华人文化已公布的计划中，他们将会合作拍摄一部灾难片以及一部基于中国传统历史故事和人物的系列魔幻影片，但影片的具体内容和上映时间仍然处于保密之中。
