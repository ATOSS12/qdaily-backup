特斯拉 3 月 31 日召开新车发布会，谍照同时曝光_智能_好奇心日报

马若飞 2016-03-18 21:48:28

买得起的特斯拉，还不摇号。

日前，特斯拉汽车向部分车主俱乐部成员发送了一封邀请函，邀请函显示，特斯拉将于 3 月 31 日召开新品发布会。而特斯拉汽车的官方推特发布的预告图显示，发布会的主角就是传闻已久的 Model 3。

特斯拉 Model 3 的消息很早就已经公布。相比在售的 X 与 S，Model 3 将会是一辆定位更低，价格更亲民的全新电动轿车，售价据推测大约在 35000 美元，这将与宝马 3 系，奔驰 C 级，奥迪 A4 正面竞争。

在递出发布会预告的档口，有媒体却收到了一张据称是特斯拉 Model 3 的效果图。

这张照片是一张明显从另外一台电脑屏幕上盗摄的渲染图，而且看起来不大正常。这张据称是特斯拉员工发来的“谍照”看整体轮轮廓看起来和 Model X 非常相似，既不是传统三厢轿车，又不太像是 SUV。对比发现，与 Model X 的媒体图非常相似，疑似是基于 Model X 图的 PS 作品。

这个发谍照的人为了证明自己的身份，还发了一份 800 多人的特斯拉汽车核心员工目录。但特斯拉汽车的发言人 Khobi Brooklyn 却表示，这个图不是 Model 3，看起来像是 PS 的。

特斯拉汽车在国内备受关注。当然了，老板马斯克的个人魅力吸引了很多目光，但作为最新兴和最激进的汽车公司，特斯拉已经发布的 Model S 和 Model X 大幅更新大家对量产汽车的认识。

同时，在中国限制汽车牌照的一线城市，比如北京、上海、广州等城市，纯电动的特斯拉汽车可以直接上牌，并无需遵守尾号限行。

特斯拉汽车是直接销售的模式，税前售价和美国相差无几。如果新车 Model 3 价格区间真的在 35000 美元左右，在国内的税后售价就大约在 32 万人民币左右。这个价格对于苦于车牌指标的一线城市中产准车主来说，相信很有吸引力。
