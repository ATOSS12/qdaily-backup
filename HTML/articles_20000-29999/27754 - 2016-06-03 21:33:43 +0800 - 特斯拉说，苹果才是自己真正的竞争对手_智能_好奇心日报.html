特斯拉说，苹果才是自己真正的竞争对手_智能_好奇心日报

唐怡园 2016-06-03 21:33:43

2 年前，人们将特斯拉比做苹果时，马斯克还挺高兴的。现在，画风不一样了。

最近，新媒体 Vox 举办的 Code 大会上，特斯拉 CEO 伊隆·马斯克说，自己的直接竞争对手是苹果而不是在做无人车的 Google。马斯克还说自己预计苹果会在 2020 年量产电动汽车。

实际上，苹果早就被特斯拉视为竞争对手了。

今年特斯拉汽车第一季度财报会议上，被问及如何提高 Model 3 产能时，马斯克就表示，Model 3 还是会用第三方供应商，如果这些供应商不可靠，那就自己生产。总之，特斯拉不会效仿苹果采用合约供应商。

当时，他还揶揄说，造车可比造小的电子产品难多了。

热爱到处为自家公司做宣传的马斯克，没少对苹果冷嘲热讽。这次 Code 大会上，他就暗示苹果进入电动汽车市场的时机有点晚。

去年 10 月，一些媒体报道说大量特斯拉工程师跳槽去了苹果。马斯克反击说，苹果汽车团队是特斯拉的坟场。

不过之后从特斯拉跳槽去苹果的趋势并没有停下来，包括它的工程副总裁 Chris Porritt 也在一个月前去了苹果。

两年前，马斯克不是这么看待苹果的。各种传闻也是另外一种面目。比如苹果可能收购特斯拉、马斯克和 iPhone 工程师一起开会讨论如何把三元锂电池放到手机里。

那会儿，特斯拉被很多媒体称为汽车界的苹果。
