「这世界」英国王室的孩子穿得是否有些......太复古了？_文化_好奇心日报

孙若空 2016-05-04 21:57:19

dress code：二战风。刺绣衬衫、膝盖袜、玛丽珍风格的童鞋、高腰短裤、彼得潘

最近，英国的夏洛特小公主一周岁了。王室官方发布了一组照片，看起来可萌可萌了。

不过，当看完乔治小王子的打扮再看到夏洛特小公主的穿着之后，一些人开始产生了些疑问：为啥英国王室的孩子穿得看起来都像是从……二战时期，大概 1940 年代来的。

“可怜的乔治王子，他看起来就像是 1940 年代的儿童模特。搞不好他可能也只是像其他孩子一样想穿蝙蝠侠连体衣。”网友 @maryhitchman 说。

“我觉得乔治王子被打扮得像在闪电战中去世的男孩的亡魂。”网友 @Scriblit 说。

“夏洛特公主的洗礼现场显然是来自 1940 年代的实况。”网友 @mikeysmith 说。

那么，为什么说他们像 1940 年代的小孩呢？不妨看看网友 @LutheBlue 的点评：乔治小王子超可爱的，但是他看起来像是从 1940 年代来的。现在蹒跚学步的孩子谁还穿着膝盖袜和矫形凉鞋呀？

看看下面这些二战时期孩子们撤离伦敦时拍的照片。 get 到那个点了吗？

和 2014 年时候的乔治小王子以及 2016 年时候的夏洛特公主的 dress code 真的很像吧？刺绣衬衫、膝盖袜、玛丽珍风格的童鞋、高腰短裤、彼得潘领和柔和的色调……现在真的没有孩子这么穿了。

话说，乔治小王子穿着长袍睡衣见奥巴马也挺令人震惊的。太不正式了是一方面，而且现在也没有小孩穿这样的居家服吧？

那么是不是这就是所谓的王室小孩的 dress code 呢？好像也不是。对比一下其他欧洲王室家庭，就可以看出英国王室的小孩有多独特了。

比如丹麦王储费雷德利克和王储妃玛丽的孩子穿得就和现在的孩子差不多。

西班牙国王菲利普和他的妻子莱蒂齐亚的一双女儿，出现在公众场合的时候穿的也是短袖衬衣、短裤和露脚趾的凉鞋而已。

荷兰国王威廉·亚历山大和他的女儿们在海滩度假时小姑娘们也都是穿着符合时代气息的宽松上衣和休闲的中裤。

然后我们在回过头看看英国女王 90 岁生日时候公布的那组和孙辈们在一起拍的合照。真的很像是从旧时代来的吧？

啊，好想看一次乔治小王子和夏洛特小公主穿那种恐龙连体衣或者有个小尾巴的毛绒小熊连体衣啊。

不过，可能乔治小王子会觉得这样的衣服穿起来太像乡巴佬了也不一定哦。
