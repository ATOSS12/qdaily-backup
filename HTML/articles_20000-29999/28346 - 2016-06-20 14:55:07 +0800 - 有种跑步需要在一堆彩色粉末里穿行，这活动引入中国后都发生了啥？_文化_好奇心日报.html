有种跑步需要在一堆彩色粉末里穿行，这活动引入中国后都发生了啥？_文化_好奇心日报

张田小 2016-06-20 14:55:07

这活动还有另一个名字：“地球上最欢快的 5 公里”

从巴黎来的 Francoise Ly 是个“老北京”，在这住了 10 年。虽然无时无刻不在拥抱中国文化，但她依然保持着“国际化”的生活风格，周末通常喜欢在国外北京人社区 Beijinger 上找一些娱乐活动来玩耍， The Color Run 算是她其中一个发现。2013 年 8 月，这个从美国飘扬过海来的活动在北京首发时，Francoise Ly 就约上了三五个中国好友去了北京丰台区的园博园。不过，因为今年错过了抢票时间，她没能报上名。

的确，The Color Run 越来越火了。与最初 Francoise Ly 能轻松买到票不同，今年这个活动北京站从 4 月中旬开始售票，一周之内票就卖光了。要知道 3 年前，美国国际管理公司 （IMG）刚把 The Color Run 从美国引进到中国市场时，北京首站那 1.2 万张票卖了整整 3 个月。

“当时我们活动网上报名窗口开了一个半月，结果只有一千多人报名。我急得脑袋都晕了，晚上睡不着觉。3 个多月的时间，才报满一万人。” IMG 副总裁张宏说。张宏是跳伞运动员出生，后退役去了美国学习了“运动场馆运营”，加入 IMG 后在经营北京国际铁人三项赛等项目，现在是 The Color Run 中国区负责人。

当时张宏的解决方案是找到在北京的外国商会帮忙推广。“参加第一届北京 The Color Run 的外国人是最多的。这些东西靠口碑，外国人报名了以后，又特别会晒，所以慢慢就有人参加了。”张宏说。看起来，Francoise Ly 就属于张宏所说的“能带动口碑的、会晒的外国人”。不过，在中国这个活动自然不是外国人的天下，根据 IMG 给出的数据显示，参加  The Color Run 的 99% 都是年轻人，而且大部分是女性，占到总人数的 65% —70%。

2011 年，Travis Snyder 在美国亚利桑那州的凤凰城首次举办了 The Color Run，并创立了 The Color Run 公司。据说，这项活动从印度的洒红节取得灵感，玩法如其名，需要参赛者穿着白衣服从起点出发，途中经过 4 个彩色站和一个泡泡站，每过一个彩色站，选手们都可以用彩色粉末“互殴”，冲过终点到达泡泡站时， DJ 会组织最后一场有音乐的彩色粉末狂欢。The Color Run 的第一次活动有 6000 人参加，随后风靡了美国，相继在一百多个城市举办。

2013 年 2 月，IMG 和 The Color Run 公司签订了合作协议，负责这项活动在美国之外的组织和推广。3 年来，IMG 已在全球包括阿布扎比、巴黎、雅加达、首尔、阿姆斯特丹、曼切斯特等城市举办了 500 多场 The Color Run 彩色跑活动，吸引了 400 多万人参加。

“ IMG 在全球推广 The Color Run 这个 IP ，现在，这项活动已经变成了一项有价值的体育运动和生活方式的 IP 。” IMG 体育运动部高级副总裁 James Leitz 说，“我们的数字化传播策略也成功创造了一个全球的彩色跑步者的社区。” 根据文化传播公司 Marketwired 给出的数据显示，截止在 2015 年 4 月，以 The Color Run 为主题的内容在 Facebook 上有超过 3500 个点赞，在 YouTube 上的内容有 900 多万次的观看，Instagram 上有 10.4 万粉丝。

看起来， The Color Run 在全球红火程度节节攀升，市场上不断涌现出它的模仿者。在美国有和它神似的 Color Me Rad ，在中国则有荧光跑、Rainbow Run 之类的彩色跑满天飞。这些模仿者的山寨活动也让拥有正版 The Color Run 的 IMG 头痛不已 。

去年 6 月，台湾新北市八仙水上乐园举行了一场山寨彩色跑，由于彩色粉末的质量问题，粉尘爆炸，造成 200 多人灼伤。事故一出，这种“一边奔跑，一边洒粉，一边疯狂”的玩耍模式终于脱掉了“洋外套”，理性的“讨伐”都来了：粉尘在开放空间集聚可能会爆炸，呼吸道会受影响，另一个形而上的问题也被提了出来——一些彩色跑活动中“参赛者喊着脏话追打，让人想起打砸抢”，这甚至“让中国人看起来依然不懂公共空间结构的转型”。

这些问题可能比最初张宏为 The Color Run 挑选落地城市时的顾虑更现实，也更急迫。按照张宏的说法，“当初选择北京、上海、广州、深圳作为活动的首发地，是看上了这些城市的时尚度和接受度”，但自去年 6 月后， The Color Run 引发的讨论关注点都放在了安全问题上。

IMG 的 The Color Run 经营也收到了影响。“基于去年，取消了很多站的活动，大概有一年左右没有任何的更新和消息。一年的时间，我们没有很多的沟通。这次希望更有系统地和媒体和受众重新做沟通。” IMG 亚太区总监 Michelle Ku 在谈到 The Color Run 的社交媒体推广时说道。

今年，就在 The Color Run 要举办的前夕，中华人民共和国公安部治安管理局联合北京城市系统工程研究中心，公安部天津消防研究所，以及 IMG 专门起草了一份《大型群众性活动中彩色粉末使用的规定》。当然，张宏是这份标准 9 位主要起草人当中的一位。

这份标准将“彩色粉末”定义为“用于渲染气氛的不同颜色的微细颗粒物”，并规定了活动中的彩色粉末必须符合美国材料与实验委员会标准，也必须通过材料特性-可燃粉尘的测试方法。

IMG 的规范操作方法是坚持让  The Color Run 在全球的彩色粉末都由印度的供应商提供，这次北京站所使用的彩色粉末的包装上都清楚地写着 ”Made in Inida "（印度制造）。

“不管在世界上任何一个城市做 The Color Run，我们都是把安全放在第一位。台湾这件事情出了以后，对中国任何一个活动都敲响警钟，不仅是我们 The Color Run。”张宏说，“国家做得非常好，不会因噎废食，有规范了，大家都按照这个标准去做。”

尽管 IMG 的正版 The Color Run 在努力按规范小心行事，但今年的活动场上，你可能见不着那么多带有  New Balance Logo 的装饰物了。在装备袋中，你也找不到 New Balance 牌的白色 T 恤了。由于安全隐患的影响， New Balance 今年没有再继续做 The Color Run 的主赞助商。

不过，这一切都没能阻止 The Color Run，今年北京站门票还是卖光了。如果你去微博搜一搜，就会看到到处都有人在发帖求门票。在这个活动的主要消费群体年轻人的心目中，这大概是不可多得“既能厮混在一起，又没有四目相对尴尬”的社交活动。用 Michelle Ku 话来说“每年在 The Color Run 都有人求婚，微信后台时不时还有人发来信息说他（她）们在这个活动上遇到了真爱”。

这种效果大概是张宏最初没有预见到的。2011 年， The Color Run 刚在美国火起来时，IMG 总部的人“就强烈推荐过”，直到 2013 年年初，张宏去美国亲眼观看了一场旧金山 The Color Run 后，才决定把这个活动带到中国。

“很震撼，觉得这真的是一个年轻的海洋，全部都是小孩，我一看这个将来在中国要火。”张宏说，“考虑到中国市场，IMG 做赛事，知道什么样的赛事是什么样的客户群体，但完全针对孩子的赛事活动，我掰着手指头算也没有。篮球排球是传统项目，有多少人打篮球？没有。真正的娱乐活动就是音乐会，音乐会太老调了，我觉得这是一个非常好的机会。” 

2013 年 The Color Run 先后去了北京和广州，今年举办的城市增加到了 8 个。成群结队的年轻人把他们“五彩缤纷”的玩耍自拍自觉地放在微博、微信上。但为了让这个在全球都能火起来的娱乐活动也能适应中国市场，IMG 也对它做了一些调整。

The Color Run 的另一个名字是“地球上最欢快的 5 公里”。虽然 IMG 并不太清楚为什么这个活动要设置成 5 公里，不是 6 公里又或者是 4 公里。但现实情况是，在寻找场地的过程中，能满足 The Color Run 活动要求的 5 公里场地太少。

“找 3 公里很容易，在中国找 5 公里很难。找了半天，场地特别好但就是只有 4 公里，这怎么办，至少在中心区里绕一圈，这个很绞尽脑汁的，虽然不是完完整整的5 公里，但基本上都是 5 公里。”张宏说。此外，还要考虑靠近市中心又有地铁。因为 The Color Run 也带有环保理念，IMG 最终确定的北京的园博园、上海森兰公园、沈阳开全国运动会的全运村也都考虑到了地铁的绿色出行因素。处于安全考虑，最后狂欢的广场也得大，路面得宽 8 米左右，并且不能交叉，否则会暴露下一站彩色粉末的颜色。“所以很多城市想让 The Color Run 去，我去了很多城市找不到好地方。”张宏说。

虽然彩色粉末都是从印度进口，但根据不同的城市的反馈，IMG 也对粉末的颜色进行一定的调整。去年北京，用的红色比较多，有人反映说这个红色太多了，一流汗红色就褪下来，IMG 就把红色粉末换成了一个浅一点的颜色。

抄袭的问题一直困扰着 IMG，遵守粉尘规范可能是一种自觉的区分方式，但如何让人明白这个明白真正的  The Color Run 在中国“只此一家”的确是一个问题。今年，Michelle Ku 决定让活动的官方社交媒体账号都用上 IMG 在德国、英国等地举办的 The Color Run 的照片。“IMG 是授权的，所有我们的道具，活动举办的规划都是正规的。这也是为了这个国际品牌做的沟通。想要说的信息是，我们是权威的主办方，安全性是最高的。” Michelle Ku 说。

不过，另一个问题是，虽然 IMG 想要把 The Color Run  原汁原味地带到中国，但美国人能在这个活动上玩的某些东西，中国市场的消费者暂时还无法享受。2014年， The Color Run 就在美国组织了夜跑 Color Run Night 的活动，但 IMG 一直没能组织夜跑。

“ 中国的公安放开夜跑时间不好调整，一般都是批到晚上 6 点。我们做这个特别希望天比较黑一点，但地铁到晚上 10 点多就关门了。”张宏说 。

在那份今年刚出台的《大型群众性活动中彩色粉末使用的规定》里还有一条细心的规定：不得在夜间释放彩色粉末。
