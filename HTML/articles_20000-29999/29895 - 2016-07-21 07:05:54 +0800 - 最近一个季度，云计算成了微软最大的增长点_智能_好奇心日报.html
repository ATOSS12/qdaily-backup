最近一个季度，云计算成了微软最大的增长点_智能_好奇心日报

马若飞 2016-07-21 07:05:54

微软正向一家服务公司大步前进。

昨天，微软公布了截至 6 月 30 日的 2016 年第四财季财报和全年财报，这次，云计算成了他们最大的增长点。

微软的“智能云计算”部门包括 Windows Azure 公有云服务和服务器业务。而公有云服务 Windows Azure 的营收增长 102%，是微软所有产品中最亮眼的表现。

相比之下，微软的“个人计算”部门，也就是包含 Windows 操作系统的传统部门，过去四个季度里只有一个季度实现微弱的盈利，三个季度都是亏损。

在微软的所有业务里，云计算是增长最为稳定的业务。微软的云计算业务，目前主要都是攻大公司的生意。

最近的消息是，民用飞机制造巨头波音和旗下的两个子公司 AerData、Jeppesen 决定把云服务迁移到微软 Azure 云计算平台上。稍早前，通用电气也和微软签了合同，把自己的软件平台迁移到了微软云平台上。而此前，波音的云计算是运行在亚马逊 AWS 和 Century Link 云计算平台上的。

尽管跟最大的竞争对手亚马逊相比，微软云计算服务还有一定的差距。

按照 Statista 的统计，2015 年全球企业云计算市场份额中，亚马逊以 31% 排名第一，微软为 9%，IBM 占 7%，Google 占 4%。

和微软不一样，亚马逊云服务更多地服务创业公司，因为赶上了互联网创业潮所以发展迅速，之后受到了大量大企业的欢迎。比如 Netflix 和 Dropbox 就是完全依赖亚马逊云服务发展起来的。

相比之下，微软更擅长和大企业合作。大企业除了公有云计算服务之外，往往需要离线运行的私有云服务。在这个方面，微软给企业用户搭建私有云服务更有经验，运营人员也更充足。

但云服务外包不是所有企业的未来，在企业的规模发展到一定程度的时候，就会考虑自建云服务，比如苹果。

目前，苹果的 iCloud 有多家承包商，微软、亚马逊和 Google 都在其中，但苹果的数据中心正在建设，未来这些服务一定都会迁移到自家的服务器上去。

云存储之外的差异化服务，可能是未来和微软和亚马逊争夺的重点。
