世界上“最丑”的颜色也是有用的，香烟包装盒需要它_时尚_好奇心日报

张田小 2016-06-13 23:27:37

你讨厌它么？

如果让你说一说最喜欢的颜色，心胸宽广的人大概能挑一箩筐；反之，让你数落几个看不上的颜色，好像也并不是什么难事。

但如果现在有人正襟危坐地告诉你，在所有颜色中，最丑陋的颜色是一种介于绿色和棕色之间的颜色，它的官方的名字叫 Pantone 448 C，或者又叫 “单调深棕色”。

这个研究结论来自消费调研公司 GfK 做的一个调查。 2012 年， GfK 用了 3 个月时间，调查了 1000 名年龄从 16 岁到64 岁不等的烟民，做了 7 个研究。最后发现，大家都很讨厌这个 Pantone 448 C。简单说来就是，烟民们纷纷表示： Pantone 448 C 太丑了，如果用它来做香烟包装，“最不吸引人，还会让香烟看起来质量低劣。”

当然，在 GfK 的这个调查中，Pantone 448 C 可不是唯一躺枪的颜色，石灰绿、淡棕色、深灰、芥末色有幸排在它之后。最有竞争力的颜色是深棕色，它差点就超过了Pantone 448 C 成了最丑色。不过，由于深棕色看起来有一些“富贵”、“巧克力”、“上层社会”的调调，所以活生生输给了 Pantone 448 C 。

Pantone 448 C 能大获全胜的原因很简单，香烟消费者对它的描述比较一致，可以用三个词来概括：“死亡”、“肮脏”、“像沥青一样”。

“我们做这个研究相当于是反美学的，不想要找有魅力颜色，也不是想要帮助零售商制作好看的包装来吸引更多消费者。相反，这项研究是帮助我们的客户，找到减少这种产品销售量的办法。” GfK 的市场研究员 Victoria Parr 说。

Victoria Parr 说的客户是澳大利亚联邦政府，GfK 的任务是帮助政府部门找到减少香烟销量的办法。随后，“单调深棕色”被用到了所有澳大利亚的香烟包装上。与此同时，爱尔兰、法国和英国政府纷纷效仿了澳大利亚这种控制香烟销售的办法。今年，英国政府也开始用包装来控制消费，Pantone 448 C 色取代了香烟盒上所有明艳的颜色。

英国政府的目的之一就是尽量阻止每年都会加入吸烟队伍的 20 万儿童，让他们远离香烟，也让更多的人能在买烟时三思而后行。

不过，潘通好像对这事不太感冒。“潘通色彩研究所对所有的颜色一视同仁。“潘通研究所的执行理事 Leatrice Eiseman 说，“根本就不存在所谓的‘最丑陋’的颜色。”

的确，颜色是一种很具主观性的语言，它看起来是一个样子，所指却因人而异。有人就说了， Pantone 448 C 色也可以被解读成“深沉的、富有大地泥土气息的”，而且它也被广泛用在了文艺青年的衣裙上，你还可以看到一些这种颜色的鞋子或者沙发。
