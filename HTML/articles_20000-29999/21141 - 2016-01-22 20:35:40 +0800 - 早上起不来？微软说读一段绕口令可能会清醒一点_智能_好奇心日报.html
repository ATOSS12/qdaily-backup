早上起不来？微软说读一段绕口令可能会清醒一点_智能_好奇心日报

钟舒婷 2016-01-22 20:35:40

如果你用了这个闹钟来帮助起床，应该是一天最糟糕的开始吧。

微软又捣弄奇怪的东西了，继爆红的 How-Old.net 测年龄网页、双胞胎测试 Twinsornot 之后，今天又发布了一个反人类的闹钟——Mimicker Alarm。

Mimicker 目前只有 Android 版本，与一般闹钟小心翼翼地、尽量不惹你生气地让你从睡梦中醒来的方法不一样，Mimicker 认为每天叫醒一个人的方法既不是梦想也不是早餐，而是一些不太能理解的任务。

Mimicker 界面

这些任务包括：拍一个指定表情的自拍、拍一个有特定颜色的物体，以及读一段绕口令，这些都要在 30 秒内完成，否则闹钟会默认你又睡过去了，待会又会响起来。

当然你也可以关掉所有这些游戏，把它当成一个普通的闹钟来用，界面设计其实还是挺好看的。

Mimicker 同样来自微软的牛津计划，微软会把一些有趣的“黑科技”放到这里供开发者使用，

在提到为什么制作一个 Mimicker 闹钟时，微软负责牛津计划的项目经理艾莉森·莱特（Allison Light）表示：“我们想用开源的牛津 API 做一个简单的应用 ，但是独特又有趣，来给开发者们展示怎么开脑洞。”

API （Application Programming Interface），又称为应用编程接口，开放的 API 能让程序员直接调用具体的功能，去年成立的牛津计划，主要开放出四大类 API，分别是人脸识别、语音识别、计算机视觉和语义理解服务，更多的 API 也会逐渐接入到这个计划内。开发者只需要申请一个免费的密钥，就可以访问到这些工具的 APIs，只要有个好点子，做出 how-old、twinsornot 这些应用就完全用不着自己折腾人工智能。
