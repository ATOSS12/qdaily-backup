Chrome 砍掉应用启动器，它不再想着变成操作系统了_智能_好奇心日报

周韶宏 2016-03-23 21:48:33

已经是操作系统的 Chrome OS，到现在也只是少数人会用的东西。

Google 的 Chrome 浏览器又迎来一次简化，去年末他们关闭了这个通知中心，现在应用启动器（Chrome App Launcher）也要被送进坟墓了。

这是 Chrome 在 2013 年推出的一个拓展功能，应用在 Mac OS、Windows、Linux 以及 Chrome OS 系统中。安装之后，你可以在浏览器关闭的状态下，在桌面直接打开 Chrome 的各种插件，比如 Google+、YouTube、日历……就像是一个小小的操作系统那样。

Google 在声明中直接了当地解释了关闭的理由：“我们发现，用户更喜欢在 Chrome 内打开应用。”说白了就是没人用。接下来几周，Chrome 浏览器的新用户将看不到应用启动器，今年 7 月开始，除了在 Chrome OS 操作系统中保留，这个启动器将彻底消失。

三个月前关闭通知中心时，Chrome 也用了相同的理由，说它的实际用户太少。这项功能也同样让你跳过浏览器，接收来自 Chrome 的应用通知。

Chrome 正在逐渐回到它最初的样子，一个简洁的浏览器，而不是复杂的操作系统。

变成操作系统的 Chrome 的确不怎么成功。2011 年，Google 找了三星和宏碁生产了第一台 Chromebook，Chrome 彻底变成了一个操作系统 Chrome OS。它所有的应用都基于云端，用户不用下载就能直接使用。

不过到现在， Chrome OS 和 Chromebook 依然只是个小众的产品，它在教育领域的成绩还是不错的。

市场调研公司 Futuresource Consulting 12 月发布的报告显示，Chromebook 在教育市场的份额已经超过了苹果和微软的总和，达到 51%。

此前 IDC 高级分析师 Rajani Singh 曾表示：“尽管微软在销量上是第一、苹果在收入上是第一，但从增长幅度来看，没人比得上 Chromebook。”

除了教育，Chrome OS 还没办法拿到更多的消费者，根据 Technalysis 的估计，2015 年 Chrome OS 的市场份额是 2.9%，即便到 2020 年，这个数字也只能上升到 3.6%。 

对 Google 来说，把心思都花在更加成功的 Android 上显然更加现实。所以从几年前开始就有消息指出，Google 正尝试把 Android 和 Chrome OS 合并，并打算会在 2017 年正式推出融合后的系统。
