你其实只是游戏里的一段代码，马斯克说有这个可能_智能_好奇心日报

谢若含 2016-06-03 21:21:03

虽然只有十亿分之一的可能。他还聊到了人工智能、计算机模拟的世界以及火箭载人计划

最近在 Code 大会上，马斯克说到很多话题，比如火箭公司 SpaceX 载人火星登陆的计划：

“如果事情进展顺利，我们能够在 2024 年开始将人类送上火星，并于 2025 年抵达火星。”

但该时间也可能被推迟，毕竟载人火箭不像已经成熟的无人驾驶技术，在人类真正踏上火星以前，它有许多伦理问题以及技术问题需待解决。

此前 SpaceX 旗下的猎鹰 9 号火箭已经成功完成了发射与垂直回收任务，并且拥有非常强的运载能力。

只有一个真正可能毁灭人类的人工智能公司

此外，在 Code 大会上，他还表示真正让他畏惧的人工智能公司只有一家，但是他并不愿意透露这家公司的名字，不过有现场记者观察到在说这话时，马斯克将目光转向了 Google。

马斯克从不掩盖他对人工智能可能对人类造成威胁的担忧。去年他还与其他公司一起成立了一家专门研究人工智能的新公司 OpenAI ，该公司会把所有研究成果向公众开放，以此来避免人工智能技术的误用。

在 Code 大会上他认为人们实际上已经成为了半机器人，现在人们已经无法摆脱智能电子产品来生活和工作。

“你通过使用手机、电脑还有其他设备拥有了超级能力，现在你已经比 20 年前的美国总统都更有能力。”

人类可能只是更高级文明所编写的代码

谈到计算机游戏时，一位记者询问到 “我们是否有可能活在一个模拟的电脑游戏当中。” 马斯克回答说很有可能，他解释到：

“四十年前，我们玩的游戏由二维图像的矩阵和点构成。40 年后仿真度极高的 3D 游戏已经如此普遍。游戏以这样的速度发展下去，很快我们将无法分辨真实与虚拟的边界。”

这种状态类似于我们处在一个黑客帝国描绘的世界里，人们生活在由母体创建的虚拟现实世界当中，而大脑其实无法分辨真实世界与虚拟世界的区别。著名的思想实验 Brain in a vat 也曾提出这一猜想。
