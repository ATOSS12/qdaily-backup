微软停止更新，Windows 8 和 IE 终于快退出历史了_智能_好奇心日报

李硕明 2016-01-14 21:43:40

大多数人或许还在用着 Windows 7 呢

如果你是为数不多还在用 Windows 8 的人，那以后你可能得自己解决系统 Bug 和安全更新了。因为微软已经在 1 月 12 日结束了 Windows 8 的更新支持。

你可以继续用 Windows 8，但不会再收到任何来自微软的安全更新了。

不过这一次和之前 Windows XP 停止更新不同，并没有国内做杀毒软件的厂商跑出来说：我来负责 Windows 8 的安全。

过低的市场占有率应该是出现这种情况的原因之一。在 Net Application 去年 12 月的统计结果中，Windows 8 只有 2.76% 的份额，另一边的 Windows XP 还有 10.93％ 的占有率。

Net Application 给出的统计数据

这在中国市场更明显，根据百度统计的数据，Windows XP 在去年仍然存在于近 40％ 的电脑上，而 Windows 8 只有 5.78% 的占有率。

百度统计的 2015 年操作系统占比

同时和 Windows 8 一起停止更新的还有旧版的 IE 浏览器，包括 IE 8、IE 9、IE 10。在 Net Application 的另一份统计中，这三个浏览器 2015 年的份额加起来仍然还有 26.5％。这代表着 IE 可能还和 Windows XP 或者 Windows 7 一起运行在你的旧电脑上。

其实微软也早就该放弃 IE 了，它甚至因为 IE 错过了移动互联网时代的发展机会。从 2014 年 12 月开始算，IE 只占到了移动互联网流量的 13.42%，而 Chrome 的数据是 42%。
