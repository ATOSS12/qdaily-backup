人生，就是拼个车都会出现一些人改变你的生活_文化_好奇心日报

Caroline Tell2016-01-07 03:00:35

有人拼出一段姻缘，有人一不小心听了个“恐怖片”

利用 Via 拼车的乘客：Dee Tseurate（左前）、Joanne Gamel（右前）、Kelly Lee（左后）、Seth Hirschhorn（右后）。

Via 是纽约的一款拼车服务，而乔安妮·加梅尔（Joanne Gamel）喜欢把 Via 专车的后座当成是自己的移动办公室。加梅尔小姐是纽约的房地产经纪人，她经常在利用 Via 拼车服务叫到的车上查收邮件、给客户打电话和整理房产清单。

类似于 Via 这样的拼车服务能给人们带来很多好处。对加梅尔小姐而言，这种分享经济最棒的地方在于有时候与她同行的其他拼车顾客正好也是个房地产经纪人。

35 岁的加梅尔小姐说：“我已经从一起拼车的其他经纪人那里拿到了至少 15 张不同的名片。拼车能帮我扩展人脉。有时候，一起拼车的房产经纪人同行还会和我分享房产清单或者给我推荐客户。我很喜欢与其他房产经纪人同乘一辆车，如果他笑起来很帅气的话，那就更棒了。”

UberPool 是 Uber 公司推出的多人拼车服务。上个月，坦纳·威尔斯（Tanner Wells）在拼车时遇到了自己的梦中情人。当时他刚在布鲁克林参加完派对，准备叫车返回位于曼哈顿的公寓。返程途中，那个姑娘使用 UberPool 服务叫到了他乘坐的车，和他开启了拼车之旅。

37 岁的威尔斯先生说：“我们在返回市区的路上相谈甚欢。她给我说了自己接下来要去见的几个人的情况，我们还聊了一些傻乎乎的话题。当我们到达她的目的地时，她告诉我很高兴与我同行，接着就下车离去了。司机转过头问我，‘你为什么不去要她的电话号码呢？’”

第二天，威尔斯先生在大型分类广告网站 Craigslist 上发布了一条“失去联系”的寻人启事，希望那个女孩看到之后能主动联系自己。（到目前为止，他还没有收到回复）

类似 Via 这样的拼车服务让纽约人互相认识。

在过去的一年里，类似于 Via、Lyft Line 和 UberPool 这样的拼车服务在纽约蓬勃发展起来。如果人们不想乘坐费用昂贵的出租车或者使用不靠谱的公共交通，那么可以选择拼车服务来出行。这些拼车服务旗下拥有大批 SUV 专车，为顾客提供奢华、合算而且省时的交通服务，给拼车行业带来了新的气象。公司高管喜欢拼车去上班，企业合伙人在孩子上学的交通高峰时段也使用拼车服务避免拥堵。到了晚上，一大批年轻的白领精英更是成群结队地在狂欢之后拼车回家。

相比越发不景气的出租车而言，拼车服务的价格要亲民很多。在 110 大街以南曼哈顿地区的任意位置上车，只要目的地还是在 110 大街以南的曼哈顿，那么拼车服务的价格只要 5 美元多一点而已。

便利性是拼车服务的一大优点。除此之外，拼车还是人们进行愉快交流和暧昧调情的温床。当几个纽约人挤在车后座上时，他们可以尽情地社交、调情甚至争论问题。拼车给他们提供了一个相对自由的环境，让他们几乎可以随心所欲。

拼车的时候，乘客们要挤在一起。身体的靠近和彼此之间的匿名使得拼车的车厢成了人们寻找刺激调情的新场所。威尔斯先生说：“拼车让人产生了一种奇怪的心理，这有点像以前很火的一个夜间电视节目《出租车内的自白》（Taxicab Confessions）。在那个节目里，所有人的心理状态都很放松，愿意与出租车司机交谈。从某种角度来说，拼车的乘客之间也是这种心理状态。车辆的后座上，乘客们的情绪都很放松。”

上周四晚上，卡门（Carmen）小姐利用 UberPool 拼车服务叫到了一辆车。上车后，她发现后座上还坐着一位帅气的男士。这两个人一见如故，立刻就开始了调情。男士不断称赞卡门小姐的裙子，请求她同意与自己出去约会。后来，这位男士甚至握住了卡门小姐的手。

卡门小姐是纽约一位著名高管的助理，她要求我们在报道时隐去她的姓氏，以免危及工作。她说：“那位男士不断要求我和他一起出去玩，邀请我去跳舞。我说自己刚刚参加完派对，感觉很累，因此希望以后找时间再和他一起出去。”当车辆达到男子的目的地时，他突然弯下腰给了卡门小姐一顿热吻，然后下车离去。

图中三人分别是Jon Lowenstein（中间）、Gregg Arenson（后排）和司机David Franco。

对于当代的媒人来说，拼车服务给他们牵线搭桥提供了绝佳机会。乔纳斯·简·巴普蒂斯特（Jonas Jean Baptiste）是 Via 拼车服务旗下一名司机，上个月他接到了两名拼车的女性乘客。他说，这两位乘客上车后就开始聊自己的孩子。一位女士有一个 30 岁的女儿，目前单身。另一位女士有一个 30 岁的儿子，刚刚从一场痛苦的分手中挣脱出来。

55 岁的巴普蒂斯特先生说：“她们俩在后座开始给对方看自己孩子的照片。接下来，她们就交换了邮箱地址和电话号码，试图把孩子们撮合到一起。”

加布里埃尔·麦凯格（Gabrielle McCaig）是 Via 公司的新闻发言人，她表示乘客经常在拼车之后联系Via公司，试图索要同行之人的联系方式。这种情况频发使得 Via 公司创设了一个政策：如果乘客想要联系另一位同行的拼车乘客，他（她）必须先与 Via 公司沟通。如果公司确认两人都有兴趣与彼此保持联系，那么才会帮助他们交换联系方式。

麦凯格女士说：“这样一来，没有人会受到拘束。如果乘客不想与拼车的同伴继续联系，可以礼貌地拒绝。”她近期拼车时听司机讲了一个故事。这位 Via 公司的司机曾经接了一位男乘客和一位女乘客，上车后两人在后座聊得非常投缘。六个月后，司机又一次接到了这位男乘客。他告诉司机，自己已经与那位女乘客开始正式认真地交往了。

不过，不是每一次拼车之旅都能开启一段浪漫的爱情故事。拼车之后的的故事走向如何，还要看乘客的自身情况。有的乘客经历的拼车之旅就完全是“恐怖片”了。凯莱布·安杰尔（Caleb Angel）是叶史瓦大学的大三学生，他有一次使用 Via 拼车服务的经历颇为惊悚，就像是现实版的《黑道家族》（一部反映美国黑手党题材的著名美剧——译者注）上演。

在车里，同行的拼车乘客一直详细地向他阐述自己威胁前女友的种种计划。安杰尔先生说：“那个人给我讲了他前女友如何偷了他 6000 美元。他以前有过违法记录，所以他不能亲自动手，只能雇佣两个打手去打碎前女友的头骨和膝盖。我很害怕，提前下了车。”

凯蒂·谢伊（Katie Shea）今年秋天使用了 Via 服务拼车。她并不是一个人乘车，身边还带着一个由自己照看的 5 岁女孩。她们上车后发现，与她们一起拼车的还有三个喝高了的男子。他们不断地大声说脏话，还用粗俗下流的语言描述自己刚刚在酒吧遇见的姑娘。

幼教老师谢伊小姐说：“他们的举动非常不得体。当时，我身边的小女孩一直问我他们为什么如此大声说话，还举止奇异。我本打算给小姑娘放一段《冰雪奇缘》的音乐，谁知她拿出了《彼得兔》这本书，请求其中一名男子读给她听。车上原本喧闹的气氛瞬间安定下来，整个场面变得既滑稽又温馨。”

拼车时，乘客们可能因为任何事情起争执。从车内空调的温度到谁坐在副驾驶位置，任何事都能成为争论的导火索。鲁本·阿劳斯（Ruben Araoz）是一名 Via 司机，他今年九月时接到了一个女乘客发来的拼车订单。这位女士当时只定了一个座位，但他接到人时才发现她还带着自己的女儿。根据 Via 公司的规定，乘客在发出拼车订单时必须说明自己有几个人，以便司机根据车辆座位数量接单。

阿劳斯先生说：“这位女乘客还是将女儿带上了车，觉得自己可以把女儿安置在车辆后面的第三排座位上。但是，其他乘客都挤在后面的前两排座位上。我不得不请车上的另一位女乘客先起身出来，以保证自己能走到最后一排座位那里安置好小姑娘。起身的那位女乘客感觉受到了惊扰，一个劲儿抱怨带孩子的女乘客‘总是如此’。我猜她们两个应该是之前认识。”

在之后的整个行程之中，两位女乘客一直争论不休。

据麦凯格女士透露，Via 公司的各项乘车规定均是由 Via 乘客论坛创立。这样一来，所有的乘客才能更乐于遵守。目前为止，乘车规定包括：准时达到、彬彬有礼、尽量少打电话和禁止饮食。

“总是有些乘客不愿意迁就他人，不同意挪座位或者在寒冷的天气开车内空调。根据我们的规定，如果乘客之间有冲突，我们会一个一个与拼车的乘客联系，保证能够掌握事情的完整细节。我们希望所有用户都有舒适的拼车体验，但这不意味着无限退让。我们的价值观是：尊重每一位司机和乘客。”

有的时候，拼车还能带来一些惊喜。去年十月，25 岁的艾斯特·诺维克（Esther Novik）想从自己位于曼哈顿中城的公寓去 SoHo 商业区。她利用拼车软件发出了订单，没想到在车上遇到了自己的妈妈。

诺维克小姐说：“车停下后我打开车门，竟然发现自己的妈妈坐在后排。我都不知道她是怎么学会使用拼车服务的。我们俩哈哈大笑，司机也跟着笑了起来。现在她已经是拼车服务的忠实用户，逢人便讲我们俩这段有趣的偶遇。”为了纪念这次惊喜，诺维克小姐还在 Instagram 上发了一张照片。
