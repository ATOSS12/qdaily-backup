Windows 10 移动版要来了，这不是旧新闻_智能_好奇心日报

马若飞 2016-03-20 18:54:56

已经不知道多少个月了

2015 年初就公布预览版本的 Windows 10 Mobile 现在终于要发给 Windows Phone 8 用户了。

但是无论是在微软粉丝还是不关心的 Windows Phone 的人中，这条消息相信都没什么存在感——还在用 Windows Phone 的用户不少恐怕早就用上了测试版。至于其他人大概会问，Windows Phone 还没彻底死掉呢？

Windows Phone 的状况看起来一点也不好。市场份额只有 1.6%，没有第三方大厂愿意制造 Windows Phone，收购诺基亚的买卖现在看来也完全没意义。

最近原来被深度捆绑的 Here 地图被汽车厂商收购后改旗易帜，宣布将退出 Windows 平台。这种状况，换作其他厂商可能已经放弃了，比如黑莓的 BB 手机操作系统。

Windows Phone 被微软寄托了重要的愿景，无论以前的“三屏一云”还是现在“Windows 即服务”，手机都是微软不能放弃的重要市场。

微软正在用一系列手段实现 Windows 10 全平台通用的计划。

Xbox 即将支持 Windows 通用应用平台（UWP），Xbox 上的部分微软发行的游戏宣布跨平台至 Windows 10 PC，邀请开发者开发通用应用等等。

所以，哪怕 Windows Phone 的全球份额只有个位数，一旦退出若想再返回，比现在咬牙坚持下去恐怕更难。

用户可登陆 Windows Store 应用商店下载Windows 10 Upgrade Advisor 应用来取得升级帮助。

首批更新的手机是最新发布的 Lumia 系列，比如 Lumia 930、820、1520 等，诸如运营商定制机等其他机型将分批推送更新。微软不会向所有的 Windows Phone 8 设备推送 Windows 10 Mobile 的升级包。
