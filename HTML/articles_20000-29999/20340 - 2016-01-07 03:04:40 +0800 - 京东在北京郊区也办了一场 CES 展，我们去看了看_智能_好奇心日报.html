京东在北京郊区也办了一场 CES 展，我们去看了看_智能_好奇心日报

唐云路2016-01-07 03:04:40

觉得挺无聊的。除非，你就是想买买买。

一年一度在美国拉斯维加斯举办的美国消费电子展（CES）还没正式开幕，京东在北京亦庄郊区的新总部大楼办的「CES京东」体验展已经开始了。

包括索尼、三星、华为、努比亚、联想、HTC 在内的 72 个品牌带来的 142 个产品分布在京东大厦一楼大厅的六个展区，分别是电脑办公-游戏、手机数码、影音、摄像、虚拟现实及智能产品和 HTC 虚拟现实专区。

京东集团 3C 事业部电脑采销部、品牌合作与发展部总经理汪延领告诉《好奇心日报》，早在 2015 年初京东去拉斯维加斯参加 CES 展时，CES 主办方就与京东有过接触，主要还是想要扩大在亚洲的影响力，夏季在上海举办 CES 亚洲展会的时候就敲定了继续合作、授权的相关事项。

”这次 CES 主要是给了名称授权，联系参展商、布展等都是我们自己做的，筹备最难的就是展品，京东在智能硬件方面的策略让我们和厂商的关系比较好。”汪延领说道。“如果没有意外明年还会继续做类似的合作，甚至明年合作规模更大一些。”

电脑办公-游戏展区

主题既然是“CES 在京东”，主办方宣称有十几款产品是与美国 CES 同时发布的新产品，不过从现场来看，主要是雷神游戏本、机械师游戏本、LG 曲面屏幕等。离成为 CES 的同步分会场，还是有不少距离的。 

现场还举办了小型的英雄联盟战队比赛

手机展区以国产品牌为主，努比亚布拉格手机、中兴 AXON MAX、乐视 MAX、Moto X Pro、HTC One M9 等发布于 2015 年下半年的手机都能在展台上找到。似乎是为了添加前沿电子展的气氛，不太大的展台上有各个品牌的手机供参会者体验，还放上了机械恐龙、小型舞蹈机器人。

最热闹的展区则是虚拟现实体验展区，在《好奇心日报》记者观展的几个小时里，现场一直有不少人排队轮候。

HTC 虚拟现实体验区

大鹏 VR 的展台同样一直有人排队体验，驻守展台的商务崔智宣告诉《好奇心日报》，他们是先确定参加拉斯维加斯的 CES 展，才被邀请来的，不过最新一代的设备被带去了美国，在京东展示的还是上一代产品。

大鹏 VR 游戏展台

在音乐展区，除了索尼、森海塞尔、飞利浦的耳机，我们也找到了由音乐人汪峰创立的耳机品牌 FIIL，它们的产品前不久刚刚上线。而在京东众筹上待了大半年的猫王收音机也出现在其中。

音乐展区

至于智能硬件展区则相对令人失望，可能由于场地所限，只是简单展示了一些智能手环。当然，如果你想把几种手环放在一起比较一下再买，这里应该能够满足你的要求。不过也可能，这么一来你就不想买了。

卓棒手环

智能硬件展区

绝大多数的产品旁边都附有产品描述和专属二维码，用手机微信或者京东 App 扫一扫，就可以进入对应的产品界面，所谓“边玩边买”，就是说在现场摸到真机，立刻就能下单购买。

出于现场互动游戏的需要，一些展品没有附上产品说明，而是带有此次活动专题页面的二维码。

京东 3C 事业部新上任的负责人胡胜利表示，这并不是京东第一次关注 CES，去年京东的网站上就开了一个 CES 的专栏，主题叫“边看，边买”。今年主题是“边玩，边买”。 除了扫码进入的专题页之外，在京东首页也有“2016  CES 新品同步开抢”的醒目入口。

在京东总部大楼举办的线下体验展为期两天，除了邀请媒体和网络红人前来参观之外，也开放给京东员工和京东的用户。只不过，相对偏僻的地理位置可能让它的作秀意义大于实用价值。

“今天现场来了两三百粉丝，不过主要还是在线上。”汪延领告诉《好奇心日报》，“毕竟交通各方面因素会影响，如果真的愿意体验，目前坐地铁来京东一趟也比较容易。”

即将正式开幕的美国消费电子展吸引了 4000 家厂商，展区面积有 50 个足球场那么大，是创办 49 年来规模最大的一次。不过，去不了拉斯维加斯，在线上看看能买的数码产品，也算是跟了个风。

在一年一度消费电子最前沿的活动里，借着 CES 推广的电商并不止京东一家。亚马逊中国也在首页推出了 CES 产品专题，主推将海外创意产品引进中国的发明家项目（LaunchPad）。
