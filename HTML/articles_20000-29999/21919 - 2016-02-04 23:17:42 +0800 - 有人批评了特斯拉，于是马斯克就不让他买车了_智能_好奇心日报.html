有人批评了特斯拉，于是马斯克就不让他买车了_智能_好奇心日报

唐怡园 2016-02-04 23:17:42

不过，Model X 有钱也买不到，不是么？

特斯拉汽车 CEO 埃隆·马斯克又在推特上和人吵了起来。这次是风险投资人 Stewart Alsop。马斯克说他是“超级粗鲁（Super rude）”的用户，并取消了他的 Model X 订单。

可能不止 Alsop 被取消了订单。马斯克 2 月 3 日更新推特说，不做粗鲁用户的生意，早就不是什么新闻了。

2 天前， Alsop 在协作博客平台 Medium 上更新了一篇文章《特斯拉禁止我买车（ Banned by Tesla！）》 。他描述了下自己有钱也买不到 Model X 的经历，居然是因为骂了特斯拉的 CEO。

2015 年 9 月， Model X 第一批车出厂。特斯拉邀请了一部分预订客户去参加首发仪式。被邀请的 Alsop 开了 2 个半小时车去参加活动，结果预计晚上 7 点开始的活动，被延迟了接近 2 个小时。大约 3000 多位嘉宾一起在场外侯着 Model X 和马斯克。官方却毫无解释。

重点是，这场活动宣称让预定车主开着 Model X 去参观工厂。最后在现场，Alsop 却连车长什么样都没看到。第二天，他写了一篇叫做《亲爱的马斯克，你应该为自己感到羞耻（Dear @ElonMusk: You should be ashamed of yourself）》的博文。

除了标题耸动一点之外，文章不过陈述了一个事实：被特斯拉举办的活动浪费了大半天时间。Alsop 不开心之余，还给马斯克提了建议：请建一个像样的市场部。结尾处，他甚至还在给了特斯拉机会，“虽然我没看到 Model X，但还是很想要一辆。……如果特斯拉能给这些相信它产品的人道歉，我还是会一如既往地支持这家公司。”

结果，特斯拉把他拉黑了。

前段时间，特斯拉和苹果互相挖墙脚时，马斯克说苹果汽车项目就是特斯拉的坟场，对竞争对手刻薄点倒也没什么。但是，他对自家客户这么狠，是不想卖车了吗？很难想象有天在微博上黑了库克，苹果就不让你买新款 iPhone 了。

倒真有人觉得马斯克学会了乔布斯的“专制”。不过，觉得乔布斯性格古怪的人不在少数，可也没见他们被贴上“不许购买苹果系列产品”的标签。

对了，那些订了 Model X 并讲过马斯克坏话的用户也不必太担心，反正 Model X 也造不出来了。
