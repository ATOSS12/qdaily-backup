报告说，Windows Phone 的用户更喜欢 Android？_智能_好奇心日报

周韶宏 2016-05-14 21:15:55

Windows Phone 在全世界范围内下滑，连微软都几乎要放弃它了。

现在还有多少人在用 Windows Phone？

市场调研公司凯度消费者指数（Kantar Worldpanel）最近发布了一个关于智能手机操作系统的调查报告，按照地区展示了 Android、iOS 和 Windows 的市场占有率变化。

增长最快的是 Android 系统。10 个被调查区域内，它都在占据着第一的位置，也在除日本的地区有 3%-12% 的增长率。与此同时，因为 iPhone 销量的萎缩，有 8 个区域 iOS 的份额出现下滑。

Windows Phone 最为惨淡，原本份额就不高的情况下还有不小的下跌。比如中国，Android 的占有率超过 70％，留给 Windows Phone 的空间只有不到 1％。

Kantar Worldpanel 表示，Windows Phone 的用户正逐渐转移到 Android 世界中。以欧洲 5 国（英国、德国、意大利、法国和西班牙）为例，6.6% 的 Android 新用户来自 Windows Phone，只有 3.3% 来自 iOS。第一次使用 iPhone 的消费者当中，也有 2.6% 之前用 Windows Phone。

“对这些放弃微软的人来说，Android 提供了更好了用户体验，同时也有更多的品牌、机型和价格。”Kantar Worldpanel 业务总监 Dominic Sunnebo 说，意大利和法国用户就更喜欢华为和华硕的中端手机。

这样的情况一点也不让人意外。在遭遇了无数次挫折之后，微软已经逐渐放弃在 Windows Phone 上投入大量资源，转而生产 Android 手机、把 Cortana 等应用转移到 Android 平台、以及强调对利润贡献最大的云计算服务。

根据 IDC 的数据，2015 年 Windows Phone 的整体市场份额是 2.2%，远低于 Android 的 81% 和 iOS 的 16%。留给微软的空间几乎消失了。

如果我们用一个更长的时间段来观察，会发现比 Windows Phone 还要惨淡的操作系统是黑莓。

4 年时间，黑莓操作系统的市场份额从 15.2% 滑到 0.8%。这条下滑的曲线对应iOS 的上涨，这意味着有相当比例的黑莓用户最后买了 iPhone。
