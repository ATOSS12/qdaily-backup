二十年前，第一款全 3D 的 FPS 游戏诞生了_游戏_好奇心日报

唐舒畅 2016-06-24 17:37:26

没有《雷神之锤》，就没有后来的《半条命》或者 AMD 了

1996 年 6 月 22 日，id Software 发行了一款名为《雷神之锤》的第一人称射击游戏（FPS）。在这款游戏中，玩家需要扮演一位名为 Ranger 的战士，前往失控的传送门去阻拦一群代号为 Quake 的怪物。（Quake 即为游戏英文名）

虽然在此之前，id Software 已经靠着《德军总部 3D》和《毁灭战士》两款游戏定义了现代意义上的第一人称射击游戏，并将其发扬光大。但是随后的《雷神之锤》仍不失其经典意义：这是史上第一款真正意义上的 3D FPS 游戏。

别看 1992 年发行的《德军总部 3D》里带着 3D 的字样，实际上这款游戏依旧用的是 2D 的贴图，然后用渲染和一些透视的方式达到 3D 的效果。在这种模式下，你依旧只能左右移动，无法体会上下的空间感。

而到了《雷神之锤》，不管是玩家的形象还是其中怪物的形象都是使用真正的 3D 建模，而玩家也终于可以实现“跳跃”这一在如今射击游戏中再寻常不过的动作。并且在光效上，这款游戏也真正使用了动态光源，让火光和射击的场景更加逼真了些：你第一次在空间中可以体会到什么叫做明暗对比。

这成为当时兴起的 3D 显卡的重要推手。这款游戏及其续作的火热，使得其成为大多数玩家在当时购买一款支持 3D 显卡的理由。当时，id Software 和著名的显卡制造商 3dfx 合作，使得他们的 Voodoo 显卡能够最好地支持游戏，并协助 3dfx 成为当时的行业垄断者。随后《雷神之锤》的续作，《雷神之锤 3》，更是见证了一系列硬件商围绕其展开一系列竞争：游戏引擎支持当时多种图形硬件技术，那么谁能让这款游戏的画面表现最好，谁家的硬件就更厉害。

除此之外，其中的多人对战模式也为后世的游戏留下了深刻的影响。虽然多人模式最初见于 id Software 的《毁灭战士》系列，但是这款游戏支持最多 16 人联网对战不仅让多人组队模式成为射击游戏中不可缺少一部分，也在某种程度上使得电子竞技进一步现代化。

在上世纪 90 年代末的英国，每周都会有《雷神之锤》的狂热者们聚集在一起角逐出游戏的冠军。在那个网络普遍只有 28 K 的年代，玩家们还是互相联网进行一对一的对战。后来，id Software 直接开发了 QuakeWorld，这一抛弃了单人模式的版本，给玩家提供更直接的多人游戏体验。甚至在《雷神之锤 3》中，原本的游戏就只有多人模式。

当然，就和 id Software 一样，这款游戏的开放度也是这款游戏的伟大之处。除了各种各样的 Mod，id Software 研发这款游戏所使用的引擎也被公开。Valve 和建立起其最初名声的《半条命》，就是靠着《雷神之锤》游戏系列的引擎所开发的。

在刚刚结束的 E3 上，id Software 宣布将制作《雷神之锤》的最新系列：《雷神之锤：冠军》。这距离上一部 2007 年发行的作品已经过去了快十年。诚然，这款游戏的知名度已经不如现在已经可谓家喻户晓的《使命召唤》以及最近像毒品一样吸引着大家的《守望先锋》，但这款游戏对整个游戏发展所作出的贡献，也承担得起新作公布之后，那些玩家的欢呼与期待。
