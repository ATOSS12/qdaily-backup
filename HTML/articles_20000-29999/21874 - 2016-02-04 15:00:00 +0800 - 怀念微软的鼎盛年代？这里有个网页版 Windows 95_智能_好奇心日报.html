怀念微软的鼎盛年代？这里有个网页版 Windows 95_智能_好奇心日报

唐怡园 2016-02-04 15:00:00

排队买操作系统的日子已经过去 20 多年了。

最近，Andrea Faulds 的个人网站被挤爆了，因为她把 Windows 95 搬到了网页里。

你在网页上就能体验到从开机到开始菜单的每一个历史细节。选择全屏之后，画质会有点渣。

在浏览器里跑操作系统并不难。除了一款 Java 编译器之外，Faulds 用到了免费的平台移植软件 DOSBox，可以很方便地在网页端模拟 DOS 系统。

可能因为做起来太容易，Faulds 自己也没想到这个网页会火。时机也比较讨巧。微软刚刚发布了 2015 年第四季度的财报。显然，靠操作系统授权费赚钱的已经不再是微软的主要生意，Office 和云都比 Windows 更赚钱。

人们开始有点想念曾统治全球超过 9 成电脑的微软。而  Windows 95 是那个划时代的产品。

1995 年正式发售的 Windows 95 比 Faulds 年纪还大。这个操作系统包含了科技行业和微软的许多个第一次。比如：

  • 对于用户来说，进入 Windows 再也不用输入 Dos 命令行了。图形界面，第一次成为 PC 开机默认的样子。
  • 在雷蒙德大学的体育场，微软首次使用卫星传播让 7 万人看到了 Windows 95 的发布会。
  • 在 Windows 95 之前，没有哪家科技公司为自己的产品这样打过广告。微软花 800 万美元买了广告背景音乐（滚石乐队的 Start Me Up）的版权，甚至拿帝国大厦当户外广告牌。

2001 年的最后一天，微软宣布停止支持 Windows 95 。从此之后，它被放进了公司的历史时间线里。如今，每当回顾微软这家公司，甚至是科技历史时，你都无法跳过它。

对了，怀旧的人有点多，Faulds 做的这个网页在几个小时前被挤崩溃了。
