研究说，在程序员的世界，Mac 甩开了 Windows_智能_好奇心日报

赵长存 2016-03-18 23:31:58

数据来自 StackOverflow

创业者办公室、开发者大会一水的 MacBook 已经是常态。不过一般印象里，很多大公司配发的依然是 Windows 电脑。

最近，著名 IT 技术问答社区 Stack Overflow 在其发布的最新调查报告中指出， Mac 的 OS X 已经成为全球开发者们使用最多的系统，到达了 26.2%。

如果越来越多的开发者都换上了 Mac，普通用户对 Windows 平台上出现更多优秀的产品的希望，恐怕也就更小了。

相比之下，Windows 系统的占比一降再降。在 2016 年的开头降幅就有 2.3%。Stack Overflow 推测照此趋势，今年用 Windows 的开发者比例就会低于总用户的一半。

就连被微软寄以厚望的 Windows 10 系统，表现得也不好。2015 年该系统推出后，为了推广它，微软甚至动用了免费升级，全屏弹窗提醒，以及自动下载的办法。结果，在这次统计中，它的普及率也不过 20.8%，低于 Linux 的使用率。

目前 Windows 10 在整个桌面系统市场已占了 12.8% 的份额。说明开发者们对它的积极性也只比普通用户高出一点而已。而 2009 年就已发布的 Windows 7 依然占有第二的位置。
