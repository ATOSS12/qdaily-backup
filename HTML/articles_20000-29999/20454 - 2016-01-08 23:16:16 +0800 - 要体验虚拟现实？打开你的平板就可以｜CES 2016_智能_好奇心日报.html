要体验虚拟现实？打开你的平板就可以｜CES 2016_智能_好奇心日报

徐弢 2016-01-08 23:16:16

Occipital 的 3D 扫描硬件，可以让你在更多电子产品上“看”到被虚拟的世界

在美剧《神盾局特工》中，曾经出现过一个尸体融化的场景。以往是需要特效公司在后期完成，而剧组用了 3D 扫描来做这个特效，所需时间变少了。

技术的提供方来自一家名叫 Occipital 的公司，他们在今年的全球消费电子展（CES）上推出了新的 3D 扫描硬件产品 Structure Core 3D Sensor。

在 3D 扫描时，除了感应器，Structure Core 还使用了 2 个红外快门相机、以及 1 个宽视野相机。这次推出的新产品原理跟上代一样，是通过丈量空间内的障碍物之间的距离进行工作，目前可以扫描的最高精度在 0.3 米，最大可以覆盖到 5 米以上。

这家公司希望把它提供给硬件厂商，让用户可以借助 Structure Core 直接在各类电子产品上体验被虚拟化了的空间，例如平板、无人机和 VR 头盔等。

Structure Core 3D Sensor

从上图就可以知道，Structure Core 实际上是一个基础的硬件组合。它的尺寸为 100.6mm x 20mm x 14.11mm，很小，可以内嵌到很多硬件里，跟配套的软件服务 Bridge Engine 搭配使用。

2015 年的 CES 展会上，Occipital 就展示了他们以 Structure Core 为核心设计的 iPad 配件 Structure Sensor（看上去就像是加了个外壳），《好奇心日报》也体验过这款产品。

Structure Sensor 是一个 iPad 配件

有趣的是，2008 年创立的 Occipital 公司原本是一家纯软件公司，没有涉足硬件。他们最早的创业方向，是对用户照片进行管理和组织，并利用这些照片建立 3D 影像。

但在搭上了 VR 概念之后，公司的 3D 扫描技术，在外界看来就变得很厉害了。在 2015 年 7 月，Occipital 还完成了一轮 1300 万美元的融资。
