不知不觉，你在Windows里扫的已经不是当年那个雷了_游戏_好奇心日报

唐舒畅 2016-01-12 03:37:40

现在谁还记得 Windows 自带的那些小游戏呢。

灰色的方格，红色的 LED 数字，各种颜色的数字以及红色小旗子，这估计是大部分人对《扫雷》这款经典游戏的记忆。但随着 Windows XP 离我们远去，扫雷也已经不是我们熟悉的那个游戏了。
最近，微软宣布对《扫雷》这款游戏进行了一次更新。更新的内容包括对 Windows 桌面版和触屏设备的游戏优化，修复了一系列 bug，并增加了重制统计积分功能等。虽然上一次对《扫雷》这款游戏进行更新还是两年前的事情，但是微软从来没有忘记这款元老级别的小游戏。
自从 1990 年诞生以来，这款小游戏基本就没有变过。不管最初的版本，Windows 98 版或者 Window XP 版，扫雷一直是那种灰色的界面，加上花花绿绿的数字和旗标。但是从 Win 7 起，《扫雷》的界面变成蓝灰色，原本在界面上方的数字和地雷显示移到了画面下方。

Win 7 版大概是史上最丑的一版扫雷了。

更大的改动从 Windows 8 开始。不仅数字的显示方式、显示位置以及扫雷的界面发生了巨大的变化，并且添加了新的模式：冒险模式。你需要操纵一个小人在满是土墙的地道中探索。挖掉那些没有陷阱的土块，收集其中的金币或者额外的生命值，最终达到下一关的入口。同时，玩家依旧可以玩到经典的扫雷，并且能更方便地定义地图大小和扫雷数量。
不仅玩法和画面上有了新的变化，《扫雷》的平台也在丰富。随着 Win 7 开始，《扫雷》有了手机版本。而在 Win 8 上，《扫雷》从系统自带改为从应用商店下载，并且开始有广告入驻这款小软件。通过微软游戏服务 Xbox Live，玩家还可以获取积分、排名和进行每日挑战。

光看这个画面，谁会想到这是个扫雷呢

当初那个小游戏一点点复杂起来的背后，是微软在游戏方面一点点的扩张。在移动游戏大行其道的今天，从家庭娱乐套装中走出来的《扫雷》逐渐变得像一个跨平台的休闲移动游戏。微软从来不想让这款游戏就这么埋没下去：他们需要这款游戏提醒你当初你是个多么忠诚的 Windows 平台永用户，提醒微软的娱乐系统曾给你带来多么多欢乐，以及需要扫雷来让他们的游戏系统为更一般的大众所知。
所以，那么微软的小游戏删了又改，但是只要有微软的地方，就有扫雷。
