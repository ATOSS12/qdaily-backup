微软的年度开发者大会都说了什么？_智能_好奇心日报

马若飞 2016-04-01 02:02:36

微软的人工智能一点也不邪恶。

昨晚，微软 Build 2016 开发者大会在旧金山开幕，微软 CEO 萨提亚·纳德拉主持开幕演讲。

微软 Build 开发者大会的人气远远不如苹果和 Google 的发布会，Build 大会的主旨也确实和普通消费者关系不大。去年的 Bulid 2015 大会是围绕着 Windows 10 进行，今年的 Build 大会依旧如此。

Windows 10 周年更新

Windows 10 是微软最重要的产品，也是“Windows 即服务”的核心。负责 Windows 的执行副总裁 Terry Myerson 在开幕演讲上宣布了 Windows 10 一周年更新（Anniversary Update）。

周年更新主要包括以下几个方面。

微软首先介绍了手写笔技术 Windows Ink。发布会上微软展示了一项令人惊讶的手写应用：在一张三维地形图上随意画上两点，将两点连接起来，应用就能计算出三维空间中这两点中的实际距离。

微软表示，依然有 72% 的人依赖纸和笔，他们喜欢手写笔的体验。Windows Ink 也能与 Edge 浏览器和 Office 办公套件配合使用。

生物识别技术 Windows Hello 之前已经在 Surface Pro 4 和 Lumia 950 上配备，借助专门的红外线摄像头识别人脸，方便快速登陆。在周年更新后，用户可以还可以使用生物信息管理 Edge 浏览器上的浏览记录。

语音识别技术 Cortana 也会在周年更新中进一步进化。萨提亚·纳德拉说，Cortana 可以实现“对话即服务”，让不熟悉技术的用户也能自如使用。

新的 Cotana 能够识别手写便笺上的日程并自动安排，能像真人一样在 Skype 里和你聊天，识别你说过的日常安排，甚至自动预定酒店。在屏幕锁定的状态下也可以使用，就像 Siri 一样，但功能强大许多。

Cortana 和 Windows Hello 未来都会支持 Xbox One 游戏主机，Windows 10 周年更新适用于 PC、平板、手机、Xbox One、Hololens 以及嵌入式设备，今年夏天免费更新。

开发者福利

微软本次发布了一款转换器，开发者可以将基于 Win32 和.NET 开发的旧 Windows 应用直接转制成 Windows 10 通用应用并上线到商店，过程中仅需要少量修改。

为了打通 PC 和 Xbox 平台，微软为游戏开发者也推出了转制工具。微软现场展示了运行在 Windows 10 上的通用应用版《帝国时代 2》和《巫师 3》，并表示以后 Steam 平台上的游戏都可以方便转制成通用应用。

微软的拿手好戏开发工具也更新了。

  • Visual Studio 2015 Update 2 RTM
  • Bash shell 登陆 Windows，原生支持 Visual Studio 2015
  • 原生 Linux 二进制安装包支持在 Windows 运行，微软与 Ubuntu 合作
  • UWP 应用优化对显示芯片的支持

Hololens 炫技

微软现实增强眼镜 Hololens 正式发货，微软在发布会上展示了价值 3000 美元的开发套件和包装盒。

微软在发布会现场舞台上展示了数个 Hololens 应用，其中一个应用有关医学方面，多个佩戴 Hololens 的使用者可以一起观察、操作同一个虚拟人体，不在一个房间的其他 Hololens 使用者可以远程加入，在其他使用者视野里会显示出一个虚拟使用者。

微软也提到，日本航空、大众、沃尔沃、奥迪、NASA 等都会开发基于 HoloLens 应用。

微软的情怀

除了“对话即服务”微软也提到了一个新概念“对话即平台”。简单的说，就是用户通过与计算机的自然交流实现对计算机的控制。

纳德拉在最后的总结中说道，未来的计算操控将围绕用户、语音助手以及聊天机器人展开，用户永远是中心。人工智能会了解用户的喜好和习惯，扮演与计算机的中介。

像是为了验证这样的理想，微软最后介绍了一位微软员工 Saqib。他组建了一个团队，并开发了一个盲人用的手机 app，可以检测人脸，对方的动作，性别和年龄，然后转换成语音发送给使用者，甚至可以帮助盲人点菜。

特别的是，Saqib 自己就是盲人。这就是微软的愿景。
