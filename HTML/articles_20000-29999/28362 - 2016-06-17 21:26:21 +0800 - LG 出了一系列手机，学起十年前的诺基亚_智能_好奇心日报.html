LG 出了一系列手机，学起十年前的诺基亚_智能_好奇心日报

马若飞 2016-06-17 21:26:21

据说是为了差异化

在各个厂商都在追求“高大全”，拼命推出各种高配置或者低价格手机抢占市场的时候，LG 做了一件相反的事情：推出了 X 系列手机。

这个系列有四款手机，分别是 X Power、X Mach、X Style 和 X Max。就像名字那样，它们分别都有一项配置比较强。

  • X Power 主打续航，电池容量 4100 mAh，拥有快充功能，同时机身厚度只有 7.9 毫米。
  • X Mach（马赫）主打下载速度，支持 LTE Cat 9 网络，最高 410Mbps 的无线网络下行带宽。
  • X Style 主打外观设计，有弧形的超薄机身。
  • X Max 主打大屏幕。

除了主打的卖点之外，LG X 系列手机的其他性能都只是中端 Android 手机的配置，并没有公布细节。这个系列的手机宣传语不追求高大全，每一款都针对特定需求的人群。但换句话说就是，没有一款的体验是完整的。

其实这不是 LG 第一次这么干了，今年初他们还推出了 X Cam 和 X Screen 两款手机，分别主打双摄像头和双屏幕。但 LG 的思路恐怕是错的：这些主打的功能本来是不能分离的。

一台主打快速数据网络的手机处理器却成了瓶颈，一台主打续航的手机却没法吸引人一直用下去，更不用所谓“大屏幕”和“好设计”，这其实就是两台普通中端智能手机而已。

而它的竞争对手，iPhone、三星试图在最贵的产品上做好一切；中国手机厂商则试图在一两千的价位上取得平衡。总之都是想让用户接受“这是最好的选择”。

十年前诺基亚做了类似的事，用 N、E、C、X 等系列针对不同的应用场景，这个努力在 iPhone 和 Android 手机的冲击下终结。没想到 LG 这时候又做了相同的事。
