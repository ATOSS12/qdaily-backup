用 IE 的人越来越少，微软浏览器份额第一的位置要不保了_智能_好奇心日报

赵长存 2016-03-03 21:09:04

Chrome 是最热的替代品。

凭借和 Windows 系统搭售，IE 已主导浏览器市场近 20 年的时间。但这样的地位恐怕也维持不了多久了。

近日，ComputerWorld 援引 Net Applications 的数据称，上个月微软的 IE 和 Edge 两款浏览器市场份额之和下降了 2.1%。这也是 IE 在近 11 年里，单月的最大降幅。

如今微软的浏览器份额已不到整个市场的一半。而在它独大的 2002 年，这一数字却为 95%。

于此同时，Chrome 的份额出现了持续的增长，上个月份额已提高了 1.5%。照此速度，ComputerWorld 预测，到今年 5 月，IE 排名第一的位置就会被 Chrome 取代。

对于微软而言，IE 早已是被放弃的产品。在 2014 年 8 月，这家公司就宣布将停止老版本 IE 的更新维护，并要求用户更新到最新版的 IE 11。

按照 ComputerWorld 的说法，这一举措非但没有获得用户的支持，反而促使他们重新开始考虑换掉 IE。根据数据来看，自该消息公布至今，IE 的份额就已丢掉 13.7%。

被微软放弃的不仅仅是老版 IE ，而是整个 IE 的品牌。2015 年，该公司就为 Windows 10 系统默认的浏览器起了个新名字 Edge。这也意味着 IE 要彻底淡出了。取代它的 Edge 影响力依然很弱，毕竟后者只能在 Windows 10 系统上使用。

在浏览器市场表现不如意的也不只有微软。曾一度威胁 IE 垄断局面的 Firefox 的份额也一直在原地徘徊。去年年末他们还放弃了曾被寄以厚望的 Firefox OS 系统。

今日 Firefox 母公司 Mozilla 还宣布要和 Chrome 团队联手，去开发虚拟现实（VR）和浏览器结合的 WebVR 技术标准。至于虚拟现实是否能给他们带来新的机会，依然还是个未知数。
