惠普又来做手机了，这次能当 PC 用_智能_好奇心日报

赵长存 2016-02-23 03:02:35

你还记得惠普的上一部手机么？

在昨日的世界移动通信大会（MWC）上，惠普发布了它旗下一款 Windows 10 系统的手机 Elite x3。

Elite x3 的主要参数如下：

  • 骁龙 820 处理器、4GB 内存、Adreno 530 GPU
  • 内置 64GB 存储空间，最大支持 2TB MicroSD 卡
  • 1440*2560 分辨率的 5.96 寸屏幕
  • 1600 万像素后置摄像头，800 万像素前置摄像头
  • IP67 级防水、无线充电、虹膜和指纹识别

基本上就是手机里的最高配置。相比同样采用 Windows 10 系统的微软 Lumia 950 XL，惠普也更强。

采用高配置的原因之一就在于，Window 10 Mobile 已加入了一个名为 Continuum 的功能。用户可用鼠标、键盘控制接入显示器上的手机——手机被当做一台桌面主机使用。

此外，惠普还为之推出了一个笔记本底座 Mobile Extender。这个配件拥有 48WHr 电池，12.5 寸屏幕，全尺寸键盘和触控板。搭配它，Elite x3 就变成了一部笔记本电脑。

这种概念不怎么新鲜，2011 年摩托罗拉就为其 Atrix 手机配备过与 Mobile Extender 同类型的 Webtop 扩展坞。

但这个方案在一年后被放弃。如果要说惠普的新配件和摩托罗拉的 Webtop 最大的不同，就是它采用了无线链接的方式。即便手机放在背包或口袋里，你也能正常在键盘上打字。

但问题是，如果要想把它作为一台电脑使用，还是要面对不少问题。比如，针对 X86 处理器的应用就无法用它运行，而这也是 PC 用户继续用 Windows 的主要原因。

与此同时，使用 Windows 系统手机的用户已经越来越少。截至 2015 年第四季度 Windows 系统在智能手机市场的份额不过 1.1%。而在 2014 年末，这一数字还是 2.8%。

如此一来，开发者又为什么要花费精力，为它推出应用呢？

想靠 Windows 回归手机市场，惠普可能晚了六年。
