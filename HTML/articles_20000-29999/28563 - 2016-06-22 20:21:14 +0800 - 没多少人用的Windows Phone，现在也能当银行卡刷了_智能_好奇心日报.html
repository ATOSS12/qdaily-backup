没多少人用的Windows Phone，现在也能当银行卡刷了_智能_好奇心日报

唐云路 2016-06-22 20:21:14

Apple Pay、Android Pay 之后，微软钱包也来了……如果你身边还有人有 Windows Phone 的话

Windows 10 手机的用户，在线下实体店铺用手机支付的时候，现在可以不用那么羡慕苹果或者 Android 手机的用户了。

就像用 Apple Pay 和 Android Pay 一样，微软新推出的 Microsoft Wallet 应用，让拿着 Windows 手机的人，也可以把手机当银行卡刷了。

用户只要在 Microsoft Wallet 里绑定自己的 MasterCard 或者 Visa 银行卡，就可以在线下的实体店铺里消费了。 用手机靠近带有 NFC 功能的 POS 机，就可以完成支付。

微软在一篇官方博客里表示，这个项目将先在美国上线，用户加入 Windows Insider 内测项目之后就可以使用。目前在线下有超过 100 万个零售店铺里使用 Windows Wallet，只是首批支持 Windows Wallet 支付的只有三款手机：Lumia 950、950 XL 和 Lumia 650。

除了银行卡，Windows Wallet 还能够储存线下合作商铺的会员卡，在消费的时候，只要在手机上打开应用让店家扫描，就可以把手机当成实体会员卡用了。

如果你用过 iPhone 上的 Wallet 应用，对这些应该不会陌生。Windows Phone 的电子钱包功能来得比其他平台都晚不少，Apple Pay  2014 年在美国上线，三星也在 2015 年下半年推出了 Samsung Pay，最近华为、小米等也推出了自家的线下支付。

咨询公司 IDC 的研究总监 James Wester 表示，和相机、音乐播放器、GPS 导航一样，电子钱包已经成为消费者心目中，智能手机必备的工具之一。但是有没有移动支付功能，对于人们购买手机的影响也没有那么大。451 Research 的资深分析师  Jordan McKee 表示，三月的一份调查显示，只有 10% 的受访者会因为移动支付功能影响购买手机的决策。

即使如此，就 Windows Phone 目前的市场份额来说，增加移动支付的功能可能也无法让微软把手机卖得更好一些。
