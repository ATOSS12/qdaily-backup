赶在财报截止，特斯拉要求 Model 3 预定用户再交 2500 美元定金_商业_好奇心日报

王毓婵2018-06-29 14:53:34

否则拿不了车

据 CNBC 报道，特斯拉汽车公司本周向所有北美地区 Model 3 的预定用户发送电子邮件，要求他们额外支付 2500 美元定金以完成订单。

此前，Model 3 的定金是 1000 美元。特斯拉表示，如果不愿意加钱，买家可以在 3 天内取消订单，并获得全额退款，3 天之后将不再享受退订优惠。所有预先支付的定金都将被自动归入购车款。但是特斯拉没有给这些买家一个具体的交付日期，也没有表示车价是否会上升。

在第一季度致股东的信中，特斯拉透露称，Model 3 的净预订（包括尚未交付的配置订单）在第一季度末超过了 45 万份。所以，如果不考虑退订情况，预计此次加收定金的做法会给特斯拉提前带来 11.25 亿美元收入。

今年一季度结束时，特斯拉的现金余额为 27 亿美元。但当时该公司预计今年仍将支出至少 30 亿美元。加收定金也是为了缓解资金链的压力。

对于特斯拉来说，这又是一次冲击目标数字的努力。特斯拉首席执行官兼董事长伊隆·马斯克曾承诺，特斯拉不仅将在本季度末之前实现每周生产 5000 辆 Model 3 的目标，而且应该在今年下半年实现盈利和现金流转正。

这不是特斯拉第一次为了数字而牺牲其他环节了。《好奇心日报》早先报道，从 2016 年开始，每到三月、六月、九月，特斯拉的 Model S 和 Model X 交付量就会激增，因为这正是每季度财报截止前的最后一个月。而接下来的四月、七月和十月，交付量又会陡然下跌，环比跌幅达到 70% 左右。

举例来说，Model X 在 2016-2018 年，每年三月的交付量分别是 1860、2750、2825，而每年四月的交付量分别是 795、715、1025。

相比之下，美国市场的其它三款销量最高的电动车日产聆风、雪佛兰沃兰达和福特蒙迪欧插电混动的销量增长都比较平稳。完全没有这种随着财报节点而调控产能的情况。特斯拉这么做，只有一个合理的解释——将原本下季度交付的车往前赶，让当季的数字更好看。

但是 Model 3 是个例外。从今年一月开始，该车型的交付量一直在逐月上升，即使在四月和五月也没有下滑，因为特斯拉对它的预期是“每周生产 5000 辆”，而这个目标的截止日期马上就要到了。

在过去的几个月里，这个担负整个企业盈利希望的车型麻烦缠身。今年 2 月，特斯拉位于加州费费利蒙市的组装工厂里的 Model 3 生产线停工了五天。根据发言人的说法，停产在计划之内，有正面作用并且在业内也并不罕见。

但是到了 4 月，同一生产线再次停产了五天。特斯拉代表说，停产的时间将用来解决生产线的自动化问题，目标是提高生产率。开始在公司睡沙发的马斯克在 Twitter 上说，过度追求自动化是他的错误。伯恩斯坦分析师认为，这些机器人在一些传统汽车制造工序中，比如最终组装上却比工人差得太多。

5 月，特斯拉表示，3.5 万美元的基础版本 Model 3 生产计划将被搁置，公司将优先制造更贵的双电机车型。马斯克非常直白地在 Twitter 上说“现在造 3.5 万元的 Model 3，特斯拉就会亏钱亏到死。”Model 3 自此放弃了大众车型的路线，价格升至 4.9 万至 7.2 万美元之间。

6 月初，马斯克称，所有地区的 Model 3 周产能超过了 3500 辆，但油漆车间、3 号装配线、4 号总装线、终点线，以及 Gigafactory 1 号工厂的 4 号模块区都需要进行“彻底的改进”。不过 BI 披露的内部文件显示，特斯拉为了达到这个产量可能是“不惜后果”的。自 2018 年以来，整个生产环节特斯拉因为报废率高，在原材料方面就浪费了 1.5 亿美元。同月，公司宣布裁员 9%。

最近的消息，是本周特斯拉经理 John Totah 发 Twitter 称，特斯拉已经解决了 Model 3 的主要产能瓶颈之一——电池问题。为 Model 3 生产电池组件和驱动单元的 Gigafactory 1 号工厂已经率先完成了既定的生产目标。

不管怎么说，六月即将结束，我们预计很快就可以看到这一次全员冲击的成果。
