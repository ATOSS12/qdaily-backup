#Meme：知道这些生活“作弊代码”，让你行走江湖更easy_文化_好奇心日报

孙若空2018-11-23 14:27:30

同一个问题用收益的视角去描述和损失的视角去描述给人的感觉大大不同。

早年在红白机时代，不少游戏都隐藏着作弊代码（ cheat code ）。其中最著名的莫过于《魂斗罗》，在手柄按下“上上下下左右左右 BABA ，就可以获得 30 条命，大大降低过关难度。

那生活可以这样作弊吗？其实是可以的，只是要换一种方式。

本周， Reddit 用户 Sun_Zuu 发帖问大家，“你在生活这个游戏中发现过什么作弊代码吗？”没想到，真的收到了很多热心网友的认真回答，其中有不少确实很有道理。而且这些建议可以跨越种族、国家，在全世界都行之有效。

1、如果你对一家餐厅的服务和食物很满意，没什么要投诉的，那可以试试去恭维一下他们的经理。

网友 heelstoo 说，通常餐厅经理比较常处理投诉，但恭维话谁都喜欢，因为敢说恭维话，他已经好几次在餐厅获得免费的开胃菜、饮料或者折扣了。

（孙若空： 6 年间我在成都一家准备推出甜点业务的麦当劳夸他们试吃的芝士蛋糕好吃，然后经理大方的送了一块给我。这个建议亲测有效。）

2、如果你做错一件事，但勇于道歉并主动做改正，那么会令人更尊重你。

网友 methedunker 的这则建议其实很类似于我们常抱怨的后进生的进步比好学生的正常发挥更容易获得表扬。不过 methedunker 的角度更为积极。

3、网友 neurotica5000 是一位幼儿园的助教，她分享了一个轻松管理一群小朋友的小技巧

如果让小朋友围坐在一圈给他们上课，可以试试说“让我看看那个小朋友的‘坐下肌肉’和‘聆听肌肉’更厉害。”为了表现自己的“强壮”，小朋友们都会不约而同的“很乖”——虽然可能他们是在拼肌肉。

4、如果你要一个人走走，那么网友Just_Curious_Dunder 建议给你的这次出动赋予一个目的。这样一来大家就会觉得你很忙，不来打扰你。

5、网友 tenderbranson301 说没有人会阻拦一个带着披萨的姑娘或小伙。所以你可以用这方法走到一个后台去听音乐会。这条建议可能需要身处美国的朋友来实测一下了。

6、mippnles 提了条对于想背着小朋友藏零食的爸妈挺管用的建议，但我们使用时得化用一下。他说，外国的 Costco 超市有一种可以重新密封的装蔬菜的袋子。家长们可以把冰激凌藏在这些袋子里，尽量分散些藏，这样孩子就找不到了。然后孩子们不注意的时候偷偷吃就行了。

（如果你没有孩子，对这条建议不明就里，可以上网上搜搜那些家长藏零食被小朋友戳破的视频看看。）

7、insertcaffine 觉得，面对赞美，最好的回应应该是谢谢。如果谢完你还想说些什么，可以再去谈谈这件事。比如有人说喜欢你的衣服，你可以说，谢谢，它还有个口袋很实用。如果有人说喜欢你的演奏，你可以说，谢谢，我一直在练习。这样一来，既可以让你显示自信，又不会太过自信。

8、“作为一个成年人，你冲任何一个在跑步的孩子说不要跑步，他们就会停下来不跑。”网友blankbeard 说。

9、网友 Taiwanderful 说，当有人说的话你也这么认为，说一句“你是对的”比说“我也这么想”更让别人舒服，而且也展示了你的聪明。

10、承诺不要说太满，做事反而要做过。网友 Nrich 5 说，如果一开始调低预期，然后努力做到超出这个预期许多，那么大家都会更认可你。

11、HydroSword 说，他的人生 cheat code 来自他爸爸。他爸爸说，不要想着出门交朋友，想着出门不要交敌人就好。

以上种种待人处事的小方法论现在有了一个统一的名字， cheat code 。其实，这些建议中有不少只是教大家一些改变说话角度的小技巧，但行走江湖，避免摩擦、与人舒心、与己方便才是提升日常幸福感的根本。所以会说话很重要。

而这种会说话的能力其实可以举一反三。李永乐老师比较新的一节课“如何说话让人喜欢？怎么做生意更吸引顾客？”里教授了一种框架效应的概念，同一个问题用收益的视角去描述和损失的视角去描述给人的感觉大大不同。很多人觉得这个问题很简单，但是，知易行难呀。
