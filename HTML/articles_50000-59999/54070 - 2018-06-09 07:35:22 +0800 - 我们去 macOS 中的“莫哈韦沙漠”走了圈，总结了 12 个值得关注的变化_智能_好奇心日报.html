我们去 macOS 中的“莫哈韦沙漠”走了圈，总结了 12 个值得关注的变化_智能_好奇心日报

罗骢2018-06-09 07:35:22

一次修修补补性质的系统升级

从猫科动物到绵延山脉，这一次最新的 macOS 10.14 使用了苹果总部所在地加利福尼亚州中的莫哈韦沙漠（Mojave）来命名。沙漠中早晚的光影变化，对应了 macOS 10.14 中最大的一个更新深色模式 Dark Mode。

在新版本中，苹果改进了 macOS 内诸多细节，增加了一些呼声已久的功能，例如更加护眼的深色模式 ，整理桌面的堆栈，升级改版的 App Store 等。Mojave 更像一次完善性的“小”升级，同时也进一步增加 iOS 和 macOS 之间的联系。

并不像 iOS 12 开发者版本目前较为流畅的体验，macOS 10.14 中还是存在众多 Bug，建议升级前用 Time Machine 做好备份。

我们测试了苹果表示已经适配 Mojave 的几款系统应用，包括 Finder（访达）、App Store、Message、Keynote、Safari、Calendar、Xcode 等应用。还有一些日常会使用的应用，例如腾讯视频、爱奇艺、优酷、有道词典、微信、QQ等。

适合夜晚工作的深色模式 Dark Mode

在开发者版本安装好之后，就会提示用户是否选择深色模式。你也可以在系统设置—通用-外观中看到两种模式的选择。

在深色模式下，整个 macOS 的界面都会统一调整为灰黑色，并使用深色的背景，包括右上角显示的消息，屏幕右侧划出的内容中心等都作出了变化。这样在夜晚的情况下，减少屏幕上的白色来避免眼睛疲劳。

 

另一个较为明显的变化是顶部的菜单栏。为了和整体统一，苹果将几乎所有顶部常驻的应用图标都换成了灰白色。

「深色模式」的工作原理并不是简单的反转配色，是一个需要应用开发者主动适配的功能。由于是开发者版本，大部分常用的软件都没有适配深色模式。

由于 Bug 问题，Keynote 在我的 Macbook Pro 13.3 寸 2017 款电脑上无法正常打开，每一次都提示崩溃。而令人奇怪的是，Xcode 虽然可以正常使用，但是并没有出现深色模式，界面还是白色的。在其他苹果应用中，深色模式适配得很完善。

在文件管理器 Finder 中，明暗不同的黑色区分了窗口内的操作区域，字体则通过白色来显示。顶上的菜单栏内里，背景也调整为为黑色。Message 中苹果则采用了磨砂的效果来区分联系人界面和内容界面。

变化较大的还有日历界面，为了在黑色背景下突出日程事项，苹果使用了紫色、黄色等亮度较低的颜色来提示。通过字体颜色变化来区分本月的时间。

在一些需要重点强调的地方，Dark Mode 下苹果选择了白色来突出。例如 AppStore 内付款选项都是纯白色。

有一个需要注意的问题是，例如 Safari 本身界面都呈现黑色主题，但目前大部分网站都是白底的，晚上突然打开依然会很刺眼。并不知道后续会不会有方法来解决这个问题。

我们还测试了旧版本的 Keynote、Numbers 这类的应用也还没有支持黑色模式。同样，日常使用的 QQ、微信、优酷、QQ 浏览器、Chrome 等常用应用都没有适配。

反而是之前就使用了黑色主题的腾讯视频、爱奇艺以及 Final Cut Pro、Photoshop 、Lightroom 这类自带黑色模式的应用看上去和深色模式很匹配。

如果追求最好的效果，我建议在外观设置中将强调色换成灰色，同时高亮显示颜色也换成灰色（Gray），整体会看得更加舒服。

还有一些细节是，垃圾桶的颜色也变成了黑色，同时最近使用的 3 个应用也会出现在 Dock 上，这是 10.14 中的变化.

随时间变化的桌面 Dynamic Desktop

另一个在发布会上演示的特点是按时间变化的沙漠壁纸。开启这个功能后，桌面上的沙漠壁纸会随着现实时间变换着环境。

这个功能被称为 Dynamic Desktop，常规的外观模式和深色模式都可以按时间变化，只不过深色模式只有两种天色变化，而常规的模式下，壁纸会随着早中晚的时间有 5 种不同的天色。

想要开启，首先必须关闭系统设置—安全性和隐私—隐私中的定位服务。然后在设置—桌面和屏保 中选择 Mac OS 10.14 中第一个壁纸 Mojave（Dynamic）。随后就可以调整时间直接查看壁纸的变化或者在日常使用中发现天空的不一样吧。

自动把桌面文件归类的堆栈功能

堆栈是 WWDC 上让场下开发者惊呼的一个功能。但主要还是因为瞬间整理桌面的效果看上去让人印象深刻罢了。

实际上，堆栈并不是将文件分类新建文件夹整理了，只是 macOS Mojave 提供的视觉效果，文件在桌面上自动收纳进相应的类型。文件管理器中的桌面依然会显示所有未归类的文件。堆栈点击展开后，对应类型的文件会重新出现在桌面上。

堆栈的开启按钮在桌面菜单栏的显示栏中或者在桌面上右键打开，开启后，系统可根据文件「种类」、「上次打开日期」、「添加日期」、「修改日期」、「创建日期」、「标记」等分类来生成对应的堆栈以及更改堆栈内文件的排序方式。

简单来说，这有点像一个会被设置好类型后，就会自动收纳文件的地方。但只能收纳文件，我们测试中堆栈并不能收纳文件夹或文件夹中的文件。同样不会被归纳的还有 dmg 这类的安装包文件。

将 iOS 上的截图功能放到了 macOS 中

Mac OS 10.14 中对于系统的截图也进行了改变，将 Mac 系统的截图做的和 iOS 一样。

在用 shift + command + 数字键 3、4、5 截图后，屏幕右下角会出现截图的预览，点击预览图片，可以给截图进行标注，分享或者删除等操作。这是 iOS 11 上的截图功能，提升了 Mac OS 中对于截图的操作，也是很实用的功能。

而这次还新增了截图模式自定义 ，按下 shift + command + 数字键 5 后就可以在屏幕中选择截图功能的各种选择，从截图范围，到录制屏幕再到储存位置都可以进行改变。

截图的编辑中，10.14 还增加了两个有趣的工具，一个是放大镜功能，可以把截图画面进行放大突出，另一个是区域高亮标识，降低你想要突出的区域之外地方的亮度，用阴影来显示出标注区域，有点像视频网站上的关灯功能。

Finder 中 Gallery View 模式

这一次 Mac OS 为系统内多个应用都增加或升级了功能。在文件管理器 Finder 中，苹果增加了一个新的浏览模式 Gallery View。

这种模式下，文件夹中的所有文件会用中间大图加底部滚动条的形式展示。同时在右侧的区域内会出现文件的详细信息，例如图片中的 EXIF 信息、创建时间、尺寸等信息。

Gallery View 模式可以更加直观地查看每个文件本身和信息，而且苹果为了增加工作效率还在这种模式界面的右下角增加了一些简单的「Quick Action」快捷操作。

例如图片文件可以直接在右下角进行旋转、标记、创建 PDF、加密等操作。视频文件则可以进行修剪、添加密码等操作。

除了照片之外，Gallery View 模式还可以预览文档、视频、PDF 等文件。此外 Finder 还新增了根据用户选择，给出操作建议的功能，比如在选中多张照片时，Finder 会提示用户「是否用这几张照片合成一组幻灯片？」

在我们的测试中，Gallery View 模式目前有一个问题在于不能在已按不同标签进行过排序的文件夹中使用。排序后的文件夹内开启 Gallery View 模式只会显示第一个文件，下方并不会出现滚动条。

预览功能加入了更多编辑功能

在 macOS 中选中文件后点击空格键可以进入预览模式。在 macOS 10.14 中，苹果加入了更多的编辑功能，从一个预览工具变成一个轻度编辑工具。

现在，可以直接在预览中对 PDF 文件、图片文件进行标注、画圈或画图等操作，实际上，是把之前系统自带查看图片功能中的标注直接放在了预览中。

同时，现在直接空格预览 Mp3、M4A 文件会直接播放声音，使用时候要注意。

重新改版的 App Store

不只是在截图等功能上参考了 iOS ，在 Mac OS 10.14 中，苹果将 APP Store 进行了较大幅度的改版，直接参考 iOS 上的应用商店，调整了整个界面和内容展示的方式。

在新的 App Store 中，苹果在左侧按照探索（Discover）、应用的不同分类、已有应用的更新来增加了 7 个栏目。

其中探索可以理解为首页推荐，在其中置顶的部分编辑会每周推荐某款应用，按照应用本身的功能，应用开发者幕后的制作理念和故事等信息来展示整个应用。你也可以将这篇推荐文章分享出去。

在下面较小的位置，放上了编辑的话、专题等内容。macOS 上 App Store 的编辑描述了这次重新设计应用商店背后的思考。

应用的详情页也进行了改版，和 iOS 的样式一样。由会自动播放的视频和图片进行介绍，同时下面会展示评分和较高认同的评论。

另一个要注意的是，下载应用的速度比之前快很多了。几乎不会再出现慢吞吞才能下载完或是频繁断开连接的下载体验。

macOS 上新增的 iOS 应用

虽然在发布会上苹果表示目前不会将 iOS 和 Mac OS 应用合并，但是苹果依然在尝试将 iOS 应用引入到 macOS 中，在 Mac OS 10.14 中，苹果就把手机上的新闻、股票、语音备忘录等应用添加到了系统中。

股票应用在深色模式下看起来很舒服，你可以在左侧边栏中选择要关注的股票和市场变化。第一个则默认显示市场的新闻动态。苹果使用了雅虎的股票数据源。

语音备忘录和控制智能家居的产品 Home 应用都比较简洁，也和深色模式进行了匹配。

由于受限于网络和政策，Apple News 在国内的 10.14 开发者系统中没有自带。从截图上看，应用的界面和股票类似，在左侧边栏可以选择相应的新闻源。最顶上则默认为今日新闻。

苹果在 WWDC 2018 上表示，让 macOS 兼容 iOS 应用是他们目前正在进行的工作。现阶段，他们已经将 iOS 系统的部分框架带入了 macOS Mojave 中，并计划在未来将更多 iOS 应用移植到 macOS。不过目前该计划仍处于「第一步」，可能要到 2019 年才能看到更多成果。

进一步保护用户数据

macOS 10.14 中另一个较为重大的改变是对用户隐私数据的管理。

首先是应用调取用户定位、照片、打开摄像头等行为时都会进行提示，这和 iOS 上是一样的，增加了必须要提示用户的环节。

然后在 Safari 浏览器的设置中，已经增加了对于网站跟踪 Cookie 的设置，用户可以在系统偏好—隐私中勾选“阻止跨站跟踪”和“要求网站不要跟踪我”。

这样避免了你在一个网站搜索过一个东西后，会发现你看任何网站都会给你推荐同一个广告内容。同样，用户还可以阻止所有的 Cookie 记录。

还无法体验的 Continuity Camera

在 macOS 10.14 之前, 我们如果用 iPhone 截图或者拍了照片, 需要使用 Airdrop 来发送到 Mac 上进行编辑，或者做 Keynote 要用到手机上的照片也需要进行传输。

现在不需要了，在 Mac 上可以直接激活手机的相机进行拍摄，拍摄完成后照片立刻就到了 Mac 上可以进行继续编辑。

但目前这个功能因为 Keynote 尚未推出新版而没办法体验。

系统的自动升级

在 macOS 中，以往需要用户主动去 App Store 中查询最新系统的更新，而现在，系统偏好设置中加入了新的软件更新功能。

用户可以在其中勾选“自动让我的 Mac 保持最新的更新”。同时也可以选择更详细的更新选项。包括检查更新、下载更新、安装更新各个环节都可以控制。

一些值得注意的细节

macOS 10.14 中有些应用会改成中文的名字，例如 Mission Control 会改名为「指挥中心」 、Dashboard 改名「仪表板」、Launch Pad 正式改名为「启动台」、Time Machine 正式改名为「時光机」，Bootcamp 辅助程式重新改名为「开机切换辅助程式」。

升级前，一定要先设置好密码，因为空密码将无法登陆开发版的 Mojave，同时第一次登陆直接按 Enter 并没有反应，这也是小 bug 之一。

我使用的罗技 MX Master 2S 无线鼠标则遇到了滚轮无法正常使用的问题。

总体来说，macOS Mojave 是一个功能比上个版本更加完善的系统，也在工作效率方面进行了改善。同时，苹果也在尝试手机和电脑之间更加直接的联系。

正式版将于今年秋季上线，目前只能加入苹果开发者计划下载升级。同时新系统也只支持 2012 年之后推出的苹果 Mac 系列硬件。
