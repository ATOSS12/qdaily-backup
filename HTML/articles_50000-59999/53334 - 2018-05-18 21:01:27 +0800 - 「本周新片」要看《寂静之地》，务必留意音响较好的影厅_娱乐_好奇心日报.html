「本周新片」要看《寂静之地》，务必留意音响较好的影厅_娱乐_好奇心日报

魏倩2018-05-18 21:01:27

声音变成情节的主要力量之一，是怎样一种体验？

《寂静之地》

  • 导演：约翰·卡拉辛斯基
  • 编剧：布莱恩·伍兹 / 斯科特·贝克 / 约翰·卡拉辛斯基
  • 主演：艾米莉·布朗特 / 约翰·卡拉辛斯基 / 米利森特·西蒙兹 / 诺亚·尤佩 / 凯德·伍德沃德
  • 片长：91 分钟（中国大陆）
  • 格式：2D

在一个世界，你绝对不能发出任何声音，否则就会被巨大的怪兽吃掉——这是影片《寂静之地》的原始设定。一种听觉灵敏的怪兽入侵地球，人类被其大肆屠戮。一家人躲在郊区的农场里安静生活。令人欣慰的是，他们的两个孩子已经习惯了这种生活。但母亲伊夫林却发现，自己又一次怀孕了。

观看《寂静之地》的体验远比简介中还要恐怖，“声音”是本片最大的卖点。在影片前 1/3 部分，如异形般的怪兽神出鬼没，始终没有展露形貌，但身边人的可怖死法却让观众由衷地为主角祈祷着：千万，千万不能发出声音。

于是，依靠屏幕内外的互动，影片完成了恐怖片最重要的任务，通过轻微的噪音，让观众与故事感同身受，当主人公踩到钉子的时刻，影院内几乎所有观众都默默吸了一口气。

与其纠结“为什么主人公一家不搬到瀑布边生活”和”为什么人类不用炸弹消灭怪兽“等古怪 bug，《寂静之地》其实还创造了另外一种恐怖感，即长久的压抑生活究竟会对人类的成长造成怎样的影响。在感到幸福和恐惧时，我们会发出尖叫，这是一种表达的原始冲动，而影片直接性地将本能变成了自杀的手段。这种扭曲的设定，会让观众同时感受到解脱和伤感的多重复杂体验。

《寂静之地》的导演约翰·卡拉辛斯基曾经是一位喜剧演员，他之前执导的作品多为电视剧。因此，也很容易从片中的家庭生活细节里体会到类似《吸血鬼猎人巴菲》的精致和简洁。

而作为一部以“寂静”为名的作品，影片的声效是为之加分的最好方式。除了前文提及的临场感，导演为乡村、瀑布、农舍等不同场景选择的背景音乐，对听觉敏感状态的观众（和怪兽）都发挥了巨大的影响力。在某些部分，它甚至成为推动情节进展的唯一力量。

因此，如果你本周决定去观看《寂静之地》，务必留意音响设备较好的影厅。

《超时空同居》

  • 导演：苏伦
  • 编剧：苏伦
  • 主演：雷佳音 / 佟丽娅 / 张衣 / 于和伟 / 王正佳
  • 片长：101 分钟（中国大陆）
  • 格式：2D

本周的《超时空同居》算是近期尴尬度较低的一部爱情片。很难得。

一间时空重叠的公寓，身处 2018 年的谷小焦与 1998 年的陆鸣意外“同居”在了一起。这种新奇的相遇方式使那些常规的互相爱慕过程显得趣味十足。而影片对陆鸣在 1998 年生活的细节描绘，则展现出类似“请回答 1988”的复古观感。

两相结合，影片前半段笑点和节奏控制均属上乘，简短的爱情小品流畅动人。这一点也得益于雷佳音的表演，虽然他和女主角间“cp 感”还是弱了一些，但和同类影片中的男主角相比，已经堪称自然。

但到了影片后段，编剧对整个故事的控制力开始下降。两位主角间的价值观碰撞、时空穿越的桥段对情节已经失去魅力。此时，陆鸣的前后反差表现也就呈现出些许国产电视剧质感——说到底，还是“男人有钱就变坏”的平面形象。

因此，作为简单轻松的爱情片来说，《超时空同居》算是合格，可惜和两周前同样由徐峥监制的另一部影片《幕后玩家》类似，它的开头要远好于结尾。

《路过未来》

  • 导演：李睿珺
  • 编剧：李睿珺
  • 主演：杨子姗 / 尹昉 / 李勤勤 / 王婷 / 娜仁花 
  • 片长：127 分钟
  • 格式：2D

本周四，青年导演李睿珺的新作《路过未来》开始在全国艺术电影放映联盟的合作影院专线放映。

去年，这部影片曾入围第 70 届戛纳电影节“一种关注”单元。和李睿珺之前导演的《告诉他们，我乘白鹤去了》和《家在水草丰茂的地方》两部作品类似，《路过未来》同样展示了一种复杂的情绪体验。

影片讲述了一个“异乡人”的故事。从小在深圳打工的耀婷与父母回到甘肃老家，短短几天内，她遭遇了父母赡养、买房卖房、健康医疗等诸多麻烦。在抗争和妥协之间，耀婷发现自己已经成为生活的旁观者。

不得不说李睿珺选择了一个有趣的视角，出生于甘肃的他在影片中呈现出家乡人口流失的现状和诸多边缘人的生活情态。可惜的是，导演“通过影片反映社会问题”的企图表现得过于明显，众多想象中的边缘危机被集中在主角耀婷身上，让影片对具体事件的讨论显得流于表浅。

事实上，他前几部作品中惯用的散文化风格并不适合这类题材表达。到观影过程中段，观众很难不感受到诗歌般的漫长和拖沓。如何统一风格和叙事间的关系，可能是艺术电影联盟的类似影片进入市场前必须考虑的一些问题。

本周还有一部电影版《昼颜》

本片 2017 年已在日本上映过，如果不是特别喜欢电视剧版《昼颜》，还是不推荐影院观看了。事实上，如果喜欢电视剧版本，你可能会更失望吧。
