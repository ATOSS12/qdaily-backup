C 罗1.12亿转会干脆利落，尤文股票累积上涨近 40%_商业_好奇心日报

宣海伦2018-07-11 17:23:02

爽快的东家、宽容的球迷和友好的税收政策，C罗选择尤文图斯有太多理由了

无论你是哪支球队的拥趸，任何联赛的球迷，抑或你不用是多么虔诚的足球迷——当皇马球星 C 罗转会尤文图斯的消息传来，大抵都能感受到这条消息背后的震动。

西班牙最大的体育报纸《马卡报》在头版头条写道：“不会再有像 C 罗一样的球星了，C 罗走了，只留下了 451 个进球。”封面铺满了 C 罗在过去 9 年为皇马攻入的全部进球。

5 届金球奖、15 座俱乐部奖杯、以及史无前例的欧冠三连冠记录，有超乎常人的自我追求和身体素质，33 岁的 C 罗依然能在世界杯单场独进三球，谁能对他的职业生涯轻下结语呢？

不同于 2017 年的转会风波，这一次 C 罗走得干脆利落。很多人刚刚准备上树（形容球迷苦等转会消息，不官宣不下树），今年夏天的“C罗去哪儿”大戏似乎才拉开序幕，一切已经尘埃落定。

皇马在今天凌晨宣布当家球星 C 罗以 1 亿欧元转会尤文图斯，将分两年付清，另有 1200 万为青训补偿费用和附加费用。按照国际足联规定，尤文图斯须向里斯本竞技和曼联支付 500 万欧元的青训补偿费用，附加费用为经纪人佣金等。

尤文图斯非常爽快地为 C 罗送上了 3000 万欧元的年薪合同，相比 C 罗此前在皇马的 2100 万年薪大幅提升，在 2018 年初的球员薪资排行榜上 C 罗仅排名第六。

1.12 亿欧元在如今高度溢价的转会市场里并不是多惊人的数字，考虑到 C 罗能够为尤文图斯带来的想象空间，很多人觉得“老妇人”赚了。但作为一名 33 岁球员的转会合同，这个数字恐怕很难被超越。巧合的是，同为皇马功勋的劳尔离开时同样是 33 岁，当时劳尔以自由转会的方式免费加盟沙尔克 04，年薪为 400 万欧元。

转会消息官宣后，尤文图斯在意大利米兰上市的股票 Juve.MI 应声上涨 5.77%，报收于 0.898 欧元，总市值为 9.05 亿欧元。自 7 月以来，一直独占 C 罗转会话题的尤文图斯股价累计上涨近 40%。

C 罗最后选择了近年来略显萧瑟的意甲，让很多人出乎意料。但实际上，尤文图斯几乎就是眼下最适合 C 罗的选择。这笔交易对于皇马、尤文来说都相当满意，转会进程如此顺利也是情理之中。

首先，C 罗在皇马已经无法得到全心的支持，除了加薪谈判无法达成一致，皇马对 C 罗开出的众多附加条款，包括将成绩、个人表现与涨薪挂钩等要求令葡萄牙人感到不被信任。

而另一方面，皇马对于内马尔、姆巴佩这些更年轻的选择，又表现出了强烈的兴趣和疏爽大方。随着 C 罗坚定的支持者齐达内的离开，转会注定要在今夏发生。

其次，相较于英超和西甲，意甲宽松的纳税环境对海外球员也极具吸引力。根据意大利近期出台的财政政策：海外纳税者为其海外收入缴税时，最低可以仅缴纳 10 万欧元。这对于多次陷入避税纠纷的 C 罗来说是再好不过的事。C 罗此前曾向西班牙税务局支付了 1650 万欧元的罚款以免除牢狱之灾。

在俱乐部实力层面，刚刚完成意甲七连冠的尤文图斯财政情况稳步增长，阵容班底臻于成熟，教练阿莱格里也已续约到 2020 年。屡战而不可得的欧冠冠军是尤文图斯的执念，C 罗将成为尤文图斯冲击欧冠最锋利的杀器。

除此以外，C 罗作为球员本身极富商业价值。C 罗的转会将为尤文、甚至意甲带来大量流量和商业价值。C罗是第一个在社交网络突破 2 亿粉丝的运动员，Business Insider 分析 C 罗一年在社交网络上发布的照片与视频，能为耐克公司创造高达 5 亿美元的商业价值。

市场调研公司 Opendorse 则综合 C 罗的社交媒体粉丝数量、粉丝活跃度以及粉丝质量等因素，估计 C 罗的一条社交媒体消息价值 23 万欧元。而尤文图斯在社交媒体方面表现一般，以 2016 年为例每 100 万社交媒体粉丝为尤文带来的收入不足 10 万欧元。目前队中的当家球星迪巴拉在 Instagram 上的粉丝量（1780 万）不及 C 罗的零头（1.33 亿）。

尤文图斯的比赛日收入从 2012 年的 1160 万欧元增长至 2014 年的 4000 万，但近几年这一数字趋于平稳，再无较大起伏。C 罗的到来将为尤文图斯带来新的收入增长点。
