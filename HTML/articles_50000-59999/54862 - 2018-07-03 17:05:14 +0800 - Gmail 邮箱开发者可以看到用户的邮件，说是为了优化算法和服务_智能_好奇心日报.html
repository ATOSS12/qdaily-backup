Gmail 邮箱开发者可以看到用户的邮件，说是为了优化算法和服务_智能_好奇心日报

周韶宏2018-07-03 17:05:14

但谁能知道是否存在滥用行为？

一年之前 Gmail 放弃邮箱广告，承诺不再扫描用户的邮件内容用于广告营销。原本这是一项收入来源，Gmail 用户喜欢看什么买什么，数据用于精准推送广告。

不过 Gmail 并没有阻止邮箱服务商继续读取用户的邮件。

《华尔街日报》周一报道，多家第三方邮箱开发者可以通过 Gmail 获取用户邮件信息。区别于原生 Gmail，这些软件用于邮箱聚合，提供一些附加功能比如超大附件的存储和发送。

这类似于给 Gmail 加了一层包装，开发者通过 Google 的接口就能直接访问 Gmail 收件箱。

其中一家是名叫 Return Path 邮箱营销公司。“我们了解邮件。”Return Path 在领英主页上介绍说，该公司帮助企业优化邮件营销的效率和效果，他们接入超过 70 个邮箱服务，包括 Gmail、雅虎邮箱、微软邮箱，覆盖 25 亿个收件箱，这是全球总数的 70%。

知情人士透露，一年半之前 Return Path 的员工阅读了超过 8000 封用户邮件，用于训练他们的软件算法。鉴于他们的主要业务就是提供广告技术方案，这等于用户的邮件内容被直接用于广告营销。

另一个案例来自 Edison Software 公司，另一个 Gmail 邮箱开发者，他们推出一款 iOS 移动应用帮助用户管理和收发邮件。为了优化“智能回复”功能，他们也曾让员工阅读“数百名”用户的邮件内容。

尽管 Gmail 不再偷看邮件，但 Google 依然允许第三方开发者这么做。 Google 表示开发者经过审核，还要得到用户的许可才能访问电子邮件。通常情况下访问也是出于安全目的比如排查 Bug。

Edison 和 Return Path 都表示他们在用户协议里做了申明，会在特定情况下读取邮件。只不过谁会看用户协议呢？

Google 一名发言人说，Gmail 团队会人工手动审核每个开发者，防止开发者滥用数据，确保他们都受到信任。遇到没有清晰披露隐私政策的做法，Google 会“立即展开行动”。

目前不能确定这场隐患涉及的具体规模，包括 Edison 和 Return Path 在内的邮箱服务还会使用机器人扫描邮件。也没有直接证据表明他们存在 Facebook 那样滥用数据的行为。使用用户数据来优化自己的算法是邮箱服务界的惯常做法，雅虎和微软等绝大多数邮箱服务都提供接口给第三方。

Google 在也隐私越界上谨言慎行，不久前刚刚推出一个隐私数据管理页面，用户可以看到自己在 YouTube、搜索、地图等等应用上的数据去向，选择删除、下载或保留。

这不能消除用户的担忧，尤其是 Facebook 丑闻后，全世界的互联网用户陷入了对个人隐私的担忧。不过用户也没有选择，对于第三方邮箱潜在的风险只能选择停止使用了。
