特斯拉公布产能提升进展，6 月最后一周里造了 5031 辆 Model 3_智能_好奇心日报

罗骢2018-07-03 07:05:01

如果能维持下去并且降低成本，特斯拉就解决了它最大的问题。

北京时间 7 月 3 日凌晨，特斯拉公布了第二季度的生产情况。在过去的三个月里，特斯拉总共制造了 53339 辆汽车，比第一季度增长 55％，成为特斯拉历史上产量最高的季度。

这其中受市场关注的 Model 3 型汽车的产量为 28578 辆，也是第一次超过了特斯拉其他两款车型 Model S 和 X 的生产总和 24761 辆。

在刚刚过去的一周时间内，特斯拉生产了 5031 辆 Model 3 和 1913 辆 S 型和 X 型车。周产 5000 辆 Model 3 型汽车是马斯克今年 4 月两次下调 Model 3 生产计划后设定的目标，终于在紧赶慢赶中完成了。

差不多也正是去年这个时候，特斯拉宣布开始量产第一辆 Model 3 型电动汽车，这款售价 3.5 万美元的电动汽车备受市场期待。作为特斯拉最重要的一款大众化乘用车，Model 3 的产量直接决定了特斯拉是否能实现真正盈利。

但随后的产能情况远远达不上马斯克在交付 Model 3 时公布的生产计划，当时他表示 Model 3的产量会实现指数级增长，在 9 月实现 1500 辆的产能，然后逐渐将生产速度提升到每周 5000 辆，争取到 2017 年 12 月实现月产 2 万辆。

实际的情况是 2017 年第三季度特斯拉生产了 260 辆 Model 3 ，只有计划的 1.3%。第四季度，特斯拉交付了 1550 台 Model 3 轿车。

到了 2018 年，特斯拉的麻烦更是不断。首先是 3 月底的 Model X 起火导致乘客死亡，随后彭博社开始对 Model 3 车辆识别号码进行追踪，报道称目前特斯拉每周大概生产 1190 辆 Model 3，与之前的目标相差甚远。

有媒体称，特斯拉迟迟未能解决生产问题，主要还是自动化生产推进太快，反正成品率低，达不到手工制作的效率。特斯拉在 2 月和 4 月分别宣布暂停生产线来解决生产线的自动化问题，目标是提高生产率。马斯克也在生产线工厂扔了一个睡袋，住在工厂解决产能问题，“汽车商业就是地狱”是那段时间他常说的话。

为了完成周产 5000 辆 Model 3 的目标，特斯拉开始“不惜成本”地冲产能。上个月月初 BI 披露的内部文件显示，特斯拉千兆工厂有 40% 的电池原材料在生产中成为不合格品，这些不合格品中一半在返工后重新使用，剩下的直接报废。自 2018 年以来整个生产环节特斯拉已经因为报废率高，在原材料方面就浪费了 1.5 亿美元。借此在报废率较高的情况下加快生产。

最终的结果是特斯拉第二季度的产能增加了 55%，完成了周产 5000 辆 Model 3 型的目标。在官网的报告中特斯拉表示，预计下个月月底前产能增加到每周 6000 辆。

虽然目前交付了 28386 辆 Model 3 ，但二季度积压的订单还有接近 42 万辆。特斯拉并不愁造出车后的销量，但能否能继续维持这样一个季度增长 55% 的产能速度是一个新的问题。

毕竟马斯克不会天天住在工厂里，而不计成本来冲产量的模式也会进一步挤压特斯拉本就紧张的现金流。而且，造 3.5 万美元一辆的 Model 3 对特斯拉来说利润并不高。

马斯克 5 月底在推特上说，“现在造 3.5 万元的 Model 3，特斯拉就会亏钱亏到死。”，随后宣布发布了更贵的 Model 3 的新型号，双电机顶配型号卖 7.8 万美元。

同时他表示，当 Model 3 周产达到 5000 台之后的 3 - 6 个月，低配车型将恢复量产。
