腾讯又发布了一批内容开发计划，IP 仍然是今后项目重点_娱乐_好奇心日报

周哲浩2018-04-25 07:15:26

项目的角度从娱乐拓展到了文化。

近日，在以“向爱而生”为主题的 UP2018 腾讯新文创生态大会上，腾讯发布了一批影视、动漫、游戏等方面的计划。从谈论的内容和实际的计划来看，IP 仍然是之后的重点。

六年前，腾讯提出了泛娱乐战略，要围绕核心 IP 衍生出小说、漫画、动画、游戏、电影等不同领域的内容产品。腾讯动漫、阅文集团、腾讯影业等品牌之后纷纷成立。在今年的发布会上，腾讯集团副总裁、腾讯影业首席执行官程武表示，腾讯要从泛娱乐向新文创升级。简单说，就是强调对 IP 的理解和塑造更加关注。

从各个娱乐版块的计划来看，IP 的存在感也很足。这其中不仅有腾讯自己的 IP 在不同领域的互动，比如从腾讯动漫上走出来的《我叫白小飞》出现在了腾讯影业的片单中，还包括与其他公司的合作，比如腾讯动漫与敦煌研究院和漫画家蔡志忠将一起传播敦煌文化——这也是新文创语境下的一个案例，即把敦煌这样的文化标志视作一个可以开发的 IP。

像上文提到的《我叫白小飞》以及与蔡志忠的合作，在过去其实都已经传出过消息。为期两天的 UP2018 腾讯新文创生态大会是一个集体展示的平台。

腾讯影业的片单上分为时代旋律、东方故事（也可以看做新文创的案例）、次元破壁、中国科幻和国际探索五个系列单元，包括已上映和计划两个方面。从 2015 年成立以来，相比制作，腾讯影业更多是以投资合作的方式输出作品。

新发布的片单上，最令人瞩目的恐怕便是与美国影视公司 Skydance 就新版《终结者》第一部的合作，腾讯影业将投资出品电影并负责中国地区的发行。开创了终结者系列的詹姆斯·卡梅隆将担任制片人，《死侍》的导演蒂姆·米勒执导。电影预计下个月开机，2019 年 11 月北美上映。阿诺·施瓦辛格以及琳达·汉密尔顿主演。

腾讯影业还成立了全资子公司腾影发行，将负责电影的发行环节。《头号玩家》是腾影发行公司参与的首个项目。根据介绍，腾影发行已覆盖中国内地 85% 票房份额的地区。在发行方面，腾讯已经是猫眼的投资方，程武表示，腾讯自建的腾影团队能够依托腾讯，产生更有创意的发行方式。

腾讯拿到古龙作品的版权以及对《王者荣耀》的开发，也都是 IP 方面的例子。

腾讯与古龙长子郑小龙达成了古龙作品十年全系列、全版权的合作，这意味着影视、游戏、动漫等领域都有可能开发出古龙武侠相关的内容。

已经成为现象级的《王者荣耀》的计划更具体，网综《王者出击》和历史文化节目《王者历史课》已经推出，由《全职高手》作者蝴蝶蓝创作的授权小说《王者时刻》已经发布并已进入影视策划阶段。另外，《王者荣耀》将与腾讯动漫推出官方漫画，衍生品等也在开发中。

从单纯的娱乐拓展到文化或者是文化与娱乐的互动，固然是拓展了 IP 衍生的范围，不过一个问题在于，这样跨界后的 IP 是否能够像在娱乐本身环节内互动的 IP 那样有吸引力，毕竟娱乐和文化，尤其是被腾讯之后视为重点的传统文化之间的受众重叠度未必很高，抱着娱乐动机的人应该不会借助娱乐产品获得文化相关的知识，反过来也一样。
