Model 3 三季度交付量激增 211%，帮助特斯拉终止连续七个季度的亏损_商业_好奇心日报

谢金萍2018-10-26 06:45:16

今年 7-9 月，特斯拉交付 5.6 万辆 Model 3。

受益于 Model 3 三季度交付量激增，特斯拉单季取得 4.17 亿美元营业利润，近两年半来首次季度盈利。特斯拉在财报开篇用了“真正具有历史性意义（truly historic）”来形容这一季度。

今年 7-9 月，特斯拉取得 68.24 亿美元收入，同比增长 128.6%，环比增长 70.51%。它一共交付超过 7 万辆新车，其中 5.6 万辆为 Model 3，环比激增 211%，交付量激增一个原因是特斯拉的以旧换新制度，至少有一半 Model 3 是以这种方式卖出去的，车主支付大约 3.5 万美元。正常购买则至少要 4.5 万美元。

5.6 万辆的交付数据也让 Model 3 排在全美国三季度交付量榜第 5 名，在它前面的是丰田、雪佛兰、福特等大众化品牌的中低端车型。

马斯克在财报电话会上说，“我之前就说过，我们会证明特斯拉可以盈利，本季度就迈出了重要的一步。”不过，他之前允诺的提高 Model 3 产量却只做到了一半 —— 7-9 月，Model 3 周产量约为 4300 辆，低于此前马斯克提出的 5000 辆周产目标。

由于特斯拉卖车收定金、交付才收尾款，产能高地直接决定了它业绩好坏。Model 3 当季交付量大增也带动特斯拉毛利率提升七个百分点至 25.8%，经营活动现金流也变成净流入 13.91 亿美元，今年一、二季度分别为净流出 3.98 亿美元和 1.3 亿美元。季度末现金储备也增加至 29.67 亿美元。

马斯克之前回应外界关于特斯拉债台高筑时的问题时曾说公司不需要融资。但是造车是个异常烧钱的生意，当季现金流重新为正的特斯拉需要保持这种势头，才能兑现它前董事长的说辞。截至 9 月末，特斯拉应收账款较去年年底增加约 12 亿美元、达到 35.97 亿美元。   

另外，中国工厂的问题这次也被特斯拉主动提及。马斯克在电话会议上说，会在 2018 年底前接收中国区 Model 3 订单， 2019 年将以“资本高效”的方式在中国本土生产 Model 3 和 Model Y —— 一款尚未发布的紧凑型 SUV，中国工厂计划年产 25 万辆，届时出厂车辆将只供应中国市场。
