三星发布可折叠手机和专属 UI，所有 Android 应用都要被重新设计_智能_好奇心日报

王毓婵2018-11-08 14:16:43

Google 已经表示支持。

三星在 2018 年开发者大会上展示了自己的首款可折叠智能手机，但这款手机还没有完成最终命名和设计。它使用了一种新的显示技术，三星称其为 Infinity Flex Display。在这种技术支持下，设备可以左右对折，展开成平板电脑，叠起来变成手机。

三星移动产品营销高级副总裁贾斯汀·丹尼森（Justin Denison）称，新手机使用了一种高度耐用的新型胶粘剂，这款主屏幕 7.3 英寸的设备可以折叠“数十万次”而不会受损。三星还展示了Infinity-U（水滴屏）、Infinity-V（V 形槽）和 Infinity-O（屏幕挖孔）三种异形屏风格。

在参数上，该设备在折叠状态下的显示屏是 4.58 英寸，被硬件科技媒体 PCWorld 评价为“小得令人难以相信这是 2018 年的手机”，分辨率为 1960x840；展开状态下屏幕为 7.3 英寸，比市面上任何一款手机都大得多，分辨率为 2152x1536。但折叠前后的厚度不明。

为了适配这种设备，三星设计了新的用户界面 One UI。One UI 包含了许多新的视觉效果，比如更多圆角、彩色斑点以及重新设计的图标，但它的最主要目的是减轻操作超大屏手机的压力。One UI 将关键的交互区域放置在屏幕底部，免除用户费力去用拇指碰触边缘区域的麻烦。

One UI 的界面能够同时显示三个手机应用，并配合手机的折叠开合调整页面状态，放大或缩小显示内容。这对于开发者来说又是一个新的挑战。显然所有 Android 应用的程序员都要为该产品重新编写应用程序，以确保手机在折叠成平板电脑时运行顺畅而不是简单地被拉大或缩小。

好消息是 Google 比较支持这一新的 UI 界面。三星开发者大会的当天，Google 正好在加州山景城总部召开安卓开发技术大会（Android Dev Summit）。在三星展示可折叠屏手机之前，Google Android 系统副总裁 Dave Burke 抢先表示，Google 正与三星合作，优化 Android 平台面向未来的可折叠设备，“Google 将与三星电子紧密合作，2019 年前期共同致力于推出可折叠屏手机。”

“我们希望让 Andorid 系统充分地利用这种新形态，并尽可能地减少开发者的工作量。”Dave Burke 说，“2019 年，我们会看到数款采用折叠屏的 Android 设备。”这意味着三星可能并不是唯一一家要做折叠屏的厂商。

三星表示，将在未来几个月准备好量产工作，2019 年有望将真正的折叠屏手机推向市场。

海外主流科技媒体对这场发布会反响不一。硬件媒体 PCWorld 在一篇题为《三星的可折叠 Galaxy 手机令人失望》的文章中提出了几个三星没讲明白的问题：手机名字叫什么？卖多少钱？电池寿命怎样？内存多少？有耳机插孔吗？展开状态下能接打电话吗？

“我们甚至不知道它是怎么工作的。在贾斯汀·丹尼森短短十几秒的展示中，他始终把 UI 和设计藏着掖着。三星甚至专门调暗了灯光，好不让我们看清。”文章写道。

市场研究公司 IDC 的分析师 Marta Pinto 则认为可折叠手机未来也许是带动三星增长的新动力。“差异化非常重要。三星的智能手机销量正在下​​降，因为它面临来自华为和其他中国品牌的激烈竞争。如果它能够为市场带来一种新的、有趣的设备，则可能重新获得动力并恢复增长。”

IDC 数据显示，今年 7 月至 9 月，三星智能手机出货量较去年同期下降 13.4％。尽管整个行业都在萎缩，但这家韩国公司很明显表现不佳，其市场份额从 22.1％ 下滑至 20.3％。

三星不是全球第一家推出可折叠屏幕手机的公司。不久前，柔宇科技曾发布世界首个量产折叠屏手机 FlexPai，但看起来很明显是一个正处于过渡期的设备。它的铰链式设计无法很好地将上下两层结构贴合在一起，其中弯曲的部分会留出一大块缝隙，这导致该机折叠后的厚度达到了 15mm，而目前主流旗舰手机的厚度仅为 7-8mm 而已。

从三星自己给出的概念图来看，它的设备似乎能让双层结构更好地贴合在一起，但也意味着机身内部的屏幕需要承担更大角度的对折，这对耐用性是一个考验。具体上手效果如何，还需等待明年的正式发售。
