微软重组，纳德拉一步步让微软完成了去 Windows 化_智能_好奇心日报

龚方毅2018-03-30 15:54:15

Windows 及设备部门被拆并进两个新成立的事业部。

当地时间 3 月 29 日一早，微软 CEO 萨提亚·纳德拉（Satya Nadella ）以全员信的方式宣布重组 Windows 及设备部门，将其拆散之后分别并入新成立的两个事业部：

  • 体验及设备事业部（Experiences and Devices），由原 Office 业务负责人 Rajesh Jha 掌管；

  • 云计算及人工智能平台事业部（Cloud + AI Platform），由原云计算与企业部负责人 Scott Guthrie 掌管。

Windows 从部门名称里消失了。微软来来回回已经改组数次，但这还是盖茨时代至今的第一次从部门名字里拿掉了“Windows”。

重组之后，体验和设备部门将负责 Windows 和 Office 的软件开发工作，并负责开发 Xbox、Surface 及其他硬件产品。纳德拉表示该部门旨在使微软不同产品间的体验更趋一致，将 Windows、Office、及其他第三方应用和设备，整合形成一个更完整的 Windows 365 体验。

云计算及人工智能平台部门则将以微软日益增长的企业服务为重心，从分布式计算架构以及人工智能软硬件技术开始，在技术栈的所有层面推动平台一致性发展。该部门负责的业务将包括 Azure云服务、各种分布式计算解决方案、人工智能感知以及混合现实等。

原 Windows 及设备部门负责人、执行副总裁（EVP）Terry Myerso 也将离职，他已经在微软工作 21 年。

纳德拉正在围绕微软增长最多的平台，云计算和人工智能，塑造公司的未来，Windows 将成为其中的一部分。最新一个季度里，Azure 收入增长了98％，而 Office 365 增长了 41％。在同一时期，Windows 和设备部门收入只增长了 2％、约为 121 亿美元。

Windows 势弱是从纳德拉 2014 年出任 CEO 后开始的。在微软组织架构中，每个事业部有各自的财务、法务、市场团队，事业部负责人（EVP）各自背负盈亏考核压力。

2015 年，纳德拉以增进效率的名义让硬件业务并入 Windows 部门，赶走了包括埃洛普在内的全部前诺基亚高管，并裁员数千人。原来一直带来丰厚利润的 Windows 部门也一下子背上了硬件部门的亏损。在 Android 和苹果面前，微软 Windows Phone 已经没有翻身机会，在业绩考核的巨大压力下，它们在 2016 年砍掉 Lumia 手机部门。

并且，纳德拉对硬件和 Windows 下手的同时，他出售互联网部门的 Bing 地图，这个部门的主要工作也变成了购买各种应用和游戏，并将它们改造成微软在其他平台上的工具 —— 而不是给 Windows 用。

还有 Windows Azure。2014 年纳德拉把云计算称为微软的战略核心，而不再强调操作系统的价值，投入大量人力开发 Linux 使其支持 Azure 平台，在这一过程中，Windows 又被边缘化。最后 Windows Azure 改名为 Microsoft Azure。

在微软成立 43 年的大部分时间里，Windows 几乎是微软的同义词，微软的商标也还是 Windows 视窗。但如今在公司组织架构上，Windows 的名字却消失了。

“他（纳德拉）显然没有受到历史的羁绊。”证券公司 Stifel Nicolaus＆Co 的分析师 Brad Reback 对《华尔街日报》说。
