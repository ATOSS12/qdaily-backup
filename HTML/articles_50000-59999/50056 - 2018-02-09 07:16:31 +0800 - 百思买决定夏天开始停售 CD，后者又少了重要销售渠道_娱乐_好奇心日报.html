百思买决定夏天开始停售 CD，后者又少了重要销售渠道_娱乐_好奇心日报

周哲浩2018-02-09 07:16:31

以百思买为代表的商场一度是 CD 销售的重镇，他们如今与 CD 的渐行渐远形成了一种不利的循环。

CD 在百思买的时代要结束了。

美国电子消费零售商百思买宣布，从 7 月 1 日开始，店内将停止售卖 CD。

鉴于 CD 销量的连年下滑和流媒体的兴起，这是一个容易理解的决定。2010 年，包括 CD 和黑胶在内的实体专辑还能够为音乐市场带来一半的收入，但到了 2016 年，实体专辑在美国带来的收入只占到了音乐产业收入的 21.8%。根据 RIAA 对 2017 年上半年的统计，这个数据进一步缩到了 16%。

CD 自身的情况也不乐观。根据尼尔森的统计，2017 年，CD 在美国卖出了 1.048 亿份，相比前一年减少了 16%。2018 年，这个数据预计会少过 1 亿。

另外一家零售公司 Target 虽然没有摒弃 CD，但是也采取了新的策略。它试图同唱片公司达成新协议，将只付给后者卖出 CD 的钱，根据此前的协议，Target 还需要自费把多余的 CD 运送回唱片公司。

“我们对于经营模式做出的调整，反映出我们在娱乐业持续的投入，同时也反映出行业和消费者行为的变化，”Target 在一份声明中这么说。

其实就在三年前，百思买和 Target 还曾表示，虽然 CD 情况不乐观，但它们不会舍弃 CD。如今一个放弃，一个改变策略，还是能体现出 CD 窘境的超乎想象。

商场一度是 CD 贩卖最主要的渠道。根据 Billboard 的说法，百思买一度是美国音乐销售最为依仗的地盘。电商的兴起一定程度上改变了这个格局——2013 年，亚马逊在 CD 市场的销售份额达到领先的 24%，不过线下仍然有竞争力，沃尔玛当年的销售份额达到 22%。

CD 的销量下滑让商场避开 CD，商场的远离又会进一步加剧 CD 的寒冬，这样形成了一个不利的循环。目前沃尔玛并没有传出停售 CD 的消息，不过它货架上的 CD 很多还不到 5 美元。

虽然 CD 要消失了，不过百思买仍然会贩卖实体的黑胶专辑。近年来涌起的怀旧热潮让黑胶的商机显现了出来。相比之下，拥有 30 多年历史的 CD 处境要尴尬许多。只是在百思买要停售的消息传出后，人们才开始在推特上怀旧。
