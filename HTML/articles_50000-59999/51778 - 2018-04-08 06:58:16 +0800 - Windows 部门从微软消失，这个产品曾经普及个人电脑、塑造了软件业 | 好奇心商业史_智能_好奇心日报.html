Windows 部门从微软消失，这个产品曾经普及个人电脑、塑造了软件业 | 好奇心商业史_智能_好奇心日报

龚方毅2018-04-08 06:58:16

Windows 兑现了盖茨对于软件行业的想象，将商业带入信息和网络的时代，但它自己没能跟上移动时代。

按下电源键，指示灯伴随硬盘噪音快速闪烁起来，一行行术语在屏幕上浮现。接着，屏幕暗下，再亮起，四色视窗标志和硕大的“Windows”出现在屏幕上。

这是数亿人对微软的印象，也是计算机在人类社会普及的开始。

曾经，微软就是 Windows 公司，一切其它业务都构筑在 Windows 的成功之上。

现在，Windows 公司不再有独立的 Windows 部门了。

西雅图当地时间 3 月 29 日一早，微软 CEO 萨提亚·纳德拉（Satya Nadella ）发全员信宣布重组，主要业务部门之一的“Windows 及设备部门”被拆散之后并入两个事业部，不再有单独的 Windows 部门。

“这是我记忆中最彻底的重组。”布拉德·西尔弗伯格（Brad Silverberg）对《华尔街日报》说。他曾是微软 Windows 95 时期 Windows 部门负责人。

微软执行副总裁 Terry Myerson 随着改组宣布离开微软，成为最后一任 Windows 部门负责人。两年前微软砍掉 Lumia 手机业务时，Myerson 曾抱怨 Windows 在移动市场有过机会，但没有得到公司足够的支持。Myerson 离职后，Windows 相关的高管，级别最高的也比他在任时低两个级别，不再直接对 CEO 汇报。

如果发生在几年前，这次重组在微软大概会被视为异端邪说。直到 2012 年，时任微软 CEO 鲍尔默还在说“在微软没有什么比 Windows 更重要的了”。

最能说明世界变化的可能是，6 年后没多少人在意 Windows 公司拿掉了 Windows 部门。

资本市场更表示支持。美股从上月 29 号到本月 3 号跟着其它几个大科技公司暴跌了好几天，微软在此期间只跌了 0.11%。 摩根士丹利认为微软此前过于专注 Windows 操作系统使其股价涨幅在很长一段时间内都落后于其他科技公司。券商 Stifel Nicolaus＆Co 的分析师 Brad Reback 称赞纳德拉“显然没有受到历史的羁绊”。

只看 Windows 的业绩，各方的反应似乎也理所当然。

过去两个财年，Windows 只占微软总收入 9%。接下去 Windows 还会继续更新很多年，数亿台运行 Windows 的个人电脑还将驱动整个世界的运作。但就像电线、铁轨不再像 100 年前那么让人兴奋一样，Windows 作为基础设施并不让人激动——我们都习惯了。

从无到有，到改变世界，再到变成了基础设施。Windows 一共花了 33 年。

前两年杰夫·贝索斯回顾亚马逊诞生的时候说“它也只有在那时候（1995）年才行得通。当时我们是个 10 人的公司，我开车去邮局送包裹。我们背后有一系列基础设施的支持。如果不是这些设施的存在，一个这样的小公司永远不可能启动 Amazon.com，就是做不来。”

贝索斯说的基础设施包括邮政、包括互联网也包括 Windows 电脑。

1995 年，美国已经有 5000 多万个人电脑用户和 1600 万网民，这是互联网起步的基础设施。第一个无人不晓的 Windows 系统 Windows 95 也在那一年发布。

对于大多数公司和家庭来说，Windows 95 是接触个人电脑的开始。“每个桌上一台 PC”最后没有停留为盖茨鼓舞员工士气的口号。

可以说今天最大的几个科技巨头都和 Windows 脱不开关系。扎克伯格 2004 年在一台索尼的 Windows 电脑上写出了 thefacebook.com 网站。对苹果复苏至关重要的 iPod 在第三代支持 Windows 系统之后才真正开始热卖。腾讯创办很长一段时间的唯一产品是一个叫 OICQ 的 Windows 软件。

对微软启动的软件业，最好的总结可能还是来自史蒂夫·乔布斯，他在 2007 年和盖茨最后一次公开对谈。在被问及如何评价彼此对于计算机行业的时候，乔布斯说：“比尔创造了行业里的第一个软件公司。我认为他创办第一个软件公司的时候，我们这个行业里还没人知道‘软件公司’是什么东西。”

Windows 不是微软的第一个软件。但从 Windows 开始，微软才真正开创了软件行业，并最终改变了世界。而这家公司，最终也被 Windows 所改变。

盖茨把软件开发从黑客的个人爱好变成一个真正的产业

1976 年，从哈佛辍学还不到两年的比尔·盖茨给计算机爱好者们写了封题为《致爱好者的公开信》，指出：

“多数电脑爱好者必须意识到，你们中大多数人使用的软件是偷的。硬件必须要付款购买，可软件却变成了某种共享的东西。有谁会关心开发软件的人是否得到报酬？”

“如果分文无获，谁会从事专业的软件开发？哪有业余爱好者会花费 3 人/年的精力去编写软件、修正软件、编写使用手册却免费发放给别人使用。事实上，只有我们大量投资来为个人电脑做软件。”

公开信落款是 Micro-Soft 普通合伙人，比尔·盖茨。

盖茨跟保罗·艾伦 1975 年创办微软，为 MITS 公司的电脑开发了 Altair BASIC 程序，指望买了电脑、用了他们开发软件的用户付版权费。

之后盖茨发现只有不到 10% 的 Altair 用户购买 BASIC 程序，但市场上在用 BASIC 的用户数则远超这一比例。1975 年年末，微软一卷储存着预发行的 Altair BASIC 的打孔纸带消失了，然后有 50 份 Altair BASIC 未授权的拷贝出现在一个黑客聚会上。这也是催生盖茨那封宣言的导火索之一。

用软件、付钱，这在今天看起来理所当然的事情。但当时，崇尚使用计算机不受任何限制、信息应该全部免费的黑客感到愤怒。

关注早期个人电脑硬件和软件开发的通讯社，Micro-8 Newsletter 创始人、编辑 Hal Singer 自述他收到公开信后觉得“最理性的反应就是把这封信撕烂然后不去想它。”

当时软件业尚未成型，绝大部分的软件开发都是由硬件厂商完成，它们把软件开发或采购成本加到待售硬件里。

个人电脑市场也还没有卖软件这种说法，很少有人意识到软件有版权和专利的问题。大多数的数据拷贝，要么是实验室里交换信息，要么是计算机爱好者们的口耳相传，他们常以“黑客”自居。

黑客们写软件、开发好玩新奇的硬件多半也都出于兴趣。Steven Levy 1984 年写的那本关于计算机发展的《黑客》提到数十位对早期计算器发展有贡献的人。

当中包括创造 Unix、UTF-8 的 Ken Thompson，15 岁就破解北美防空指挥系统 Kevin Mitnick，自由软件基金会创始人 Richard Stallmam，以及盖茨、乔布斯和沃兹尼亚克。

他们之中大概只有盖茨在 1970 年代就精于商业运作。就连乔布斯那时候创办苹果以后，也专门请了个 CEO 来管理公司，自己跟沃兹尼亚克去捣鼓新产品。其他人则更像一群嬉皮士，试图在电脑上创造艺术和美，认为所有信息都应该自由传播。

当然，个人电脑世界最后按照盖茨设想的规则运作着。从 1975 年的微软开始。之后 1977 年甲骨文的数据库、1982 的 Lotus Excel 办公软件以及再之后 Adobe 公司，一批批软件公司诞生。

而盖茨和微软的财富也是借着 Windows 和之后的 Office 一系列围绕 Windows 开发的产品越积越厚。1985 年，有着图形化界面、不再需要用户对着屏幕敲命令行的 Windows 1.0 发布，隔年微软上市，持股 45% 的盖茨一跃成为亿万富豪。

1995 年，Windows 95 发布让盖茨财富彻底爆发、是年以 129 亿美元身家成为全球最年轻的首富。从那之后，盖茨在 23 年里当了 18 年首富，今年才被贝索斯超过。

Windows 诞生于微软成立第十年，让它成为一个成熟的公司

1980 年，西雅图电脑产品公司雇员蒂姆·帕特森（Tim Paterson）花费了四个月时间编写出了 86-DOS 操作系统。微软隔年以五万美元购得该产品的全部版权，并将它更名为 MS-DOS。

随后 IBM 发布了第一台个人电脑，当时采用的操作系统是西雅图电脑产品公司的 86-DOS 系统，但微软很快改进了 MS-DOS，并使它成功地成为 IBM 电脑采用的操作系统。

IBM 一开始不知道 MS-DOS 的出处，而帕特森在交易时也不知道微软和 IBM 的合作。微软做了个精明的买卖。

多年来以官僚、封闭形象出现的 IBM，在 IBM PC 之父的 Don Estridge 主导下，突然开放起来。它对外开放了个人电脑的硬件标准，允许微软将 MS-DOS 卖给其他电脑公司。

与此同时，英特尔兑现摩尔定律、快速迭代半导体工艺，为个人电脑市场提供了新的处理器。IBM PC 以及其他克隆机的市场份额在短短三年时间内达到苹果的两倍。而每一台 IBM PC 都在用微软的系统。

一个买来的系统、英特尔的硬件、IBM 的开放标准，这些让微软成了早期 PC 市场的控制者。

但真正让微软成为微软的，还是 Windows。

1980 年代初，乔布斯开始开发最早的 Macintosh 电脑，他已经从施乐那里学到图形化界面的点子。乔布斯找到苹果多年的合作伙伴盖茨，希望微软能为 Macintosh 开发一套简单易用的编程语言，并开发一些应用软件，比如文档、图表和数据处理程序。

在听乔布斯描述完 Macintosh 将是一款拥有图形化界面、 为大众而开发的电脑后，盖茨与乔布斯签约。那是一个特别艰难的项目，盖茨后来回忆说大多数参与 Macintosh 的苹果和微软工程师做完那个项目就离职了。

双方合作并不愉快，因为苹果很快发现微软在搜集新系统的交互细节。

1983 年，盖茨宣布微软计划为 IBM PC 开发 Windows 操作系统，并在纽约广场饭店发布预览图。微软抢在 Macintosh 之前，对外演示了采用图形界面，有窗口、图标和可以指向并点击的鼠标的新系统。

乔布斯在苹果总部与盖茨对质，他对盖茨喊：“你欺骗了我们！我信任你，但你却从我们这偷东西！”

盖茨只是坐在那儿，给出了那句著名的回答，“我觉得这更像是说我们都有一个富有的邻居叫施乐。我破窗而入打算偷他家电视，然后发现你已经偷了。”

1985 年，Windows 1.0 在微软成立十周年的时候正式发布。就像大多数微软的第一代产品一样，Windows 1.0 体验糟糕、能运行的软件也不多。等到 1986 年 Microsoft Works 办公应用出现在 Windows 上，后者销量才得以提升。

但微软善于修复问题。之后微软差不多每 2-3 年带来一次大版本更新。到 Windows 95 的时候，Windows 操作系统已经占据超过 80% 的市场份额、微软年销售额首次突破 10 亿美元。

到 1991 年，《财富》杂志准备封面问题，回顾“个人电脑 10 年发展史”的时候，只找了乔布斯和盖茨，没人再关心 IBM。

Windows 95 让个人电脑变得足够易用，开始在家庭和企业办公市场普及。

但它有名的蓝屏一直没有解决，它依赖的操作系统内核不够稳定。企业已经接受 Windows 和 Office 作为日常办公所用，但在 1980、1990 年代都还在用不同于个人电脑的工作站、服务器处理对稳定性和性能要求更高的工作。

让微软真正控制整个计算机市场的是 Windows NT。

1988 年，微软从 DEC 公司挖来大卫·卡特勒（David Cutler）和他的团队开发新的操作系统。卡特勒早年在杜邦公司编程，1971 年转投 DEC 公司，开发操作系统 RSX-11M，因为一个筹备已久的项目被 DEC 砍掉而萌生去意。

卡特勒独立于当时的 Windows 开发团队，组建了一个更资深的团队。经过五年开发并伴随着中途数次延期，1993 年 7 月，Windows NT 发布到工厂刻盘生产。

之后 WIndows NT 不但接管了微软的服务器系统，还在 Windows Me 的巨大失败之后替换掉原有内核，成为 Windows XP、Windows 7 更个人操作系统的内核。从 Windows NT 开始，微软让个人电脑开始代替工作站、专用服务器，毁了 IBM、Sun 最后的硬件生意。

Windows 让微软多年来有一个完美的商业模式

微软一直在软件商店卖盒装 Windows 光盘，但这只占微软收入很边缘的一部分。它最主要的收入来自企业客户，它们或通过预装的 Windows 或批量采购，支付微软授权费。微软 2000 财年总收入超过 229 亿美元，其中企业客户贡献 204 亿美元。

而且差不多从 Windows 95 发布至今，全球造出来的个人电脑都在用 Windows 系统。各家电脑厂商起起伏伏，有公司专做游戏电脑、有公司做高端机、也有公司打价格战……这些公司有破产的、有上市的、还有像戴尔这样上市、退市再上市来回折腾的。

但不管电脑做成什么样子，苹果之外的个人电脑品牌基本只用 Windows，不同电脑之间无法形成足够的差异化。

最终 PC 市场还是以打价格战为主。过去几年，所有电脑厂商收入之和是苹果卖 Mac 的四五倍，但利润却只有 1/8。PC 市场的绝大多数利润去了两家公司——英特尔和微软。

PC 厂商跟着英特尔和微软的研发节奏更新产品，它们把软硬件交出去的同时，也交出了市场里的利润。

Windows 对消费者来说只是个操作系统，但对 PC 厂商来说，就像一家拼命压榨供应商的连锁超市。PC 厂商想把产品卖给消费者，就要给 Windows 交一笔进场费。

最终 PC 厂商把价格越压越低，更便宜的电脑也让更多人能用上，愈发普及的个人电脑也把 Windows 推给更多用户。循环往复，微软的利润越积越多，Windows 市场占有率接近九成。

2012 年，全球 PC 出货量同比减少 4%，但微软和英特尔的经营利润之和，还是比全球前三大 PC 厂商之和多 700%。

Windows 自身特别赚钱，同时它还成为微软推动其它生意的基础。

大企业成为微软客户后，会跟微软签企业协议（EA）。作为这项协议的一部分，微软会为协议中的软件部分提供折扣。例如，Windows 授权费或 SQLServer 数据库授权费或者 Office。

企业花钱用上 Windows 之后，又买了微软的更多软件。1994 年前后，Word、Excel、PowerPoint 这三个应用的市场份额超过 50%，1998 年接近 100%。

甚至今天微软说的云端办公套软件 Office 365，也是这个体系的延续。

微软一层一层的控制加速了企业的信息化，也为微软带来了丰厚利润。

1991 年，美国企业主的信息技术花费约为 1240 亿美元，只占美国 GDP 2%。到 2000 年，这一比例上升到 4.6%、绝对值为 4728 亿美元。在此期间，微软的收入从 18 亿美元上升至 230 亿美元，净利润从 4.63 亿美元上升至 94 亿美元。

相比之下，虽然苹果在 iPhone 之后变成了更赚钱的公司，但它的生意就没有微软这么轻松——它需要总是做出最好的软件和硬件，用足够大的体验差异让用户付出更多钱。这正变得越来越难。

相比之下，微软几乎每隔一代 Windows 就会出一次大问题，但垄断地位纹丝不动。

当然，直到智能手机变得比 PC 重要之前。

Windows 的最后的机会是在 2010 年

微软多年以来的成功，某种程度上在于它成功设置了种种障碍，阻止竞争对手进入，也让自己的客户拥有高昂的切换成本。

至今在 PC 市场，微软的垄断地位还是不变，厂商的利润还是被其攫取。但手机是个新市场。

关于微软和 Windows 如何丢掉手机市场，《好奇心日报》此前曾通过五个片段总结过。

今天回头来看，Windows 在移动市场的最后机会是 2010 年，那年苹果刚推出 iPhone 4，一个从硬件到造型到软件体验都远远甩开业界其他对手的产品，开始加速进入全球市场。

Android 经历一系列修修补补之后刚成熟起来，三星、摩托罗拉和 HTC 各自推出了手段销量过百万的 Android 手机。

苹果极其强势，而 Google 对几个传统行业的颠覆也引起运营商和手机厂商的警觉。在 2010 年，手机厂商和运营商都愿意再来一个操作系统。重新设计的 Windows Phone 7 在那年夏天来了。

但微软的一系列选择让它最终错过了那个机会。

在商业模式上，微软还想复制个人电脑市场的做法：微软做操作系统，手机生产商生产硬件，并为每一部搭载 Windows 系统的手机支付授权费。与此相对的，Android 不要钱。

为了保证用户一致的使用体验，微软强迫其他手机厂商不能预装应用、不能修改用户界面设计、还要求手机硬件规格符合微软要求。

手机厂商和运营商却都想要差异化，大家都知道 PC 产业没有差异化的下场 —— 成为代工厂。

今天 Google 一家独大、逼迫手机厂商放弃自己的用户界面，但它 2010 年不是这样。2010 年会看到摩托罗拉定制了一个首屏满是社交网络的状态更新的新潮界面；HTC 把以前在更老系统上试过的天气、日期插件搬到 Android 平台，就连三星都用自己的手机界面风格。

更糟糕的是，WP7 发布后，微软没有继续迭代系统，而是提出一个统一手机、电脑的大计划，让 Windows 在手机市场停了两年。

等到纳德拉 2013 年接手微软时，Windows 败局已定。时任微软首席运营官 Kevin Turner 在 2014 年承认全球只有 14% 的联网设备运行 Windows 操作系统。再往前倒退十年，这一数字为 97%。

之后纳德拉通过一系列改组削弱了 Windows 在微软的地位。其中最重要两步一个是让硬件业务并入 Windows 部门、让盈利丰厚的 Windows 背上了硬件部门的亏损。微软各部门都要对盈亏负责，最终 Windows 部门不得不选择放弃手机部门。另一个是允许 Office 等核心业务优先开发其它操作系统产品，比如为触控重新设计的 Office 最早出现在 iPad 上，这在此前是不可能的。

当 Windows 占据九成市场份额时，微软所有产品尽可能的绑着它，可以避免用户流失。但当 Windows 不断下滑沦为一个小众产品时，其他产品再绑着它只被会拖累。纳德拉的去 Windows 化是不得已的选择。

现在纳德拉成功的帮助微软去 Windows 化，摆脱 Windows 束缚的 Office、云计算等业务快速增长，微软品牌价值提高到 6950 亿美元，排在苹果和 Google 后面（最近亚马逊股价大跌，又被微软超过）。

但所有这些业务最初都建立在 Windows 的垄断之上。Windows 不再重要了，它们之后能在多长时间锁住用户也不好说。

可苹果、Google，以及亚马逊、京东、阿里巴巴、腾讯等大科技公司，它们在用户眼里都有具体的形象——iPhone 公司、搜索和 Android 公司、电商公司、社交公司……

没有了 Windows 部门的微软是个什么公司？它在用户眼里的形象模糊起来，一如早先失去了计算机市场的 IBM。
