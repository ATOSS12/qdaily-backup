主创介绍

《生活万岁》（Vive la Vie）是由 INTERFACE 剧团创作的戏剧作品。

INTERFACE 剧团成立于 1900 年，根据剧团官网介绍，想要理解剧团的发展历史和目标可以按 interface 一词的字面意义进行理解——连接两个完全不同领域的接口。

INTERFACE 剧团一直在寻找戏剧、舞蹈和音乐的共同之处，同时也在探寻艺术创作与日常生活和文化历史、社会经济之间的联系。他们创作的作品表现形式往往比较丰富，作品的主题也非常多样，从女权主义的发展到不同社会形态的历史都有涉及。

INTERFACE 剧团代表作有《安息日》（SHABBATH）、《你在何处？》（OÙ ES-TU ?）和《异教徒》（LES HÉRÉTIQUES）等戏剧作品，曾获 2014 年阿维尼翁戏剧节最受大众欢迎奖，作品巡演至法国、意大利、德国、伊朗和韩国等多个国家。

内容介绍

《生活万岁》是一部表现十九世纪末二十世纪初法国农业家庭生活风尚转变的戏剧作品。

据相关宣传资料显示：

剧情始于滞固不前的农业社会，这个社会拥有自己的法则和规范。渐渐地，技术的进步打乱了人们原本的习惯，打破了生活的确定性。科学领域的发现，尤其是电气时代的到来，促使了新的思维方式的诞生。技术的进步极大地方便了人们的日常生活，使人们能够更好地生活在一起。然而随着时间的推移人们发现，在这个近乎完美的世界里，某些事情却进展得不太顺利……

除了呈现生活风尚的变化，《生活万岁》还会以戏剧的形式探讨电气技术和计算机技术对人生活的影响等问题。

这部作品的演出时间只有 60 分钟，歌剧、舞蹈、戏剧、马戏和阿卡贝拉等不同的舞台表现形式都会运用到演出之中。
