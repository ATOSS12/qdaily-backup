云服务 Cloudflare 推出消费者 DNS 服务，主打速度和隐私_智能_好奇心日报

马若飞2018-04-03 07:13:11

网站响应更快，同时你的访问历史不会被拿去分析广告投放。

互联网云服务公司 Cloudflare 日前推出了一项面向全球的 DNS 服务，主打隐私和速度。Cloudflare 说，所有访问这个 DNS 的地址查询请求都会在 24 小时内清除，而且速度比目前流行的 Google DNS 更快。

DNS 在国际互联网中相当于电话号码本的功能。简单说，当你在浏览器的地址栏中输入诸如“google.com”类似的地址时，DNS 服务会将他们翻译成类似 1.2.3.4 的 IP 地址，完成网站的访问。

于是，DNS 这个“电话号码本”的性能就和用户访问国际互联网的响应速度有关了。理论上来说，所有国际互联网用户最终都依靠全球 13 个根域名服务器来查找地址，为了减少延迟，这些根域名服务器在全球各地都有缓存和镜像。

但因为互联网用户和网站的数量都极多，实际上依靠根域名服务器和镜像还不能满足性能的需求。另外如果需要更多的功能，比如消除广告、反钓鱼、未成年保护等功能，根域名服务器这种基础服务就不能满足。目前，免费的 Google Public DNS，以及提供更多商业化企业功能的 OpenDNS 比较流行。因为一些原因，在中国大陆 114DNS 的访问要更快速一些。

同时，本地的网络运营商默认也会有自己的 DNS 服务器，如果用户没有设置自己的 DNS 服务器就会默认使用它们。

Cloudflare 新推出的 DNS 服务域名是 1.1.1.1，最主要的特性是隐私保护。无论是 Google 还是 OpenDNS，用户访问 DNS 服务器的记录都是会被保存的，作为分析和广告等用途。但 Cloudflare 宣称他们的 DNS 服务器会在 24 小时内删除记录，保证完整的隐私，会雇佣审计公司毕马威来保证这个保证被正确执行。

在广告中 Cloudflare 说，Cloudflare DNS 的全球响应时间为 14 毫秒，比 OpenDNS 的 20 毫秒，以及 Google Public DNS 的 34 毫秒都更快。

作为一家商业公司，Cloudflare 最主要的业务是云服务、网路托管、Web 防火墙、商业 DNS 等。过去，Cloudflare 也曾参与了反对右翼言论的斗争，2017 年 8 月，Cloudflare 停止为新纳粹网站提供网络服务。作为全世界流量最高的互联网服务商之一，Cloudflare 停止右翼网站的服务令它们失去了防止诸如分布式拒绝攻击（DDos）的功能，事实上给他们的生存增加了极大的难度。

目前，中国大陆还可以正常访问 Cloudflare DNS 的网站 https:1.1.1.1 以及 DNS 服务器。作为普通用户，在自己的互联网设备上配置 1.1.1.1 这个 DNS 服务非常简单，它可以令访问部分网站响应速度更快，以及避免可能的 DNS 污染。要注意的是，设置新 DNS 服务器之后，访问部分中国大陆的网站可能并不会更快。
