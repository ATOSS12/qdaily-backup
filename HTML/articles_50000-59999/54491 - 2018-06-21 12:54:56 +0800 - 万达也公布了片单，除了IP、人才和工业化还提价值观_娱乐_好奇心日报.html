万达也公布了片单，除了IP、人才和工业化还提价值观_娱乐_好奇心日报

周哲浩2018-06-21 12:54:56

在谈工业化的时候，其实是在谈东方影都。

电影节期间，万达公布了新的片单和计划。在“万达之夜”上，IP、人才培养以及工业化是它谈论的三个重点。

在公布的片单中，电影《唐人街探案 3》以及动画《全职高手之巅峰荣耀》是典型 IP 的例子。由陈思诚执导、刘昊然主演的唐人街系列第二部目前是 2018 年内地票房第二高的电影。系列口碑维持得也不错。至于《全职高手之巅峰荣耀》的动画，在万达宣布的前一天，阅文集团在其举行的 IP 生态大会上就透露了根据该小说改编影视的计划。

发布会上，万达与央视动画达成版权共享战略合作，同时央视也向万达开放衍生品的开发权力。《新大头儿子和小头爸爸 3》是双方合作的项目之一。另外，万达与网易文漫共同创立了“IP 实验室”，面向全球征集 IP。

IP 开发计划最多的围绕网络小说《巫颂》展开。万达将与中文在线就《巫颂》开发三部电影、六部衍生剧。这个 IP 也将出现在万达的广场、万达影城等实景娱乐和商业地产中。

在提到对 IP 价值的判断理解时，万达影视集团总裁曾茂军提到的第一点是关于 IP 的文本价值：“特别是在我们中国，我们一定要考虑是不是符合主流的价值观。否则很多电影在拍摄后期发现价值观有问题，可能电影没有机会上映。”

这次的发布会还分享了去年启动的菁英＋导演计划的最新情况。两位导演白雪和邹鹏的作品《过春天》和《地球人都不知道》都已拍摄完毕进入后期尾声。

打造 IP 的片单、青年创作者的扶持以及工业化都是本届上海电影节常见的元素。华谊兄弟公布的片单中有专门 IP 向的单元，各种青年导演/编剧以及工业化相关的论坛也纷纷举办。

万达把东方影都视作工业化的重点。据介绍，东方影都享有总额 50 亿元影视产业发展专项资金，对入驻东方影都拍摄的影视作品给予制作费用 20%-40% 的补贴。

不过，在万达将东方影都转让给融创后（项目运营管理仍然由万达负责），东方影都有自己的麻烦存在。《三声》去年在报道时描述，“影都中大多数底商都在亏损经营”、吸引剧组前来拍摄成为了它的问题：虽然前来青岛拍摄的剧组很多，但拍完就走，没有找到一个合适的办法将其留下。

万达在发布会上介绍，东方影都已吸引 20 家中外剧组签约拍摄，包括好莱坞电影《环太平洋：雷霆再起》、黄渤导演处女作《一出好戏》、中国科幻电影《流浪地球》等。乌尔善投资达到 30 亿元的《封神三部曲》也正在东方影都产业园筹备。
