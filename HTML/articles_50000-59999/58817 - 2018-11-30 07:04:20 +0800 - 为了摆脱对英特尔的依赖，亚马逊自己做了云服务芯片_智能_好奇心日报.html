为了摆脱对英特尔的依赖，亚马逊自己做了云服务芯片_智能_好奇心日报

王毓婵2018-11-30 07:04:20

与英特尔、英伟达、Google、IBM、微软竞争

过去的两天，亚马逊 AWS 在拉斯维加斯的 AWS Re:Invent 大会上密集发布了十余个新产品，品类涉及机器学习芯片、区块链、存储、数据库、机器学习和混合云等等。

首先是两款自己研发的芯片。

27 日，亚马逊推出了首款自研 Arm 架构云服务器 CPU Graviton，28 日，亚马逊又推出了首款云端 AI 推理芯片 Inferentia。

Graviton 芯片采用 ARM 技术，将支持新版本的 EC2 云计算服务。在此之前，亚马逊和其他大型云运营商几乎完全依赖由英特尔研发的 Xeon 芯片。据亚马逊全球基础设施和客户服务副总裁 Peter DeSantis 表示，使用新的芯片能让 AWS 虚拟机特定工作负载的价格再降低 45%。

凭借这款定制化 CPU，亚马逊成为主流云计算服务商中第一家推出定制 CPU 的企业。这有可能对英特尔的市场主导地位构成挑战。

英特尔处理器运行着全球 98% 以上的服务器，亚马逊、微软和 Google 等大型数据中心都是英特尔的客户。虽然这些互联网巨头正在通过自主编程降低大部分组件的成本，但英特尔的 Xeon 芯片仍然维持着较高的市场依赖性。

在 28 日的大会上亚马逊又推出了一款机器学习推理芯片 Inferentia。该芯片支持 TensorFlow、Apache MXNet 和 PyTorch 深度学习框架，以及使用 ONNX 格式的模型。

AWS 首席执行官 Andy Jassy 在发布中介绍说，Inferentia 将会是一款超高吞吐量、低延迟、性能强大，且功耗比极佳的处理器。

目前，英特尔的处理器目前在机器学习推断市场也占据主导地位，晨星分析师预计该市场到 2021 年的规模将达到 118 亿美元。今年 9 月，英伟达也推出了自己的推理芯片，与英特尔展开竞争。

Google 云计算部门在 2016 年推出了一款与英特尔、英伟达竞争的人工智能芯片，但这款芯片的使用成本很高，而且需要对软件进行定制。Google 云计算业务针对 TPU 芯片收取每小时 8 美元的使用费，而英伟达芯片仅为每小时 2.48 美元。

Andy Jassy 表示，Inferentia 芯片主要通过其云业务向外提供服务，并不打算直接出售，因此不会对芯片巨头英特尔和英伟达构成直接威胁。客户可以通过 AWS 的 EC2 instances、Amazon SageMaker 和 Elastic Inference 等产品获得这一服务。与其他 AWS 服务一样，用户可以根据使用量付费。

但这一趋势最终仍可能会影响传统芯片厂商的业务，因为 AWS、Google 和阿里这些云服务大厂是英特尔和英伟达等重要客户。随着上述云服务厂商的自研 AI 芯片越来越成熟，他们将会减少对芯片供应商的依赖。

另外，亚马逊也进入了此前一直比较排斥的混合云领域。AWS 宣布与全球云基础架构和移动商务解决方案厂商 VMware 的合作，转型为内部硬件供应商。从明年起开始，向客户销售硬件，并通过新推出的 AWS Outposts 服务，在其自己的数据中心中运行，两家公司共享收入。

混合云是一种计算环境，它通过允许在公有云和私有云之间共享数据和应用程序将两种云组合起来。当计算和处理需求波动时，混合云计算使企业能够将其本地基础结构无缝扩展到公有云以处理任何溢出，而无需授予第三方数据中心访问其整个数据的权限。

亚马逊在云计算领域的对手微软在 2017 年 7 月推出了名为 Azure Stack 的混合云解决方案，并允许用户使用 Azure 公有云服务与存储在堆栈中的数据，并在公共 Azure 云和堆栈上部署相同的基于 Azure 的应用程序。IBM 也在 2017 年推出 IBM Cloud Private，帮助“on-premises”（本地）数据中心开发云本地应用程序，以作为中间件使得打造混合云变得更容易。

亚马逊进军混合云领域将加剧其与微软、IBM 在云计算领域的竞争。

在数据库业务上，Andy Jassy 表示，到 2019 年中期或年底，亚马逊将完成“去甲骨文化”，将所有在甲骨文上运行的亚马逊数据都迁移到亚马逊自己的数据库上运行。

亚马逊正在减少对甲骨文数据需求的依赖。Andy Jassy 表示，到明年 1 月份，目前在甲骨文上运行的 88％ 的亚马逊数据都将在 Amazon DynamoDB 或 Amazon Aurora 上运行。今年 11 月 1 日，亚马逊已经把其数据库从甲骨文迁移到了自己的服务器 Redshift 上。

在这之前，甲骨文曾多次抨击对手。甲骨文 CTO Larry Ellison 公开表示，亚马逊的云服务存在安全漏洞，并在甲骨文 OpenWorld 大会的主题演讲上比较亚马逊 AWS 和甲骨文云服务的性能和价格，认为相比于“第一代云”的亚马逊，“第二代云”的甲骨文更能应对频繁而智能的网络攻击，而且处理效率更高，价格更便宜。

Larry Ellison 的言论引起了亚马逊的不满。Andy Jassy 在社交媒体上曾针锋相对地回应对方。

在本次大会上，亚马逊公开了一副“全球云计算市场份额图”，显示 AWS 的市场份额远高于甲骨文。而且，亚马逊有意调侃 Larry Ellison，把他的头像放在了图上。

虽然此前 Andy Jassy 曾表示 AWS 对区块链不感兴趣，但在本次 re:Invent 大会上，AWS 却宣布正式推出 2 项区块链服务，分别是 Quantum Ledger Database（量子账本数据库），和 Amazon Managed Blockchain（亚马逊管理区块链），可用于管理点对点支付、处理贷款并帮助企业与分销商和供应商进行交易。

量子账本数据库是一个全托管的中心权威机构拥有的分布式记账数据库，提供透明、不可更改、加密的交易认证。而亚马逊管理区块链则是一个区块链托管服务，它支持超级账本平台 Hyperledger Fabric 和以太坊架构。

AWS 承诺，该服务将扩展到数千个应用程序，并允许用户运行数百万笔交易。亚马逊管理区块链的用户只需几次点击即可建立和管理可伸缩的区块链网络。

根据 Gartner 的数据，2021 年全球公共云市场规模将从今年的 1760 亿美元增长到 2780 亿美元。据 Jefferies 分析师 Brent Thill 称，2022 年，AWS 的销售额将达到 710 亿美元，这将使该部门的估值达到约 3500 亿美元。
