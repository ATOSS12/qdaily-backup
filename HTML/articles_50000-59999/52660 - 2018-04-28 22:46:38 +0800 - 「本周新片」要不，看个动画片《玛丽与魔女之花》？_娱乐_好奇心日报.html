「本周新片」要不，看个动画片《玛丽与魔女之花》？_娱乐_好奇心日报

魏倩2018-04-28 22:46:38

其他的并不值得推荐。

《玛丽与魔女之花》

  • 导演：米林宏昌
  • 编剧： 坂口理子 / 米林宏昌 / 玛丽·斯图尔特
  • 主演： 杉咲花 / 神木隆之介 / 天海祐希 / 小日向文世 / 满岛光
  • 片长：103 分钟（中国大陆）
  • 格式：2D

如果想要在影院里重温对吉卜力画风的爱意，这部影片应该不会太让你失望。可惜，米林宏昌毕竟不是宫崎骏。

《玛丽与魔女之花》是 Studio Ponoc 的首部电影作品。在离开吉卜力工作室后，本片导演米林宏昌和西村义明共同成立了这家全新的工作室，工作室名中的“Ponoc”在克罗地亚语里意为“午夜”，两位合伙人希望以此代表全新一天的开始。但从《玛丽与魔女之花》来看，他们依然没能摆脱，更遑论超越老东家吉卜力。

影片取材于英国作家 Mary Stewart 1971 年发表的童话故事《小扫帚》。一个名叫“玛丽”的小女孩无意间发现神奇的“魔女之花”，凭借它的力量来到魔法学院学习，在这里，她发现了魔法世界的种种奇妙和邪恶阴谋。

影片的前期宣传中， Studio Ponoc 采用“魔女再临”作为口号，试图与 1989 年的那部《魔女宅急便》建立联系，同时也希望观众用不同的眼光来看待这两部影片。《玛丽与魔女之花》确实与《魔女宅急便》风格迥异，但故事主角和不少桥段却不时令观众想起其他几部熟悉的影片，比如《哈尔的移动城堡》或者《千与千寻》。

即使对于米林宏昌本人来说，《玛丽与魔女之花》也不能算是他的最佳作品。2010 年，他以《借东西的小人阿莉埃蒂》正式出道，不仅继承了吉卜力特有的手绘画风，还讲述了一个充满想象力和柔情的故事。但在《玛丽与魔女之花》里，我们很难看到那种出乎意料的创意和让人晕眩的世界观。一切都太套路，也太平静了。

不过，也许是观众期待值太高。如果抛开对比，将它纯粹当成一部完成度很高的童话改编作品，《玛丽与魔女之花》还是值得一看的。对了，配乐很不错。

《后来的我们》

  • 导演：刘若英
  • 编剧：袁媛 / 何昕明 / 潘彧 / 安巍 / 刘若英
  •  主演：井柏然 / 周冬雨 / 田壮壮 / 苏小明 / 张子贤 
  • 片长：120 分钟 
  • 格式：2D

刘若英首次导演作品，算是及格的青春片。

“后来，我们什么都有了，却没有了我们”，这句宣传词几乎可以概括全片内容——小晓和见清是一对从东北来北京工作的老乡，他们在回家的火车上相识，继而相恋，二人在北京相依为命，希望能过上更好的生活，但最终却因互不理解而分开。多年后，他们在飞机上再次相遇，回忆过往。

这类简单到每个人都能预测的剧情，尤其需要导演的氛围营造和情绪渲染能力。从雪地上的绿皮车等场景切换中看得出，刘若英已经尽力，或者说有点用力过猛了。原本清晰的故事，她用了两种并行的方式讲述，一面是时间戳式的彩色回忆线索，讲述二人的恋情始末；一面是黑白的当下重逢，刻画时间变迁流逝。在增加观影层次的同时，它也把故事拆得七零八落。观众每每进入某一场景情绪，便马上被抽离到现状，对同一个故事的反复讲述，又让影片显得各外漫长。

节奏不佳同时暴露着影片的剧本问题。在影评中，有观众将《后来的我们》比做“低配版《甜蜜蜜》”，与后者切切实实的感情相比，更能体现该片的逻辑错漏，比如片中充满程式化想象的所谓“北漂生活”，以及加上刻意煽情的偶像剧式的台词设置，无论是痛苦还是喜悦，影片几乎每一个片段都与真实生活无关，因此也无法得到青春片最重要的“共鸣”。

最后，田壮壮演得不错，他部分填补了这部片子欠缺的真实感。

《幕后玩家》

  • 导演：任鹏远
  • 编剧：任鹏远
  •  主演：徐峥 / 王丽坤 / 王砚辉 / 段博文 / 任达华
  • 片长：114 分钟 
  • 格式：2D

烧脑、悬疑、金融犯罪、幕后黑手……《幕后玩家》很像那种在电影节的创投单元做路演的电影，可惜最终质感看起来有点像网大。

一夜醒来，金融犯罪高手钟小年发现自己被囚禁在了家中。凶手拿着他所有的犯罪证据，逼他做五道选择题，在公众面前承认自己的罪行。钟小年努力逃出牢笼，但也陷入了更重大的选择之中。

影片英文名叫“A or B”，听起来像是那种“如果我和他同时落水，你会救谁”的极端选择，刺激、有趣，但也不切实际。于是，《幕后玩家》的导演兼编剧任鹏远在提出这个新奇的点子后，最需要的就是让它变得更真实一些。

他失败了。坐拥百亿财富的钟小年住在一所远离城市的阴森豪宅里、他的妻子爱吃保温杯里的云吞、救助钟小年的“财经记者”为了农业节目露宿荒野…… 全片几乎所有的人物动机和剧情走向都显得莫名其妙，完全为了导演那个““A or B”的点子服务。这种随心所欲的叙事，显然无法给观众带来真正的情感体验，不少悬疑感人场景出现时，观众席上甚至爆发出笑声。

这些显然离导演任鹏远和监制徐峥所期待的，让人感受到“成长”和“救赎”的影片相距甚远。最可怕的是，他们还将全片的价值押在了徐峥和王丽坤的感情戏上。

还有两部投资不低的电影

  • 《低压槽：欲望之城》：张家辉首次执导警匪片，他希望能拍摄属于自己的风格。他自己担任编剧，设定了一个犯罪横行的城市，再扮演一位渴望打击罪恶的警察。（听起来像是蝙蝠侠和哥谭市的设定），不过尽管投资巨大，辗转多地取景，《低压槽》的“电影感”还是很弱，无论是审美还是叙事节奏，都无法与张出演过的那些经典警匪片相比。何炅等大牌明星的加入，则莫名为影片带来一丝塑料感。
  • 《战神纪》：陈伟霆和林允主演的历史题材影片，原名“ 铁木真传说”，讲述一代草原英雄铁木真的成长故事。陈伟霆主演少年铁木真（和胡军同框），林允是他的伴侣孛尔帖。看完全片后，只能说蒙古族导演拍的草原挺美的。
