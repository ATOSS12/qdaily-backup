一个能预测城市交通灯状态的 API，也许可以帮助避免车祸_智能_好奇心日报

聂丽平2018-06-04 07:03:36

预计在 2019 年第一季度末全面推出。

5 月 30 日，一家以色列初创公司 Nexar 宣布将推出一个名为 Lights 的 API，它将提供城市每个交通信号灯的实时信息，预测信号灯的颜色和时间。

这家总部位于以色列特拉维夫的公司成立于 2015 年，是一家基于 AI 的行车记录仪 APP 来监控道路安全的初创公司。主要产品就是一款名为 Nexar 的应用程序。这款 APP 通过智能手机的摄像头和感应器，依靠计算机视觉和传感器的结合来检测事故并分析路况，然后将实时警报发送到其 V2V （vehicle-to-vehicle）网络上的其他车辆。

Nexar 还开发了一款 Nexar Dashcam APP，提供一个位置追踪器，可以记录汽车内外的视频，以及连接到云端存储数据。这些数据可以被司机用作碰撞事故的证据，也可以收集用于分析和开发。因此 Nexar 的技术也被保险公司还原车祸现场，以帮助他们进行索赔调查。

将要推出的这款 Lights API 使用的数据库正是来自于 Nexar 的用户。装有行车记录器的汽车在行驶过程中，会不断捕捉和传送周围事物的数据，其中就包括交通灯。收集的这些信息将被实时分析处理并通知，比如交通灯即将改变颜色，会在一秒内传送。通过计算机视觉 AI 和 V2V 网络，当汽车刚行驶进路口时，Lights 就可以提前显示道路尽头的交通灯状态，并显示倒计时，汽车转弯时能迅速切换当前道路的交通灯信息。

Nexar 的 CEO 兼联合创始人埃兰·希尔（Eran Shir）表示，因司机错误地计算了交通灯的时间（美国很多交通灯并不显示倒计时），交叉路口常常发生因司机闯红灯而引发的碰撞事故，Nexar 希望通过提供交通灯预测服务来解决这些问题。他说，Nexar 研究后发现，向司机提供交通灯预测可以降低他们的焦虑程度，甚至他们在开车时发短信的频率也会下降，后者也是车祸的主要原因之一。

Nexar 相信，导航和共享汽车服务也将能利用 Lights 优化行驶路线，缩短行车时间。埃兰·希尔说：“有了 Lights，我们可以预测城市每个交通灯的状态，”“实时查看和绘制道路地图，将使 Nexar 和我们的合作伙伴在任何时候都能对整个城市和街道有详细的一个了解。这种理解将让合作伙伴解决实际问题，并最终为现在和未来的自动驾驶车辆创造更安全的道路。”

Lights API 最终还是为 Nexar“一个没有车祸的世界”这个“使命”服务的。Nexar 希望创建一个汇聚了道路上每个物体的新地图，包括交通灯、车辆、行人、路标、骑自行车者、路灯、急救车等。他们希望 Nexar APP 最终将提供包括交通灯监控、交通堵塞、道路危险、基础设施故障、停车、行人、路标、骑自行车者和紧急车辆等的一系列功能。埃兰·希尔表示，Nexar 的目标是“在自动驾驶车辆实现之前，消除车祸，让我们的道路更安全”。

不过，目前来看，Nexar 离这个目标还很远。自 2016 年推出以来，目前有 160 个国家的 740 个城市使用了 Nexar 的 APP，行驶里程超过 1.6 亿公里。目前其最大的市场是纽约市（Nexar 称，共享汽车司机中超过 10% 都在使用），旧金山、特拉维夫和拉斯维加斯。使用人数并不算多。

Lights API 最终的效果如何，也有待它正式投入市场。Nexar 方面称，Lights 将于今年第三、第四季度在第一批合作伙伴中推出，并在 2019 年第一季度末全面推出。目前，Lights 已经得到了纽约、旧金山和拉斯维加斯的支持，计划明年再增加 30 个美国城市。

今年 1 月，Nexar 在 B 轮融资中获得 3000 万美元。该轮融资由 Ibex Investors 领投，阿里巴巴创新投（AIV）、全美互惠保险公司（Nationwide Insurance）和现有投资者参投。鉴于阿里的参投，也有评论认为，Nexar 未来可能会进入中国市场。

除了 Nexar，奥迪也推出了一项名为“交通灯信息”（Traffic Light Information）的功能。与 Nexar 依靠 V2V 网络预测交通灯信息不同的是，奥迪采用的是 V2I（Vehicle to Infrastructure）技术，它的部分车型能通过车载 4G LTE 接收城市交通管理系统传达的实时交通灯信息，在驾驶者等待信号灯变绿的过程中仪表盘上会显示倒计时。

2016 年，奥迪在拉斯维加斯首次发布了这项技术，目前已经扩展到华盛顿、达拉斯、丹佛、休斯顿和帕洛阿尔托等 7 个城市。目前该系统支持美国 1600 多个交叉口的交通信号。鉴于很多城市在规划基础设施系统时没有将 V2I 考虑在内，奥迪的扩展并非易事。

此外，Nexar 的竞争对手之一，去年被英特尔以 153 亿美元收购的以色列机器视觉公司 Mobileye 的软件和硬件也可以检测道路标记，识别和读取交通标识、交通灯和方向标识，还可以分析视野以预测可能与人类、车辆和其他类型的障碍物发生的碰撞，并绘制自动驾驶车辆的可驾驶路线图。
