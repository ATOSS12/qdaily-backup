#Meme：Netflix 新片成了 meme 制造机，克苏鲁题材总是很抓人_文化_好奇心日报

孙若空2018-12-29 12:48:13

在克苏鲁类电影中，克苏鲁本身并不是主角，因克苏鲁的存在而发狂的人类如何表现，有强大意志的主角会如何冒险才是故事聚焦的核心。

圣诞新年党通常都是合家欢影片的市场，但今年 Netflix 却反其道而行，在 12 月 21 日圣诞到临前上线了一部恐怖惊悚影片 Bird Box 。

但这个大胆的行为却成功吸引了大家的注意。用媒体的话来说， Bird Box 在圣诞期间接管了这个互联网，成为了 meme 制造工厂。

几乎看过的人，都忍不住在讨论它，甚至模仿它。

Bird Box 根据 Josh Malerman 的同名小说改编，由桑德拉·布洛克、崔凡特·罗兹等演员主演。

有人说 Bird Box 像是《寂静之地》的姐妹篇，《寂静之地》是不能说， Bird Box 是不能看。有中文名干脆直接把它翻成了《蒙上你的眼》。

这个故事的主线是世界上突然出现了一种想要消灭人类的恐怖势力。人们一旦看到这股难以名状的力量，就会丧失理智地自杀或杀人。桑德拉·布洛克饰演的母亲为了躲避这股可怕的势力，带着两个孩子一路去往崔凡特·罗兹等人也在寻找的安全之地。在这个过程中，为了不“看到”那股可怕的力量，他们主动蒙上了自己的双眼，跋山涉水，在完全丧失视觉的情况下面对重重困难。

不得不说，故事的感染力和桑德拉·布洛克的精湛演技成功让许多人入了戏。以至于，一时之间互联网上出现了层出不穷的对 Bird Box 的模仿 meme 。

比如有人学桑德拉·布洛克的造型，蒙上自己和孩子的眼，一手一个拉着孩子出门去探险。

有人学桑德拉·布洛克蒙上眼后的台词，在一段一个小伙猛打一棵树的视频下配上“强风”“有人在动”“有人呼吸的声音”这些台词，还原了电影里桑德拉·布洛克有些如惊弓之鸟般歇斯底里的状态。

还有人把所有蒙着眼打转的场景都看成了 Bird Box ，即使是海绵宝宝。很有可能，从此以后，蒙着眼出行可能就直接被冠上一个俚语，就叫 Bird Box 。

当然，很多人也在积极用表情包分享着观影体验——仿佛坐上情绪过山车般的体验。

关于 Bird Box 的受欢迎，除了影片主创本身的努力，我们还不得不提一个名词，克苏鲁。

克苏鲁是美国小说家霍华德·菲利普·洛夫克拉夫特所创造的克苏鲁神话中的一个邪恶存在。它的一大异能是能在远处与人类发生精神接触。与克苏鲁有精神接触的人类都会因为受到了强烈的精神震荡而发狂。

许多影视剧沿用了类似的设定，由此形成了一个固定的克苏鲁流派。《寂静之地》是克苏鲁， Bird Box 也是如此。

有趣的是，几乎所有人在看完克苏鲁电影后都会忍不住去研究这个故事中的克苏鲁究竟是何形象。 Bird Box 上映之后也是如此。一些网站也很积极为大家揭秘。例如 PopBuzz 网站就发文说， Bird Box 中，导演曾经拍出过那股势力的样子，以剧中一个角色绘制草图的方式出现，只是之后被剪掉了。

很多导演会隐藏克苏鲁。因为在克苏鲁类电影中，克苏鲁本身并不是主角，因克苏鲁的存在而发狂的人类如何表现，有强大意志的主角会如何冒险才是故事聚焦的核心。而对人类来说最恐怖的，永远是恐惧本身。
