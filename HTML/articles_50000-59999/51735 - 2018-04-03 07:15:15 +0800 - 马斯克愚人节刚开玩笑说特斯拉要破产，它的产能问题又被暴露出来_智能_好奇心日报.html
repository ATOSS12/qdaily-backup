马斯克愚人节刚开玩笑说特斯拉要破产，它的产能问题又被暴露出来_智能_好奇心日报

汪江敏2018-04-03 07:15:15

Model 3 目前产能比目前折半下调的目标还要低一半。

“特斯拉已经彻底完全破产了，破产到你难以置信。”

马斯克以新闻稿的格式写了一段破产消息，并附上一张昏倒的照片。

他调侃自己：

“伊隆被发现昏倒在一辆特斯拉 Model 3 旁，身旁有许多的‘Teslaquilla’（特斯拉牌龙舌兰，Tesla+tequilla）的瓶子。他的脸上仍能看到泪水的痕迹。”

这是一个愚人节玩笑，但现在看起来没那么好笑了。

首先是 3 月底的 Model X 起火导致乘客死亡，现在确认特斯拉的自动驾驶系统当时开启。此外特斯拉这几年生产的 Model S 中有近一半被召回。

现在，它又有了个新麻烦。

根据彭博社对 Model 3 车辆识别号码的追踪，目前特斯拉每周大概生产 1190 辆 Model 3，与之前的目标相差甚远。

特斯拉两次下调了 Model 3 的目标产量。它在 2017 年第三季度财报中表示，到 2018 年第一季度末，它将每周生产 5000 辆 Model 3。而在 2018 年 1 月 3 日，它将目标周产量下调到 2500 辆，说是要更重视电动车的质量。即便是下调后的目标，现在也只完成了不到一半。

现在它的目标是到今年第二季度每周生产 5000 辆 Model 3。

就像马斯克自己曾说的：“很多人都跟我说，我定的时间规划都过于乐观。”

现在来看，马斯克也意识到了自己的乐观。昨天在推特上，马斯克表示：他现在正回到工厂里睡觉，为了加班加点增加特斯拉的产量。这不是没有发生过，Model X 量产前一周，他在生产线旁边扔了一个睡袋，准备累了倒头就睡。

“汽车商业就是地狱”这是马斯克最近说的话。

上周二，穆迪将特斯拉的评级从 B2 下调到了 B3。原因有很多，比如产能不足无法按期交付 Model 3；市场对 Model S 和 X 的需求开始下降；没办法短期融到大笔资金。评级下调的直接影响是特斯拉之后募资的利息更高。而特斯拉一直亏损，募资是维持后续生产所必需的。

Vilas Capital Management 对冲基金经理 John Thompson 表示，如果拿不到融资，特斯拉撑不过 12 个月。根据推测，要在未来 18 个月里活下去，特斯拉大概需要 80 亿美元资金来应对各项支出。

而要扭转这一切，还是得够快造出 Model 3。
