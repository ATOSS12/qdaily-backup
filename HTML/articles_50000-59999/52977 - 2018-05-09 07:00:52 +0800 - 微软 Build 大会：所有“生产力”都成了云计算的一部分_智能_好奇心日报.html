微软 Build 大会：所有“生产力”都成了云计算的一部分_智能_好奇心日报

马若飞2018-05-09 07:00:52

内容围绕物联网、增强现实和人工智能，和消费者有关的仅剩 Windows。

微软年度开发者大会 Build 2018 已经开幕，惯例由 CEO 出席的开幕演讲已经结束。在微软将旗下产品转向“服务化”之后，Build 大会上谈的与普通消费者相关的内容越来越少，新开发工具方向的内容越来越多。今年 Build 大会也不例外，简单的说，微软这次宣布的大部分新东西都和云计算有关。

和消费者有关的 Windows 10 更新、语音助手

上周，微软刚刚向所有用户推送了版本号 1803 的 Windows 10 Redstone 4 功能更新，这次 Build 大会宣布了下一个阶段 Redstone 5 的主要内容，正常情况下将于秋季推送给用户。

微软现在已经没有了自己的手机硬件和操作系统平台，和手机有关的功能都集中在和 PC 的协作上。Redstone 5 将包含一个跨平台的移动设备协作工具“你的手机”。这个功能目前在现在版本的 Windows 10 中已经内置，可以添加 iOS、Android 和 Windows 10 Mobile 三个平台的手机，通过手机上安装的 Edge 浏览器将内容分享到电脑上，包括网页和电子邮件等等。

新的“你的手机”比现在的功能更强大，用户可以从 PC 直接访问短信、照片、通知，也可以从手机快速把照片分享到 PC，还是支持 iOS、Android 和 Windows 10 Mobile 三个平台，但因为系统限制，Android 上的功能将更全，比如在 PC 上可以完整显示手机上的短信、照片，手机上的通知也会直接在 PC 上弹出。

新的协作功能还包括从 PC 到手机。目前微软为 Android 系统推出了主打办公功能的启动器 Microsoft Launcher，在新的版本中 Microsoft Launcher 将支持 Windows 10 的“时间线”功能，可以实现如下类似的功能：在 PC 上正在编辑的 Word 文档，会自动在安装了 Launcher 的手机上显示，只要点击手机上的图标，编辑进度就会自动在手机上的 Word 中继续。

要注意的是，此项功能目前都是围绕微软自己的应用，比如 Office 套件和 Skype 等。同时 iOS 因为系统更封闭，并不支持第三方启动界面，微软把类似的功能放进了 Edge 浏览器中：在 PC 上浏览的页面可以快速转移到 iOS 上继续访问。

在开幕演讲中，微软花时间展示了另一个功能：亚马逊 Alexa 和微软 Cortana 的跨平台运行。

去年 8 月，微软已经宣布和亚马逊合作，两家公司会打通智能语音助手的平台限制，但直到 2018 年 1 月，两家公司都没进一步的消息。这一次开幕演讲终于展示了这种合作究竟是什么样子。

简单的说，亚马逊 Echo 系列智能音箱和 Windows PC 都将完全内置 Alexa 和 Cortana 两个语音助手。在使用 Echo 音箱时可以说“Alexa, open Cortana”，在使用 PC 时可以说“Cortana, open Alexa”，就可以在两个平台上任意使用两个语音助手。

目前，亚马逊 Alexa 在生活场景上比较强大，比如读个新闻、买个东西之类；而 Cortana 在办公场景上更厉害，比如设定日程，回复和追踪电子邮件等。微软和亚马逊的合作功能上可以互补，从业务上来说两家公司都没有移动平台，合作可以确保优势最大化。

但遗憾的是，Alexa 目前还没有中文，虽然 Cortana 是最好的中文语音助手之一，但 Alexa 和 Cortana 合作的这个新功能对中国用户大概没什么用。

物联网硬件都要和 Azure 云服务连接

微软去年提出 Intelligent Edge 的品牌概念，意思是要把各种开发平台都和云计算平台连接。这次开幕演讲上，微软主要宣布了两个物联网云计算方面的新东西。

第一个是与大疆的合作。微软宣布将和中国无人机厂商大疆合作推出开发套件，未来企业用户将可以使用 PC 利用无人机的行业功能。微软展示了一种场景：一台大疆消费级无人机 Mavic Air 在工厂上方进行巡查，摄像头看到的画面被实时传回 PC，PC 上运行的一个应用使用图像识别结束看到了一处管道损坏，在应用窗口中被实时标记了出来。

这意味着开发者可以在 Windows PC 上直接开发应用，可以直接使用大疆无人机上搭载的传感器的功能，比如可见光、红外摄像机，避障传感器等等。开发者也可以利用微软的云计算平台开发类似的图像识别功能，比如可以让无人机自动巡查高压电缆，发现是否有杂物挂在上面。这种企业应用场景也是大疆一直在强调的行业业务。

微软说，两家公司的合作还包含云服务，未来大疆旗下产品的云服务将优先使用微软 Azure 平台。

第二个是关于微软 Kinect 的“重生”。微软宣布的 Project Kinect for Azure 新项目，就是将过去饮料瓶大小的 Kinect 传感器变成了一个手指大小的传感器模块，并且将计算转移至了云端。

过去，第一代 Kinect 是微软 Xbox 360 游戏机上的配件，主打的是体感游戏。第二代 Kinect 出现在 Xbox One 游戏机上，主打的依然是体感游戏，但微软将它引入其他行业，作为 PC 的配件开发体感应用，比如无需接触的互动式交互屏幕，方便手术室中有消毒需求的医师。后来微软把 Kinect 的深度摄像机技术放进了增强现实头盔 Hololens 中，成为了它的基础功能。

这次 Project Kinect for Azure 是彻底的小型化，和过去的 Kinect 传感器需要独立电源才能驱动不同，Project Kinect for Azure 可以运行在低功耗和低网络连接速度的场景下。开发者可以据此开发更便宜的人工智能应用，可以运行在移动平台或者片上系统上。

在 iPhone X 发布的时候，新增加的 Face ID 功能其实就是和 Kinect 类似的 3D 环境传感器。不过苹果选择的是红外线结构光传感器，Kinect 使用的是基于激光的结构光技术。

除上述内容之外，微软还宣布了和物联网相关的诸多内容，比如离线使用机器视觉人工智能功能的新版 Custom Vision，与高通合作开发基于高通 Vision 智能平台的人工智能应用，以及方便开发者开发车载、家用语音助手的 Speech Devices SDK 开发套件，套件内置降噪、语音识别、远场声音等功能。

增强现实和虚拟现实还是主打商用客户

微软的 Hololens 增强现实头盔已经发布了好几年，这次 Build 大会依然强调了它的企业功能属性。微软现场播放了两段视频，展示了增强现实功能的实际应用。

视频展示了德国汽车零配件制造商采埃孚（ZF AG）和重工业公司蒂森克虏伯（ThyssenKrupp AG）在工厂中部署了 Hololens 来帮助一线员工工作：当遇到生产线上的实际问题时，现场的一线员工戴上 Hololens 之后，可以直接在实际看到的画面上进行“标记”，比如某个开关的位置，某条管线的走向等等，后方的技术人员可以看到一切并进行指导；当员工在进行施工场地布置的时候，戴着 Hololens 可以把计划布置的机械以实际大小直接叠加在真实画面中，可以直接看到是否与其他设备冲突，还可以用手势测量间距。

视频中的两个主要产品是微软 Hololens 混合现实头盔以及微软 Layout 套件。

Layout 工具可以看作是现有三维设计功能的增强版，可以将原来只能存在于 PC 的三维设计模型以实际大小显示在增强现实或者虚拟现实环境中，设计师也可以直接在虚拟现实中进行设计；结合 Hololens，设计完成的模型可以“混合”进现实中，还能与工作伙伴共享。

在消费者熟知的领域，虚拟现实的热潮实际上并不长久。一方面 Oculus 和 HTC Vive 过于高昂的价格没有让这种设备成为大众流行的潜力，另一方面并不成熟的技术和艺术环境导致没什么可玩的，说得上好玩的虚拟现实游戏非常少见，只有足够硬核的飞行模拟游戏 Elite:Dangerous、足够沉浸的生化危机 7、跳票很久之后在最近刚刚推出的音乐游戏 Beat Saber，除此以外受到好评的就很少了。

微软的 Hololens 在几年前发布时还包含一个 Minecraft 游戏的增强现实版本，后来便不再提了，专心将 VR 和 AR 放在企业应用上。事实上，与消费级用户相比，虚拟现实和增强现实目前确实更适合企业用户。

微软进一步走向幕后

每年一度的 Build 名为开发者大会，但也是比较好的品牌宣传事件，几年前的时候还和消费者关系比较密切。2012 年到 2015 年，Windows 以及 Windows Mobile 都是比较优先介绍的内容。

但是随着手机平台停止开发，Windows 10 逐渐服务化，Build 大会就越来越没法像苹果 WWDC 开发者大会那样吸引普通消费者关注了：因为除了 Windows 之外，微软几乎没有什么面向一般消费者的软件产品了。

4 月底刚刚发布的微软财报却有一点点意外。上个季度，微软增幅最大和营收最多的业务部门依然是包含 Office 服务的“生产力与业务流程”和包含云计算的“智能云”部门，增长率均为 17%，贡献了共 170 亿美元的营收。

但在同比增长 13% 的“个人计算”部门，售卖个人电脑产品的 Surface 部门营收 11 亿美元，同比增长超过 32%，增长率仅次于刚整合的 Linedin 部门。硬件销售的增长应该和微软新推出的 Surface Laptop 和 Surface Pro 产品销售有关。

总的来说，重组之后的微软正在将所有的业务流程整合进优势的云计算领域，更进一步走向幕后。

走向幕后不是问题，微软营收依然漂亮、推动了市值最近重新超过 Google，次于苹果和亚马逊。但微软锁定企业服务市场曾经靠的是 Windows，以及靠 Windows 建立了大企业合作关系、Office 生态。现在随着 Windows 不再那么重要，微软的护城河不如以往。你看曾经管云计算的纳德拉被提成 CEO 这么几年，亚马逊在云计算市场的第一位置还是毫无变化。
