喜茶和塔卡沙圣诞季联名合作。既有联名圣诞杯套，还推出了联名圣诞福袋，包含围巾、包包、日历等联名限定，目前圣诞杯套已经在喜茶各门店上线，圣诞福袋还需消费者于平安夜在指定门店，通过喜茶的圣诞贺卡来获取。

“双 12” 期间，网易云音乐和知乎联合推出 RedBlue 联合笔记本，选择了双方的各自代表色红蓝，并用一份“知音日报”详细介绍了此次合作。

今天是星巴克的天猫超级品牌日，限定发售直通星级星礼卡，有定制“直通玉星星礼卡”和“直通金星星礼卡”两款。还推出了圣诞礼盒，包括圣诞杯、星礼卡等，以及相框纪念徽章等。

Windows 官方推特今天正式发布了 Windows 95 毛衣。这款“丑丑毛衣”（Ugly Sweater）饰以 Windows 95 的开机 Logo。微软使用 “Soft Wear” 来指代这款产品，读音一语双关。不过这款毛衣目前只能通过抽取幸运儿的方式获得。
