阅文发布半年报，依然垄断但想做的 IP 运营没什么进展_商业_好奇心日报

姚书恒2018-09-28 17:50:57

在拥有 IP 和运营 IP 之间，还有漫长的距离。

阅文集团早已主导着中国网络文学市场：72% 的作品、88% 的作家（同一作家可以同时在不同的平台写作）、50% 的读者，但它还是靠一小部分的用户付费阅读来赚钱，在 IP 版权运营业务上进展缓慢。

现在的阅文，其实是由起点中文网、盛大文学、腾讯 QQ 阅读组成的联合体。拥有近千万部作品储备、700 万名创作者，覆盖  200 多种内容品类。根据其半年报，上半年阅文营收 22.83 亿元，盈利 5.04 亿元，同比增长 136.2%；其中，付费阅读营收 18.51 亿，占总营收 81%。

但网络文学市场的规模非常小。根据 Frost&Sullivan 的估计，2017 年网络文学付费规模大约为 63 亿元；而北京市委宣传部同期的估算大约是 90 亿元。而同期，国内影视与游戏行业的市场规模达到了 5400 亿，是网络文学市场的 60－80 倍。

在这个小市场里，阅文已经有接近 50%的市场占有率，未来的提升空间不大。事实上，这也反应在半年报中了。今年上半年，阅文的付费用户数增长幅度远不及用户数量增幅，付费用户的比例比去年同期下降了 16.42%。

阅文也意识到网络文学市场天花板太低，要把版权通过影视、游戏等方式来进一步变现。

在 2016 年，阅文就举办了第一届 IP 运营生态大会，提出了“IP 共营合伙人”制度：作家、版权方、影漫游等开发商、投资方等在内的产业链各端的不同合作方即合伙人纳入同一体系，围绕 IP 不同形态不同阶段的衍生开发协同合作，合伙人共担风险，共享收益。也是因为阅文手握大量版权、有这样对标漫威迪士尼的 IP 运营前景，当它 2017 年在港股上市时，市场为 2016 年仅盈利 3040 万元的阅文，给出了千亿估值。

但是，尽管有大量 IP，运营难度却比想象中大。2016 年，在票房最高的 20 部国产电影、收视率最高的 20 部电视剧、观看量最高的 20 部网络剧、下载量最高的 20 款网络游戏中，都有至少 60%的 IP 是从阅文手上买的。只不过，阅文的 IP 运营收入只有 2.47 亿，占收入比重 10%。

跟两年前相比，没有太多进展。阅文把 60 部网文改编权授予合作伙伴，例如电视剧《武动乾坤》、《扶摇》；网剧《萌妻食神》；动画《全职高手》、《斗破苍穹》；漫画《修真聊天群》等等。但收入才 3.17 亿，占总收入比重 14%。

也就是说，在整个内容产业链上，从 IP 生产到影视制作、衍生品，看上去是最上游的 IP 授权最强势，但实际并非如此。更像是影视、网游决定了什么才是好的 IP。拿着 72%的作品版权，却只在 5400 亿的娱乐市场中获得 2、3 个亿的收入，阅文积累版权的付出与收入是不成正比的。

为了改善 IP 变现能力，上个月阅文还以 138 亿收购了新丽传媒。新丽传媒近年出品了《我的前半生》、《煎饼侠》、《夏洛特烦恼》、《羞羞的铁拳》、《如懿传》等爆款作品，IP 改编和影视制作能力还是不错的。

在预期中，阅文的 IP 供应、新丽的 IP 改编、腾讯的渠道分发、游戏的转化，整个链条看上去是很美好的，但腾讯一直不太擅长整合收购来的公司与业务。这次也不一定能有什么变化。
