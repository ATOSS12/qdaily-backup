微软 75 亿美元收购 Github，比上一轮估值翻了近 4 倍_商业_好奇心日报

张丹2018-06-05 16:55:06

主要为了云计算

赶在苹果 WWDC 前，微软宣布 75 亿美元收购 Github，比之前传闻的价格高出了 25 亿美元。

这是微软第二次在 WWDC 开始前宣布大额收购消息，上一次是 2016 年，宣布以 260 亿美元收购领英。

本次收购预计在今年年底完成，交易完成后，Github 会保持独立运营，它的营收将纳入微软 Intelligent Cloud 业务板块，微软副总裁 Nat Friedman 将出任 GitHub CEO，GitHub 联合创始人兼董事长 Chris Wanstrath 将成为微软技术研究员，向微软 Cloud + AI 业务执行副总裁 Scott Guthrie 汇报。

Github 成立于 2008 年，是世界上最大的代码存放网站和开源社区，个人和企业都可以在这个平台上协作和共享代码，截至今年 3 月，它共托管了 8000 万个代码存储出库，有 2400 万活跃用户。Github 的企业客户有 Twitter、Facebook、Google 等科技巨头，也有美国政府。

Github 主要通过给企业提供代码托管相关增值服务来赚钱。这些增值服务包括无限制的存储空间、隐私和安全保护等。个人会员和企业招聘广告也是收入来源的一部分。据 Github，截至去年秋季，会员服务营收为 2 亿美元，其中 1.1 亿美元来自企业用户。它于 2015 年完成最后一轮融资，当时的估值是 20 亿美元，现在翻了近 4 倍。

从收购后在人事和财务上的安排可以看得出，微软此次收购 Github 是为了拓展云计算业务。微软的云计算业务已经连续 11 个季度增长超过 90%，是公司营收增长的主要推动力。据市场研究公司 Canalys 数据，现在云计算市场规模为 156 亿美元，微软以 14% 的市场份额排名第二，仅次于占有 32% 份额的亚马逊。

早在 2006 年，微软也曾上线过开源软件托管平台 CodePlex，但由于 bug 太多，让 Github 后来居上，去年 12 月微软关闭了 CodePlex，并把存储在上面的代码库转移至 Github，在今年的 Build 2018 开发者大会上，二者还合作把微软 Azure 的开发运维一体化在线服务开放给 Github 用户。

Github 最近也遇到了一些管理问题。在收购前的近十个月里，Github CEO 处于空缺状态。2014 年联合创始人 Tom Preston-Werner 因性别歧视丑闻离职后，Chris Wanstrath 接过了 CEO 的职位。Wanstrath 于去年 8 月辞去了 CEO 职位，说想把精力放在社区和产品战略上，会为 Github 找到合适的 CEO。一直自称 2011 年就开始盈利的Github，还在两年前被爆出亏损问题。

从一个独立发展的平台，成为大公司的一部分，有用户叫好，也有用户感到不安。为了消除这些疑虑，微软再三表示，会把开发者优先和平台独立性的原则放在首位。

5 月 30 日，在云计算业务的推动下，微软市值超过 Google 母公司 Alphabet，并在过去 12 个月里，股价上涨了 40%，是 Alphabet 涨幅的 5 倍。
