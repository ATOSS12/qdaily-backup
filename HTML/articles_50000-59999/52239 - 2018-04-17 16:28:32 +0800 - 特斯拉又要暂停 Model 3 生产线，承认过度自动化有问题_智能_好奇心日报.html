特斯拉又要暂停 Model 3 生产线，承认过度自动化有问题_智能_好奇心日报

马若飞2018-04-17 16:28:32

马斯克曾对自动化工厂非常骄傲，但产能一直都不乐观。

美国电动汽车公司特斯拉日前宣布，计划停产公司位于加州弗里蒙特总装厂的 Model 3 生产线。这是特斯拉继 2 月份以来第二次暂时停产 Model 3，本次停产时间计划为 4 到 5 天。特斯拉代表说，停产的时间将用来解决生产线的自动化问题，目标是提高生产率。

在这次停产之前，特斯拉的 Model 3 在 3 月份刚刚走上正轨：1-3 月份 Model 3 生产了 9700 辆，在 3 月最后几周产能达到了 2000 台每周，虽然离特斯拉定下的 2500 台每周的目标还有些距离，但已经是 Model 3 投产以来的最好消息。

上周末，马斯克对特斯拉的生产模式发表了一个转折较大的评论，他在推特上说，过度追求自动化是他的错误。这是马斯克罕见的对特斯拉现有模式的表态，他曾经对特斯拉高度自动化的工厂非常骄傲，2016 年 8 月他对分析师说，特斯拉的工厂非常先进，和其他汽车工厂相比就像“外星无畏舰”。他展望说，当特斯拉的工厂迭代到 3.0 版本，生产线上将没有一个工人。

但事实并不尽如人意。在 3 月底伯恩斯坦分析师的分析中，就指出特斯拉在全自动化工厂上挣扎了太久，花了大笔金钱订购库卡公司的工业机器人，但这些机器人在一些传统汽车制造工序中，比如最终组装上却比工人差得太多。

按照彭博的估计，特斯拉 Model 3 的生产率将在 4 月内达到周产 3000 台，月产 12000 台的水平。马斯克在推特上也比较乐观，他说今年 3 到 4 季度公司会因为交付更多的车实现现金流的正增长，无需通过债券等手段筹措更多资金就可以维持运转。

但特斯拉的问题不止生产这一部分，作为一家汽车企业，传统上这个行业最在乎的是产品的稳定性和安全，但特斯拉对这件事的反应从来都显得非常强硬。

3 月 23 日 一台 Model X 在加州撞上护栏导致车主死亡之后，4 月 11 日特斯拉公布了针对这起事故的声明：声明首先表示了遗憾，然后主要阐述了车主充分了解“高级辅助驾驶”的性能和事故路段的情况，最后说我们特斯拉没错，而且高级辅助驾驶功能有效减少了事故率。

从 2016 年佛罗里达州 Model S 撞击大货车致人死亡事故最终的调查结果，以及特斯拉目前公布的数据来看，这起事故最主要的原因很可能是因为驾驶员自己疏忽。但是，特斯拉如此强硬的表态对这件事几乎没有任何帮助。
