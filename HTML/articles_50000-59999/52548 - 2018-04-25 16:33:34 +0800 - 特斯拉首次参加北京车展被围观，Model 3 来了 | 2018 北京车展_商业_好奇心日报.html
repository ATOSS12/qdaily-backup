特斯拉首次参加北京车展被围观，Model 3 来了 | 2018 北京车展_商业_好奇心日报

龚方毅2018-04-25 16:33:34

Model 3 首次亮相亚洲。

没有新品发布，展位面积是同行的 1/4，只带来三辆展车的特斯拉，大概是今年第十五届北京车展上最引人关注的汽车品牌。

因为 Model 3 来了。这是始终没有摆脱产能问题的特斯拉平价车型首次在亚洲亮相。

在今天（4 月 25 日）北京车展的首个媒体日上，媒体和展商工作人员围聚在红色长距离版 Model 3 周围。人群里甚至还有保时捷、奥迪等豪华车厂的人，它们这次也有全新新能源车型参展，其中保时捷的 Mission-E 还被视作特斯拉 Roadster 2 以及 Model S 的竞争对手。

因为留在场地内以及场地外打算静态体验 Model 3 的人太多，排队体验、查看内饰大概也要等 20 分钟。特斯拉从大约早上 11 点半开始限流，只允许媒体进入展台，特斯拉工作人员也通知现场维持秩序的安保，不要再放厂商的人进来。

“今天来了很多竞品公司的人，威马、蔚来，保时捷的 CEO 也来，我们其实欢迎他们过来，”一位在现场的特斯拉中国工作人员说，“但这是我们第一次对非 Model 3 车主打开车门，展示内饰，所以还是更希望把时间留给媒体。”

Model 3 用一个放在仪表台最中间的 15 英寸大屏幕取代传统的仪表盘，所有的行车信息和绝大部分车辆功能控制，都在这个屏幕上。其中屏幕左半边显示包括车速、档位、剩余电量在内的信息，右半边则留给地图服务，服务商是 Mapbox，地图数据现场显示是腾讯地图。

这跟 2014 年 Model S 在中国开售时不一样，最初的六个月，中国没有引入地图导航和互联网娱乐服务，之后通过软件更新才补上这两大功能。

但这次在美版 Model 3 上，中文系统和专门给中国市场准备的地图应用已经直接可用，那么大概率表明特斯拉已经至少为中美车型统一开发系统，根据地区适配不同的应用版本，就跟你用 iPhone 在中国和美国用地图服务类似。

方向盘后部，只有两个控制拨杆，左侧的控制车灯、雨刷器，右侧的拨杆控制五个档位，倒车、停车、空档、行驶和自动（辅助）驾驶：自动驾驶被单独设计成了一个档位。

Model 3 也没有传统的空调控制面板和空调出风口。空调面板自然是集成在中控屏幕里面，空调出风口变成了挡风玻璃下方的一个横跨整个车厢的长条状开口，风向也是在中控屏幕里控制。

对于外界关心的产能、中国区订单量、定价、国产等问题，特斯拉工作人员均表示不便透露。中国发改委本月中旬宣布年内取消新能源汽车外资持股比例限制，对希望在中国独资建厂的特斯拉来说，最大的行政障碍消失。

根据马斯克上周的内部邮件，其产能直到上月才做到连续三周突破 2000 辆/周。而这本该是它们在去年第四季度某个节点达到的产能。现在，Model 3 新的产能被定在今年 6 月底达到 6000 辆/周，还要留出冗余，意味着满负荷产能将超过这一数字。

目前中国区 Model 3 交付时间的对外说法仍是 2019 年。
