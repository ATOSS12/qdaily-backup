如果有些新闻素材跟电影一样，如何把它们拍成真的电影？_娱乐_好奇心日报

孙今泾 温欣语2015-07-24 02:19:59

在好莱坞，这是件值得骄傲的事：即使电影人只用了个点子，他们也得支付同样一笔授权费。

回想起来， Todd Hoffman 得感谢 2005 年《纽约时报》上一篇关于伊拉克战争的文章。当时，他是 ICM 代理编剧和导演的经纪人，每天西装笔挺，忙得不可开交。一位编剧客户找到他，称自己打算写个关于这场战争的剧本。 Hoffman 带着他一齐去了《纽约时报》，买下了这篇报道的授权。完成后的剧本最终高价卖给了派拉蒙。
转机差不多就是在那个时候发生。一年后， Hoffman 和《纽约时报》签订了一份协议，协议的大致内容是，“在那以后，如果再有人想要《纽约时报》授权文章去拍电影，就要通过我。” Hoffman 在洛杉矶世纪城的一间办公室里说。 2013 年，他离开了 ICM 成立了一家叫 Storied Media Group 的公司，生意里最大的一块就是为近 10 家媒体客户代理 IP (Intelligence Property) 。

过去，包括《纽约时报》、 NPR (National Public Radio) 在内的成熟媒体拥有自己的版权部门和法务部门，来应对偶有上门的 IP 购买者，但多少有些守株待兔。 Hoffman 打算结束这种局面。现在，如果有人说他想拍个关于摇滚明星 Elvis Presley 的故事， Hoffman 会告诉他，这个主意很棒，随即，他就会在《纽约时报》的文库里翻找，“有 376 篇。” Hoffman 说。

更多时候，他带着一组几个人每天读报、读杂志，再主动去找那些需要 IP 的电影人。上个周日，Hoffman 在《纽约时报杂志》上读到一个关于双胞胎交换的故事。“这肯定可以拍成一部电影！” Hoffman 第一时间给故事的主人公打去了电话，除了代理这篇故事，他打算同时代理文中的四个男孩。说不定以后会有真人秀。

不是所有人都相信新闻里能够找到好故事，大型的代理公司还是更倾向于翻书，找到一本畅销、或者伟大的小说。但新闻 IP 存在的时间也不短了，在月初的一次采访中，独立制片人 Ron Yerxa 说，每天早上翻看报纸对他来说既是休闲也是工作，“说不定哪个故事就能被拍成电影”。

媒体每天生产无数的新闻故事，Hoffman 希望挑选其中有影响力的媒体品牌，这样一来，撇开单个的 IP，媒体本身就是一块谈判的招牌，于是他飞快地签下了 NPR, Smithsonian, New York 和 60 Minutes。他称，这些合作都是独家的，并补充解释说，通过把故事发展成电视或电影，可以提升媒体的的品牌：“他们看完这部剧，下次就会想到去纽时看文章。”
这很难量化，不过，提高 IP 的价值是显而易见的。像 Hoffman 这样的中间人会把 IP 进行包装，让它看起来更值钱，而不是简单地把文章的链接或一本杂志寄给制片厂。十年前卖给派拉蒙的剧本就搭上了 Tom Cruise。这是经纪公司的老办法——“打包”，独立制片人去筹钱时，通常也得这么说服投资人。但 Hoffman 拒绝透露他从高价售出的 IP 中的收取的佣金。
在一些时候，Hoffman 会让他手下的媒体客户和编剧客户协作，最后将一个完整的剧本卖给制片公司。但在 95% 的案例中，Hoffman 更倾向于为 IP 找到最合适的编剧，而非捆绑创作，好让产业链的分工更加明晰。
考虑到大部分的客户是制片厂，这门生意也在面临新的挑战。但 Hoffman 不会相信人们说的“好的 IP 早没了”，既然制片厂这会儿正醉心于系列片，那就给他们想要的。今年 3 月， Hoffman 卖给了狮门一个新闻故事，关于摩托车的偷车贼，当时他同狮门说，这会是一个很好的系列片题材，就和《速度与激情》差不多。他向我们解释说，这项交易证明了“到处都有系列片的素材”。
过去制片厂和旗下的广播网每年发行大量的影片，还需要把时段填满，但现在他们确实制作得少多了。这是另一个问题。尽管开发的剧本还是远远多于最后完成的电影，但 Hoffman 说，整个行业都因为制片厂影片数量的下滑产生了“多米诺效应”。“如果制片厂在开发上的投入少了，就意味着编剧赚得少了，经纪人赚得也少了。” Hoffman 说， “过去他们开着奔驰到处招摇，现在只能开本田。” 他的办公室挨着好莱坞最大的经纪公司 CAA 。
和大多数人一样， Hoffman 寄希望于新平台的兴起，他用了一句老套的谚语“如果生活给了你柠檬，不如把它做成柠檬酒”， 紧接着说：“电视剧需要更多好点子。”在好莱坞，这是件值得骄傲的事：即使电影人只用了个点子，他们也得支付同样一笔授权费。CBS 刚刚买下了《纽约时报杂志》的一篇长文章，讲述 Alabama 小镇的一位律师，如何为选美比赛的姑娘做顾问。但 CBS 的这部电视剧并不会以这个传奇的男人为主角。

当然，他没有忘记一个巨大的新市场。我们刚坐下，他就翻出了一篇和中国有关的新闻故事，一个重庆橄榄球队的美国教练带领队伍拿到了全国冠军。“我们会拿着剧本去见中国公司，故事本身就决定了它的市场。” Hoffman 说，这是他有意为之。
