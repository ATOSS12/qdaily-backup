Firefox 手机终结，还是给 iPhone 开发应用去了_智能_好奇心日报

赖江锟 2015-12-09 21:48:38

你见过 Firefox OS 的手机吗？

Mozilla 总算想通了，想在 iOS 和 Android 之外再做成一个智能手机系统是多么不切实际。他们于今天在奥兰多正式宣布关停手机项目。

原本，Firefox 浏览器背后的 Mozilla 打算用 Firefox OS 系统切入低端市场，并尝试拉来了 LG、中兴、阿尔卡特等公司做了十几款产品。但 Firefox OS 手机没能得到预想中从功能机换到智能手机的用户，因为现在的低端 Android 手机体验越来越好。

Firefox OS 手机

缺少应用，似乎是所有 iOS 和 Android 外的系统所面临的大问题。这两个已经让开发者们忙不过来，也就没什么人去开发 Firefox OS 应用。而 Firefox OS 手机也没多少用户，去年共卖出 100 万台，相当于小米们的几轮“秒杀”销售。但 Mozilla 还未说明将如何处理现有用户。
去年底，Mozilla 曾激进地计划推出 25 美元的手机，后来却发现做出来的这么便宜的手机，体验过于差劲。他们也在今年 5 月底放弃了这个设想。

从 2011 年宣布，到 2013 年推出 Firefox OS 系统，再到如今勉强推出一些卖不好的手机。四年多的努力，Mozilla 很难把自己的手机系统扔到垃圾桶。其物联设备的高级副总裁 Ari Jaaksi 发 Twitter 表示，Mozilla 将把 Firefox OS 应用到物联网领域。

Focus 广告拦截应用

放弃手机之后，Mozilla 又在 iOS 上推出一款广告拦截应用 Focus，仅支持 Safari 浏览器。与一般的逢广告必拦的应用不同，Focus 主要拦截网站用来追踪用户上网行为的那部分代码，具体包括广告追踪、分析追踪和社交追踪。所以，将 Focus 定位成网络追踪器拦截应用似乎更加合适。 

不过市场上也已经有很多公司在做了。
