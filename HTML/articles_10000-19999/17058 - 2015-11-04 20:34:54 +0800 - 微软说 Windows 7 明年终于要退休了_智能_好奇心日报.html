微软说 Windows 7 明年终于要退休了_智能_好奇心日报

李硕明 2015-11-04 20:34:54

因为它已经成了微软推行 Windows 10 的障碍

为了能让更多人用上 Windows 10，微软决定明年让 Windows 7 退休了。至少，你不会在明年买到预装 Windows 7 的电脑了。
根据 The Verge 的消息，微软要求各家 PC 厂商在明年 10 月 31 日之后，停止销售预装 Windows 7 或者 Windows 8.1 系统的电脑。对于微软来说，这样的做法就是为了能够卖出更多的 Windows 10 设备，提高 Windows 10 的市场占有率。
一般来说，微软会在新的操作系统发布 2 年之后，选择结束上一代系统设备的销售。不过由于 2012 年发布的 Windows 8 情况太过糟糕，微软只能继续延长了 Windows 7 的结束时间。

根据微软给出的最新数据，现在已经有 1.1 亿台设备使用了 Windows 10 系统。微软对于 Windows 10 的目标可是在 2-3 年的时间内，会有 10 亿台设备安装 Windows 10。在 Net Application 给出的 10 月份 Windows 系统数据中，Windows 10 只有 7.49% 的占有率。相比之下，这个数字在 Windows 7 上是 55.77%。

73% 的用户打算在 2 年内升级 Windows 10

看起来 Windows 7 还是推广 Windows 10 的最大障碍，不过 Spice Works 的一份调查结果也安慰下微软。这份调查显示，打算在两年内安装 Windows 10 的用户比例达到了 73%。

这可能是因为，他们都可以免费升级 Windows 10 吧。
