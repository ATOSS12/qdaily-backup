Windows 更新了，要把 Skype 变成即时聊天工具_智能_好奇心日报

李硕明 2015-10-13 20:18:59

在花了 85 亿美元收购 Skype 之后，微软想要把它植入 Windows 系统。这一次，他们要把 Skype 变成操作系统内置的聊天工具。

微软想要他们的聊天软件 Skype 无处不在，变成人们（或者说是 Windows 使用者）生活方式的一部分。

今天，微软就走其中一步棋，让 Skype 变成 Windows 上的即时聊天短信。

微软在今天通过 Windows Insider 推送了最新的 Windows 10 Threshold 2 测试版，在系统应用中加入了 Skype 的消息、语音、视频功能。

具体来说，你可以通过 Windows 10 内置的消息应用切换短信或者 Skype 和别人沟通，获得和 iPhone 手机里面的网络短信 iMessage 或者视频聊天应用 Facetime 的体验。

微软曾经在今年 1 月的 Windows 10 发布会上展示过这项功能，不过在后来发布的一系列测试版本中并没有推送这个功能。

相比之前，单独为 Windows 8 开发一个版本的 Skype，微软这一次在 Windows 10 中采取了把 Skype 集成到系统应用中的做法。

以后的 Skype，就不仅是我们熟悉的网络视频电话应用了。
