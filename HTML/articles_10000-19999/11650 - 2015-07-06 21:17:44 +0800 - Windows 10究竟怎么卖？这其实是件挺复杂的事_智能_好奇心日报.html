Windows 10究竟怎么卖？这其实是件挺复杂的事_智能_好奇心日报

王民超 周韶宏2015-07-06 21:17:44

等了这么久，不介意再等等吧？

7 月 29 日是微软打算发售 Windows 10 的大日子，但那天到底哪些人能免费升级到 Windows 10？这个本来年初“新篇章”发布会上就该说清楚的问题，直到最近才确定下来。

Windows Insider 会员能在第一时间收到更新

上月底，负责操作系统的执行副总裁 Terry Myerson 在一篇博文中说：“从 29 日开始，我们会为 Windows Insider 用户开放 Windows 10 正式版的下载，之后我们会开始逐步地为其他用户推送。”

也就是说，如果你不是 Windows Insider 成员，并不会第一时间得到更新正式版 Windows 10 的机会。至于其他用户还要等多久，Terry Myerson 并没有明确指出。

Windows Insider 是微软在去年针对 Windows 10 推出的会员服务，加入这项计划可以获取预览版的 Windows 10，并且在论坛中反馈体验和问题。目前 Windows Insider 会员数量已经超过了 500 万。

这一升级政策也意味着，那些还用着 XP、Vista 或者索性是盗版 Windows 的用户，早先如果加入 Windows Insider 项目，也是能在第一时间免费升级成正版 Windows 10，不过这部分人也付出了代价：忍受预览版更新的各种 Bug。

Myerson 这篇博文中也为用户温馨提示了升级准备。首先，如果你预定了 Windows 10 的升级，微软会确认设备的兼容性，在某些情况下，如果你的设备并不合格，微软会向你提供硬件厂商的联系方式。当然，你也可以选择直接忽略，强制升级。

另外，选择下载哪个版本也需要认真考虑。Windows 10 将至少有七个版本：家庭版、移动版、专业版、企业版、移动企业版和物联网核心版（IoT Core）。

给不给盗版用户免费升级？微软自己搞混了

发布了新系统该如何升级，这本来不是个复杂的问题。但半年来微软不停发出自相矛盾的信息，把所有人都搞得晕头转向，甚至他们自己也有些迷糊。微软左右摇摆的基本过程是这样的：

1 月 21 日：在“新篇章”发布会上，微软称 Windows 7 和 8.1 的用户将在第一年免费升级到 Windows 10，这一政策也一直延续至今。

3 月 18 日：Terry Myerson 在接受路透社采访时第一次透露将允许所有 Windows 用户免费升级到 Windows 10，不管是否是正版系统。

微软发言人在给 The Verge 的声明中也称，盗版用户也能升级到 Windows 10，但没有明确是否免费，声明认为：“我们相信久而久之用户会意识到获取正版 Windows 的价值，我们将让获得合法版本（Windows 系统）的方式变得简单。”

3 月 19 日：微软改口称盗版用户升级之后的 Windows 10 系统仍然是“非官方版本”，但没有明确这个令人匪夷所思的“非官方版本”和正式版之后究竟会有什么区别。随后，根据另一家科技网站 ArsTechnica 公布的一份来自微软的声明称，盗版 Windows 系统升级到 Windows 10 之后仍为盗版状态，微软不会给予技术支持。

5 月 15 日：微软又摆出一副“严打盗版”的姿态，发公告称要给所有盗版 Windows 系统打上水印，还宣布根本不会让盗版 Windows 系统升级到 Windows 10；但微软也没把话说死，他们表示会给盗版用户提供“有吸引力的方式”去升级成正版 Windows 10。可当时微软又对这个“有吸引力的方式”秘而不宣，只是说 Windows 10 还在发展中，“目前无法回答全部疑问。”

6 月 19 日：微软官方发布公告称：“只要你正在运行会员预览版并绑定微软账户，（7 月 29 日）就能收到 Windows 10 的最终发售版，并能继续激活。”

换句话说，任何用户，即便是盗版的 Windows 用户，只要安装了 Windows 10 预览版便能在一个月后免费升级到正版 Windows 10，照此说法，“转正”的代价几乎为零。至于那个 7 月 29 日之后还将继续运行的 Windows 会员项目，微软当时只是“希望”其成员能留下来，并无任何强制举措。

6 月 20 日：根据科技媒体 The Verge 的报道，微软删除了“收到 Windows 10 最终发售版”后面的那句“并能继续激活”。

6 月 22 日：微软更是大幅更新了那份公告，承认自己最初的说法导致了混乱，并重新解释了预览版用户获得 Windows 10 免费升级的具体方式。

7 月 2 日：微软发表博文，所有 Windows Insider 会员均可以在 29 日获得更新。

摇摆背后是进退两难的境地

盗版用户升级到底该不该收费，本身应该是一个非常明确的问题，而直到发售前一个月微软还在摇摆不定，这背后其实是 Windows 的尴尬处境。

微软依靠正版操作系统以及 Office 工具获取利润，曾经“让每家都有一台电脑”的微软如今将重点偏向生产力工具和企业服务。

相比之下，苹果采取“硬件赚钱+系统免费”的策略；Google 采用“互联网服务赚钱+系统免费”，这都对微软的系统收费模式产生巨大冲击。针对竞争，微软在 Build 2014 大会上宣布，将对其智能手机以及 9 英寸以下小尺寸平板电脑实行 Windows 系统免费策略。加上 Surface 和诺基亚，微软希望以此形成“硬件赚钱+系统免费”的模式。

但是，无论是平板还是手机，都不足以支撑微软重拾 PC 时代的荣光，尤其是销量惨淡的诺基亚，自从被收购，诺基亚手机的出货量一路下跌。今年一季度，微软的手机部门收入为 14 亿美元，但是总体成本比收入高出了 400 万美元。换言之，微软每销售出一部手机，亏损 12 美分。行业分析师表示，最早在二季度，微软将对手机业务进行“灾难性”的资产减记。

同样是在今年第一季度，Office 的情况就好很多，虽然营收下滑，但 Office 365 消费者版的营收增长 8300 万美元，注册用户同比增长 179%；此外，Office 365 商业版注册用户同比增长了 84%。

这么看来，“Windows 免费，Office 赚钱”能否行得通呢？
