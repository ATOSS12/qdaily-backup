为拓展亚洲市场，C罗将肖像权卖给了新加坡大亨_商业_好奇心日报

宣海伦2015-06-30 20:06:00

球场和商场上的 C 罗，哪个更强呢？

当球员 C 罗做生意的时候，那股卖力的劲头和绿茵场上是如出一辙的。日前，C 罗在其官方 Facebook 宣布，授权新加坡商人林荣褔旗下的媒体公司 Mint Media 使用自己的肖像权，并表示此举是为了在亚洲更好宣传个人品牌 CR7 。

“很高兴与 Mint 媒体公司达成合作，这家公司归我的好朋友林荣福所有，他们将拥有我的肖像权。对我本人和我的管理团队来说，这是一次很有战略性的举措。C 罗这个品牌将提升到另一个档次，特别是在亚洲。”

有趣的是，林荣褔同时也是西甲劲旅瓦伦西亚的老板，上个赛季瓦伦西亚在联赛中排名第四，皇马则位居第二。而两年前，C 罗的肖像权一度也成为与皇马续约谈判中的关键因素，最终皇马同意让球员本人拥有大部分的肖像权。


林荣福与 C 罗一家聚餐合影

CR7 是 C 罗（Cristiano Ronaldo）的名字缩写加球员号码(7 号)，也是球迷对他的昵称，最早得名于 C 罗为曼联效力期间。2009 年，C 罗以天价转会到皇家马德里，第一个赛季身披 9 号战袍。次年夏天，随着皇马传奇劳尔远走沙尔克 04，C 罗得以重披 7 号球衣，也成为了伯纳乌的 CR7 。

CR7 主要经营服饰，以 C 罗的个人品牌内裤起家，去年也推出了衬衫和鞋类产品。作为目前世界上身价最高的球员之一，C 罗在商业领域的吸金能力十分恐怖。去年其年收入达 7,960 万美元，在《福布斯》体坛吸金榜力压梅西，位居全球第 3 。

C 罗和前女友俄罗斯超模女友 Irina Shayk 拍摄 VOGUE 大片

除了工资收入和分红，其中 2400 万来自于赞助商佣金，C 罗的代言品牌包括耐克、清扬、阿联酋航空、泰格豪雅，他还是第一位登上 VOGUE 封面的足球运动员。

C 罗的 CR7 同款内裤生日蛋糕

作为一名富有争议的球员，C 罗在商场上也颇具经营头脑。无论是自己的生日，还是和女友之间的悄悄话，都可以作为自己品牌的营销话题。

去年，一组拍摄清凉的 C 罗内衣广告，被投射到了米兰、伦敦、巴黎等地的标志性建筑上，一时间赚足了眼球和口水。
