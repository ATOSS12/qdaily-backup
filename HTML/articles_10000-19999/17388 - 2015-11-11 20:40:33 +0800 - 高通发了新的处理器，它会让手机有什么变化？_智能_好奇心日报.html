另外，这芯片可能是高通最后的希望了。

芯片厂商高通现在还有一个难题，怎样把新的高端芯片“骁龙 820”卖得好。因为早前版本的高通骁龙 810 发热严重，坑了一众手机厂商，也让高通股价持续下跌，所以这次的 820 芯片能否翻身，也吸引了不少眼球。

根据 Engadget 的消息，高通昨天正式发布了骁龙 820 芯片。相比上一代的 810 处理器，高通说这次 820 处理器的性能更好。它现在可以支持 4k 分辨率的屏幕显示，以及更好的图像处理性能。除了这些之外，高通还在 820 处理器中加入了第三代的快速充电技术和跟更多的 4G 网络制式。

最有意思的更新，应该算是能够靠超声波识别指纹的 Sense ID 了。

按照高通的描述，Sense ID 使用了一种依靠超声波的感知技术，你甚至不需要接触手机就能完成识别。同时由于采用了超声波，Sense ID 也可以兼容多种材料，包括玻璃、蓝宝石、金属、塑料。

Sense ID 可以扫描出指纹的 3D 画面，而此前类似 Touch ID 的电容式指纹识别方案只能识别 2D 平面。从理论上说，Sense ID 的安全性要更高一些，并且手指有水也不受影响。

高通的 Sense ID 技术可以用超声波来识别指纹

在过去一年的时间内，高通的日子并不好过。由于之前 810 处理器被发现存在过热问题，索尼旗舰机 Xperia Z4、小米的米 Note 顶配版、HTC 的旗舰设备 M9 都是因为这款芯片造成了手机过热。这种糟糕的质量，让今年生产新 Android 手机厂商不愿意使用。

尤其是三星，就在今年的 Galaxy S6 上换成了自己的处理器。其它厂商也是选择了退而求其次的骁龙 808 处理器，例如摩托罗拉的 Moto X Style。

实际上高通正在逐渐丢掉自己在手机处理器市场的份额。根据数据机构 Strategy Analytics 在 7 月发布的报告，高通的份额已经下降到了 47%，而这个数字在去年是 55%。在高通最新的 7-9 月的财报中，整体收入 55 亿美元下降 18%。至于利润要更惨一些，只有 11 亿美元下降幅度达到了 44%。

要想不成为下一个被人遗忘的厂商，820 处理器也许是高通最后的希望了。
