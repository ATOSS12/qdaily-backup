Chrome 终于要施舍一点电力给你了_智能_好奇心日报

赵长存 2015-06-07 16:20:00

为了省电，这一次 Chrome 把矛头指向了 Flash。

一直用 Chrome 浏览器，却觉得它太费电？今日，Google 宣布了一个解决办法，就是主动帮你把 Chrome 上的耗电大户 Flash 给关掉。

如果你的笔记本平时只是浏览网页和视频，就开始发烫，而且耗电迅速，甚至影响了电池的寿命，这可能就是网页中 Flash 元素惹得祸。当初，苹果公司就宣称 Flash 过于耗电，不适合移动设备，于是在 iOS 系统上便始终都不支持这一技术。

Windows 系统为了省电，会让处理器在不太繁忙时休息，以一定间隔唤醒。而 Chrome 就因为自身问题，使得处理器在每秒里的唤醒次数是 Windows 上其他浏览器的 64 倍。微软称，相比之下其它浏览器，Chrome 会多消耗 25% 的电量。

而对于苹果的 OS X 系统，Chrome 也是耗电能手。The Verge 在 4 月做了一个测试，在一台最新的视网膜版 Macbook Pro 上，使用 Chrome 的时间竟然要比苹果自家的浏览器 Safari 少 3 个半小时。因此，Chrome 也有了电池杀手的称号。

Google 一直也想要解决这一问题，不过这次可不是要像苹果那样和 Adobe 正式宣战，而是采用了一种折衷的办法。当你浏览网页时，如果 Chrome 检测到非页面中央的位置有 Flash 内容，就会将这些元素自动暂停。但是网页中央正在用 Flash 播放的视频并不会关闭或中断。倘若你对被关闭的内容感兴趣，却被 Chrome 错误地关闭，也可以点击播放键，恢复被暂停的 Flash 元素。

如果你已升级到了最新版的 Chrome，却发现这一功能没有被默认打开，可以试试手动的办法启动它。打开浏览器的设置菜单，在“隐私”项目下点击“内容设置”按钮，找到“插件”一栏，再选择“检测并运行重要插件内容”就好了。未来这一特性将会被运用到各版本的 Chrome 浏览器中，并成为默认项。

不管这么做能为 Chrome 提高多少效率，能为用户省点电，并提高电池的寿命总是好的。同时，Google 也承诺要在省电上为 Chrome 做出进一步的改善。

目前在相当多的网站上，广告、甚至图片和动画依然是以 Flash 的形式展现给用户。如果越来越多的浏览器也效仿这样的设计，网站们也会清静点吧。
