便宜的 Windows 要来了，还来得及么？_智能_好奇心日报

赵长存 2015-06-08 16:59:28

微软要在低端机上把 Windows 10 和 Bing 捆绑，来降价。

长久以来，内置 Windows 操作系统的笔记本电脑总比打擦边球，装 DOS 或者 Linux 等你自己装盗版的同规格更贵一些。这背后的原因显而易见。在 Windows 8 操作系统刚推出时，曾有消息传出，微软向笔记本电脑厂商收取每台电脑 70-80 美元的操作系统授权费用。

随着 Android 和苹果 OS X 的咄咄进逼，微软高昂的授权费在榨干合作伙伴利润的同时，也威胁着自己的生意。联想、宏碁、戴尔、惠普等忠实的 Windows 支持者，都各自推出了自己的 Android 平板，甚至笔记本电脑。在这样的背景下，微软似乎打算做一些调整。

近日，一张英特尔的内部幻灯片被泄露了出来。其显示该公司正在开发数款迷你电脑棒，而它们所使用的操作系统则是一种绑定了 Bing 搜索的 Windows 10 特殊版本。

今年 4 月，微软曾宣称要在未来的两到三年里，要让它的新系统 Windows 10 的安装量突破 10 亿。免费提供操作系统，就是这家公司的解决办法之一。

如果消息属实，微软或将以较低甚至免费的授权费，来让 OEM 设备制造商们预装这一版本的 Windows。这也意味着，厂商的成本会随之降低，消费者也就能购买到更便宜的 Windows 设备了。

这种做法，已不是微软的第一次尝试。去年，微软就已开始用这种办法向厂商提供类似的 Window 8.1 With Bing 系统。但是只有低端的机型才能被授权安装，且这种授权仅限于 OEM 厂家，消费者自己并不能独立地安装或升级。

微软的意图其实很明显，它不会白白放弃高端市场的利润，此举只为在低端市场寻求扩张。由于 Windows 10 囊括了桌面版、手机版，同时也可运行在平板设备上，再加上此次曝光的便携电脑棒，微软或许可以从价格上和对手展开全面的竞争。可是，这么做还来得及吗？

这些年，Google 用廉价的 Chromebook 在教育市场上崭露头角，同时意欲在企业市场进一步扩张。而在手机市场中，它又推出了 Android One 系列手机抢占低端市场。

除了来自 Google 的压力，Windows 自身也阻碍自己前进的脚步。根据netmarketshare 的统计数据，目前市场上仍有 14.6% 的桌面设备运行着 14 年前推出的 Windows XP 系统，比最新的 Windows 8.1 的装机量还要大。这其实表明人们守着旧的系统和机器，不愿意购买新的设备。而微软承诺要让 Windows 7 和 Windows 8 的设备免费升级到 Windows 10，也意味着用户大可不必急于购买新的廉价设备。

不过，对于想要购买这种特殊版 Windows 10 设备的用户而言，完全不用担心自己使用上的问题。尽管 OEM 制造商被限制不能修改系统默认的 Bing 搜索引擎，但此前在 Windows 8.1 With Bing 系统中，微软并没有限制用户把 Bing 改为 Google 或者其他搜索引擎。这一次或许也会如此。
