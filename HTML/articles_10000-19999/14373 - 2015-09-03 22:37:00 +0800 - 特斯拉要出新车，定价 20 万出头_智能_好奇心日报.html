Model S 卖了三年后，特斯拉终于提供了更便宜的选择

“首先，你得有一辆特斯拉。”

再过半年，这句网络语得加上一个定语了。 

今天埃隆·马斯克连发 2 条推。抢在迟迟未发布的 SUV Model X 之前，公布了更便宜的特斯拉 Model 3。  

马斯克发推公布 Model 3 定价

“超级电池工厂要在未来 2 年内火力全开，准备生产特斯拉 Model 3 。” 

“3.5 万美元，这是价格。明年 3 月汽车亮相，预订随即开始。” 

折合人民币 22 万元左右，按照马斯克在中国推出 Model S 时的定价公式，引入中国可能会在 30 多万。

2014 年 6 月 Model 3 当时还叫 Model E 。被福特起诉侵权之后，换了名字，却一直活在马斯克对特斯拉未来的描摹中。

“会比 Model S 小 20%。”、“外形一定很酷，但没那么怪，肯定是辆汽车。”在各种采访中，他对 Modle 3 的性能只字不提。 

不过，不少电动车行业分析人士认为，为了刺激消费者购车，续航里程得是 Model S 的低配水平。车充一次电，能跑 320 公里左右。毕竟，Model 3 的定位主打高性价比嘛。 

期待外观设计和性能的同时，最后你到底能不能拿到车？现在只建了四分之一的 Gigafactory，在 2016 年全部建成后，能满足 Model 3 产品线所需的全部产能。这是马斯克的预期。但 Model X 到现在仍未能发布。 

BMW i3

除此之外，2016 年 3 月开始预订，用户可能到 2017 年才能拿到车。那时，宝马纯电动 SUV BMW i3 也已经上市。
