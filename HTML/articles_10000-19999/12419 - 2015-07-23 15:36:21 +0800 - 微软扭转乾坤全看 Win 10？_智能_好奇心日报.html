微软扭转乾坤全看 Win 10？_智能_好奇心日报

Nick Wingfield2015-07-23 15:36:21

有分析师称，微软在企业市场的地位并不如人们普遍想的那么稳固

西雅图电 — 微软最新的操作系统 Win10 将在下周发布。目前共有超过 15 亿台电脑上装的都是 Windows 的系统，这一次 Win10 将给微软用户带来一系列或熟悉、或新鲜的体验。

软件中将自带一个虚拟助理，帮助追踪用户的日程，微软公司将定期在互联网上发布功能更新。Windows 公司标志性的开始菜单也将在新系统中正式回归。

但最明显的变化应该是系统的价格——用户对电脑进行更新将不再需要支付任何费用。由此我们就可以看到技术产业动态的一些变化。

曾经微软的系统收费一般在 50 到 100 美元，这一收费的取消，也让我们看到了收费软件的日薄西山。因为有庞大广告业务的支撑，像 Google 一类的公司正在通过免费手机软件和各种服务吞噬微软的业务。苹果在最近几年实现了手机应用和操作系统的免费更新，而利润来源则转向了硬件销售。

Windows 系统一以贯之、具有标志性的开始菜单将在 Win10 系统中正式回归。此前的系统界面经过重新设计后，使得一些用户无法正常使用。 


微软的核心业务是软件，微软公司希望尽可能久地利用其软件业务盈利。但苹果和 Google 的介入在不断向微软施加压力，不得不迫使微软寻找新的盈利机会。

 “这会改变用户的期望，让他们认定更换操作系统是应该免费的，”Jackdaw 研究公司的分析员詹·汤森在谈及 Win10 系统时说道。关于微软的其他设备，他还说：“除了 Xbox、Lumia 手机和 Surface 平板，微软已经很难从消费者那里赚钱了。”

微软公司已经推出了 Office 应用，比如  Word 和 Excel 的客户端，而这一举动也让微软原本薄弱的移动端焕发了新的生机。目前微软已经开始免费向生产移动设备（如智能手机和平板电脑）的公司免费提供 Windows 操作系统，也借此推广微软的软件产品。

Win10 的免费战略思路与此相似。微软决定牺牲 Windows 操作系统这一部分利润的原因很简单，那就是微软需要更多的用户使用 Win10 系统——并且是在短时间内。最近几年，计算机的势头已经远远跟不上智能手机和平板电脑了。微软上一个操作系统 Win8 没能挽救微软的市场地位，Win8 大胆尝试重新设计的界面反而让事情变得更加棘手，更换系统之后丢失了许多 Windows 老用户。

据高德纳咨询公司统计，第二季度，全球范围计算机销量下降 9.5 个百分点。高德纳估计今年计算机成交数量大约有 3 亿台，而手机则有 19 亿台。在全球手机的总销量中，搭载 Windows 系统的智能手机还不到 3%，而 Google 的安卓系统还有苹果的 iOS 系统占了剩余的大半部分。

 “在全球范围内，微软正在与苹果和安卓争夺在操作系统的统治地位，”艾维克情咨询公司的分析师比尔·维曼说道，“这就是真正的挑战。”

微软 CEO 萨蒂亚·纳德拉和其他高管正在寻找利用 Windows 系统盈利的新方式，而 Windows 系统的更新也将首次免费向用户开放。


操作系统的发展必须要与手机软件的发展步调一致。但在最近几年，但很多软件开发者已经不再把 Windows 系统作为首选了，大部分的软件开发商都已经看准了更受用户青睐的智能手机市场。这就让不擅长做移动端的微软处在了一个很尴尬的位置。

为了让更多的开发商对 Windows 的操作系统感兴趣，微软最新的 Win10 系统不仅可以在计算机上运行，也同样适用于智能手机和其他设备，这就为软件开发商编写适用于 Windows 系统的应用提供了便利的条件。微软公司坚定地表示，在未来的两三年里，将有十亿设备运行微软的操作系统，这也将给软件开发商带来巨大的盈利空间。

 “我认为将会由很大一批用户会采用，”Win10 系统，高端游戏计算机制造商 Razer 的系统产品市场营销主管凯文·撒泽说道。

如果安装 Win10 的用户数量能够在短时间内大幅增加，那么就一定能弥补微软放弃的那一部分利润。微软并未透露新的操作系统更新为公司带来的收益。但分析师估计，相比于单纯对操作系统收费，操作系统升级后，微软能够通过其他途径获得更高的利润。

微软的首席财务官艾米·胡德最近告诉投资者，截至 6 月 30 日，微软在过去一年中仅靠 Windows 系统就创造了 150 亿美元的收益。

大部分的收益都来自于公司市场，微软在这方面要强于其他竞争者。微软大约有四分之一的收益来自于与企业签订的合同，这些企业大多会一次性购买数年 Windows 的系统更新，这样便于管理公司内部网络的众多用户。

无论用户是否意识到了这一点，但大多数人使用的都是付费的 Windows 系统。因为人们购买电脑时都是自带 Windows 系统的。微软公司近半数的收益来自专业计算机生产商，而这些企业购买微软系统都是瞄准了专业市场。微软四分之一的收益（40 亿美元）来自用户计算机制造商。

微软的操作系统目前仍然是计算机市场的主流，除非微软的这一地位受到动摇，否则微软是不会停止向计算机生产商收费的。但微软的竞争者在不断侵占着微软的传统业务，这就使得微软不得不在某些领域降低价格，去年，微软对低端笔记本制造商降低了操作系统的售价，因为微软面临着搭载免费 Google 操作系统的三星笔记本的巨大冲击。

微软的首席执行官萨蒂亚·纳德拉同样决定将 Windows 系统对九寸以下的设备免费开放，也就是说，手机和平板电脑等电子产品也将使用免费的 Windows 操作系统。

微软的高管们已经开始寻找新的利用 Windows 操作系统盈利的方式了。高管们认为，必应的广告收入是盈利的途径之一。必应是微软的搜索引擎，融合了 Windows 系统的诸多功能。如果公司能够吸引足够的人在 Windows 商店购买游戏和其他应用，那么微软的放弃就是值得的。

微软试图在其他领域寻求一种平衡。例如，微软推出了移动客户端的 Office 办公软件，鼓励人们订购这项应用，还新增了云存储和其他诸多新功能。在技术领域，微软还在不断探索硬件设备方面的出路，因此推出了 Surface 平板电脑和 Lumia 智能手机，但由于智能手机在市场上的欠佳表现，微软已经准备收回这项业务了。

针对企业市场，微软公司有十分丰富的软件产品，包括数据库、短消息和云计算等服务，微软的这些业务十分出色，可以抵消因为免费升级给公司带来的损失。

但杰弗瑞咨询公司的分析师约翰·迪福奇称，微软在企业市场的地位并不如人们普遍想的那么稳固。他说他注意到，在投资者会议上，越来越多的参会者都是用 iPad、安卓平板和三星的笔记本来记笔记的。

 “我认为人们还忽略了一些风险，或者说人们并没有给予足够的重视，”他说道。
