Google 要帮微软和苹果淘汰旧系统，靠 Chrome_智能_好奇心日报

赖江锟 2015-11-11 23:18:00

XP、Vista、Mac OS 10.8 都在 Google 要处理的列表上

你还在使用 Windows XP、Vista、Mac OS X 10.8 或更早的系统吗？微软和苹果应该早就鼓动过你升级了。现在，Google 也来催你了。
Google 最近在其官方博客宣布，从明年 4 月开始，上述系统上的 Chrome 浏览器将不再获得补丁和更新，尽管你仍然可以用它上网，但将越来越不安全。
Google 在博文中表示，微软和苹果届时已不再主动更新那些旧版本系统，而使其无法获取重要的安全更新。这种情况下，Google 的 Chrome 浏览器肯定也无法确保用户的上网安全。另外，时下的移动化趋势已使软件开发团队将更多力量用于手机端应用，在电脑端对旧版本提供支持并不是他们优先考虑的事情。

目前，Windows XP、Vista、Mac OS X 10.6-10.8 加起来共有约 14.6% 的份额，比全球 Mac 系统的用户总数还多。

14 年前推出的 Windows XP，份额比整个 Mac 系统还多

与 Google 相反，惠普和戴尔对 Windows 最新系统的态度是拒绝的，此前曾曝出他们的技术支持人员建议用户不升级至 Windows 10，原因是新系统仍有 bug，并且软件兼容性还有待提高。
