曾经的私密社交鼻祖 Path，终于还是被卖了_智能_好奇心日报

赵长存 2015-05-30 00:50:01

小而美的 Path 已被韩通讯应用 Kakao 收购。

曾经一度流行、并吸引到一系列中国效仿者的私密社交应用 Path，昨天宣布被 Daum Kakao 收购，后者正是韩国热门的聊天软件 Kakao Talk 的开发商。

收购后 Path 和它即时通讯应用 Path Talk 仍将保持运营，不过它们未来的重点将放在东南亚市场。和微博之类的社交网络不同，在 Path 中 ，用户只能将自己的状态更新分享给有限数量的人。

这种设计起初的确火了起来。2012 年发布 2.0 版本后，它一度在 App Store 美国区的社交应用排行中达到前十名。到了 2013 年，随着加入“表情商店”的 3.0 版本发布，Path 甚至在前三名的位置上蝉联了一个多月，仅次于 Facebook、Twitter 这两个超级巨头。

时至今日，Path 在欧美的影响力已经大不如前。只有在亚洲，尤其是在印尼还有些用户。 

四年后，Path 拥有 3000 万的用户，并收获了超过 7500 万美元的融资。但对比主流的社交应用而言，这个数量显得微不足道，来自中国的微信，韩国的 Line，每个都有数亿的用户量级。被 Facebook 190 亿美元收购的 Whatsapp 用户数量已经达到了 8 亿。虽然 Path 有着独到的设计，当年甚至被评论说“美得让人飙泪”，但始终，Path 都只是一个小众市场的产品。

所以去年开始，这家公司就一直有被收购的传闻，在去年 9 月的时候，绯闻甚至烧到了苹果的身上。不过，最终这个传闻并没有成真。

收购 Path 的 Kakao，最近也并不好过。去年末它就因被韩国政府指出有监控行为，而被用户抵制，一下失去了 40 万的用户。在今年第一季度，它的营收上也表现不佳，降幅达到了 23% 。

被收购后，Path 能否像它所说的那样能在社交领域有所作为，这还不好说。但它在上月发布的新应用 Kong 中却被遗弃了。在 Kakao 的这次收购中，并不包含这款以自拍动态图为社交方式的应用。
