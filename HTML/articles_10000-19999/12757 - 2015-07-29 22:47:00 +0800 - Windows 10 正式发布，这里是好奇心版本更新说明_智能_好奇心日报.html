Windows 10 正式发布，这里是好奇心版本更新说明_智能_好奇心日报

唐云路 2015-07-29 22:47:00

微软说，Windows 10 让你感觉很熟悉，“您从一开始就是专家。”

Windows 10 终于来了。

北京时间 7 月 29 日零点起，微软正式开始向全球 190 个国家和地区的 Windows 用户推送 Windows 10 正式版安装包。如果你已经收到推送，那么恭喜：“你已参与有史以来最大规模的软件升级活动。”

此前，微软已经公布了运行 Windows 10 的最低配置要求：

  • 1GHz 或者更快的处理器
  • 操作系统：Windows 7 SP1 或者 Windows 8.1
  • 硬盘空间：16 GB（32 位操作系统）或者 20 GB（64 位操作系统）
  • 显示器：1024x600 分辨率

理论来说，只要能够运行 Windows 7 或者 Windows 8 的电脑，都可以安装 Windows 10。除了 Windows RT 的更新包可能需要等到 9 月，基本上从今天开始，Windows 用户都可以免费升级到最新的操作系统了。

成为 Windows Insider 会员，盗版也能“转正”

Windows Insider 是微软在去年针对 Windows 10 推出的会员服务，加入这项计划可以获取预览版的 Windows 10，并且在论坛中反馈体验和问题。目前 Windows Insider 会员数量已经超过了 500 万。

6 月底，负责操作系统的执行副总裁 Terry Myerson 在一篇博文中说：“从 29 日开始，我们会为 Windows Insider 用户开放 Windows 10 正式版的下载，之后我们会开始逐步地为其他用户推送。”

这意味着，如果你已经加入了 Windows Insider 项目并且安装了 Windows 10 预览版，那么无论此前你用的 Windows 操作系统是正版还是盗版，是 Windows XP、Vista ，还是 Win 8，都可以第一时间“转正”。

正版 Windows 7/8 最新版本升级，只需鼠标点击几次

作为《好奇心日报》记者中为数不多的 Windows 用户，今天早上打开电脑时，我就已经在桌面的右下角发现了这条推送。

Windows 10 推送

任务栏上的这个 Windows 图标是一个专门简化升级过程的应用，点击之后就可以进入升级程序。如果你已经是正版 Windows 7 SP1 或者 Windows 8.1 用户，只需点击“预订免费升级版”按钮，接下来的事情，交给电脑自己就可以了。

安装包的大小在 3GB 左右，在后台下载完毕之后，用户只需按提示升级即可。安装包有效期为 1 年，也就是说，接下来的这一年， 你什么时候想升级了都可以。

如果你已经能够在 C 盘里看到“$Windows.~BT”命名的隐藏文件夹，这意味着，安装包已经开始下载了。升级到 Windows 10 并不是强制的，这个时候你的系统盘里应该还会出现一个 Windows Old 文件夹，升级之后觉得用着不顺手，还是可以“愉快地滚回旧版”。

如果你是 Windows 7/8 的正版用户，用的不是最新版本

如果找了几遍，在任务栏的右下角还是没有看到那个 Windows 图标，就不要再找了。

这说明你的电脑至少没有跟上 Windows 7SP1 或者 Windows 8.1 的更新，这个时候你需要先检查 Windows 更新，把操作系统升级到当前系统的最新版本。

完成之后，只要配置满足 Windows 10 的兼容条件，你就可以更新 Windows 10 了。

如果，你还在用 Windows XP

首先，你确定要把这台电脑升级到 Windows 10 吗？要知道，一旦升级，XP 可就回不来了。

好吧，如果你确定要，那么加入 Windows Insider 计划就可以免费升级。

如果你想要的是一台全新的设备，起码有 200 款搭载 Windows 10 的设备已经在路上了。早前，戴尔就已经宣布，搭载了 Windows 10 的 Windows PC 已经于 7 月 28 日出货。

理论上来说，你应该已经可以买到了。至于单独的 Windows 10 软件则将于 8 月 30 日正式开始销售，中文家庭版售价 888 元。

更新完毕，我们能期待什么？

业界认为，至少 5 年之内，微软都不会再发布新的操作系统，今日正式发布的 Windows 10，可能会陪伴 PC 用户很长的一段时间。如果你要去问微软的销售人员为什么自己应该升级，很可能收到的答案会是“Windows 10 让你感觉很熟悉”。

在微软的官方网站上，它们这样形容“迄今为止最好的 Windows”：“尽享全新功能和熟悉的体验，您从一开始就是专家。”，以及“现在用的是 Windows 7 或 Windows 8？您会爱上 Windows 10！”

Windows 8 得有多失败，才能让 Windows 10 打出了这样的宣传语？

微软在销售指南里把所谓的官方四大卖点归纳为：相比以往更加熟悉，也更好；最个性化和最自然的体验；跨平台使用；更多生产力。

对于普通的用户来说，值得期待的可能是以下这几点：

  • 首先，开始菜单回来了

开始菜单

如果一定要给升级 Windows 10 一个理由，应该就是开始菜单了。Metro 风格界面是 Windows 8 系统最大的卖点之一，可惜许多用户并不买账。从传言中版本还叫“Windows 9”的时候起，开始菜单的回归就是其中最重要的一点。

大动作切换全屏应用、没有开始菜单、找不到关机按钮等用户经常的问题将很快随着新的系统版本的到来做一个了断。新的开始菜单将分为两栏，融合传统的开始菜单和 Metro 风格的应用可视化界面，用户还可以自定义设置开始菜单项目。


  • IE 浏览器也可以说再见了

微软的新浏览器叫做 Microsoft Edge，此前它的内部代号是斯巴达人计划，来自 Xbox 游戏 Halo。

Edge 里内嵌了 Cortana 智能语音助手。此外，为了更好的平板体验，你可以直接在网页上涂涂画画、做标注。

为适应新环境，Edge 用了新的同名渲染引擎。作为一个从头开始的浏览器，它将直接兼容 Chrome 和 Firefox 上的丰富插件。

一个新的 Spartan 品牌可以洗脱本来 IE 给人以“二十世纪初的 PC 端网页浏览器”的印象，而数据也证实了 IE 浏览器的江河日下：根据 Net Applications 的数据，IE 浏览器市场占有率已经从高峰期的 90% 下降到了 58%，在移动端，IE 的占有率更加惨，只有 2%。

  • 你可以在电脑上用语音助理 Cortana 了

升级 Windows 10 之后，用户可以在电脑呼叫微软的个人语音助手 Cortana 进行搜索，让它提供语音服务，例如告知天气、设置提醒等。

PC 厂商显然还对 Windows 10 抱有期待。其中之一就来自于智能助理 Cortana。有一些甚至将 Cortana 做成了实体按键。

6 月，东芝发布了新的笔记本电脑产品线，所有的笔记本，在键盘功能栏里都有了一个新的按键叫做 Cortana。

联想虽然没有在硬件上进行改动，但上月发布的软件 REACHit 能进一步拓展 Cortana 在 PC 上的搜索能力，帮助 Windows 10 用户跨平台（PC、平板和云端服务）搜索图片、邮件和音乐等，例如在 ThinkPad 平板上搜寻另一台联想笔记本上的内容。

  • 多个平台，一套系统

Windows 10 和以前任何一个微软操作系统都不太一样。它是微软第一次尝试让手机、平板、电视、电脑、VR 设备、游戏机使用同一套系统、同一个应用商店，以此让分裂的 Windows 家族统一起来赚钱。

此外，在增加效率方面，Windows 10 的多窗口并排功能得到了加强。通过快捷键，任务窗口能够快速变为不同的尺寸。

这是微软三年前就该做的事情

微软推销这一代 Windows 操作系统的出发点，在于“熟悉”，而不是“与众不同”。正如它们近期更新的销售指南所说，“Windows 10 至少让你有回家的感觉”。

目前，共有超过 15 亿台电脑上装的都是 Windows 的系统。为了鼓励用户升级到 Windows 10 ，微软在定价上做了有史以来最大的改变：用户对电脑进行更新将不再需要支付任何费用。

微软决定牺牲 Windows 操作系统这一部分利润的原因很简单，那就是微软需要更多的用户使用 Win10 系统——并且是在短时间内。

微软上一个操作系统 Win8 没能挽救微软的市场地位，Win8 大胆尝试重新设计的界面反而让事情变得更加棘手，更换系统之后丢失了许多 Windows 老用户。在 Windows 10 预览版推出之后的半年多里，用户对它的评价还不错。

和 Windows 7 的时代不同，最近几年，计算机的势头已经远远跟不上智能手机和平板电脑了。根据 Gartner 的数据，全球范围内，计算机销量在第二季度下降了 9.5 个百分点。如今，微软的竞争对手是苹果和 Android。

相比起 Windows 8，Windows 10 才是那个能够带领微软迎接全新挑战的操作系统。终于打通的移动端和 PC 端系统也许会令软件开发商重新对 Windows 提起兴趣。不过，它还是来晚了三年。

不过，长期追踪报道微软 ZDNet 老牌记者 Mary Jo Foley 带来了“别着急升级的建议 ”。到目前为止，这个操作系统在功能上仍有需要完善之处，它看起来更像一个半成品而不是成熟的操作系统。

微软将在今年秋天为 Windows 10 加入更多的新特性。在此之前，升级到 Windows 10 并不是一件特别着急的待办事项。或许，你可以等更多搭载 Windows 10 的设备上市之后，再看看。

不过，我已经第一时间选择了升级。起码，它再糟糕也糟糕不过 Windows 8 了。

一分钟也不想再忍？现在就升级吧。
