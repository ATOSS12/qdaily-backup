现在，推荐朋友买特斯拉也能赚钱了_智能_好奇心日报

钟舒婷 2015-07-31 14:15:00

首先，你要有一台特斯拉，然后还要有想买并且买得起特斯拉的朋友若干名

为了达到今年卖出 55000 万辆电动车的目标，特斯拉近来实行了和 Uber 一样的销售政策，老用户邀请新用户购买特斯拉，双方都能获得 1000 美元的优惠。

老用户指的是 Model S 的车主，只要将自己的推荐链接分享给朋友，并且对方在 10 月 31 日前购买了 Model S，这样不仅新车主获得 1000 美元的优惠，老车主也能获得 1000 美元的积分，可以用来购买服务、配件，或者，买一辆新的特斯拉。

此外，如果老用户能累积推销 5 辆 Model S，将被邀请参加在内华达州的超级工厂 Gigafactory 的开业典礼。如果能成功推销 10 辆，就能以普通价格买到 Model X SUV 的限量订制车款。

另外，第一个成功推荐 10 个人买 Model S 的人，将免费获得一台 Model X SUV。

据特斯拉 CEO 马斯克透露，目前特斯拉卖出一辆电动车的销售成本大约为 2000 美元，于是他们想透过“口碑传播”的方式，将钱回馈给顾客，并减少未来开设实体门店的数量。

特斯拉采用的是直营销售模式，用户直接在官网下订单。但根据美国某些州的法律，汽车制造商不允许直接销售车辆，必须经过第三方经销商，因此特斯拉部分线下的门店只能提供体验和售后服务，不允许销售。

Model X SUV

特斯拉今年的销售预期是交付 55000 辆电动车，在刚过去的第一季度财报中，特斯拉交付了 10045 辆，未达到目标的四分之一，7 月 2 日，特斯拉公布 Model S 第二季度的交付量为 11507 辆，这样加起来达到了目标的 40%。

目前，特斯拉的产能已经实现每周生产超过 1000 辆，同时超级工厂 Gigafactory 也在建设中，2016 年估计就能加入到生产中。

马斯克在不久前接受采访指出，目前已经接到了 20000 份 Model X 的订单，但 Model X 生产难度较高，使得原本计划在去年上市的 Model X 一再推迟交付时间。

根据计划，今年第三季度特斯拉就要开始交付新车型 Model X SUV，假如  Model X 不能按时交付，恐怕还是要靠老客户们口口相传，依靠 Model S 来达到年度目标了。
