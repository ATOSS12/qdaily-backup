炫酷的智能布料，指甲盖大小的微型雷达芯片，还有可以实时直播全景的视频设备和即插即用的安全系统。

Google 的神秘部门 ATAP（前沿科技与项目）一直以做有未来感的设备闻名，如模块化手机 Project Ara、3D 扫描手机 Project Tango……

今年在 Google I/O 大会上，ATAP 摆了一个大展台来展示最新的项目，里面展示了智能布料、微型雷达芯片还有实时直播全景视频的系统 Spotlight Studio，上前询问的开发者络绎不绝。《好奇心日报》也去逛了 ATAP 的展台，还听了一场他们的年度新品发布会，看看今年 ATAP 都带来了什么黑科技？

LEVI'S 品牌设计师 Paul Dillinger 也出现在 Google I/O 大会上

在可穿戴设备技术的演进上将电路织入布料的做法，Google 并非第一家。《好奇心日报》今年在美国消费者电子展 CES 上看到做类似技术的初创公司。但对于 Google 来说，最大的优势不仅在于研发，而是可以快速找到合作伙伴。

Project Jacquard 刚刚发布，就已经宣布了第一个合作品牌——美国本土牛仔裤品牌 LEVI'S。尽管无论是 Ivan Poupyrev 还是 LEVI‘S 的品牌的现任设计师 Paul Dillinger 都不愿意公布进一步合作的服装产品的信息，所以我们目前还不知道会不会有智能布料做的牛仔裤面世。

指甲盖大小的雷达芯片，想让你用手指精确交互

接下来就是 Project Soli，一个体积很小的雷达芯片。Kinect 使用高清的摄像头和一套解析程序，还原人在空间中的运动；Leap Motion 的方式是用图像捕捉。Project Soli 不同，它实际是一个由微型芯片驱动的微型雷达装置，通过无线电波反射探测周围环境。

Project Soli 芯片的大小

Project Soli 一心想要把雷达芯片放到像智能手表这么小的设备当中。可以想象，一旦这么做，智能手表的交互方式将会发生非常巨大的变化，我们可能不需要在 1.5 英寸的小屏幕上用手指触屏操作，而是在手表附近用手指精细操作。这种操作会有多精细？下面这个现场视频可以给你一些概念：

一个装有安全电脑“SD 卡”

Project Vault 也是第一次曝光的黑科技，它看上去只是一张平淡无奇的 MicroSD 卡，就是许多 Android 手机可以插的那种。但它内置了一个微型电脑，因为里面有一个保护隐私和安全的操作系统 ARTOS，还配有 NFC 和天线。你可以用这张 SD 卡储存重要的信息，如密码等等，更复杂的用法，Project Vault 能够帮助你做文件加密。


更加有趣的是，Project Vault 用起来也跟一张普通的 SD 卡无异，不需要额外的复杂操作，无论是 Android 手机还是普通的 PC 或者 Mac，你只要像平常储存文件到 SD 卡一样，把文件放到 Vault 的两个文件夹当中就能完成通讯的操作。
