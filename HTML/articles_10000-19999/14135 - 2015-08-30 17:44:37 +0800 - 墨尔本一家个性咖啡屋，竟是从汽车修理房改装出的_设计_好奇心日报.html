墨尔本一家个性咖啡屋，竟是从汽车修理房改装出的_设计_好奇心日报

廖婷 2015-08-30 17:44:37

工业感和摩登味的奇妙混搭

将汽车修理房改装成咖啡屋，是澳大利亚建筑事务所 ZWEI 最近的项目。虽然看上去他们毫无关联，但当打开天窗，围着墙摆满一圈桌椅，放上咖啡机和各类点心，这里还是一个通风安静的喝下午茶的好地方。

这家店位于北墨尔本，墙面刷白后依然露出了砌砖的缝隙，也是唯一保留的原来房子的痕迹。屋内的墙面画上了简单的彩粉画，很有质朴的童趣。尽管屋外的墙面已经严重风化到难以补救，从外面看上去也不失为一种特别的自然。

外墙与内墙

小屋的中心是用木材搭建的两层工作区，楼上是储藏间，楼下是开放的操作间和收银台。咖啡屋里尽可能摆满了桌椅，鼓励客人们在这里放松地一直坐下去。自然光照射在温暖色调的墙面上，淡淡的工业感与都市的摩登味奇妙地融合在一起。

澳大利亚的墨尔本是有名的移民城市，也被称为咖啡之都。自上世纪 50 年代，意大利的咖啡机传入澳大利亚，靠近海边的墨尔本就培养了强大的咖啡市场。对他们来说，咖啡不仅是一种饮料，也是联络感情的地方，不同的咖啡店也会将咖啡师作为招牌，拥有固定常客。所以，来自美国快餐文化的星巴克才会在墨尔本乃至澳大利亚都屈指可数，当地人称他们的口味早就养得非常挑剔了。

咖啡老店深深扎根在当地的文化中，新开的咖啡店也要寻找自己的特色。这家 Code Black coffee 就是颇具实验性质的品牌，对黑色艺术和技术感兴趣。从他们的 LoGo ，一个插着花带着微笑的黑骷髅头，就可看出嘻哈风格。搭配着咖啡屋有意布置的机械装饰，非常吸引年轻人。不仅如此，他们还非常注意咖啡的品质，从农场挑选高质量的咖啡豆进行烘焙，打造健康的形象。

Code Black coffee

Code Black coffee偏好黑色艺术

另外，负责装修的 ZWEI 是 2006 年成立的专注于室内设计的公司，他们还装修了不少与咖啡屋类似的餐饮零售小店。为 Code Black coffee 在今年的墨尔本国际咖啡展（Melbourne Int. Coffee Expo）上搭建的零时展示屋，入围了 Eat Drink Design 奖。详情参见他们的主页 zwei.com.au
