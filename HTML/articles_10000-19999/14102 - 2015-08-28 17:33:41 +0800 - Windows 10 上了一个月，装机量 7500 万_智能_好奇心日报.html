Windows 10 上了一个月，装机量 7500 万_智能_好奇心日报

赖江锟 2015-08-28 17:33:41

Windows 10 的推出，又掀起了一轮黑 Windows 8 的小高潮。

Windows 10 推出 1 个月 (4 周) 后，装机量达到 7500 万。根据 IDC 分析师 Al Gillen 的说法，这 7500 万用户里大部分来自于 Windows 8 的升级，后者在全球约有 1.4 亿用户。你可以想象 Windows 8 用户们有多想赶紧换系统。

此前的两代系统，Windows 8 第一个月有 4000 万用户，Windows 7 的成绩则是 2 个月 6000 万，均低于 Windows 10 的普及速度。

免费是最重要的原因。现在的主流做法就是软件免费，以便吸引大量用户使用。Windows 10 不是微软第一次使用免费的策略。此前，微软所有的手机应用都已供免费下载安装，包括 Office、Sunrise日历、邮件 OWA for iPhone 等。Google 一直都在提供免费软件，苹果也在最近几年将电脑操作系统改为免费。

Windows 8 系统界面设计糟糕也“功不可没”。Windows 8 同时拥有两套操作界面，桌面的和平板的。它们只是被放在了一起，并没有很好地结合。最明显的例子，是“开始界面”(原来“开始按钮”的位置) 会让桌面用户不知所措。在 2014 年初 (Windows 8 推出一年半后)，HP (惠普) 甚至开始在新出厂电脑上重新预装 Windows 7，他们还表示因为大家都有这个需求。

所以 Windows 10 的出现，还掀起了一轮黑 Windows 8 的高潮：The Verge 的记者 Tom Warren 表示“Windows 10 很好地解决了 Windows 8 上存在的问题”，Wired 的记者 David Pierce 则写到“你 Windows 10 用得越多，你就会觉得 Windows 8 有多荒谬”。

不过，如果你也升级了 Windows 10，那就要看好自己的钱包了，因为微软下一步是为你提供软件内的收费服务。在此之前，微软已经不让你在 Windows 10 上用盗版软件和游戏了。
