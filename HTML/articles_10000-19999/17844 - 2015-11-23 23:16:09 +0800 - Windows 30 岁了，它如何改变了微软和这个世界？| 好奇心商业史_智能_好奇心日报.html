Windows 30 岁了，它如何改变了微软和这个世界？| 好奇心商业史_智能_好奇心日报

李硕明 2015-11-23 23:16:09

“每家的桌上都有一台电脑”

回头看去，1985 年 11 月 20 日是微软历史上最重要的日子之一。1.0 版本的 Windows 操作系统在这一天被装在方形软盘发给用户。不到 10 年时间，这个操作系统便成为绝大多数个人电脑用户开机之后见到的第一样东西。

但在当时，Windows 的发布好像不是什么值得一提的消息。直到发货七天后，《纽约时报》这个前互联网时代美国人最重要的信息来源才在一篇谈电脑展会文章的结尾花了不到 30 字介绍了这个改变微软、乃至整个计算机产业发展的产品：

“在位于华盛顿贝尔维尤的微软公司内，一个并没有多少影响力的部门最终发布了它们的第一个图形化操作系统。Windows 最初发布在两年前，它可以让 PC 用户一次通过窗口化的方式运行多个不同的程序。”

在 1985 年，微软确实还不是什么科技巨头。这个后来成长为因为垄断先后被美国司法部和欧盟起诉的公司要到 1986 年才挂牌上市。

微软当时的主要业务还是卖 MS-DOS 操作系统，主要客户是 IBM。不过关注 PC 产业的人当时已经可以看到微软的潜力。诞生刚两年的 IBM PC 已经远远甩开了苹果的 Macintash 电脑，成为个人电脑的标准。

Windows 1.0，微软有了第一个自己做的操作系统

无论是 MS - DOS 或者 Windows 都是一种操作系统，但究竟什么才是操作系统？

美国作家 G.Pascal Zachary 在他关于微软的著作 ShowStopper 做了一个易懂的类比：

想象一个 19 世纪的英国豪宅。整幢房子井井有条，家政人员打扫卫生、司机在清洗明天要用车辆，厨师做着一日三餐。楼上住的贵族，并不需要过问卫生、厨房的细节，他需要什么，只需要对管家嘱咐一句，就会处理妥当。

如果将个人电脑比作这桩豪宅，操作系统（管家）协调着一切硬件（仆人以及屋里的设施）的工作，软件就是屋里的贵族，它只需要和操作系统沟通就能完成工作，不管这台电脑用了哪家公司的处理器、哪个品牌的内存——操作系统会处理好一切。

操作系统的存在让软件开发变得简单，而软件则是普通用户使用个人电脑的理由。

MS-DOS 就是在 1985 年扮演了这样的角色。这个盖茨当年花费了 5 万美元从别人手里买回来的操作系统，帮助微软成为 IBM PC 的标准软件平台。

DOS 操作系统的界面，需要你输入完整的指令才能运行应用

DOS 看起来就像是一个文字版的 Siri，你输入一段命令，它给出一个回应。但不同于有一定理解能力的 Siri，你只能给出极其精确的提问。比如打开 WORD 软件这样简单的操作，你需要输入 c:\program files\word\word.exe 这样一段话。

想象一下一个只能听懂“主屏斜杠电话斜杠妈妈斜杠通话确认拨打电话”的 Siri 吧。对于大多数用户来说，DOS 太难用了。

为了降低电脑使用门槛，让更多人用上自己的软件。微软决定要做图形化的操作系统。其实这个想法还是盖茨在 1982 年的 Comdex 大会上看到 Visi 公司的图形系统演示之后产生的。

Windows 1.0 的界面设计

微软在 1983 年 11 月公布了自己的第一个图形界面操作 - Windows 1.0。

这是微软在图形界面操作系统的第一次尝试。Windows 1.0 可以让 PC 用户通过窗口化的方式，同时打开多个应用程序，并且能够利用鼠标在这些窗口之间进行切换或者关闭。

同时微软在 Windows 1.0 上也预装了一些其他应用，比如日历、计算器、控制面板……，这些后来都成为 Windows 系统默认的预装附件。

比尔·盖茨稍晚在接受科技媒体 InfoWorld 采访时候说：

“我们将会把所有的资源和精力完全投入到 Windows 中去，就像之前在 MS-DOS 和 Xenix 所做的一样。第三方应用只有利用 Windows 提供的优势，才能获得长期的竞争力。”

Windows 1.0 可以让 PC 用户通过窗口化的方式，同时运行多个不同的程序并在之间进行切换，这是 DOS 系统无法做到的一点。开始有了下拉式菜单、滚动条、图标、对话框。

尽管 Windows 1.0 仍然是一个非常粗糙的产品，微软还是兑现了盖茨所说全力投入的承诺，继续开发了 Windows 3.0 版本。

Windows 3.0 的图形界面

正如你在上面图片中看到的一样，Windows 3.0 也才是微软真正描述的图形化操作系统。尤其是文件管理器的图标在 Windows 3.0 上面变成了一个真正有图标构成的东西，取代了之前Windows 版本依靠的 MS-DOS 文件系统。

从这一代的 Windows 开始，微软开始得到众多第三方开发者们的支持。

实际上 Windows 3.0 此时仍然是运行在 DOS 上的一个图形程序。指令界面还是用户看到的第一个东西，需要输入正确的指令才会进入图形化的操作界面。

Windows 和微软公司的霸主地位，仍然要等到 1995 年才会到来。

Windows 95，微软统治个人电脑

1995 年8 月 24 日，微软推出了 Windows 95 操作系统。随之而来的，还有一场当时在科技行业前所未见的广告攻势。

微软当时在雷蒙德大学的一个运动场内举行了 Windows 95 的发布会，并且首次用卫星传播的方式让 7 万人看到 Windows 95 发布会。

Windows 95 的界面设计

相比之前需要在 DOS 下先输入一段指令代码才能进入 Windows 的做法，微软在 Windows 95 上第一次把图形界面做成了 PC 真正的默认界面。Windows 95 的大多数图标设计在这一代成了以后 Windows 的标准配置，比如回收站、我的电脑…….以及开始菜单。

对于当时第一次接触个人电脑的用户来说，他们可以通过“开始”菜单找到任何想要的东西，而不必去考虑文件被放在了哪个位置。

为了推广 Windows 95，微软花费了 3 亿美元的广告费在这一年的 8 月份开始了一场疯狂的宣传。它们专门用 800 万美元，买下了滚石乐队的《Start Me Up》版权用来当作 Windows 95 广告的背景音乐。微软甚至把 Windows 95 的开始菜单按钮和 Microsoft 标志放到了纽约的帝国大厦和多伦多的国家电视塔上面。

微软为 Windows 95 做的户外广告

至于苹果这个竞争对手，它的创始人乔布斯已经在几年前被赶出了公司，单独做起了 NeXT 电脑。苹果自己的 Macintosh System 7 操作系统也停滞不前被微软甩开。Mac 在 1990 年代被挤到市场的边缘，苹果一度面临破产。

伴随 Windows 的普及，微软建立了一个长达 20 年的高效率生意：从 PC 厂商们卖出的每一台 Windows 电脑抽取一部分收入。

那个时候像戴尔和惠普这样的大型 PC 厂商们，每年都会生产至少上千万台的个人电脑，而每一台电脑上面都搭载着微软的的 Windows 操作系统。尽管微软也会单独推出零售版的 Windows，但它们的绝大部分收入还是来自 PC 厂商在发货前预装的 Windows 授权费。

在这个模式之下，微软可以只需保持一个规模相对不大的销售团队，把注意力都放在大客户的软件生意上面。至于 PC 市场内部的斗争，不论谁是赢家，都得为 Windows 付钱。

如果从 1995 年算起一直到 2000 年的短短 5 年时间内，微软就依靠这种授权费模式让自己的市场从 540 亿美元增加到了 6000 亿美元的最高点，整整提高了 11 倍。

Windows NT，微软统治了一切计算机

Windows 95 和此后的 Windows 98 让微软取得了空前的成功，实现了盖茨能让每个家庭都有一台 PC 的梦想。但是微软控制的只是个人电脑，Windows 9x 系列的不稳定性让微软无法把这样的操作系统用在企业的服务器上面。。

对于 1990 年代的 PC 用户来说，个人电脑崩溃死机是经常能遇到的事情。典型情况就是 Windows 的蓝屏界面隔三差五就会出现一次。

Windows 的蓝屏死机画面，XP 之前的用户会经常遇到它

当企业需要性能和稳定性更好的电脑时，它们还是会购买 Sun、Silicon Graphics 的工作站——一种硬件配置极高的计算机，运行更为稳定的 UNIX 操作系统。

为了能够得到这笔利润最高的软件生意，微软决定重新开发一个操作系统——Windows NT。盖茨在这个项目中并没有选择自己公司的员工，而是请来了当时在计算机巨头 DEC 工作的著名程序员大卫·卡特勒。

大卫·卡特勒（Dave Cutler）之前曾经在 DEC 公司负责 VMS 操作系统的开发工作，也因此被别人称作是最好的操作系统设计者。

卡特勒在加入微软之后，从 DEC 中带来了一批软件工程师组成自己的开发团队。相比微软的大部分员工，卡特勒和他的团队显得有些格格不入。他们自视更有才华也更加成熟，会嘲笑那些之前开发 Windows 95 系统内核的微软程序员是“微废人”（MicroSlop)。

Windows NT 的开发团队从一开始要对付的就是高强度的工作。想要重新构建一个操作系统并不是一件容易的事情，尤其微软还想要把 Windows NT 做成一个通用的系统。开发团队中负责文件系统的汤姆·米勒（Tom Miller) 表示，“Windows NT 可能是最后一个操作系统，很难想象会有人从零开始构建一个现代的操作系统”。

微软从 1989 年开始准备 Windows NT 的开发工作，盖茨原本打算在 1991 年之前就完成全部的工作。由于 NT 涉及的工作量太过庞大，卡特勒和团队给出的交付日期也是一再延迟。最终微软在 1993 年才完成了 NT 的大部分开发工作，而这个持续了 5 年的项目也花费了 1.5 亿美元的资金。

等到 Windows NT 开发完成时，许多初期成员都筋疲力竭离开微软。至于卡特勒自己也有过离职的想法，最终还是被盖茨挽留了下来。卡特勒本人也成了微软传奇一样的人物，后来在 2008 年和 2012 年的时候，以超过 60 岁的高龄参与了 Azure 云服务和 Xbox 的开发工作。

Windows NT 3.1 的界面设计

但 Windows NT 物有所值。它取代了过去企业常用的 UNIX 系统。一方面 Windows NT 的稳定性和性能更好，也是第一个可以运行 32 位程序的 Windows 系统。另一方面，NT 支持和个人电脑相同的 X86 处理器，企业不需要再购买昂贵的专用硬件。

Windows NT 以及之后的 Windows Server 一度成了企业市场的重要存在。一直到今天企业服务每季度都能为微软贡献至少 50 亿的收入。

企业之外，NT 后来也作为 Windows XP 的内核，帮助微软挽回了 Windows Me 带来的惨重失败，确保了 Windows 在个人电脑市场延续至今的统治地位。

鲍尔默卖 Windows XP 的广告

Windows 7，最“好”的 Windows 系统，也是微软衰落的开始 

但我们今天提起微软的时候，你再也不会把它看作是统治科技行业的霸主了。今天最热门的消费类科技产品已经不再是个人电脑，而是手机——运行 iOS 和 Android 的手机。Windows 已经不再是占据大多数人最多时间的电子产品。

讽刺的是，Windows 乃至微软的衰落却发生在许多人认为最好的 Windows 系统——Windows 7。

Windows 7 的界面设计

微软在 2009 年的 10 月 22 日发布了 Windows 7，它取代了那个麻烦故障不断的 Windows Vista 系统，成了这家公司历史上最成功的操作系统。Windows 7 发布一周之后就推动 PC 市场增加了 40% 的销量，在整整一年之后微软就卖出了 2.4 亿份之多的 Windows 7 拷贝。

微软霸主地位的消失，不是因为麻烦故障层出不穷的 Windows Vista 系统，也不是苹果每年更新一次的桌面系统 OS X。而是在 2007 发布的 iPhone 以及之后一年 Google 推出的 Android 系统。

乔布斯在 2007 年演示的多点触控技术完全改变了用户过去认知的操作方式。相比之下，微软当时的 Windows Mobile 操作系统还在试图把 PC 上的体验搬到手机上来。苹果在 2008 年推出的应用商店，更是让第三方应用的开发者们不再把 Windows 作为优先考虑的平台了。

我们虽然把 iPhone 或者 Android 叫做智能手机。但其实它们就是个人电脑，是一个比 PC 更加个人的个人电脑。

几乎你想到的任何应用场景，现在都是发生在手机上面，比如社交网络、游戏、即时聊天、阅读文章……你更多的时间都是在智能手机上处理这些事情。

与此同时，整个 PC 市场也开始陷入了长期下降的困境之中。

市场的长期下滑直接导致了戴尔和惠普这样的传统 PC 厂商选择了转型，这两家公司各自通过十余次的并购或者业务拆分从卖 PC 转而为企业提供服务。剩下的公司也基本都开始推出越来越多的 Android 产品，不管是手机、还是平板。

甚至在微软内部 Windows 也已经不再重要，Office 和企业服务带来的利润早已经超过了它。

萨提亚·纳德拉（Satya Nadella）在 2014 年 2 月，接替鲍尔默担任微软的 CEO。这位微软老兵先后负责互联网服务、Office 和云计算业务——除了 Windows 以外的每一个微软主要业务。

纳德拉把 Windows 变成了展示微软服务的最佳平台，而不再看作是一个需要重点关注的生意了。他做得第一件事情就是放行 iPad 版本的 Office 应用。

微软 Office 业务营销经理朱莉亚·怀特在用 iPad Air 演示 Office 应用

微软在 2014 年的 3 月，发布了 iPad 版本的 Office，在发布一周时间便被下载 1200 万次。3 个月后，iPhone 和 Android 手机也用上了 Office。

至于专门为 Windows 设备开发的触屏 Office 应用，得等到 2015 年的 7 月底，距离发布 iPad 版本已经过去整整 16 个月的时间。毫无疑问，对于微软最能赚钱的 Office 部门来说，Windows 已经不是自己需要优先考虑的平台。

“Facebook、Google、苹果、亚马逊，这是现在科技界最重要的 4 家公司。”Google 董事长埃里克·施密特（Eric Emerson Schmidt）2011 年在参加 All Things D 大会时候这么说道。对于微软为何不再其中，施密特表示 “微软只是在企业客户方面做得不错，但并没有掀起一场消费电子革命。”

Facebook 控制着 15 亿人的社交、Google 控制着 Android 系统以及中国以外的互联网服务、苹果至今仍然控制着个人电脑硬件的发展方向以及行业里的绝大部分利润、亚马逊则同时是全球最大的在线商店和云计算平台。

到 2011 年，在全世界仍然有超过 12.5 亿台运行 Windows 操作系统被人使用。但很明显，它们代表的更多是曾经的辉煌而非未来。

Windows 统治下的桌面系统，已经无法让微软拥有和这 4 家公司相同的影响力了。就像当初被 Windows 打败的众多竞争对手一样，它在 30 年后也结束了自己辉煌时代。

今天不管社交网络、搜索引擎还是在线电商，它们的存在都仰赖于个人电脑的普及。在推动这个趋势上，没有谁比 30 年前那位总穿着毛衣出现、年轻而强硬的商业领袖做的更多。

“每家的桌上都有一台电脑”（a computer on every desktop and in every home）。这件事，Windows 做到了。
