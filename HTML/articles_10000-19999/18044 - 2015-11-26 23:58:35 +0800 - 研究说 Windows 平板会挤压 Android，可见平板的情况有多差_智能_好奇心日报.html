研究说 Windows 平板会挤压 Android，可见平板的情况有多差_智能_好奇心日报

李硕明 2015-11-26 23:58:35

问题是，你买来到底是当平板用，还是当 PC ？

数据分析机构 Strategy Analytics 最近公布了一项关于 Windows 平板 增长情况的报告。按照这份报告的结果，Windows“平板”现在已经占据整个平板电脑市场分额的 10%。

今年前 9 个月里，Windows 平板比去年这时候多卖了 58%，今年可能会有 2200 万台装着 Windows 操作系统的平板被卖出去。

如果继续按照这个速度增长的话，Windows“平板”在 2019 年的时候会拥有 18% 的市场分额，逼近 iPad 并挤压 Android 目前的空间。

Strategy Analytics 平板市场份额预测，2015 年与 2019 年对比

Windows 平板的热卖反应出的却是平板的尴尬。

Windows 10 发布已经有几个月了，但 Windows 上的触屏应用依然停留在 3 年前的水平。例如平时中国用户会经常用到的微博应用，在 Windows 的商店中仍然还是 Windows 8 发布时候的样子。甚至连应用提升都是“欢迎使用 Windows 8 版本”。

大多数用户选择 Windows“平板”是因为它提供了丰富的桌面软件。从专业的绘图应用AutoCAD 到 GTA 5 这样的游戏大作。用户花钱购买的理由，还是因为它 PC 的那部分。

从 2001 盖茨手里拿着的 Microsoft Tablet，到乔布斯 2010 年发布的 iPad，再到今年的 Surface Book 和 iPad Pro。究竟平板为什么而存在，还是一个问题。
