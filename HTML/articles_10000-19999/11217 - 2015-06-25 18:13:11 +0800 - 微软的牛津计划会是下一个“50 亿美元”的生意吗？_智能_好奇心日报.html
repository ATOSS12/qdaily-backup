微软的牛津计划会是下一个“50 亿美元”的生意吗？_智能_好奇心日报

王杰夫 2015-06-25 18:13:11

微软为 Azure 云平台找到了一个亚马逊做不到的事情。

还记得不久前火过的那个测年龄应用 how-old 吗？用户只要将照片上传到网上，这个应用就可以识别出照片里有几个人，他们的年龄分别是几岁。它简单粗暴地击中了人们对于“客观第三方评价”的渴望，于是一张张照片被人以“口嫌体正直”的姿态放到微软的人工智能面前一一过目。

据称该应用上线七天后被测过年龄的照片就突破了四亿张，所有年纪加起来比地球岁数都大了。作为该事件幕后主导者的微软享受了一番许久没有过的“头条感”后，趁热打铁地推出了一款兄弟版应用——twinsornot。

twins or not 应用

与 how-old 做了的事情类似，用户只要上传两张个人照片，微软的人工智能就可以评判出这两张脸的相似程度。（似乎我已经看到朋友圈各种晒与某明星的撞脸照了）目前已经有不少有名人趟枪，什么汪峰白岩松啦、范冰冰李冰冰啦。

不过，这篇文章的重点不在这些有趣的应用，而在实现这些有趣应用的工具——微软的牛津计划。牛津计划最早在微软开发者大会 Build 2015 上被介绍出来，简单来说，这是一个数据分析工具的“五金店”，当小公司、个人开发者想要分析一些图片、音频文件，但手边又没有便宜方便的工具时，就可以到牛津计划这里逛逛。

微软将自己的一些有趣的“黑科技”放在牛津计划里供开发者使用，用户只需要申请一个免费的密钥，就可以访问到这些工具的 APIs，只要有个好点子，作出 how-old、twinsornot 这些应用简直不费吹灰之力，微软员工表示 how-old 其实就是两个程序员用一天时间捣鼓出来的东西。

已经很久没有见微软这么有趣了，如果不是偶尔还需要打开一下某个第三方公司发来的 Word 文件，我都差点以为微软已经醉死在哪个企业的怀里。当然说笑归说笑，微软在过去几年间与普通消费者的距离似乎越来越远。虽然可以用“微软的立足点在企业级市场”这句话聊以自慰，但事实反复告诉我们，抓不住消费级市场的科技公司，也就很难抓住企业级市场的心。

纳德拉讲微软云

微软亚太研发集团云计算与企业事业部中国创新组的廖勤樱告诉《好奇心日报》，其实微软的这种改变很大程度上得益于其新任 CEO 萨蒂亚·纳德拉（Satya Nadella）。这位印度裔高管上台甫始就确定了微软今后“移动为先云为先”的战略，而牛津计划其实也是微软云平台 Azure 的一部分。

与其将自己花费重金开发出来的技术摆在研究室落灰，不如通过开放 API 和 SDK 的形式，将它借给第三方开发者使用。这与当年亚马逊决定做网络服务 AWS 的初衷倒有几分相似。

亚马逊 CEO 杰夫·贝索斯就曾将 AWS 比喻为发电站，意思是用户不需要建发电站就可以用（买）电，由此推之，程序员也不需要建主机就可以使用（买）网络空间，那么再进一步呢？纳德拉认为程序员不需要知道数据如何被处理，只需要从微软这里购买处理好的数据就行。

于是，在微软 CEO 的支持下，一个在鲍尔默时代微软难得一见的横跨几大部门、涉及十多个团队的牛津项目成立了。你可以看到，后端的人脸识别 API 来自微软亚洲研究院，语音识别 API 来自 Bing(必应)团队；前端应用层面的 how-old 是美国团队开发，而 twinsornot 则由中国团队主力开发；最后，牛津项目云服务平台开发和维护就是由廖勤樱所在的云计算与企业事业部负责。

“这是一个由云计算部门主导，可能是微软内涉及团队最多的项目之一”，廖女士表示：“该项目主力在中国，我们也为中国本地化做了不少工作”。

牛津计划刚刚成立几个月，目前主要开放出四大类 API，分别是人脸识别、语音识别、计算机视觉和语义理解服务，据廖女士表示，更多的 API 也会逐渐接入到这个计划内。

牛津计划对图片进行特征分析

当然，由于牛津计划还处于开发中，所有的数据处理服务暂时都免费提供给开发者。微软内部之所以作出的 how-old 和 twinsornot 应用，也就是希望吸引开发者加入进来，挖掘这些数据处理服务的潜力。

微软的牛津计划可谓一举多得，一方面可以集合更多人的智力，为那些花大钱开发出来的“黑科技”找到用武之地；另一方面，如果牛津计划能够顺利运作起来，微软还可以借此推广它的云服务 Azure，要知道亚马逊的 AWS 已经是一个 50 亿美元的生意；此外，微软还能够扭转它一直以来“官僚中年男人”的企业形象，重新变得像个“Cool Boy”。

最重要的是，微软或许会借此改变游戏规则。目前像微软这样进行数据处理的大公司比比皆是，但是却又都封闭保守。无论 Google、Facebook，还是百度、腾讯都在闭门造车，如果微软的牛津计划能够成功，那么更多来自这些公司数据处理方面的 API 也会相应地被开放出来。

如果这真的是个生意，那么微软已经占得先机，不过这也不是微软第一次占得先机了，不过纳德拉能否通过这个先机，把微软做成像亚马逊在云服务方面的优势地位，才是最关键的。
