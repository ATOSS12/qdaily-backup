XP 之后，Windows 7 成了微软的另一个阻碍_智能_好奇心日报

周韶宏 2015-07-03 16:49:00

Windows 10 还没出生，就已经被亲兄弟碾压了。

Windows 10 将在本月底正式发布了，这个被微软赋予重要使命的操作系统，最大的竞争对手也许是自家的 Windows 7。

Net Applications 最近发布的数据显示，今年 6 月，Windows 所有操作系统当中，份额最高的依然是微软在 6 年前发布的 Windows 7，占比超过了 60%。其次是有 13% 份额的 Windows 8，尽管去年 4 月就已经停止更新，Windows XP 也依然保留着 12% 的份额。

Windows 操作系统中，Windows 7 占据了超过 60% 的份额


让 Windows 8 更为尴尬的是，Windows 7 在这个月的份额相比 5 月增长了 3.22%，而 Windows 8 却下滑了 0.67%。Windows 8 和 Windows 8.1 二者总共的占比也只有 16%。这其实是 Windows 8 在发布两年后，第一次在市场份额上超过 Windows XP。

鉴于 Windows 10 马上就要到来，Windows 8 今后的用户会越来越少。自从发布至今，Windows 8 就没怎么受到欢迎，用户甚至在买了新电脑之后装回 Windows 7。而现在 Windows 10 也要重蹈覆辙了吗？

微软在今年初正式发布 Windows 10 预览版，它是一个统一核心的多平台操作系统，将横跨手机、平板、PC、Xbox。纳德拉正试图让 Windows 走向更多的设备，实现“移动为先、云为先”的目标，应对来自苹果和 Google 的竞争。

现在看来，微软的这一策略竟然面临着来自 Windows 7 的阻碍，毕竟用户花了 6 年的时间依然接受不了更新的 Windows 系统。

虽然 Windows XP 停止更新，但国内的杀毒软件纷纷为它站台，以至于在中国，直到今年 4 月，Windows XP 还存在于 50% 的电脑上，大部分是企业用户。根据百度流量研究院最新的数据，截止 6 月 30 日，Windows 7 以 46% 的份额排名第一，其次是 XP，占比 41%。

两年内打算安装 Windows 10 的用户达到 73%


不过一份来自 Spice Works 的调查也许能给 Windows 10 带来一点信心。这份调查显示，打算在两年内安装 Windows 10 系统的用户比例达到了 73%。这个结果也许是因为 Windows 10 的免费更新政策。
