插个U盘，就可以用 Google 开发的电脑了_智能_好奇心日报

赖江锟 2015-11-19 14:07:00

Chromebook 很贵？现在插个 U 盘就能用了，呃其实也不是电脑操作系统，只是 Chrome 而已。

时下软件服务提供商要获取更多的用户，都做起了硬件，Google 也不例外。他们与华硕一起，正式推出了内置 Chrome OS 操作系统的电脑棒 Chromebit。
Chromebit 的外观就像个 U 盘，但它其实是个电脑主机。从配置上看，Chromebit 属于手机级别。ARM 架构的 CPU，2G 内存，16G 存储空间，具备 Wi-Fi、蓝牙、USB 2.0 连接能力。

它与普通电脑的最大不同在于，你只要把它直接插进电视或显示器的 HDMI 接口即可使用。

Chromebit

而 Chromebit 其实与普通主机也没什么不同，同样需要外接键盘、鼠标。不得不提醒一下，你仍然需要在使用时为它外接电源，因为没有内置电池，这可能把你对它产生的简单、便携的第一印象给打破了......

Chromebit 内置的 Chrome OS 是一套桌面操作系统，在某种程度上，你可以把它看作是一个 Chrome 浏览器。你能用到的所有功能，均与 Google 相关。从系统的功能性和复杂度来说，Chrome OS 与 Windows 这样的桌面系统完全是两回事。

据说配置 Chrome OS 的电脑在教育市场已有一席之地，那是一种比较受限的使用场景，并不总是需要 Windows 或 Mac OS X 般强大的系统。而在其它地方呢？

你还是要外接电源

Chromebit 售价便宜，只有 85 美元，而且也要比电视棒 Chromecast 强点。在客厅或卧室用它把电视变成电脑，也许有人会用它来看看视频或干点别的简单工作吧。而这正是 Google 想要的。
Google 在今年五月的 Google I/O 上宣布 Chrome 用户突破 10 亿，而其中相当一部分均来自 Android 和 iOS 移动系统。

现在，在电脑和手机之后，Google 正在想办法占据更多入口，包括用最低成本的方式占据你家的电视和电脑。
