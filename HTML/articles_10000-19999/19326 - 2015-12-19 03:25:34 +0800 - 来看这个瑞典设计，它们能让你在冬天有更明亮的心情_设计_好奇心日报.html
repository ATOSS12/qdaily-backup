来看这个瑞典设计，它们能让你在冬天有更明亮的心情_设计_好奇心日报

谢舒敏 2015-12-19 03:25:34

北欧很多设计工作室都能让你一眼认出他们的风格，而 Note 是其中最优秀的之一。

上个月，瑞典最大的商场斯德哥尔摩商场开张营业了。它也是欧洲最大的购物中心之一，容纳了 224 家店，很多店的门面都有两层楼、将近 8 米高。

这个新商场里还别出心裁地开设了一个“餐饮体验空间”，名为“Dining Theatre”，它兼有用餐和烹饪功能，顾客可以直接看见厨师在这里准备食物。商场开发商、欧洲大型商业地产公司 Unibail-Rodamco 委托设计工作室 Note 来设计这个空间的家具与装饰，希望他们打造出一个“有触感、可感知”的用餐环境。

Note 是斯德哥尔摩本地的一家设计工作室，主要做家具和家居产品设计，也有不少室内设计和建筑项目。他们为斯堪的纳维亚商场设计的这个空间体现出自己一贯的风格，而且人们一眼便知这是来自北欧的设计：简洁，不用雕花和纹饰，色彩明亮却十分柔和，有亲和力，很好地平衡了形式与功能。

Dining Theatre 的主色调为蓝色与暖调的粉色，并以此划分出用餐区和烹饪区。每一个烹饪单元配有洗手池和烤箱，台面由不锈钢或大理石制成。厨师们就现场在这里烹制食物，简洁的设计也是为了让他们尽可能地专注于食物，而不会被环境所打扰。

用餐区摆着木质的长凳与矮桌。本来方正的木桌椅会过于平实普通，但这里的方桌椅巧妙地用“金属管”作为桌腿和凳腿，水粉色的金属管还出现在桌面/长凳表面的边缘，起装饰作用。金属管和椅子表面上覆着浅橙色的皮革。金属、皮革、木、大理石，多样的材质同时出现在这个空间中，但显得很契合，一点也不突兀。

Dining Theatre 里还摆着一些有意思的装饰，比如矮桌上的小台灯，像一个个水粉色的蘑菇，从桌面长出来。用餐区立着一个街灯类似物，高高的灯柱使用了跟烹饪区同色系的蓝，上面挂着木质灯罩的聚光灯。

整块区域都很灵活，如果在这里要举办什么活动，这些独立单元的家具完全可以重新组合、规划成适合于那些场合的样子。

Note 工作室对自己的这个项目评论道，“当你坐在那儿，坐在长凳上用餐，同时还能看到一位米其林星级大厨在你面前做好一桌美妙的菜，你就能感到其中的‘品质’，既是食物的品质，也是家具的品质。”

Note 有很多出彩的作品，设计师们也很有自己的想法。他们在主页上这样写，“我们喜欢关注自己身边的环境与事物，并且创造出能让别人也像我们这样做的东西。通过发现和强调每个项目的独特性，我们将非物质性的价值观转化为可触摸、可感知的实物与空间。”听着有些抽象，我们还是来看作品吧。

他们设计过一个“神龛（Shrine）”。是的，这是一个宗教词汇，而它其实只是个带拱形的金属框架。这个东西的用途是为了帮助人们强调与纪念那些对自己有特殊意义的东西，保存生命中那些重要的回忆。

在今年的斯德哥尔摩家具展上，他们推出了一系列被柔和色彩的纺织品包裹的灯饰。其中的配色灵感源自 Note 设计团队在斯堪的纳维亚山脉上观察到的自然色调——清晨，黄昏，正午的太阳，闪烁的湖面，野外，花草。设计师表示，他们并不想要营造强烈的光芒，而是黑暗、漫长的冬日里一抹柔光。

他们为西班牙家具品牌 Sancal 设计过一款造型独特的休闲椅，以筒形、弯曲的金属丝笼为基底，设计的灵感源自西班牙东南部 Yecla 的酿酒产区的酒筒，而此地恰好是 Sancal 品牌的所在地区。

Note 的室内设计项目相对少一些，但都跟他们的家具和家居产品一样，很柔和，给人温馨舒适的感觉。

比如斯德哥尔摩这家色彩丰富、明亮的小餐馆，早晨到这里吃早餐喝咖啡，一天的心情都会很不错吧。

他们最新的室内作品是一个公寓改造项目，翻修了一间 1930 年代的 LOFT 公寓。业主希望这里能改造得有家的感觉，但又可以大大方方展示出来。于是他们就尽可能地少用北欧家居常见的白色，转而使用其他同样温和的色系。此外，这间公寓的家具全都是 Note 自己设计的。
