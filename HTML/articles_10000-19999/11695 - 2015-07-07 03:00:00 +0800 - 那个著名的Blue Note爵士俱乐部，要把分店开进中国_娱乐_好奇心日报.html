那个著名的Blue Note爵士俱乐部，要把分店开进中国_娱乐_好奇心日报

刘平安 2015-07-07 03:00:00

国内的爵士和蓝调乐迷又有好去处了。

世界上最著名的爵士乐俱乐部之一，美国纽约的 Blue Note Juzz Club 要来中国开店了。国内的第一家店面会落地在北京，明年三月开张，未来三年内还会在上海和台北开两家分店。

“Blue Note Jazz Club 的 Live Show，和大家买票去参加仰慕已久的乐手的音乐会体验完全不同，这里更多的是强调和朋友聚在一个地方、结识新人、享受音乐，就算大家可能不认识台上的表演者，也没有关系。”Blue Note Entertainment 的总裁 Steven Bensusan 在接受国外媒体采访时说。

不少蓝调和爵士大师都会到 Blue Note 办小型的拉阔秀，美国本周的表演嘉宾就是钢琴家 Robert Glasper, 他的专辑曾登上 iTunes 爵士榜单的榜首，未来他将会去到米兰的 Blue Note 表演。

事实上，亚洲的爵士乐中心一直都是日本，而 Blue Note Club 早在 25 年前就在日本东京开了门店，随后又在大阪和名古屋开了分店，至今日本的销售额已成为 Blue Note 总销售额的重要组成部分了。

虽然中国的爵士乐市场份额比较小，但 Steven Bensusan 认为这里的氛围正在变得越来越浓：北京上海相继出现了不少别致的爵士酒吧，譬如北京的东岸咖啡，这家店由摇滚乐手崔健的朋友、萨克斯演奏家刘元开办经营；还有上海的 JZ 酒吧，这里常会有来自五湖四海的爵士乐手登台表演。中国是有着巨大潜力的新兴市场，“我们要抢占先机，帮中国建立起这个爵士市场。” Steven Bensusan 告诉《纽约时报》的记者。

Blue Note

即将在北京开业的 Blue Note Jazz Club 选址在北京天安门广场附近一栋老式建筑的地下室里，这是一个建于 1903 年的石屋，也是前美国大使馆和达赖喇嘛的住所。开业后，这里每周周二到周日晚上都会有两场现场演出。

除了日本和中国，未来 Blue Note Jazz Club 还会在韩国开店，近一步拓展亚洲市场，但是具体计划还未确定。
