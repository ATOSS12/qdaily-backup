蒂姆·波顿又拍了部诡异的片子，从预告片就开始开脑洞_娱乐_好奇心日报

刘平安2015-09-08 03:00:00

蒂姆·波顿又有一部科幻电影要来冲击我们的视觉。

我们已经在《剪刀手爱德华》、《大鱼》、《爱丽丝梦游仙境》里见识过蒂姆·波顿的脑洞了，他的新电影《怪屋女孩》（Miss Peregrine's Home for Peculiar Children）刚刚发布了首支预告片，风格和导演的气质保持了高度一致，诡异，而且非常“概念化”。

预告片全程锁定一个绿色天鹅绒首饰盒，上面放着一支精致的怀表。但这个表却有些“故障”：每次指针快到九点一刻的时候，就会倒着走，然后再恢复正常......配乐一直在循环唱着：兔子快跑（Run Rabbit， Run Rabbit， Run run run），复古气息浓厚。 

时针之所以一直倒着走、再反反复复，原因是（还未看过小说原著的朋友，此处剧透）：1940 年 9 月 3 号是作者萨姆·里格斯（Ransom Riggs）在书中设定的一个重要日子。就是这一天，小岛被德军炸毁，所以只要时钟一直能够倒着走，这一天就永远不会来临，小岛上的人都能幸免于难。

电影根据萨姆·里格斯（Ransom Riggs）的同名畅销书改编。小男孩雅各布（Jacob）在爷爷去世后，按图索骥找到了之前爷爷多次讲故事时提到的小岛，发现了“怪屋”和里面住着的拥有特异功能的神奇小伙伴。不过，事情也没有那么简单，一股邪恶的力量也在悄悄逼近他们......

名导演蒂姆·波顿向来喜好这样的科幻题材电影，《怪屋女孩》的故事和情景氛围都不由地让人想起导演之前的作品。此次合作的演员是爱娃·格林（Eva Green），她将出演“怪屋”孩子们的看护人，主演还有《雨果》的小男孩阿沙·巴特菲尔德（Asa Butterfield）。编剧是简·古德曼（Jane Goldman），代表作《王牌特工》和《X 战警》。《怪屋女孩》由二十世纪福克斯电影公司制作发行，明年 3 月上映。
