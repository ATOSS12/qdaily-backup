Chrome 要进一步解决耗电问题，苹果电脑先上手_智能_好奇心日报

徐弢 2015-06-15 23:36:47

坏消息是，广大的 Windows 用户还得等等。

Google 在认真对待耗电大户 Chrome 的问题。

本月 4 日，Google 和 Adobe 合作为 Chrome 浏览器增添了一项节电功能：主动监测并停止非页面中央的 Flash 内容，而保留网页中央用 Flash 播放的视频。 

只不过，Chrome 耗电的问题远不仅仅限于 Flash。因而只是减少 Flash 在 Chrome 浏览器上的播放，并不足够。要知道，Chrome 浏览器在 Windows 和 OS X 系统上同样是电池杀手。微软称，Chrome 浏览器唤醒处理器的次数远超其他浏览器，在 Windows 系统上会多消耗 25% 的电量，而 The Verge 的测试则证明了 Chrome 同样会影响 Mac 的续航。

为此，Chrome 团队正在努力解决 Chrome 本身耗电的问题，嗯，Mac 用户会先享受到。
根据 Chrome 高级工程师 Peter Kasting 11 日在 Google+ 上写的文章，Chrome 将会有一些新的节电功能。新的节电功能着力在两点：调整标签页渲染的优先级，以及降低跟网络搜索相关的处理器占用率。
到目前为止，Chrome 浏览器所有的标签在渲染时都是相同的优先级。Chrome 团队在做的，是将背景标签的优先级降低，将其置于前景标签页的优先级之后，以降低空闲时间唤醒次数。
其次，Chrome 团队在大幅降低 Chrome 的唤醒次数和处理器占用率。在亚马逊网站（amazon.com）的测试案例中，Chrome 唤醒次数比原来降低了 59%，CPU 占用降低了 70%。

Ian Morris 去年在福布斯写的一篇文章指出，OS X 不会像 Windows 一样受到唤醒次数的影响，但 Mac 用户仍会因为使用 Chrome 而减少续航。这就是说，Chrome 团队关于减少唤醒次数的改进可能是无用的。
