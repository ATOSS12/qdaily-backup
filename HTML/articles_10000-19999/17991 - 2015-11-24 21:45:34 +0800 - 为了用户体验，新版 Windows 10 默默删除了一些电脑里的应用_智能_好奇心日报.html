为了用户体验，新版 Windows 10 默默删除了一些电脑里的应用_智能_好奇心日报

赖江锟 2015-11-24 21:45:34

安装 Windows 10 时，你有没有读过那份用户协议？

最新版 Windows 10 激怒了一部分用户，原因是它在未通知用户的的情况下，把一些应用删除了。
这些应用包括硬件监控工具 CPU-Z、CCleaner 公司的硬件信息搜集工具 Speccy、和显示设置工具 AMD Catalyst Control Center 等。用户打开电脑可能会纳闷这些应用去哪了，因为微软并不会在动手前告知用户。

“微软应该征得我的同意，而不是寻求原谅。”一位 ID Bright-Spark 的用户发帖表示。

目前，微软方面还未对此进行回应。但其发言人曾在今年 8 月表示，Windows 10 系统的协议，就是你必须同意后才能安装 Windows 10 的那个许多页的文件，已经给予微软有权对一些具有潜在风险的应用进行处理。这些风险包括安全漏洞、性能丢失、与系统不兼容等。

这次被删除的一些应用都没有及时支持最新的 Windows 系统，会产生一些不稳定问题。

但微软没有任何提示，直接做法仍然过于粗暴。目前微软已经不再自动删除这些应用，用户也可以再次安装。

微软可能希望借此震慑 Windows 10 应用的第三方开发商，逼迫他们为最新系统升级应用，以提供更好的体验。目前，仍有许多 Windows 应用并未对高分辨率屏幕优化，例如 Windows 版的印象笔记（Evernote）就比不上 Mac 版本的体验。

此前曾有戴尔和惠普的技术支持人员建议用户别忙着升级 Windows 10，理由是 Windows 10 系统兼容不佳。这类看法不在少数，而微软显然不想再独自承受了，可能他们认为第三方应用应该尽快支持其最新系统。

不过现在的做法显然不是最好的。
