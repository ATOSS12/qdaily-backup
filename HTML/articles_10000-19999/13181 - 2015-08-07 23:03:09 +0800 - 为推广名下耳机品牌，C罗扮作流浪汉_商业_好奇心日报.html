为推广名下耳机品牌，C罗扮作流浪汉_商业_好奇心日报

宣海伦 2015-08-07 23:03:09

这有啥，C 罗还代言过瘦脸仪呢

当皇马最大牌的球星走上马德里街头大秀球技会如何？结果是：整整过去了一个小时都没人愿意陪 C 罗玩耍，就连问美女要手机号都被拒绝了呢。

看上去很久没打理的胡须和头发、臃肿的身材再配上脏兮兮的运动服，C 罗看起来不能更糟了，事实上他看上去和那位当家球星一点关系都没有。

尽管脚下的花活依然精湛，卖力吆喝的 C 罗还是没能引起路人的注意，自觉没趣的 C 罗只得躺倒在地上大喊“好累呀”，或者呆坐在长椅上百无聊赖地啃着苹果，若有所思地觑着路人。所以说嘛，你们到底是看球还是看脸呢？

这支视频录制于半年前，但完整视频最近才流出，目的是为了推广 C 罗个人的签名耳机系列 ROC by Monster，由 C 罗名下公司 ROC Live Life Loud 与魔声耳机 (MONSTER) 合作，分别有三款耳机产品：头戴式的无线耳机、入耳式无线耳机，以及耳罩式有线耳机，售价从 170 美元 - 300 美元不等。

短短两天不到，这支视频在社交平台 Facebook 上的浏览量超过 3 千多万次，被分享 90 万余次，点赞和评论分别达到了百万和十万级，浏览量以每小时百万人的速度在增长，这甚至超越了梅西和科比在 YouTube 视频网站上曾创下的纪录。

尽管在球场上，一时瑜亮的梅西与 C 罗难分高下，但在商业领域，C 罗平均代言品牌的广告费用是 143,750 美元，差不多是四届金球奖得主梅西的一倍，代言过的产品领域从豪车到营养奶昔，十分努力。
